# 刷题高命中索引

用途：做题时先按题面信号定位知识点，少翻长笔记。先用本页命中讲次；命中后只看该讲的 `刷题流程`、`公式速查`、`易错速记`。

## 使用规则

1. 题目问“怎么算”：先看 `公式极简总表`。
2. 题目问“属于哪类”：先看本页关键词。
3. 进入具体讲次后，优先看 `刷题流程`，再看 `公式速查`。
4. 不要从头读整讲；只有连续做错同类题时再看 `易错速记`。
5. 题目含多个知识点时，先定位主动作：求极限、求导、积分、判秩、求概率、估计参数。

## 高频题型入口

### 1. 求极限

优先顺序：

1. 先代入，看是否直接有值。
2. 再看等价无穷小。
3. 再看洛必达法则。
4. 含根号、指数、三角、对数且局部复杂，考虑泰勒展开。
5. 数列极限优先看单调有界、夹逼、递推极限。

入口：[[01_高数第1讲_函数极限与连续]]、[[02_高数第2讲_数列极限]]

### 2. 证明中值、等式、不等式

优先顺序：

1. 有端点值相等：罗尔定理。
2. 有函数值差和导数：拉格朗日中值定理。
3. 有两个函数比值：柯西中值定理。
4. 积分不等式：看单调性、最大最小值、积分中值、对称性。

入口：[[06_高数第6讲_一元函数微分学的应用二]]、[[11_高数第11讲_一元函数积分学的应用二]]

### 3. 求面积体积弧长

先判断对象：

1. 平面面积：上下函数差或左右函数差。
2. 旋转体体积：垫片法、壳层法、参数方程法。
3. 弧长：直角坐标、参数方程、极坐标。
4. 摆线：直接查摆线公式。

入口：[[10_高数第10讲_一元函数积分学的应用一_几何应用]]

### 4. 多元积分

先判断积分类型：

1. 二重积分：先画平面区域。
2. 三重积分：先选直角、柱面、球面。
3. 第一类曲线/曲面积分：带弧长或面积元素。
4. 第二类曲线/曲面积分：带方向或侧向。
5. 闭区域、闭曲面、闭曲线：优先想格林公式、高斯公式、斯托克斯公式。

入口：[[14_高数第14讲_二重积分]]、[[18_高数第18讲_多元函数积分学]]

### 5. 线代判定题

先问自己：

1. 是数值大小问题：看行列式。
2. 是可逆、解个数、维数：看秩。
3. 是向量组：看线性相关和极大无关组。
4. 是矩阵幂或对角化：看特征值与特征向量。
5. 是正定或标准形：看二次型。

入口：[[19_线代第1讲_行列式]] 到 [[24_线代第6讲_二次型]]

### 6. 概率分布题

先判断类型：

1. 问事件概率：先翻译事件集合。
2. 问一维随机变量：看分布函数、分布律、密度。
3. 问二维随机变量：先联合，再边缘，再条件。
4. 问期望方差：先算期望和平方期望。
5. 问大样本近似：中心极限定理。
6. 问样本和估计：数理统计。

入口：[[25_概率第1讲_随机事件与概率]] 到 [[30_概率第6讲_数理统计]]

## 最容易浪费时间的地方

1. 极限题一上来就洛必达：先看等价和泰勒。
2. 数列题硬算通项：先看单调有界和递推极限。
3. 中值题乱构造函数：先看端点条件和目标式结构。
4. 积分题不画区域：二重积分、三重积分、几何应用必须先画。
5. 线代题不看秩：方程组、向量组、可逆性很多都归到秩。
6. 特征值题只看特征值：对角化还要看线性无关特征向量个数。
7. 概率题不翻译事件：文字概率先变成集合运算。
8. 条件概率把条件忘掉：条件会改变样本空间。
9. 二维分布题直接看边缘：先看联合分布。
10. 数理统计题背公式前不分“方差已知/未知”：已知用标准正态，未知用 $t$ 分布。

## 只背这些优先级最高

### 高数

1. 常用等价无穷小与泰勒展开。
2. 导数与微分公式。
3. 单调、极值、凹凸、拐点判定。
4. 换元积分、分部积分、变限积分求导。
5. 面积、旋转体体积、弧长公式。
6. 二重积分换序与极坐标。
7. 级数收敛判别和常用展开。
8. 格林公式、高斯公式、斯托克斯公式适用条件。

### 线代

1. 行列式性质。
2. 逆矩阵、伴随矩阵、初等矩阵。
3. 秩与方程组解的关系。
4. 向量组线性相关判定。
5. 相似对角化条件。
6. 实对称矩阵正交对角化。
7. 二次型正定判定。

### 概率

1. 加法公式、条件概率、全概率公式、贝叶斯公式。
2. 常见分布的分布律或密度。
3. 常见分布的期望和方差。
4. 二维边缘、条件、独立性。
5. 中心极限定理标准化。
6. 正态总体常用统计量。
7. 矩估计、最大似然估计、置信区间、假设检验。
