## 背景设置

 - 直接设置body样式，可以更改整个页面
 - 
```css
body {

  background-image: url('../components/img/background.png');

  margin: 0;

  padding: 0;

  height: 100vh;

  display: flex;

  align-items: center;

  justify-content: center;

  flex-direction: column;

  overflow: auto;

}
```
