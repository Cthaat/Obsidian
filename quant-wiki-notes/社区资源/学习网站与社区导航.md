# 学习网站与社区导航

> [!note] 核心问题
> 社区与网站是加速器，也会变成时间陷阱。本篇按**用途**列出高信号入口：官方文档、数据、平台、监管与教材框架，并给使用纪律。

## 学习目标

1. 建立分类书签（文档 / 数据 / 平台 / 论文 / 求职）。  
2. 每个分类只保留 3–5 个主站。  
3. 会用社区搜报错，而不是要「稳赚策略」。  
4. 知道监管与投资者教育入口的存在。  
5. 与 [[网站工具与资源导航]]（阶段零）互补：本篇偏**全库与全球**，阶段零偏课程动手。  

## 使用纪律（先读）

| 规则 | 说明 |
|---|---|
| 文档 > 营销页 | 以 docs/GitHub README 为准 |
| 报错检索 | 贴版本号与最小复现 |
| 不信收益截图 | 无成本、无样本外直接忽略 |
| 每周上网时限 | 与 [[实操百科总索引]] 学习块绑定 |
| 中英文双轨 | A 股工具中文站 + 经典理论英文源 |
