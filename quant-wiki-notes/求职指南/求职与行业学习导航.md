# 求职与行业学习导航

> [!note] 核心问题
> 面经与机构故事很容易挤占研究时间。本篇把 **求职指南 + 行业洞察** 收成一条可执行路径：先定角色与作品集，再限时读路径/简历/面试，最后用行业笔记建立地图——**不替代**阶段零研究闭环。

## 学习目标

1. 选定目标角色并写出一句话定位。  
2. 用作品集清单对照自己已有交付物。  
3. 按「路径 → 简历 → 面试 → 机构」顺序抽样阅读。  
4. 设置每周求职信息时限。  
5. 区分买方/卖方/自营等雇主类型的信息用途。  

## 总原则

| 原则 | 做法 |
|---|---|
| 作品集 > 面经 | 研究包、风控卡、公司分析优先 |
| 时限 | 求职阅读建议 ≤3h/周（可自调） |
| 年份敏感 | 薪资/招聘文注意时效，交叉验证 |
| 地图而非八卦 | 机构笔记用来理解分工与文化，非内幕交易 |

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 先背完面经再做项目 | 项目证明可迁移能力 |
| 薪资截图当规划 | 方差大、结构不同 |
| 只追海外 MM 叙事 | 与自身签证/学历/地域匹配 |
| 行业八卦当 alpha | 信息与可交易信号不同 |
