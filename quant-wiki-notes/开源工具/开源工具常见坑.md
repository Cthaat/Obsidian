# 开源工具常见坑

> [!note] 核心问题
> 90% 的「框架不行」其实是**环境、版本、数据、权限、期望**问题。本篇按症状索引，减少乱换框架。

## 学习目标

1. 遇到报错先分环境/数据/权限/逻辑四类。  
2. 建立最小复现习惯。  
3. 知道何时该换工具、何时该读文档。  
4. 避免依赖地狱与星标收藏症。  

## 排错决策树

```text
能 import / 能启动吗？
  否 → 环境与安装
  是 → 能取到数据吗？
        否 → 数据源/权限/网络
        是 → 结果荒谬吗？
              是 → 复权/未来函数/成本/成交假设
              否 → 性能或扩展问题
```

## 环境与安装

| 症状 | 常见原因 | 处理 |
|---|---|---|
| `No module named ...` | 装错解释器 | `python -m pip` + 选对 IDE 解释器 |
| 编译失败 | 缺 wheel/版本不兼容 | 换 Python 小版本；用 conda |
| vn.py 与研究库冲突 | 依赖树巨大 | **独立环境** |
| Docker 相关失败 | 守护进程未开 | 先开 Docker 再 lean |
| Windows 路径/权限 | 中文路径、杀毒 | 项目放简单路径 |

## 数据

| 症状 | 常见原因 | 处理 |
|---|---|---|
| 空表 | 代码后缀、日期格式、权限 | 打印参数；读接口权限 |
| 收益爆炸 | 复权混用、单位错误 | [[复权与公司行动实操]] |
| 接口昨天还行今天挂 | 网页源变更 | 升级 AKShare；看 changelog |
| QC/LEAN 无行情 | 未下载/生成数据 | CLI 数据步骤 |

## 回测逻辑

| 症状 | 常见原因 | 处理 |
|---|---|---|
| 夏普高得离谱 | 费用=0、未来函数 | 加成本；检查 shift |
| 与另一引擎差很远 | 仓位单位、成交时点 | 对齐假设再比 |
| 参数扫完完美 | 过拟合 | 预注册；样本外一次主结果 |
| 向量化很快但实盘崩 | 规则未建模 | A 股约束清单 |

## 权限与账号

| 症状 | 处理 |
|---|---|
| Tushare 积分不足 | 换接口或 AKShare；读权限表 |
| QC lean 会员限制 | 以官网层级说明为准 |
| token 泄露 | 重置；环境变量；勿提交 git |

## 期望管理

| 错误期望 | 校正 |
|---|---|
| 换框架提高收益 | 框架不产生 alpha |
| 列表 97 库都要会 | 一条主链 |
| 论坛给完整实盘 | 学架构与排错 |
