# 工具实操总导航

> [!note] 核心问题
> 工具无限，时间有限。本篇用**场景表**帮你选一条主链，并给出官方起点，避免同时装 vn.py + Qlib + 三个回测框架。

## 学习目标

1. 按研究/回测/交易/AI 选主工具。  
2. 区分「列表收藏」与「上手闭环」。  
3. 知道中文 A 股与美股/全球工具链差异。  
4. 第一周只承诺一条链的 Hello World。  
5. 会用 [[quant_learn]] 做检索而不是通读。

## 三条推荐主链（只选一条深挖两周）

### 链 A：个人研究最快（中国学习者默认）

```text
Python + AKShare → quant-lab / Backtrader → 模拟日志
可选：聚宽并行
```

### 链 B：交易工程

```text
VeighNa 安装 → 官方 demo → 仿真 → 再谈网关
单独 conda 环境，勿与轻量研究环境混装
```

### 链 C：AI 量化研究

```text
Qlib 安装 → 准备数据 → qrun 示例工作流
先理解工作流，再换模型
```

## 第一周禁止事项

| 禁止 | 原因 |
|---|---|
| 同时 Hello World 四个框架 | 无深度 |
| 只 star 不跑 | 假进度 |
| 未读文档就改框架源码 | 不可维护 |
| 把星标当 alpha | 星标≠收益 |

## 如何使用 quant_learn 大列表

1. 用浏览器内搜索或 Obsidian 搜索关键词（如 `data`、`backtest`）。  
2. 每个分类只点开 **1–2** 个维护活跃的项目。  
3. 看：最近提交、文档、issue、许可。  
4. 记入 EXP：为何选/不选。  
5. **不要**从头读到尾。
