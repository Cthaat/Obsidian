# 多空策略入门：Long-Short Equity 的逻辑、风控与回测

> [!note] 核心问题
> 多空策略不是简单“买好公司、空坏公司”，而是通过多头和空头组合，把市场风险、行业风险和风格风险尽量拆开，集中赚取相对判断的收益。

## 你会学到什么

1. Long-Short Equity 策略如何赚钱？
2. 多头、空头、净敞口、总敞口分别是什么？
3. 市场中性和普通多空有什么区别？
4. 如何用因子或基本面构建多空组合？
5. 多空策略的回测和风控有哪些关键坑？

## 1. 什么是多空策略

多空股票策略同时持有：

- 多头：买入预期跑赢的股票；
- 空头：卖出或做空预期跑输的股票。

它的目标可以是：

1. 降低整体市场方向风险；
2. 赚取股票之间的相对收益；
3. 捕捉价值、动量、质量、成长等因子；
4. 在市场上涨和下跌中都尽量保留收益来源。

## 2. 多空策略的核心敞口

### 净敞口

$$
Net\ Exposure = Long\ Exposure - Short\ Exposure
$$

如果多头 100%，空头 100%，净敞口为 0%，组合大致市场中性。

如果多头 100%，空头 40%，净敞口为 60%，组合仍明显偏多。

### 总敞口

$$
Gross\ Exposure = Long\ Exposure + |Short\ Exposure|
$$

多头 100%、空头 100% 时，总敞口是 200%。这说明组合使用了较高杠杆或资金效率。

### Beta 敞口

净敞口为 0 不等于 Beta 为 0。若多头股票 Beta 较高、空头股票 Beta 较低，组合仍可能对市场上涨下跌敏感。

## 3. 普通多空 vs 市场中性

| 维度 | 普通多空 | 市场中性 |
|---|---|---|
| 净敞口 | 可正可负 | 通常接近 0 |
| 目标 | Alpha + 部分市场方向 | 尽量只赚 Alpha |
| 风控 | 关注多空比例 | 关注 Beta、行业、因子中性 |
| 难度 | 中 | 高 |
| 常见风险 | 市场下跌、空头挤压 | 因子拥挤、成本、模型失效 |

市场中性并不意味着无风险，只是尽量减少某些系统性风险。

## 4. 收益来源

多空策略收益通常来自：

### 1. 选股 Alpha

做多被低估或未来改善的股票，做空被高估或未来恶化的股票。

这需要结合 [[财务比率分析]]、[[估值方法入门]] 和 [[point72-idea]]。

### 2. 因子收益

按因子排序构建组合：

- 价值；
- 动量；
- 质量；
- 低波；
- 盈利修正；
- 成长；
- 规模。

可配合 [[因子投资体系]]。

### 3. 风格轮动

不同市场环境中，价值、成长、动量、质量等风格会轮流表现。策略可尝试动态调整因子权重，但过度择时容易过拟合。

### 4. 风险对冲

空头不只是盈利工具，也能降低市场下跌时的组合波动。

## 5. 一个基础构建流程

```mermaid
flowchart LR
    A[股票池] --> B[数据清洗]
    B --> C[因子计算]
    C --> D[行业/市值中性处理]
    D --> E[排序打分]
    E --> F[多头/空头组合]
    F --> G[风险约束]
    G --> H[交易和再平衡]
```

### 第一步：确定股票池

筛选条件可以包括：

- 市值；
- 日均成交额；
- 上市时间；
- 是否可做空；
- 是否停牌；
- 是否 ST 或退市风险；
- 数据是否完整。

### 第二步：计算信号

例子：

```text
价值因子 = 低 PE + 低 PB + 高 FCF yield
质量因子 = 高 ROE + 高毛利率 + 低杠杆
动量因子 = 过去 12 个月收益 - 最近 1 个月收益
```

### 第三步：排序

按综合分排序：

- 前 20% 做多；
- 后 20% 做空；
- 中间股票不持有。

### 第四步：中性化

控制：

- 行业暴露；
- 市值暴露；
- Beta 暴露；
- 风格因子暴露；
- 个股权重。

### 第五步：再平衡

常见频率：

- 每日：成本高，适合短周期；
- 每周：平衡反应速度和成本；
- 每月：适合基本面因子；
- 季度：适合财报驱动策略。

## 6. 简化 Python 思路

下面只展示研究结构，不代表完整实盘代码：

```python
import pandas as pd

def build_long_short_weights(scores: pd.Series, long_q: float = 0.2, short_q: float = 0.2) -> pd.Series:
    scores = scores.dropna()
    long_cut = scores.quantile(1 - long_q)
    short_cut = scores.quantile(short_q)

    weights = pd.Series(0.0, index=scores.index)
    longs = scores[scores >= long_cut].index
    shorts = scores[scores <= short_cut].index

    weights.loc[longs] = 1 / len(longs)
    weights.loc[shorts] = -1 / len(shorts)
    return weights
```

真实回测还需要处理：

- 数据滞后；
- 停牌；
- 退市；
- 涨跌停；
- 做空可得性；
- 交易成本；
- 行业和 Beta 中性；
- 调仓成交价。

## 7. 多空策略的主要风险

### 1. 空头风险

做空亏损理论上没有上限，还可能遇到：

- 借券成本上升；
- 借券被召回；
- short squeeze；
- 被收购导致股价跳涨；
- 交易限制。

### 2. 因子拥挤

许多基金使用类似因子。市场反转时，大家同时平仓，可能造成剧烈回撤。

### 3. 行业和风格暴露

如果多头集中科技、空头集中金融，你以为在做选股，实际可能在赌行业。

### 4. 成本侵蚀

多空策略换手率较高，成本包括：

- 佣金；
- 印花税；
- 滑点；
- 借券费；
- 融资成本；
- 市场冲击。

### 5. 模型失效

因子有效性会随市场环境变化。价值、动量、质量都可能多年表现不佳。

## 8. 回测指标

除了收益率，还应看：

| 指标 | 解释 |
|---|---|
| 年化收益 | 策略长期回报 |
| 年化波动 | 收益稳定性 |
| 夏普比率 | 单位风险收益 |
| 最大回撤 | 历史最痛苦损失 |
| 多头贡献 | 多头是否真的有效 |
| 空头贡献 | 空头是否赚钱或对冲 |
| 换手率 | 成本压力 |
| Beta | 市场暴露 |
| 行业暴露 | 是否偏离选股逻辑 |
| 因子暴露 | 是否在赌某个风格 |

可配合 [[夏普比率]]、[[回测方法论]]、[[风险管理框架]]。

## 9. 常见误区

### 误区 1：多空一定比多头安全

如果空头上涨、多头下跌，多空会双边亏损。

### 误区 2：净敞口为零就是市场中性

Beta、行业和风格暴露仍可能很大。

### 误区 3：做空只是反向买入

做空有借券、保证金、逼空和跳涨风险。

### 误区 4：因子排名越复杂越好

复杂模型可能只是过拟合。简单稳健的因子组合往往更容易长期跟踪。

## 10. 学习练习

1. 用 20 只股票构造一个等权多空组合，计算净敞口和总敞口。
2. 选择一个因子，排序后做多前 20%、做空后 20%，观察历史表现。
3. 分别计算多头和空头的收益贡献。
4. 检查组合是否行业集中。
5. 加入不同交易成本，观察策略是否仍然有效。
6. 写出这个策略最可能失效的三种情景。

## 11. 小结

多空策略的价值在于把“我看好什么”和“我看空什么”同时表达出来，并尽量剥离不想承担的市场风险。真正难的不是同时买和卖，而是控制净敞口、总敞口、Beta、行业、因子、成本和空头风险。
