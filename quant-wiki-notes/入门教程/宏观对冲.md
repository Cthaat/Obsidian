# 宏观对冲基金入门：从经济周期到跨资产交易

> [!note] 核心问题
> 宏观对冲不是预测新闻，而是把增长、通胀、利率、政策、资本流动和风险偏好翻译成跨资产交易。

## 你会学到什么

1. 宏观对冲基金主要交易什么资产？
2. 全球宏观、固定收益相对价值、新兴市场宏观、商品策略有什么区别？
3. 宏观交易如何从经济观点变成头寸？
4. 宏观策略的风险和适用环境是什么？
5. 如何用宏观框架帮助资产配置和量化研究？

## 1. 什么是宏观对冲

宏观对冲基金从自上而下的视角做投资判断。它关心的问题包括：

- 经济增长会加速还是放缓？
- 通胀会升还是降？
- 央行会加息还是降息？
- 财政政策会扩张还是收缩？
- 哪些国家资产被高估或低估？
- 风险偏好会提升还是下降？

表达观点的工具通常包括：

- 国债期货；
- 利率掉期；
- 外汇；
- 商品期货；
- 股指期货；
- 信用指数；
- ETF；
- 期权。

宏观交易既可以是方向性的，也可以是相对价值的。

## 2. 宏观交易的基本链条

```mermaid
flowchart LR
    A[宏观变量] --> B[资产定价逻辑]
    B --> C[交易观点]
    C --> D[工具选择]
    D --> E[仓位和风险预算]
    E --> F[情景跟踪]
```

例如：

1. 判断美国通胀高于预期；
2. 推测央行维持高利率更久；
3. 债券价格可能承压，美元可能偏强；
4. 可用国债期货、利率掉期、美元指数或相关货币对表达；
5. 设置止损、久期风险、事件风险；
6. 跟踪 CPI、就业、央行会议和市场预期变化。

## 3. 四类常见宏观策略

### 1. 全球宏观

全球宏观授权最宽，可在不同国家和资产类别之间寻找机会。

常见交易：

- 做多某国债券，做空另一国债券；
- 做多美元，做空高风险货币；
- 做多黄金或能源；
- 用期权表达危机风险；
- 做收益率曲线陡峭化或平坦化。

适合环境：

- 政策转折；
- 通胀和增长预期变化；
- 危机和风险偏好突变；
- 国家之间基本面分化。

不利环境：

- 低波动、政策稳定、资产同向震荡；
- 宏观数据与市场价格长期脱节。

### 2. 固定收益相对价值

固定收益相对价值（FIRV）利用债券和利率衍生品之间的定价差异。

常见交易：

- 现金国债 vs 国债期货；
- 不同期限债券曲线交易；
- 利率掉期 vs 国债；
- 不同国家利率曲线相对价值；
- 基差交易。

特点：

- 单笔价差通常很小；
- 杠杆可能较高；
- 对融资和流动性敏感；
- 平时波动低，危机时尾部风险大。

这类策略尤其需要 [[evt-var-es]] 和压力测试。

### 3. 新兴市场宏观

新兴市场宏观关注新兴国家的债券、外汇、股票和商品相关资产。

收益机会来自：

- 改革和政策变化；
- 主权信用改善或恶化；
- 外部资金流入流出；
- 商品周期；
- 汇率失衡；
- 本币利率变化。

风险也更高：

- 流动性差；
- 政策不透明；
- 资本管制；
- 汇率跳贬；
- 地缘政治；
- 做空工具有限。

### 4. 商品宏观

商品策略关注能源、金属、农产品等。

分析变量包括：

- 供给；
- 库存；
- 需求；
- 天气；
- 地缘政治；
- 运输和仓储；
- 期限结构；
- 替代品价格。

常见交易：

- 原油趋势；
- 裂解价差；
- 金银比价；
- 铜与宏观增长；
- 农产品天气冲击；
- 期货曲线近远月价差。

商品容易出现大波动，必须关注保证金和跳空风险。

## 4. 宏观观点如何转成交易

同一个宏观观点可以有多种表达方式。

例如你判断“通胀会高于预期”：

| 表达方式 | 逻辑 | 风险 |
|---|---|---|
| 做空长期国债 | 利率上升，债券价格下跌 | 避险买盘压低利率 |
| 做多商品 | 通胀与商品相关 | 增长下滑导致需求下降 |
| 做多美元 | 高利率支持美元 | 其他国家更鹰派 |
| 买通胀保护债 | 通胀补偿上升 | 流动性和实际利率影响 |
| 买利率波动率 | 政策不确定性上升 | 期权费过贵 |

宏观交易的关键不是观点本身，而是选择风险收益更好的表达工具。

## 5. 宏观策略的风险

### 1. 观点正确但交易亏钱

你可能判断对了通胀，但选错工具。例如通胀上行同时经济衰退，商品可能跌，债券可能因避险上涨。

### 2. 时间不匹配

宏观逻辑可能需要 6 个月兑现，但你的止损和保证金只能承受 2 周波动。

### 3. 政策反应函数变化

央行和政府不是机械规则。市场关注的不只是数据本身，而是政策制定者如何解释数据。

### 4. 相关性突变

危机中股、债、汇、商之间的关系可能改变。历史相关性不一定可靠。

### 5. 杠杆与流动性

宏观工具流动性通常较好，但在危机中价差和保证金要求会快速变化。

## 6. 宏观分析的学习框架

### 增长

关注：

- GDP；
- PMI；
- 就业；
- 消费；
- 信贷；
- 企业盈利。

增长上行通常利好风险资产，但如果同时推高通胀和利率，影响会复杂化。

### 通胀

关注：

- CPI；
- PPI；
- 工资；
- 租金；
- 商品价格；
- 通胀预期。

通胀影响利率、汇率、商品和股票估值。

### 政策

关注：

- 央行利率；
- 资产购买或缩表；
- 财政赤字；
- 税收；
- 产业政策；
- 监管。

政策变化往往是宏观交易的重要催化剂。

### 风险偏好

关注：

- 信用利差；
- 股票波动率；
- 美元；
- 黄金；
- 新兴市场资金流；
- 高收益债。

风险偏好下降时，很多风险资产会同时受压。

## 7. 宏观与量化如何结合

量化宏观可以把主观判断转成规则，例如：

- 当通胀超预期且趋势上行时降低久期；
- 当 PMI 改善且估值不高时增加股票；
- 当美元趋势强且利差支持时做多美元；
- 当商品期限结构 backwardation 且趋势向上时做多商品。

注意：宏观数据频率低、修订多、样本少，过度拟合风险很高。可配合 [[回测方法论]]。

## 8. 学习练习

1. 选择一个国家，画出增长、通胀、政策、汇率、股市、债市的关系图。
2. 找一个近期宏观事件，列出至少三种交易表达方式。
3. 比较“做多商品”和“做空债券”表达通胀观点的差异。
4. 用过去 3 年数据观察股债相关性是否稳定。
5. 写一份宏观交易计划：观点、工具、止损、反证条件、观察指标。

## 9. 小结

宏观对冲的难点不是知道宏观新闻，而是理解经济变量如何进入资产定价，并把观点转成风险收益合适的交易。优秀的宏观框架必须同时处理方向、工具、时间、政策、仓位和反证条件。
