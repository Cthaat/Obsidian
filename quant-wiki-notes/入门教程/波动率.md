# 波动率入门：从历史波动到隐含波动率曲面

> [!note] 核心问题
> 波动率不是“涨跌方向”，而是“价格变化幅度的不确定性”。理解波动率，才能理解风险、期权定价、仓位管理、回测波动和策略失效。

## 你会学到什么

读完这篇笔记，你应该能回答：

1. 历史波动率和隐含波动率有什么区别？
2. 为什么同一个标的、不同执行价和到期日的期权会有不同 IV？
3. 波动率曲面、微笑、偏斜、期限结构分别在表达什么？
4. 量化策略里如何把波动率用于仓位、风控和交易成本估计？
5. 为什么波动率很难预测，但又是金融市场最核心的状态变量之一？

## 1. 波动率到底是什么

最常用的波动率定义是收益率的标准差：

$$
\sigma = \sqrt{\mathrm{Var}(r)}
$$

其中 $r$ 是收益率。若用日收益率计算日波动率，常见年化方法是：

$$
\sigma_{annual} = \sigma_{daily} \times \sqrt{252}
$$

这里的 252 是美国市场常用交易日数量。不同市场可以按实际交易日调整。

### 一个直觉例子

假设两只资产年化收益率都接近 8%：

| 资产 | 年化收益 | 年化波动 | 投资体验 |
|---|---:|---:|---|
| A | 8% | 8% | 曲线相对平稳 |
| B | 8% | 35% | 大涨大跌，很容易中途退出 |

它们长期平均收益可能相近，但风险体验完全不同。波动率越高，同样的收益越不稳定，组合越需要更强的仓位管理和心理承受力。

## 2. 历史波动率：从过去估计风险

历史波动率（Historical Volatility, HV）用已经发生的价格数据计算。常见做法：

1. 取收盘价；
2. 计算日对数收益率；
3. 取滚动窗口标准差；
4. 年化。

```python
import numpy as np
import pandas as pd

def historical_volatility(close: pd.Series, window: int = 20) -> pd.Series:
    log_ret = np.log(close / close.shift(1))
    return log_ret.rolling(window).std() * np.sqrt(252)
```

### 常见窗口

| 窗口 | 用途 | 局限 |
|---|---|---|
| 10 日 | 捕捉短期风险变化 | 噪声大，容易过度反应 |
| 20-30 日 | 接近月度风险 | 最常见，但仍滞后 |
| 60-120 日 | 观察中期状态 | 对突发风险反应慢 |
| 252 日 | 年度风险基准 | 可能混合多个市场环境 |

历史波动率的核心问题是：它描述过去，不等于未来。极端事件出现前，历史波动率常常看起来很低。

## 3. 隐含波动率：从期权价格反推预期

隐含波动率（Implied Volatility, IV）不是直接观测到的波动率，而是把市场期权价格代入定价模型后反推出的 $\sigma$。

在 Black-Scholes 框架中，看涨期权价格可写成：

$$
C = S\Phi(d_1) - Ke^{-rT}\Phi(d_2)
$$

其中：

$$
d_1 = \frac{\ln(S/K) + (r + 0.5\sigma^2)T}{\sigma\sqrt{T}}
$$

$$
d_2 = d_1 - \sigma\sqrt{T}
$$

如果市场上的期权成交价是 $V$，隐含波动率就是让模型价格等于 $V$ 的那一个 $\sigma$：

$$
C(\sigma) - V = 0
$$

### 用二分法求隐含波动率

```python
import numpy as np
from scipy.stats import norm

def bs_call_price(vol: float, s: float, k: float, t: float, r: float) -> float:
    if vol <= 0 or t <= 0:
        return max(s - k, 0.0)
    d1 = (np.log(s / k) + (r + 0.5 * vol ** 2) * t) / (vol * np.sqrt(t))
    d2 = d1 - vol * np.sqrt(t)
    return s * norm.cdf(d1) - k * np.exp(-r * t) * norm.cdf(d2)

def implied_vol_call(market_price: float, s: float, k: float, t: float, r: float) -> float:
    lower, upper = 1e-6, 5.0
    for _ in range(100):
        mid = (lower + upper) / 2
        price = bs_call_price(mid, s, k, t, r)
        if abs(price - market_price) < 1e-6:
            return mid
        if price > market_price:
            upper = mid
        else:
            lower = mid
    return (lower + upper) / 2
```

> [!warning] 实务提醒
> IV 是“模型里的波动率”。它受定价模型、利率、股息、成交价、买卖价差、流动性影响。深度实值、深度虚值或成交稀少的期权，算出来的 IV 可能很不稳定。

## 4. 历史波动率和隐含波动率的区别

| 维度 | 历史波动率 HV | 隐含波动率 IV |
|---|---|---|
| 来源 | 历史价格 | 期权价格 |
| 含义 | 过去实际波动 | 市场对未来波动和风险溢价的综合定价 |
| 是否可交易 | 不能直接交易 | 可通过期权间接交易 |
| 常见用途 | 风控、仓位、回测 | 期权定价、波动率交易、市场情绪 |
| 主要问题 | 滞后 | 受模型和流动性影响 |

一个常见观察是：IV 通常高于未来实际波动率。这部分差异可理解为卖方承担波动风险所要求的风险溢价，但它并不稳定。

## 5. 波动率曲面

同一个标的，不同执行价 $K$、不同到期日 $T$ 的期权，通常对应不同 IV。把这些点放在三维空间里，就得到波动率曲面：

- x 轴：执行价或 moneyness；
- y 轴：到期时间；
- z 轴：隐含波动率。

波动率曲面包含三类信息：

1. **价位维度**：市场更担心上涨还是下跌？
2. **期限维度**：短期风险还是长期风险更贵？
3. **形状变化**：事件、财报、政策、危机是否正在被定价？

## 6. Smile、Skew 和期限结构

### 波动率微笑

波动率微笑指深度实值和深度虚值期权 IV 高于平值期权，曲线像微笑。它说明市场认为极端上涨或极端下跌的概率高于正态分布假设。

### 波动率偏斜

股票指数期权常见左偏：低执行价看跌期权 IV 更高。原因包括：

- 投资者愿意为下跌保护付更高保费；
- 市场大跌时相关性和波动率常同时上升；
- 做市商对尾部风险要求更高补偿。

### 期限结构

同一执行价下，不同到期日的 IV 形成期限结构：

| 状态 | 含义 |
|---|---|
| 近月 IV 高于远月 | 市场担心短期事件，如财报、政策会议、危机 |
| 远月 IV 高于近月 | 市场认为长期不确定性更大 |
| 整体抬升 | 市场进入高风险状态 |
| 整体下降 | 风险偏好改善或事件落地 |

更多期权结构可配合阅读 [[期权策略]] 和 [[波动率微笑_Volatility Smile|波动率微笑]]。

## 7. 波动率在策略中的用途

### 1. 仓位缩放

如果目标组合年化波动率是 10%，当前策略年化波动率是 20%，理论仓位可以缩到 0.5 倍：

$$
w = \frac{\sigma_{target}}{\sigma_{current}}
$$

这类方法常被称为波动率目标（volatility targeting）。它能让组合风险更稳定，但在剧烈行情中可能被迫追涨杀跌。

### 2. 风险预警

波动率快速上升通常意味着：

- 市场流动性变差；
- 相关性上升；
- 止损和保证金压力增加；
- 回测中的交易成本假设可能失真。

这部分应和 [[风险管理框架]] 一起看。

### 3. 策略选择

不同策略喜欢不同波动环境：

| 策略 | 通常喜欢的环境 | 不利环境 |
|---|---|---|
| 趋势跟踪 | 波动上升并形成方向 | 来回震荡 |
| 均值回归 | 中等波动、价格过度反应 | 单边趋势 |
| 做市/卖波动 | 低到中等波动、流动性好 | 跳空、危机、流动性消失 |
| 买期权 | IV 偏低但未来可能剧烈波动 | IV 贵且实际波动不足 |

### 4. 回测校准

回测不能只看收益。至少要观察：

- 波动率是否稳定；
- 高波动阶段策略是否失效；
- 交易成本是否随波动率上升而扩大；
- 最大回撤是否集中在波动率突变期。

可配合 [[回测方法论]]。

## 8. 波动率不是风险的全部

波动率很重要，但不能代表所有风险。它容易低估：

- 跳空风险；
- 流动性风险；
- 杠杆和爆仓风险；
- 模型失效；
- 收益分布的偏度和肥尾；
- “平时很稳，偶尔崩盘”的负偏度策略。

所以风险分析不能只看标准差，还要看最大回撤、VaR、ES、压力测试和情景分析。尾部风险可继续阅读 [[evt-var-es]]。

## 9. 学习练习

1. 找一只指数 ETF，计算 20 日、60 日、252 日历史波动率。
2. 观察它们在大跌前后如何变化。
3. 找一条期权链，比较平值、虚值看涨、虚值看跌的 IV。
4. 思考：如果一个策略在低波动阶段收益很好，它在高波动阶段可能会遇到什么问题？
5. 把一个策略收益序列按高波动和低波动时期分组，分别计算收益、波动、夏普和最大回撤。

## 10. 小结

波动率是连接价格、风险和期权的核心变量。历史波动率告诉你过去市场有多剧烈，隐含波动率告诉你期权市场愿意为未来不确定性支付多少价格。真正重要的不是记住公式，而是学会判断：当前波动率是否异常？它会怎样影响仓位、执行、回撤和策略有效性？
