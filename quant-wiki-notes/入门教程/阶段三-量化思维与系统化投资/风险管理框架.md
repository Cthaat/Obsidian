# 风险管理框架

> [!note] 核心问题
> 风险管理不是避免亏损，而是确保亏损发生时不会把你带出游戏。任何投资和量化策略都会失效，风险管理的作用是控制失效时的损失、速度和范围。

## 学习目标

读完这篇，你要能做到：

1. 区分系统性风险、非系统性风险、流动性风险、模型风险。
2. 理解波动率、最大回撤、夏普比率、VaR、Beta 等指标。
3. 建立仓位管理、分散化、止损、再平衡和压力测试规则。
4. 知道杠杆为什么危险，以及如何设置风险预算。
5. 为自己的投资组合写一份风险监控清单。

## 风险管理的底层原则

### 1. 先活下来

长期复利要求你留在市场里。一次不可承受的大亏，会让后续所有高收益假设失去意义。

### 2. 风险不是波动这么简单

波动是风险的一种，但真正危险的风险包括：

- 永久性亏损；
- 流动性枯竭；
- 杠杆爆仓；
- 策略失效；
- 心理崩溃后在底部卖出。

### 3. 风险要在买入前定义

如果买入后才想“跌了怎么办”，通常已经晚了。每笔投资都应该提前定义：

- 最大亏损多少；
- 什么条件说明我错了；
- 仓位是否可承受；
- 是否会影响整体组合。

## 风险分类

| 风险类型 | 含义 | 例子 | 应对方法 |
|---|---|---|---|
| 系统性风险 | 整个市场共同下跌 | 熊市、利率上升、金融危机 | 资产配置、现金、对冲 |
| 非系统性风险 | 单个公司或行业风险 | 财务造假、政策打击 | 分散化、单票上限 |
| 流动性风险 | 想卖时卖不出去 | 跌停、停牌、小盘股无量 | 流动性过滤、仓位限制 |
| 杠杆风险 | 借钱放大损益 | 融资爆仓、期货穿仓 | 控制杠杆或不用杠杆 |
| 模型风险 | 策略假设失效 | 回测过拟合、因子衰减 | 样本外、监控、暂停规则 |
| 操作风险 | 执行和流程错误 | 下错单、数据源错误 | 自动检查、日志、权限控制 |

## 核心风险指标

### 1. 波动率

$$
年化波动率 = 日收益率标准差 \times \sqrt{252}
$$

波动率衡量收益上下波动程度。

| 年化波动率 | 常见资产特征 |
|---:|---|
| < 5% | 现金、短债 |
| 5%-15% | 债券、稳健组合 |
| 15%-25% | 宽基股票指数 |
| 25%-40% | 高波动行业或个股组合 |
| > 40% | 杠杆、衍生品、极高波动资产 |

波动率不是坏事，但如果你无法承受波动，就会在错误时间卖出。

### 2. 最大回撤

最大回撤是从历史高点到后续低点的最大跌幅。

$$
MDD = \frac{峰值 - 谷底}{峰值}
$$

它比波动率更接近投资者真实痛感。

| 最大回撤 | 心理影响 |
|---:|---|
| -10% | 大多数人能承受 |
| -20% | 开始怀疑策略 |
| -30% | 很多人会停止执行 |
| -50% | 需要 +100% 才回本 |
| -70% | 多数人已经退出 |

策略年化收益再高，如果最大回撤超过承受能力，也不适合你。

### 3. 夏普比率

$$
夏普比率 = \frac{年化收益率 - 无风险利率}{年化波动率}
$$

| 夏普比率 | 常见评价 |
|---:|---|
| < 0 | 不如无风险资产 |
| 0-0.5 | 风险补偿较弱 |
| 0.5-1.0 | 可以接受 |
| 1.0-2.0 | 优秀 |
| > 2.0 | 很好，但要警惕样本短或过拟合 |

夏普不适合评价极端尾部风险很大的策略，因为它主要看平均波动。

### 4. VaR 和 CVaR

VaR 表示在某个置信水平下，可能亏损不超过多少。

例子：

- 95% 单日 VaR = -3%，表示正常情况下 95% 的交易日亏损不会超过 3%。

CVaR 进一步回答：如果超过 VaR，平均会亏多少。

VaR 的局限是：它依赖历史分布，容易低估黑天鹅。

### 5. Beta

Beta 衡量组合相对市场的敏感度。

| Beta | 含义 |
|---:|---|
| 1 | 与市场大致同步 |
| > 1 | 比市场更激进 |
| 0-1 | 比市场更防御 |
| < 0 | 与市场反向 |

如果你的组合 Beta 很高，牛市会很舒服，熊市也会更痛。

## 仓位管理

仓位管理决定你错的时候亏多少。

### 等权仓位

每个标的买同样金额。简单、透明，适合新手。

### 单票上限

常见规则：

- 单只股票不超过总资产 5%-10%；
- 学习仓单票不超过学习仓 20%-30%；
- 高波动资产比例更低。

### 行业上限

单一行业不超过总资产 20%-30%，避免政策或周期冲击。

### 风险预算

用“愿意亏多少钱”反推买入金额。

例子：

- 总资金 100000 元；
- 单笔最大亏损 1% = 1000 元；
- 止损距离 10%；
- 可买金额 = 1000 / 10% = 10000 元。

这样即使止损触发，组合总损失仍然可控。

## Kelly 公式

$$
最优仓位 = \frac{胜率 \times 盈亏比 - 败率}{盈亏比}
$$

Kelly 公式提醒我们：仓位和胜率、盈亏比有关。但现实中胜率和盈亏比很难稳定估计，完全 Kelly 往往仓位过重。

实务中更常用：

- 半 Kelly；
- 四分之一 Kelly；
- 固定风险预算。

新手不建议依赖 Kelly 精确计算仓位。

## 止损与退出

### 三类止损

| 类型 | 含义 | 适合 |
|---|---|---|
| 价格止损 | 跌到某个价格或比例卖出 | 技术交易、短线策略 |
| 基本面止损 | 买入逻辑被证伪 | 长期个股投资 |
| 时间止损 | 一段时间内没有按预期发展 | 事件驱动、趋势策略 |

长期投资不一定要机械价格止损，但必须有“逻辑止损”。如果商业模式、财务质量或估值假设被证伪，不能只因为亏损就不卖。

## 分散化

分散化要避免两种假分散：

1. 买很多同一行业股票；
2. 买很多名称不同但风险暴露相同的基金。

真正的分散包括：

- 资产类别分散；
- 地域分散；
- 行业分散；
- 策略分散；
- 时间分散；
- 流动性分散。

分散化会降低暴富概率，但能提高长期生存概率。

## 杠杆控制

杠杆会同时放大收益和亏损。

| 杠杆倍数 | 资产下跌 20% 后权益亏损 |
|---:|---:|
| 1 倍 | -20% |
| 2 倍 | -40% |
| 3 倍 | -60% |
| 5 倍 | -100% |

杠杆最危险的地方不是亏损比例变大，而是你可能在最低点被强制平仓，失去等待恢复的资格。

新手原则：

- 不借钱投资；
- 不用高杠杆衍生品；
- 不把短期资金投入高波动资产；
- 不把“低波动历史”当成“可以加杠杆”的理由。

## 压力测试

压力测试问的是：如果极端情况发生，我会怎样？

可以模拟：

| 情景 | 检查问题 |
|---|---|
| 市场下跌 30% | 组合会亏多少？是否会被迫卖出？ |
| 单票跌停连续 3 天 | 仓位是否过大？ |
| 利率快速上升 | 债券和成长股是否承压？ |
| 汇率大幅波动 | 海外资产是否受影响？ |
| 流动性枯竭 | 小盘或高换手策略能否退出？ |
| 因子失效 2 年 | 是否还能坚持策略？ |

压力测试不是预测灾难，而是提前设计应急方案。

## 风险监控清单

每周或每月检查：

- [ ] 当前最大回撤是否接近预设上限？
- [ ] 单票仓位是否超标？
- [ ] 行业集中度是否过高？
- [ ] 组合 Beta 是否明显高于计划？
- [ ] 是否有低流动性持仓过大？
- [ ] 策略换手率是否异常上升？
- [ ] 是否有一条逻辑已经被证伪但仍不愿卖出？
- [ ] 现金比例是否足够应对短期需求？

## 建立个人风控规则

| 规则 | 示例 |
|---|---|
| 最大组合回撤 | 超过 -20% 暂停加风险仓 |
| 单票上限 | 总资产不超过 8% |
| 行业上限 | 总资产不超过 25% |
| 学习仓上限 | 总资产不超过 10% |
| 杠杆规则 | 不使用融资和高杠杆衍生品 |
| 再平衡 | 每半年或偏离 5% 调整 |
| 停止策略条件 | 样本外连续失效、回撤超过历史极限 |

规则要写在上涨时，而不是下跌时。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 风险管理会降低收益 | 它降低的是毁灭性亏损概率 |
| 分散化会错过大牛股 | 但能避免单一错误毁掉组合 |
| 止损就是不坚定 | 止损是承认假设可能错误 |
| 高夏普一定安全 | 可能隐藏尾部风险 |
| 历史最大回撤就是未来最大回撤 | 未来可能更差 |
