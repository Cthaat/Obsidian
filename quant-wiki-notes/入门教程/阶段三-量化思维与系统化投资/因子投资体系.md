# 因子投资体系

> [!note] 核心问题
> 因子是能解释资产收益差异的共同特征。因子投资不是寻找“神奇指标”，而是系统性地暴露在有经济逻辑、长期可能获得风险溢价或行为溢价的特征上。

## 学习目标

读完这篇，你要能做到：

1. 理解因子是什么，以及它和主观选股的关系。
2. 掌握价值、动量、质量、低波、规模五类经典因子。
3. 知道因子收益可能来自风险补偿、行为偏误或市场结构。
4. 了解单因子、多因子、因子中性和因子择时的基本做法。
5. 识别因子投资中的数据挖掘、拥挤交易和周期失效风险。

## 什么是因子

因子可以理解为股票的“标签”或“特征”。例如：

- 估值低；
- 最近涨得强；
- 盈利质量高；
- 波动率低；
- 市值小。

如果某类特征长期能解释收益差异，它就可能成为因子。

主观投资者说“我要买便宜的好公司”，量化投资者会进一步定义：

- 便宜：PE、PB、FCF Yield 排名；
- 好公司：ROE、毛利率、经营现金流质量；
- 买入：每月选择综合排名前 50 只；
- 权重：等权或按分数加权。

这就是因子化。

## 因子收益从哪里来

| 来源 | 含义 | 例子 |
|---|---|---|
| 风险补偿 | 承担某种不受欢迎的风险，获得长期补偿 | 小盘股流动性差，要求更高收益 |
| 行为偏误 | 投资者系统性犯错，导致错误定价 | 市场过度追逐热门股，低估价值股 |
| 市场结构 | 机构约束、指数规则、交易制度造成机会 | 指数调仓、风格拥挤 |

一个因子最好既有统计证据，也有经济逻辑。只有回测好看但解释不清的因子，要高度谨慎。

## 五大经典因子

### 1. 价值因子

核心思想：便宜资产长期可能跑赢昂贵资产。

| 指标 | 公式或含义 | 注意点 |
|---|---|---|
| PE | 市值 / 净利润 | 周期行业利润高点会误导 |
| PB | 市值 / 净资产 | 资产质量差时低 PB 可能是陷阱 |
| PS | 市值 / 收入 | 忽略盈利能力 |
| EV/EBITDA | 企业价值 / EBITDA | 忽略资本开支 |
| FCF Yield | 自由现金流 / 市值 | 更关注现金创造 |

价值因子可能来自投资者过度追逐成长故事，也可能来自价值股本身风险更高。

### 2. 动量因子

核心思想：过去一段时间表现强的资产，短期可能继续表现强。

常用定义：

| 指标 | 计算方式 |
|---|---|
| 12-1 动量 | 过去 12 个月收益，剔除最近 1 个月 |
| 6 个月动量 | 过去 6 个月收益 |
| 相对强度 | 相对行业或指数的超额收益 |
| 趋势强度 | 价格相对均线或突破幅度 |

动量因子可能来自市场反应不足和羊群行为。

风险：

- 趋势反转时亏损快；
- 拥挤交易会放大回撤；
- 高换手带来成本压力。

### 3. 质量因子

核心思想：财务质量高、盈利稳定、现金流好的公司长期更稳健。

| 指标 | 含义 |
|---|---|
| ROE / ROA | 资本回报能力 |
| 毛利率稳定性 | 产品或商业模式稳定 |
| 经营现金流 / 净利润 | 利润质量 |
| 低负债率 | 财务风险较低 |
| 应计利润低 | 盈利更接近现金 |
| 盈利波动低 | 可持续性更强 |

质量因子和阶段二的基本面分析联系最紧。它把“好公司”变成可排序的指标。

### 4. 低波动因子

核心思想：低波动股票的风险调整后收益可能优于高波动股票。

常用指标：

- 历史波动率；
- Beta；
- 特质波动率；
- 最大回撤；
- 下行波动率。

低波因子看似违反“高风险高收益”，一个解释是很多投资者喜欢彩票型高波动股票，导致它们被高估；机构又受基准约束，不愿买低波低 Beta 股票。

### 5. 规模因子

核心思想：小市值股票长期可能获得额外收益。

常用指标：

- 总市值；
- 流通市值；
- 成交额和流动性。

规模因子可能来自小公司风险更高、流动性更差、信息覆盖不足。但小盘也更容易出现财务质量差、交易成本高和退市风险。

## Fama-French 模型

### 三因子模型

$$
R_i - R_f = \alpha + \beta_m(R_m - R_f) + \beta_s SMB + \beta_v HML + \epsilon
$$

其中：

- 市场因子：市场整体风险；
- SMB：Small Minus Big，小盘减大盘；
- HML：High Minus Low，高账面市值比减低账面市值比，代表价值因子。

### 五因子模型

后来加入：

- RMW：盈利能力强减盈利能力弱；
- CMA：投资保守减投资激进。

这些模型的意义不是让你背公式，而是理解：股票收益不只来自市场涨跌，还可能来自风格暴露。

## 单因子策略

基本流程：

1. 定义股票池；
2. 计算每只股票的因子值；
3. 按因子排序；
4. 买入排名最高的一组，卖出或回避最低的一组；
5. 定期调仓；
6. 评估收益、回撤、换手和稳定性。

单因子优点是清晰，缺点是周期性强。例如价值因子可能多年跑输，动量因子可能突然崩溃。

## 多因子策略

多因子策略把多个因子组合起来，降低单一因子失效风险。

常见方法：

| 方法 | 做法 | 优点 | 风险 |
|---|---|---|---|
| 等权打分 | 各因子标准化后等权相加 | 简单透明 | 忽略因子强弱变化 |
| 加权打分 | 根据 IC、ICIR 或经验赋权 | 更灵活 | 容易过拟合 |
| 分层筛选 | 先质量，再估值，再动量 | 逻辑清楚 | 可能过度筛选 |
| 优化器 | 加约束后求最优组合 | 可控暴露 | 复杂且依赖估计 |

新手建议从“价值 + 质量 + 动量”三因子开始，因为逻辑直观，也和基本面、技术面衔接自然。

## 因子处理的基本步骤

### 1. 去极值

处理异常值，避免极端数据影响排序。

### 2. 标准化

不同因子单位不同，需要转成可比较分数，例如 Z-score。

### 3. 行业中性

有些因子天然受行业影响。例如银行 PB 低，科技 PB 高。行业中性可以减少行业暴露干扰。

### 4. 市值中性

如果不控制市值，策略可能无意中变成小盘策略。

### 5. 滞后处理

财报数据必须按真实发布日期使用，不能在公告前使用未来数据。

这些处理看起来繁琐，但它们决定了回测是否可信。

## 因子评估指标

| 指标 | 含义 |
|---|---|
| IC | 因子值和未来收益的相关性 |
| ICIR | IC 均值 / IC 波动，衡量稳定性 |
| 分组收益 | 因子高低分组的收益差 |
| 多空收益 | 做多高分组、做空低分组的收益 |
| 换手率 | 调仓频繁程度 |
| 最大回撤 | 因子失效时的风险 |

一个好因子不一定 IC 很高，但应当逻辑清晰、方向稳定、成本可控。

## 因子投资的常见风险

1. **数据挖掘**：测试太多指标，总会出现偶然有效。
2. **前视偏差**：使用了当时还未公布的数据。
3. **因子拥挤**：太多人买同一类股票，回撤时一起卖。
4. **风格周期**：某些因子可能多年失效。
5. **交易成本**：高换手会吃掉收益。
6. **暴露混杂**：以为赚的是价值，实际赚的是小盘或行业暴露。

## 因子投资与基本面选股

| 维度 | 基本面选股 | 因子投资 |
|---|---|---|
| 优势 | 深度理解公司 | 广度覆盖、纪律执行 |
| 弱点 | 样本少、主观性强 | 可能忽略个体细节 |
| 信息来源 | 财报、行业、调研 | 数据库、指标、排序 |
| 决策方式 | 研究判断 | 规则打分 |

个人投资者可以结合两者：

1. 用基本面排除看不懂和质量差的公司；
2. 用因子排序做相对比较；
3. 用估值和风险管理决定仓位。

## 练习：设计一个三因子打分表

| 因子 | 指标 | 方向 | 权重 |
|---|---|---|---:|
| 价值 | PE、PB、FCF Yield | 越便宜越高 | 33% |
| 质量 | ROE、经营现金流 / 净利润、负债率 | 越优质越高 | 33% |
| 动量 | 过去 6-12 个月收益 | 越强越高 | 33% |

然后回答：

1. 这个组合更偏价值、质量还是趋势？
2. 哪些行业可能天然得分高？
3. 如果价值因子连续两年失效，你是否还能坚持？
4. 换手率和交易成本会不会太高？
