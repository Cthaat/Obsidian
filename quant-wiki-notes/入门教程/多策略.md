# 多策略对冲基金入门：组合不同收益来源

> [!note] 核心问题
> 多策略基金不是“什么都做”，而是把不同团队、不同资产、不同周期、不同收益来源放进同一个风险和资本分配系统里。

## 你会学到什么

1. 多策略基金为什么能获得较稳定的风险调整收益？
2. 机会主义、多基金经理聚焦型、多基金经理多元化有什么区别？
3. 资本配置、中心账本、风控和激励机制如何影响收益？
4. 多策略的优势和隐藏风险是什么？
5. 普通学习者如何把多策略思维用到自己的投资组合中？

## 1. 什么是多策略

多策略对冲基金把资本配置到多个底层策略，例如：

- 股票多空；
- 市场中性；
- 事件驱动；
- 固定收益相对价值；
- 宏观；
- 商品；
- 量化统计套利；
- 期权和波动率；
- 信贷和特殊情形。

它的目标不是押中单一方向，而是让不同收益来源在不同市场环境下互相补充。

## 2. 多策略的核心逻辑

多策略基金想解决三个问题：

### 1. 单策略周期性

任何策略都有不适合的环境：

- 趋势策略怕震荡；
- 均值回归怕单边；
- 并购套利怕监管失败；
- 固定收益相对价值怕流动性冲击；
- 卖波动怕跳空。

多策略试图用多个策略降低对单一环境的依赖。

### 2. 资本动态配置

当某类机会更好时，增加资本；当机会变差或风险升高时，降低资本。

这类似组合层面的“策略再平衡”。

### 3. 风险集中控制

多个团队可能看起来策略不同，但实际暴露可能相似。例如都在：

- 做多成长股；
- 做空价值股；
- 做多信用利差；
- 做空波动率；
- 依赖同一流动性环境。

多策略平台需要识别这些隐藏相关性。

## 3. 三种常见模式

### 1. 机会主义 / 动态型

特点：

- 一位 CIO 或少数核心投资人主导；
- 在不同策略之间灵活切换；
- 不强制始终满仓；
- 更依赖核心人物判断。

优势：

- 机会集差时可以减少风险；
- 决策统一；
- 策略之间更容易协同。

风险：

- 对核心人物依赖高；
- 规模不容易无限扩张；
- 如果判断错误，组合可能集中受损。

### 2. 多基金经理：聚焦型

特点：

- 多个 PM 运行独立小组合；
- 但集中在某个资产或策略领域；
- 例如股票市场中性平台、亚洲多策略平台。

优势：

- 专注度高；
- 可以复制优秀 PM 的框架；
- 风控相对集中。

风险：

- 同一领域风险拥挤；
- 风格切换时多个 PM 同时受损；
- 容量受市场和策略范围限制。

### 3. 多基金经理：多元化型

特点：

- 规模很大；
- 多个业务部门；
- 跨资产、跨地区、跨策略；
- 通常有复杂风控和通道费用结构。

优势：

- 分散度高；
- 可扩展；
- 资源、融资、交易对手和数据优势明显。

风险：

- 管理复杂；
- 杠杆和融资链条复杂；
- 费用结构可能很高；
- 极端市场中隐藏相关性会暴露。

## 4. 多策略基金的结构组件

### 资本配置

资本不是平均分给每个团队，而是根据：

- 历史表现；
- 当前机会集；
- 策略容量；
- 流动性；
- 回撤；
- 风险贡献；
- 与其他团队相关性；
- PM 能力和团队稳定性。

### 中心账本

中心账本是在 PM 组合之上进行集中管理的工具。常见用途：

1. 对冲平台整体风险；
2. 放大最稳定的 Alpha；
3. 降低行业、国家、因子集中；
4. 用公司层面信息做额外组合优化。

中心账本不是魔法。它依赖高质量数据、强风控和对 PM 行为的深入理解。

### 风险管理

多策略平台通常分层风控：

| 层级 | 风控内容 |
|---|---|
| PM 层 | 仓位、杠杆、回撤、品种、策略授权 |
| 策略组层 | 行业、资产类别、流动性、相关性 |
| 平台层 | 总杠杆、VaR、ES、压力测试、融资风险 |
| 运营层 | 估值、保证金、对手方、系统、合规 |

可配合 [[风险管理框架]] 和 [[evt-var-es]] 学习。

## 5. 多策略为什么常有较高夏普

原因通常来自：

1. 多个低相关收益来源；
2. 单一 PM 回撤可被限制；
3. 资本向更高质量机会流动；
4. 中心风控降低无意暴露；
5. 平台基础设施提升执行效率。

但这并不代表多策略“低风险”。高夏普背后可能包含杠杆、融资、流动性和复杂度风险。

## 6. 多策略的主要风险

### 1. 隐藏相关性

平时看起来不相关的策略，危机时可能一起去杠杆。例如股票多空、信用相对价值、并购套利都可能在流动性冲击下同时亏损。

### 2. 强制止损和踩踏

平台常对 PM 设置严格回撤线。好处是控制损失，坏处是在极端行情中可能集体减仓，加剧市场冲击。

### 3. 杠杆和融资风险

许多相对价值策略单笔收益很薄，需要杠杆放大。融资条件变化会直接影响策略生存。

### 4. 激励扭曲

通道费用和个人业绩激励有利于吸引人才，但也可能鼓励 PM 追求短期表现或承担难以观察的尾部风险。

### 5. 运营复杂度

多资产、多市场、多团队、多系统使错误更难排查。风控、估值、合规、交易确认都不能薄弱。

## 7. 普通投资者能学什么

你不需要复制大型平台，但可以借鉴三件事：

1. **不要只有一种收益来源**：长期投资、现金管理、趋势、因子、期权保护可以分工。
2. **按风险配置，而不是按想象配置**：一个小仓位高波动策略可能贡献大部分风险。
3. **定期检查相关性**：组合里的资产是否真的分散，还是危机时一起跌？

一个简化版个人多策略框架：

| 模块 | 作用 |
|---|---|
| 长期资产配置 | 提供 Beta 和复利 |
| 因子/多空研究 | 提供可验证 Alpha |
| 趋势或防守模块 | 应对危机和方向性行情 |
| 风险预算 | 控制最大损失 |
| 现金和再平衡 | 提供流动性和机会资金 |

## 8. 学习练习

1. 选 5 类策略，判断它们在通胀上行、股市大跌、利率上升、低波动震荡环境中可能如何表现。
2. 设计一个三策略组合：每个策略负责什么收益来源？
3. 用相关性矩阵检查三类资产是否真的分散。
4. 写一个组合层面的风险限额：单策略、总组合、流动性分别怎么限制？
5. 思考：如果所有策略都在同一天亏损，可能说明什么？

## 9. 小结

多策略的本质是组合工程：把不同收益来源、不同团队、不同资产和不同风险周期放在一个统一系统里管理。真正难的不是“策略多”，而是识别隐藏相关性、控制杠杆和流动性、让资本在机会和风险之间动态移动。
