# 量化策略分类：从收益来源理解对冲基金策略地图

> [!note] 核心问题
> 策略分类不是背名词，而是识别收益从哪里来、承担什么风险、适合什么市场、该如何验证。

## 你会学到什么

1. 为什么“量化”不是一种单独策略，而是一种研究和执行方式？
2. 常见量化策略可以按哪些维度分类？
3. 股票统计套利、市场中性、CTA、量化宏观、另类风险溢价分别赚什么钱？
4. 如何用风险、周期、容量、流动性、杠杆比较不同策略？
5. 学完阶段三后，如何把策略分类转化为研究计划？

## 1. 先理解：量化是一种方法，不是一种收益来源

量化投资的核心是把投资判断转成可定义、可检验、可执行的规则。它可以应用在股票、期货、债券、外汇、商品、期权等市场，也可以应用在基本面、技术面、宏观、事件、订单簿等不同信息源上。

所以“量化策略”不是单一类型。更准确的说法是：

> 量化是开发和执行策略的方法，策略本身仍然需要明确收益来源。

例如：

- 用财务指标选股，是量化基本面；
- 用价格趋势做期货，是量化 CTA；
- 用订单簿预测下一笔成交，是高频量化；
- 用期权价格做波动率交易，是量化衍生品策略；
- 用宏观数据配置股债汇商，是量化宏观。

## 2. 五个分类维度

### 1. 按资产类别

| 资产 | 常见策略 |
|---|---|
| 股票 | 多因子、市场中性、统计套利、多空策略 |
| 期货 | CTA、趋势跟踪、期限结构、跨品种套利 |
| 债券 | 久期、曲线、相对价值、信用利差 |
| 外汇 | 套息、动量、宏观相对价值 |
| 商品 | 趋势、库存周期、曲线交易、跨商品价差 |
| 期权 | 波动率交易、做市、波动率套利、尾部保护 |

### 2. 按信号来源

| 信号 | 例子 |
|---|---|
| 价格/成交量 | 动量、均值回归、突破、订单流 |
| 基本面 | 估值、质量、盈利、成长、现金流 |
| 宏观 | 通胀、利率、增长、汇率、政策 |
| 事件 | 并购、分拆、回购、财报、指数调整 |
| 情绪/另类数据 | 新闻、搜索、社交媒体、供应链数据 |

### 3. 按持有期

| 持有期 | 特征 |
|---|---|
| 毫秒到分钟 | 高频交易、做市、订单簿预测 |
| 日内到数日 | 短周期均值回归、事件反应 |
| 数周到数月 | 因子、多空、趋势、宏观 |
| 数月到数年 | 资产配置、价值、长期风险溢价 |

持有期越短，执行成本和数据质量越关键；持有期越长，基本面逻辑和风险周期更关键。

### 4. 按方向性

| 类型 | 含义 |
|---|---|
| 方向性 | 明确押注市场上涨或下跌 |
| 相对价值 | 做多一个资产、做空相关资产，赚价差变化 |
| 市场中性 | 尽量对冲市场 Beta，主要赚 Alpha 或因子差异 |
| 套利 | 捕捉定价偏差，但通常仍有执行、流动性和模型风险 |

### 5. 按收益结构

| 结构 | 典型策略 | 风险特征 |
|---|---|---|
| 正偏度 | 趋势跟踪、买期权、尾部保护 | 经常小亏，偶尔大赚 |
| 负偏度 | 卖期权、做市、部分相对价值 | 经常小赚，偶尔大亏 |
| 稳定小 Alpha | 市场中性、多因子 | 容易受拥挤和风格切换影响 |
| 周期性收益 | 宏观、商品、事件驱动 | 机会集随市场状态变化 |

偏度相关内容可配合 [[波动率]] 和 [[evt-var-es]]。

## 3. 常见量化策略地图

### 1. 股票统计套利

股票统计套利通常利用价格、成交量、相关性、短期供需失衡等数据，在大量股票之间寻找短期相对错价。

常见信号：

- 短期均值回归；
- 动量延续；
- 配对交易；
- 行业内相对强弱；
- 新闻和财报后的短期反应。

风险来源：

- 相关性突然失效；
- 因子拥挤；
- 流动性下降；
- 做空成本；
- 交易成本被低估。

适合结合 [[常见量化策略]] 和 [[回测方法论]] 学习。

### 2. 量化股票市场中性

市场中性策略通常做多预期跑赢的股票，做空预期跑输的股票，并尽量控制市场、行业、市值、风格因子等暴露。

常见步骤：

1. 建股票池；
2. 构建因子；
3. 标准化和去极值；
4. 预测收益或排序；
5. 组合优化；
6. 约束风险暴露；
7. 定期再平衡。

它和 [[多空策略]] 接近，但市场中性更强调风险对冲和因子暴露控制。

### 3. CTA / 管理期货

CTA 通常交易流动性较好的期货市场，包括股指、债券、商品、外汇。最经典的是趋势跟踪：

- 上涨趋势做多；
- 下跌趋势做空；
- 价格反转时止损或反向。

CTA 的优势是跨资产分散，并可能在危机趋势中赚钱。缺点是在震荡市场容易连续小亏。

它的收益结构常偏正偏度：平时亏小钱，遇到大趋势赚大钱。

### 4. 量化宏观与全球资产配置

量化宏观用宏观数据和资产价格数据，在股票、债券、外汇、商品之间配置风险。

典型变量：

- 通胀；
- 利率；
- 经济增长；
- 财政和货币政策；
- 估值；
- 趋势；
- 风险偏好。

量化宏观和 [[宏观对冲]] 有重叠。区别在于量化宏观更强调系统化模型，宏观对冲可以包含更多主观判断。

### 5. 另类风险溢价

另类风险溢价（Alternative Risk Premia, ARP）把一些长期存在的收益来源拆成可复制的规则，例如：

- 价值；
- 动量；
- 质量；
- 低波；
- 套息；
- 波动率风险溢价；
- 流动性溢价；
- 并购风险溢价。

ARP 的好处是透明、可组合、费用较低；缺点是容易拥挤，且一旦很多人复制同一规则，收益会下降或阶段性失效。

## 4. 策略比较表

| 策略 | 主要资产 | 方向性 | 典型周期 | 风险重点 | 入门难度 |
|---|---|---|---|---|---|
| 股票统计套利 | 股票 | 多为中性 | 短到中期 | 成本、拥挤、因子失效 | 中高 |
| 股票市场中性 | 股票 | 中性 | 中期 | 因子暴露、做空成本 | 中 |
| CTA | 期货 | 方向性 | 中期 | 震荡亏损、杠杆 | 中 |
| 量化宏观 | 跨资产 | 方向/相对价值 | 中长期 | 宏观模型失效 | 高 |
| 另类风险溢价 | 跨资产 | 多样 | 中长期 | 拥挤、尾部风险 | 中 |
| 高频交易 | 股票/期货等 | 多样 | 极短 | 系统、延迟、执行 | 很高 |
| 期权波动率 | 期权 | 多样 | 短到中期 | Gamma、Vega、跳空 | 高 |

## 5. 如何判断一类策略是否值得研究

可以用这张清单：

1. **收益来源是否清楚**：赚风险溢价、行为偏差、流动性补偿，还是信息优势？
2. **数据是否可获得**：是否有足够历史、是否有幸存者偏差？
3. **成本是否真实**：手续费、滑点、借券费、冲击成本是否纳入？
4. **容量是否足够**：策略能容纳多少资金？
5. **风险是否可控**：最坏情况下会怎么亏？
6. **是否过度拥挤**：很多人知道后，收益是否消失？
7. **是否能解释失效**：表现不好时，知道是环境不适合还是模型错了？

## 6. 从分类走向研究计划

建议用这个模板写策略研究计划：

```text
策略名称：
资产范围：
持有周期：
核心假设：
收益来源：
信号定义：
组合构建：
风险约束：
交易成本：
回测区间：
失效情景：
下一步实验：
```

这比“我想做一个量化策略”更具体，也更容易验证。

## 7. 学习练习

1. 选 3 个你感兴趣的策略，分别写出收益来源和主要风险。
2. 把一个策略放入五个分类维度：资产、信号、持有期、方向性、收益结构。
3. 比较 CTA 和市场中性：为什么一个可能在危机赚钱，另一个可能在危机受损？
4. 找一个策略回测，检查它是否真实考虑交易成本。
5. 用上面的模板写一页策略研究计划。

## 8. 小结

策略分类的目的不是给基金贴标签，而是帮助你理解：这类策略为什么可能赚钱、什么时候会亏、需要哪些数据和工程能力、应该如何验证。等你能把每个策略拆成收益来源、风险来源、执行约束和失效条件，就真正开始从“看策略”进入“研究策略”。
