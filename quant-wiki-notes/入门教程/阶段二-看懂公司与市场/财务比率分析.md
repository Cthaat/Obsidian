# 财务比率分析

> [!note] 核心问题
> 财务比率不是为了背公式，而是把财务报表翻译成可比较、可追问的语言。一个比率本身很少能直接给出结论，真正有价值的是看趋势、看同行、看商业模式是否匹配。

## 学习目标

读完这篇，你要能做到：

1. 知道盈利能力、偿债能力、运营效率、成长质量分别看哪些指标。
2. 理解比率必须结合行业和历史趋势，不能用统一标准机械判断。
3. 能用一组指标快速判断公司是“高质量成长”“低质量扩张”还是“财务压力较大”。
4. 知道哪些比率容易被一次性因素、会计政策或行业特征误导。

## 比率分析的三条原则

### 原则 1：先纵向，再横向

先看同一家公司过去 3-5 年的变化，再看同行对比。因为行业差异非常大，银行、白酒、软件、地产、零售的资产结构和利润率完全不同。

```mermaid
graph LR
    A[公司历史 3-5 年] --> B[趋势是否改善]
    B --> C[同行公司对比]
    C --> D[解释差异来自哪里]
```

### 原则 2：指标要成组使用

只看 ROE 可能忽略高杠杆，只看毛利率可能忽略费用率，只看收入增速可能忽略回款质量。一个可靠判断至少需要同时看：

- 盈利能力：赚不赚钱；
- 现金质量：赚的钱有没有回来；
- 资产效率：资产用得好不好；
- 财务安全：债务压力大不大；
- 成长质量：增长是否可持续。

### 原则 3：公式只是入口，解释才是核心

同样的高毛利率，可能来自品牌定价权，也可能来自短期涨价；同样的高负债率，银行可能正常，制造业可能危险。比率分析的目标是提出更好的问题，而不是让一个数字替你做决定。

## 一、盈利能力：公司赚钱是否有质量

| 指标 | 公式 | 重点解释 | 常见误读 |
|---|---|---|---|
| 毛利率 | (营业收入 - 营业成本) / 营业收入 | 产品定价权、成本控制、产品结构 | 不同行业差异巨大，不能跨行业硬比 |
| 营业利润率 | 营业利润 / 营业收入 | 主营业务扣除期间费用后的盈利能力 | 一次性费用或会计调整会扰动短期数据 |
| 净利率 | 净利润 / 营业收入 | 最终留给股东的利润比例 | 可能被投资收益、补贴、资产处置影响 |
| ROE | 净利润 / 平均净资产 | 股东资本回报率 | 高 ROE 可能来自高杠杆或净资产偏低 |
| ROA | 净利润 / 平均总资产 | 全部资产创造利润的效率 | 重资产行业天然偏低，轻资产行业偏高 |

> [!tip] 口径提醒
> ROE 和 ROA 更严谨的算法通常使用“平均净资产”和“平均总资产”，即期初和期末取平均。只用期末数也能粗看，但遇到资产大幅变化时会失真。

### 盈利能力怎么读

先看毛利率，再看费用率，最后看净利率。

1. 毛利率稳定或提升，说明产品或服务有一定竞争力。
2. 费用率下降，说明规模效应可能在发挥作用。
3. 净利率提升，说明增长没有被成本和费用吃掉。
4. ROE 提升，再用 [[杜邦分析法]] 拆解来源。

### 好现象与坏现象

| 现象 | 初步判断 |
|---|---|
| 收入增长，毛利率稳定，净利率提升 | 经营质量改善 |
| 收入增长，但毛利率下降、费用率上升 | 可能是低质量扩张 |
| 净利润增长，但营业利润不增长 | 可能依赖非主营收益 |
| ROE 很高，但负债率也快速上升 | 回报可能由杠杆推动 |

## 二、偿债能力：公司会不会被债务拖垮

| 指标 | 公式 | 重点解释 | 适用提醒 |
|---|---|---|---|
| 资产负债率 | 总负债 / 总资产 | 整体杠杆水平 | 银行、保险、地产要单独看行业规则 |
| 流动比率 | 流动资产 / 流动负债 | 短期偿债覆盖程度 | 存货质量差时会高估安全性 |
| 速动比率 | (流动资产 - 存货) / 流动负债 | 更保守的短期偿债能力 | 对存货占比高的公司更有用 |
| 现金短债比 | 货币资金 / 短期有息债务 | 手上现金能否覆盖短债 | 要排除受限资金 |
| 利息保障倍数 | EBIT / 利息费用 | 经营利润能覆盖多少利息 | 周期行业要看低谷期是否安全 |

偿债能力的重点不是“有没有负债”，而是负债是否和商业模式匹配。稳定现金流业务可以承受更高债务；现金流波动大的公司，高杠杆会放大风险。

### 负债分析要分清三类

| 负债类型 | 例子 | 判断方法 |
|---|---|---|
| 经营性负债 | 应付账款、合同负债 | 可能代表对供应商或客户有议价能力 |
| 有息负债 | 银行借款、债券 | 需要支付利息和本金，风险更高 |
| 或有负债 | 担保、诉讼、表外承诺 | 需要看年报附注，容易被忽略 |

> [!warning] 短债压力
> 新手最容易只看资产负债率，却忽略债务期限。长期债务压力可以慢慢处理，短期债务如果集中到期，而公司现金不足，就可能造成流动性风险。

## 三、运营效率：资产用得好不好

| 指标 | 公式 | 代表什么 | 如何理解 |
|---|---|---|---|
| 应收账款周转率 | 营业收入 / 平均应收账款 | 回款速度 | 越高通常越好，但要结合信用政策 |
| 存货周转率 | 营业成本 / 平均存货 | 存货变现速度 | 越高说明卖得快，过低可能积压 |
| 总资产周转率 | 营业收入 / 平均总资产 | 资产产生收入的效率 | 零售高，重工业低，行业差异大 |
| 固定资产周转率 | 营业收入 / 平均固定资产 | 产能利用效率 | 制造业、能源、物流更重要 |

运营效率能帮你理解商业模式。高毛利低周转的公司，通常靠品牌、专利或稀缺资源赚钱；低毛利高周转的公司，通常靠供应链和规模赚钱。

### 周转率背后的现金转换周期

更进一步可以看现金转换周期：

$$
现金转换周期 = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数
$$

含义是：公司从付钱买货，到卖货回款，中间要垫多少天资金。

现金转换周期越短，公司越不需要占用自己的钱；如果甚至为负，说明公司可能先收到客户钱，再慢慢付给供应商，这类商业模式现金流通常更舒服。

## 四、成长质量：增长是否可持续

| 指标 | 看什么 | 高质量增长的特征 |
|---|---|---|
| 收入增长率 | 需求和市场份额 | 增长稳定，且不是靠大幅赊销 |
| 净利润增长率 | 利润弹性 | 不长期低于收入增速 |
| 经营现金流增长率 | 现金回收质量 | 与利润增长大致匹配 |
| 毛利率趋势 | 产品竞争力 | 稳定或提升 |
| 资本开支强度 | 扩张投入 | 投入后能带来收入和现金流 |

成长分两种：一种是质量变好的成长，一种是规模变大但风险堆积的成长。

| 增长组合 | 初步判断 |
|---|---|
| 收入增长 + 毛利率稳定 + 现金流同步增长 | 高质量成长 |
| 收入增长 + 应收账款大增 + 经营现金流变差 | 可能靠赊销拉动 |
| 利润增长 + 负债快速上升 + 利息费用增加 | 杠杆推动，需要谨慎 |
| 收入不增长 + 净利润增长 | 可能来自降本、涨价或一次性收益，要拆原因 |

## 五、估值比率：好公司是否买贵

估值比率不直接判断公司质量，它判断价格和基本面的关系。详细方法见 [[估值方法入门]]。

| 指标 | 公式 | 适用场景 | 局限 |
|---|---|---|---|
| PE | 市值 / 净利润 | 稳定盈利公司 | 利润为负或周期波动大时失真 |
| PB | 市值 / 净资产 | 银行、保险、地产等资产型公司 | 不能反映轻资产公司的价值 |
| PS | 市值 / 营业收入 | 亏损但收入增长的公司 | 忽略成本和盈利能力 |
| PEG | PE / 净利润增速 | 稳定成长公司 | 增速预测不可靠，高增长公司容易误判 |
| EV/EBITDA | 企业价值 / EBITDA | 跨公司、跨资本结构比较 | 忽略资本开支和营运资本需求 |

估值要放在质量之后。便宜的差公司可能是价值陷阱，昂贵的好公司也可能透支未来。

## 六、财务比率诊断流程

可以按下面顺序做一次基础体检：

1. 先看收入和净利润 3-5 年趋势，判断公司是否在增长。
2. 看毛利率、营业利润率、净利率，判断增长是否有利润质量。
3. 看经营现金流和净利润，判断利润是否变成现金。
4. 看应收账款、存货、现金短债比，判断财务压力。
5. 看 ROE，并用 [[杜邦分析法]] 拆成利润率、周转率和杠杆。
6. 最后看估值比率，判断市场价格是否合理。

## 七、行业差异速查

| 行业类型 | 更重要的指标 | 原因 |
|---|---|---|
| 消费品牌 | 毛利率、净利率、ROE、经营现金流 | 品牌力和定价权决定利润质量 |
| 零售连锁 | 存货周转率、现金转换周期、同店增长 | 利润薄，周转和供应链最关键 |
| 制造业 | 毛利率、固定资产周转率、资本开支、负债率 | 重资产投入多，周期波动明显 |
| 软件互联网 | 收入增长、毛利率、销售费用率、留存能力 | 前期费用高，规模效应重要 |
| 银行保险 | PB、ROE、不良率、资本充足率 | 商业模式和普通企业不同 |
| 地产建筑 | 现金短债比、资产负债率、存货、预收款 | 杠杆高，现金流和债务期限关键 |

## 八、常见误区

1. **用统一标准判断所有行业**：例如用制造业标准看银行负债率，会得出错误结论。
2. **只看一年数据**：单年指标可能受周期、汇率、一次性收益影响。
3. **只看利润不看现金流**：利润增长但现金流恶化，是最常见的财务质量问题。
4. **只看高 ROE 不拆来源**：高 ROE 可能来自高杠杆，也可能来自净资产被压低。
5. **只看低估值**：低 PE 可能是市场错误定价，也可能是利润即将下滑。
