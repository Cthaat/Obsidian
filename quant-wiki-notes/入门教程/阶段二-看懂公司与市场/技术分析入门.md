# 技术分析入门

> [!note] 核心问题
> 技术分析研究价格、成交量和市场行为。它不能告诉你一家公司的内在价值，也不能稳定预测未来；它更适合用来观察趋势、情绪、供需和买卖时机。对长期投资者而言，技术分析是辅助工具，不是投资理由本身。

## 学习目标

读完这篇，你要能做到：

1. 理解技术分析的三个前提假设和局限。
2. 看懂 K 线、均线、成交量、MACD、RSI、布林带等基础工具。
3. 知道趋势、支撑阻力、量价关系如何辅助决策。
4. 区分“信号”和“交易系统”，避免凭单个指标买卖。
5. 把技术分析放在基本面和风险管理之后使用。

## 技术分析的三个前提

| 前提 | 含义 | 需要注意 |
|---|---|---|
| 市场行为消化信息 | 价格反映已知信息和交易者预期 | 价格也会被情绪和流动性扭曲 |
| 价格以趋势方式运行 | 趋势形成后可能延续一段时间 | 趋势总会结束，滞后信号会亏损 |
| 历史会重演 | 人性中的贪婪和恐惧反复出现 | 相似形态不等于相同结果 |

技术分析不是自然科学。它更像概率工具：帮助你描述市场状态，而不是保证预测正确。

## 一、K 线：价格行为的最小单元

一根 K 线包含四个价格：

- 开盘价；
- 收盘价；
- 最高价；
- 最低价。

| K 线元素 | 代表什么 |
|---|---|
| 实体 | 开盘到收盘之间的变化 |
| 上影线 | 盘中上冲后回落 |
| 下影线 | 盘中下探后回升 |
| 阳线 | 收盘高于开盘，买方较强 |
| 阴线 | 收盘低于开盘，卖方较强 |

### 常见形态怎么理解

| 形态 | 表面含义 | 使用提醒 |
|---|---|---|
| 长阳线 | 买方力量强 | 要看是否放量、是否突破关键位置 |
| 长阴线 | 卖方力量强 | 要看是否跌破支撑、是否恐慌放量 |
| 十字星 | 多空分歧，方向不明 | 需要结合前后趋势 |
| 长下影 | 下方有承接 | 下跌趋势中可能只是短暂反弹 |
| 长上影 | 上方抛压大 | 上涨末期更值得警惕 |

单根 K 线的可靠性很低，必须放在趋势、位置和成交量里看。

## 二、趋势和均线

趋势是技术分析的核心。最简单的趋势定义：

- 高点和低点不断抬高：上升趋势；
- 高点和低点不断降低：下降趋势；
- 上下边界反复震荡：震荡区间。

### 移动平均线

$$
MA(N) = \frac{最近 N 天收盘价之和}{N}
$$

| 均线 | 常见用途 |
|---|---|
| MA5 / MA10 | 短期交易情绪 |
| MA20 / MA30 | 月度趋势 |
| MA60 | 季度趋势 |
| MA120 | 半年趋势 |
| MA250 | 年线，长期趋势参考 |

### 金叉和死叉

- 金叉：短期均线上穿长期均线，代表短期趋势强于长期趋势；
- 死叉：短期均线下穿长期均线，代表短期趋势弱于长期趋势。

金叉不是必然买入，死叉也不是必然卖出。它们是趋势状态变化的提醒，必须结合市场环境、成交量和交易计划。

## 三、支撑、阻力与突破

| 概念 | 含义 |
|---|---|
| 支撑位 | 价格下跌到某区域后买盘可能增强 |
| 阻力位 | 价格上涨到某区域后卖盘可能增强 |
| 突破 | 价格有效越过阻力或支撑 |
| 假突破 | 短暂越过后迅速回到原区间 |

支撑和阻力不是精确价格，而是区域。它们之所以有效，通常来自过去成交密集区、心理价位、均线位置或资金成本。

### 突破是否有效，看三点

1. 是否伴随成交量放大；
2. 是否收盘站稳，而不只是盘中刺穿；
3. 突破后回踩是否能守住原阻力位。

## 四、成交量：价格背后的参与度

成交量代表市场参与程度。价格变化如果没有成交量配合，可靠性通常较弱。

| 量价组合 | 常见解释 |
|---|---|
| 放量上涨 | 买盘积极，趋势可能延续 |
| 缩量上涨 | 上涨动能不足，需警惕 |
| 放量下跌 | 卖压强，可能有恐慌或资金撤退 |
| 缩量下跌 | 抛压减弱，也可能无人接盘 |
| 底部放量 | 分歧加大，可能换手充分 |
| 顶部巨量 | 情绪极热，需警惕派发 |

> [!warning] 注意
> “量在价先”不是绝对规律。成交量只能说明参与度变化，不能单独判断方向。

## 五、常用指标

### MACD

MACD 由快慢均线差和信号线构成，用来观察趋势动能。

常见用法：

- DIF 上穿 DEA：金叉，动能改善；
- DIF 下穿 DEA：死叉，动能转弱；
- 柱状线扩大：趋势动能增强；
- 价格创新高但 MACD 不创新高：可能背离。

局限：MACD 滞后，震荡市场容易频繁假信号。

### RSI

RSI 衡量一段时间内上涨和下跌力量对比。

$$
RSI = 100 - \frac{100}{1 + RS}
$$

| RSI | 常见解释 |
|---:|---|
| > 70 | 可能超买 |
| 30-70 | 常规区间 |
| < 30 | 可能超卖 |

趋势很强时，RSI 可以长期超买或超卖，所以不能看到 70 就卖、30 就买。

### 布林带

布林带由中轨和上下轨组成：

- 中轨：通常是 MA20；
- 上轨：中轨 + 2 倍标准差；
- 下轨：中轨 - 2 倍标准差。

常见观察：

- 带宽收窄：波动下降，可能酝酿方向选择；
- 沿上轨运行：强趋势；
- 跌破下轨后快速收回：可能短期超卖；
- 跌破下轨并持续放量：可能不是机会，而是风险。

## 技术分析与基本面分析

| 维度 | 基本面分析 | 技术分析 |
|---|---|---|
| 关注问题 | 值不值得买 | 什么时候买卖 |
| 数据来源 | 财报、行业、宏观、估值 | 价格、成交量、指标 |
| 时间尺度 | 中长期 | 短中期 |
| 优势 | 判断质量和价值 | 判断趋势和情绪 |
| 局限 | 不擅长择时 | 不判断内在价值 |

更稳妥的组合是：

1. 用基本面筛选好资产；
2. 用估值判断是否值得买；
3. 用技术面辅助入场和风控；
4. 用仓位管理控制错误成本。

## 从指标到交易系统

新手最容易犯的错误，是看到一个信号就下单。真正可执行的交易系统至少包括：

| 模块 | 要回答的问题 |
|---|---|
| 标的池 | 交易哪些资产 |
| 入场条件 | 什么情况下买入 |
| 出场条件 | 什么情况下卖出 |
| 仓位规则 | 每次买多少 |
| 风控规则 | 错了亏多少退出 |
| 评估方法 | 如何判断策略有效 |

没有风控和仓位规则的技术信号，只是看图猜方向。

## 技术分析的局限

1. 指标大多来自历史价格，天然滞后。
2. 形态识别主观性强，不同人可能看出不同结论。
3. 越多人使用的信号，越可能被提前交易或失效。
4. 震荡市场会让趋势指标频繁亏损。
5. A 股涨跌停、T+1、停牌等制度会影响技术策略执行。
6. 技术分析无法识别财务造假、商业模式恶化和估值泡沫。

| 价格与 MA20/MA60 关系 |  |

| 成交量是否异常 |  |
| RSI/MACD 状态 |  |
| 如果买入，止损或退出条件 |  |

最后写一句话：技术面是在支持我的基本面判断，还是在提醒我暂时等待？
