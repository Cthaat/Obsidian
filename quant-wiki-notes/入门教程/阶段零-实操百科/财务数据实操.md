# 财务数据实操

> [!note] 核心问题
> 阶段二要你读三张表，但「表从哪来、字段什么意思、如何避免未来函数」常被跳过。本篇打通：**用 AKShare/Tushare 拉财务样例 → 映射到分析问题 → 做一份可交作业的公司财务笔记**，并挂上 quant-lab 脚本。

## 学习目标

读完这篇，你要能做到：

1. 说清行情数据与财务数据在频率、发布滞后上的区别。  
2. 用 quant-lab 或手动 API 拉出至少一张财务相关表并落盘。  
3. 把字段映射到收入、利润、现金流、资产负债等分析问题。  
4. 在研究中标注**报告期 vs 公告日**，警惕前视。  
5. 完成一家公司的最小财务作业包。  

## 行情 vs 财务

| 维度 | 日行情 | 财务 |
|---|---|---|
| 频率 | 日 / 更高 | 季/年为主 |
| 发布时间 | 盘中/当日 | 公告日滞后于报告期 |
| 主要坑 | 复权、停牌 | PIT、口径变更、粉饰 |
| 用途 | 价量信号 | 质量、估值、基本面因子 |

理论复习：[[三张财务报表]]、[[财务比率分析]]、[[杜邦分析法]]、[[估值方法入门]]。

## 工具路径

### 路径 A：quant-lab 脚本（先求有表）

```powershell
cd "路径\...\阶段零-实操百科\quant-lab"
python scripts/pull_financials_example.py --symbol 600519 --kind indicator
```

- 成功时：`data/processed/` 下 CSV + `*_financials.meta.json`  
- 失败时：升级 `akshare`，并到 [数据字典](https://akshare.akfamily.xyz/data/index.html) 搜「财务」替换接口名  

脚本**故意容错**：接口更名时不会假装成功；你要读 meta 里的 `attempts`。

### 路径 B：Tushare 财务接口

在具备权限时，用 Pro 文档中的利润表/资产负债表/现金流量表/指标类接口（名称以官网为准）。  
token 安全见 [[Tushare数据上手指南]]。

### 路径 C：平台研究环境

聚宽等研究环境常有 `get_fundamentals` 类 API——适合与回测对齐，但导出受限。见 [[聚宽平台操作手册]]。

## 你真正需要的不是「所有字段」

先固定分析问题，再找字段：

| 分析问题 | 常见关注 | 对应阶段二 |
|---|---|---|
| 赚钱吗？ | 营收、净利润、毛利率 | [[三张财务报表]] |
| 赚的是现金吗？ | 经营现金流 vs 净利润 | 现金流质量 |
| 杠杆高吗？ | 资产负债率、流动比率 | 比率 |
| 回报如何？ | ROE、ROA | [[杜邦分析法]] |
| 贵不贵？ | PE/PB 需市值+利润/净资产 | [[估值方法入门]] |

## 报告期 vs 公告日（前视核心）

```text
错误：用「2024 年年报最终净利润」在 2024-01 回测选股
更好：只用该信息在「公告日之后」才可知的数据
```

| 概念 | 含义 |
|---|---|
| 报告期 | 财务所属区间（如 2023Q3） |
| 公告日 | 市场首次可得时间 |
| PIT | Point-in-time，按当时可得信息集合 |

学习作业阶段：若只有报告期字段，**必须在笔记声明**「未做严格 PIT，结论仅供方法练习」。  
严肃因子研究：上专业数据或平台基本面库。见 [[回测方法论]]。

| 步骤 | 动作 |

## 从财务到「伪因子」（可选）

教学向：不要上全市场优化。

```text
1. 股票池：某指数成分（或手工 20 只）
2. 指标：如 ROE 或 净利率（定义写清）
3. 排序：每季（或每年）选前 N
4. 成交：公告日后下一交易月第一个交易日（若无公告日，作业中声明简化）
5. 成本：显式 > 0
6. 输出：EXP 笔记 + 否决/保留
```

更系统：[[因子投资体系]]、进阶 `因子投资/`。

## 与 quant-lab 行情流水线的关系

```text
pull_akshare_example.py     → 价量
pull_financials_example.py  → 财务表（样例）
你的分析笔记                → 阶段二作业
第一策略仍建议价量简单规则   → 财务策略放到第二、三策略
```

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 字段越多越专业 | 问题驱动字段 |
| 财报数字绝对真实 | 有操纵与会计估计 |
| 自动接口 = 可直接回测 | 缺 PIT 会假 alpha |
| 一次拉取终身够用 | 公司行动与重述会变更 |
| 财务策略第一周就全 A | 先单公司基本功 |
