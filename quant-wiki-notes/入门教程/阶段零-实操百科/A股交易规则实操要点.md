# A股交易规则实操要点

> [!note] 核心问题
> 回测引擎里的「买卖」若不符合 A 股真实约束，曲线只是动画。本篇用**研究/回测必须建模或至少标注**的规则要点，帮你把策略从理想国拉回沪深京市场。具体费率、税率、临停规则以交易所与券商最新规定为准。

## 学习目标

读完这篇，你要能做到：

1. 说出 T+1、涨跌停、竞价与连续竞合对策略的影响。
2. 在策略说明书里写明：最小单位、费用结构、是否允许T+0假设。
3. 识别回测中常见的「假可成交」。
4. 区分主板/创业板/科创板在门槛与涨跌幅上的差异意识。
5. 把规则清单嵌进第一策略与模拟盘检查。

## 为什么规则是 alpha 的一部分

| 忽略规则 | 典型假结果 |
|---|---|
| 当作 T+0 | 高估短线反转/做市类 |
| 忽略涨跌停 | 高估动量追涨 |
| 忽略最小 100 股 | 小资金组合权重失真 |
| 费用 = 0 | 高换手策略造假 |
| 忽略停牌 | 虚假持仓与净值 |

## 一、交易日历与时段（意识模型）

| 阶段 | 常见时段（教学近似） | 对策略含义 |
|---|---|---|
| 开盘集合竞价 | 上午开盘前 | 开盘价形成机制；极端单可能影响开盘 |
| 连续竞价 | 上午、下午会话 | 多数日频策略假设落在此 |
| 收盘相关 | 尾盘 | 收盘价信号 vs 可成交性 |
| 休市 | 周末与法定假期 | 用交易日历，不用自然日硬填 |

日频策略通常用**日 OHLCV**；若信号用收盘，成交假设必须写清「次日开盘」或「次日均价」等。

## 二、T+1

| 点 | 说明 |
|---|---|
| 含义 | 当日买入的股票，一般下一交易日才可卖出 |
| 回测 | 禁止出现「当日买当日卖」除非明确做可转债等例外品种且规则允许 |
| 影响 | 隔夜风险无法用尾盘T+0对冲（股票） |

检查你的引擎：是否允许同一 bar 开平？

## 三、涨跌停与临停

| 概念 | 研究要点 |
|---|---|
| 涨跌停板 | 价格达到限制后可能**有价无市** |
| 回测错误 | 涨停日仍按收盘价买入大额 |
| 更好处理 | 涨停不买、跌停不卖（或按成交量模型） |
| 临停 | 盘中异常波动停牌，日频数据可能只见结果 |

不同板块涨跌幅限制历史上不同（如 10%、20% 等），**回测跨年要警惕规则变迁**，不要假设 2010 与 2024 相同。

## 四、证券品种与板块意识

| 类型 | 新手策略注意 |
|---|---|
| 沪深主板 | 经典样本，仍要 ST 与停牌 |
| 创业板 / 科创板 | 交易门槛、涨跌幅、注册制特征不同 |
| 北交所 | 规则与流动性差异大，勿与主板混一池不加标签 |
| ST/*ST | 风险警示，策略应显式过滤或单独研究 |
| 指数 | 不可直接当可交易股票；用 ETF 或期货替代要另写规则 |

## 五、数量、价格与订单类型（回测最小集）

| 规则意识 | 回测建议 |
|---|---|
| 1 手 = 100 股（常见股票） | 目标权重下单后取整，允许现金残余 |
| 价格最小变动单位 | 日频可简化，分钟级需注意 |
| 市价/限价 | 回测写清用何种成交假设 |
| 撤单 | 高频才关键；日频从简但勿假设无限流动性 |

## 六、费用结构（必须写进说明书）

教学用结构表（**数字请用你券商与当前规则替换**）：

| 费用项 | 常见方向 | 回测建模提示 |
|---|---|---|
| 佣金 | 买卖 | 有最低 5 元等规则时，小单杀伤大 |
| 印花税 | 多为卖出侧（规则会调整） | 不要漏；变更年要分段 |
| 过户费等 | 视市场 | 可合并进 bps 近似，但需注明 |
| 滑点 / 冲击 | 隐形成本 | 按换手与市值分档更诚实 |

> [!warning]
> 本库不维护实时费率表。写策略时打开你的券商费率页与交易所公开信息核对，并在笔记注明「核对日期」。

## 七、停牌、退市与特殊处理

| 事件 | 回测影响 |
|---|---|
| 停牌 | 无成交；强行用前后收盘插值会制造假路径 |
| 退市 | 幸存者偏差；池子需历史可投资集合 |
| 分红 | 代码/板块变更导致拼接错误 |
| 分红 | 见 [[复权与公司行动实操]] |

## 九、对第一策略的具体约束

双均线日频示例约束：

1. 金叉信号若以收盘确认 → 成交点设为次日开盘（或文档默认）并前后一致。  
2. 次日一字涨停 → 买入失败，保持空仓，而不是按愿望成交。  
3. 死叉但跌停 → 卖出可能失败，持仓风险继续。  
4. 小资金不足 100 股 → 该标的不买或集中到可买标的。  

## 十、模拟盘时如何练规则

| 练习 | 方法 |
|---|---|
| T+1 体感 | 纸交易禁止同日卖出当日买仓 |
| 涨停 | 记录「信号有但无法买」次数 |
| 费用 | 每笔按券商规则手算一笔对比 |
| 开盘缺口 | 对比信号价与次日开盘价偏差 |

见 [[从模拟到小资金实盘]]。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 美股代码逻辑直接套 A 股 | T+1 与涨跌停改变一切 |
| 回测用收盘价当无限量成交 | 流动性是约束 |
| 规则太烦以后再说 | 以后改引擎会推翻研究结论 |
| 只过滤 ST 就够了 | 还有停牌、次新、低流动性 |
