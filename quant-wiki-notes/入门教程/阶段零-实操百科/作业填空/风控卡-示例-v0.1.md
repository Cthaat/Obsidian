## 卡 A：总账户（示例）

| 项目 | 规则 |
|---|---|
| 目标 | 长期配置 + 学习研究 |
| 期限 | 10 年+ |
| 总账户最大回撤 | **-25%** |
| 单日最大亏损 | -3% 触发冷静（当日不再开新仓） |
| 风险资产上限 | 核心权益按配置；总风险资产≤规划 |
| 单票上限 | **5%** |
| 单行业上限 | 学习仓主题合计 ≤10% |
| 杠杆 | **否** |
| 现金安全垫 | 6 个月支出，账户外 |
| 暂停开新仓 | 账户回撤达 -20%（预警） |
| 强制减仓 | 账户回撤达 -25%：学习仓清零或仅保留核心 ETF 再平衡 |
| 专项禁止 | 期权卖方、无权限期货、跟风主题满仓 |
| 复查 | 每月 |

## 压力情景（示例动作）

| 情景 | 动作 |
|---|---|
| 指数两周 -15% | 不追加学习仓；检查现金垫 |
| 跌停卖不出 | 预先限制低流动性标的；记录无法成交 |
| 换手翻倍 | 提高 cost_bps 重跑；暂停实盘念头 |
| 连续 3 日破例 | 关闭学习仓交易权限一周 |
