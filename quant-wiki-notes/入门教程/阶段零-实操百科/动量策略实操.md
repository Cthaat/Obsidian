# 动量策略实操

> [!note] 核心问题
> 双均线是第一策略；第二策略用**动量**把你推进「信号定义、股票池、再平衡、简单风控」四件事。本篇对应可运行代码 `quant-lab` 中的时序动量与小池截面动量，强调教学边界而非收益承诺。

## 学习目标

读完这篇，你要能做到：

1. 区分时序动量与截面动量。  
2. 在 quant-lab 跑通 `run_ts_momentum.py` 与 `run_cross_section_momentum.py`。  
3. 为动量策略写清 lookback、再平衡、成本与失效条件。  
4. 用可选 `--max-dd` 理解「风控会改变收益路径」。  
5. 拒绝把 5 只股票的截面回测当成可投资结论。  

## 两种动量

| 类型 | 问题 | quant-lab 入口 |
|---|---|---|
| 时序动量 | 这只资产自身过去强不强？强则持有 | `scripts/run_ts_momentum.py` |
| 截面动量 | 池子里谁相对更强？持有前 K 名 | `scripts/run_cross_section_momentum.py` |

## 一、时序动量（单标的）

### 规则（v0.1）

```text
formation_return = close[t-skip] / close[t-skip-lookback] - 1
若 formation_return > 0 → 下一期持有（pos=1）
否则 → 空仓
可选：策略权益回撤超过 max_dd → 强制空仓直至回撤收敛
```

### 命令

```powershell
cd ...\quant-lab
python scripts/pull_akshare_example.py --symbol 000001
python scripts/run_ts_momentum.py --symbol 000001 --lookback 20
python scripts/run_ts_momentum.py --symbol 000001 --lookback 60 --max-dd 0.25
```

### 参数实验纪律

| 版本 | 只改一项 |
|---|---|
| v0.1 | lookback=20 |
| v0.2 | lookback=60 |
| v0.3 | 加 max_dd=0.25 |
| v0.4 | cost_bps 10→20 |

每个版本一条 EXP，见 [[研究笔记与实验工作流]]、示例 [[EXP-示例-时序动量-v0.1]]。

## 二、截面动量（小观察池）

### 规则（v0.1）

```text
每周（W）对池中股票按 lookback 收益排序
等权持有 top_k
权重次日生效（shift）
换手按权重变化计成本
```

### 命令

```powershell
# 编辑 data/watchlists/demo_pool.txt 为你的流动性较好的代码
python scripts/pull_watchlist.py
python scripts/run_cross_section_momentum.py --lookback 20 --top-k 2 --cost-bps 15
```

### 必须写在说明书里的警告

| 警告 | 原因 |
|---|---|
| 池子极小 | 方差巨大，曲线故事性强 |
| 无行业中性 | 可能只是押注单一板块 |
| 无 ST/停牌/涨跌停 | 假可成交 |
| 非点-in-time 指数成分 | 有幸存者风险 |
| 成本常数 | 小盘冲击被低估 |

## 三、与双均线怎么选练习顺序

| 顺序 | 策略 | 训练点 |
|---:|---|---|
| 1 | 双均线 | 闭环、复权、两引擎对照 |
| 2 | 时序动量 | 信号窗口、可选回撤熔断 |
| 3 | 截面动量 | 池、排序、再平衡、换手 |

## 四、简单风控在教什么

`--max-dd` **不是**最优风控，它演示：

1. 风控是路径依赖的，会改交易序列；  
2. 「提高收益」与「活下来」常冲突；  
3. 任何熔断都要写恢复条件，否则可能永久空仓或反复鞭打。  

接 [[风险管理框架]]、[[从模拟到小资金实盘]]。

## 五、结果如何读

| 现象 | 可能解释 | 下一步 |
|---|---|---|
| 显著强于买入持有 | 样本偏牛市趋势 | 换区间、加成本 |
| 加 max_dd 后收益降、回撤降 | 风控在起作用 | 记录取舍 |
| 截面换手很高 | 再平衡过频或信号噪声 | 加长 lookback 或降频 |
| 与双均线高度同涨同跌 | 都在吃趋势 beta | 换资产或加过滤 |

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 动量=无脑追涨 | 有窗口、再平衡与空仓规则 |
| 小池子夏普高=圣杯 | 过拟合与噪声 |
| 风控只是卖点包装 | 风控改变策略本身 |
| 先上全 A 再谈规范 | 先规范再扩池 |
