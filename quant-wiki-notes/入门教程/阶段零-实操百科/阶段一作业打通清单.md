# 阶段一作业打通清单

> [!note] 核心问题
> 阶段一的作业是一页**投资者说明书**。没有它，后面所有回测和策略都缺少「为谁、多久、能亏多少」的边界。本清单带你填完整、可迭代的 v0.1，并接到阶段四风控卡与毕业 IPS。

## 学习目标

1. 完成投资者说明书全部必填项。  
2. 写出 ≥5 条禁止事项与学习仓边界。  
3. 用简单数字感受复利与回撤（可手算）。  
4. 识别自身 2–3 个高频心理偏误并写应对规则。  
5. 把说明书存档，供后续阶段引用。

### B. 风险边界

| 字段 | 填写 |
|---|---|
| 投资期限 |  |
| 最大可承受总回撤 | 例：-15% / -20% / -30% |
| 触及回撤时的动作 | 暂停加仓 / 再平衡 / 只减不加 … |
| 单日或单周情绪熔断 | 可选 |

> [!important]
> 回撤数字必须能通过「生活影响测试」：跌到该水平时是否仍能睡觉、是否会被迫卖出。抄别人的 -50% 无效。

| 字段 | 填写 |

| 再平衡规则 | 时间（半年）或偏离阈值 |

## 复利与回撤小练习（必做一题）

**题 1**：本金假设 10 万，年化 5% / 8% / 12%，20 年后终值粗算（可用表格或代码），写一句结论。  

**题 2**：若先亏 30%，需要涨多少才回本？写公式与数字。  

结论写入说明书备注：为何「少犯大错」优先于「追高收益」。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 说明书写了就永远不变 | 要有修订日 |
| 回撤填很乐观的数字 | 按最差可执行情景 |
| 学习仓无限大 | 有硬顶才叫纪律 |
| 只写目标不写禁止 | 禁止事项决定生存 |
