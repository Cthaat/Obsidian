# FRED宏观数据实操

> [!note] 核心问题
> 宏观不靠「猜明天公布」，而靠**可复现的序列 + 情景语言**。 [FRED](https://fred.stlouisfed.org/)（圣路易斯联储经济数据）是全球最常用的公开宏观库之一。本篇教你网页怎么查、研究笔记怎么记、如何接到公司/配置作业。

## 学习目标

1. 在 FRED 网站检索并打开一条序列页面。  
2. 记录序列 ID、频率、单位与下载时间。  
3. 把 2–3 条序列写进「宏观温度计」。  
4. 区分网页浏览与 API（API 通常需申请 key，以官网为准）。  
5. 不把宏观点位当日内买卖信号。  

## 从哪个网站开始

| 步骤 | 动作 |
|---:|---|
| 1 | 打开 https://fred.stlouisfed.org/ |
| 2 | 顶部搜索框输入关键词，如 `GDP`、`federal funds`、`CPI` |
| 3 | 进入序列页，阅读标题、单位、频率、最后更新 |
| 4 | 使用页面提供的下载/图表功能（界面文案以站点为准） |
| 5 | 在 Vault 记下 **series id**（URL 或页面标注，如常见的 `GDP`、`FEDFUNDS` 等，以页面为准） |

进阶 API：见 FRED API 文档站（`fred.stlouisfed.org/docs/api`）。**网站浏览一般不必先申请 API**；程序化拉取通常需要 API Key（注册流程以官网为准）。

## 新手观察清单（示例序列主题）

| 主题 | 搜索关键词示例 | 接到本库 |
|---|---|---|
| 增长 | real GDP | [[国内生产总值_Gross Domestic Product (GDP)]] |
| 政策利率 | federal funds rate | [[联邦基金利率_Federal Funds Rate]] |
| 通胀 | CPI, PCE | [[宏观经济基础]] |
| 就业 | unemployment | 宏观温度计 |

中国宏观请用国内官方统计与可靠数据源；FRED 偏美/全球对照。

## 与公司分析接线

- 该公司对利率/汇率/地产/出口哪条链敏感？  
- FRED 或国内数据里你跟踪哪 1–2 个代理变量？  

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 数据公布瞬间交易 | 噪音与已定价 |
| 只看名义不看实际 | 通胀调整 |
| 美数据直接套 A 股短线 | 传导滞后且结构不同 |
| 不记录序列 ID | 无法复现 |
