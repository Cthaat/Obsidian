# 复权与公司行动实操

> [!note] 核心问题
> 分红、拆股、配股、增发等会让价格序列「断裂」。不理解复权与公司行动，均线、动量、回测收益都会错。本篇用实操语言说明前复权/后复权/不复权怎么选，以及研究中如何记录与检查。

## 学习目标

读完这篇，你要能做到：

1. 解释为什么除权日价格跳空不一定是「暴跌」。
2. 在技术信号、长期收益、事件研究三类场景选对口径。
3. 对同一标的对比不复权与前复权序列并做差异检查。
4. 在数据元数据中强制记录 `adjust` / 复权因子来源。
5. 知道公司行动未处理时策略说明书必须声明的风险。

## 公司行动在数据里长什么样

| 事件 | 价格上的直观 | 若不当回事 |
|---|---|---|
| 现金分红除息 | 价格下跳约股利 | 假跌破均线 |
| 送转 / 拆股 | 价格按比例变 | 假暴跌/暴涨 |
| 配股 | 除权与权益复杂 | 收益算错 |
| 增发 | 股本变化 | 市值/权重错 |
| 合并换股 | 代码与序列拼接 | 历史断层 |

「公司行动」是真实权益变化；「复权」是为了让序列在研究上可比的一种调整。

## 三种价格口径

| 口径 | 思想 | 常见用途 | 缺点 |
|---|---|---|---|
| 不复权 | 当时成交价 | 对照盘面、某些事件 | 技术指标被跳空污染 |
| 前复权 | 以**现在**价格为锚回推历史 | 看盘、均线、形态 | 历史绝对值随新分红变化 |
| 后复权 | 以**过去**为锚向后累乘 | 长期收益、净值研究 | 最新价不是盘面价 |

```text
粗规则：
- 做均线/形态/动量排序 → 同一研究内统一前复权（或文档推荐口径）
- 做长期累计收益、产品净值思维 → 后复权或「总回报」序列（含息）更贴切
- 做「那天能以什么价委托」→ 需要不复权 + 当时可交易约束
```

> [!important]
> **总回报（含股息再投资）** 与 **价格回报** 不同。只做价格复权不等于完整总回报；严肃业绩比较要问序列是否含息。

## 复权因子直觉（不推导也可会用）

许多数据源提供 `adj_factor` 或直接给复权 OHLC：

```text
前复权收盘 ≈ 不复权收盘 × (因子对齐方式)
```

你不需要手推所有公司行动，但必须：

1. 只用**同一来源**的复权字段；  
2. 不混用「自己胡乱 pct_change 拼接」；  
3. 在除权日附近做抽查。  

## 场景选型表

| 研究问题 | 推荐口径 | 备注 |
|---|---|---|
| 双均线交叉 | 前复权收盘 | 全样本一致 |
| 横截面动量（过去 N 日收益） | 前复权或官方后复权收益 | 全市场同一规则 |
| 买入持有 10 年收益 | 后复权或总回报指数 | 写明是否含息 |
| 涨跌停是否可买 | 不复权现价 + 规则 | 复权价不是委托价 |
| 事件研究（公告日） | 明确窗口与是否复权 | 避免把除权当异常收益 |

## 实操：一次对比实验（必做）

对任意活跃股：

1. 拉 `adjust=""/"none"` 与 `adjust="qfq"`（参数名因库而异）。  
2. 对齐日期，算日收益。  
3. 找分红/送转公告日附近，看不复权跳空。  
4. 确认前复权序列在该日是否平滑得多。  

```python
# 示意：比较两种口径的日收益差异（列名按你的数据改）
import pandas as pd

def daily_ret(close: pd.Series) -> pd.Series:
    return close.astype(float).pct_change()
```

## 指数与 ETF

| 对象 | 注意 |
|---|---|
| 价格指数 vs 全收益指数 | 长期比较基准别选错 |
| ETF | 净值、IOPV、二级市场价格是不同东西 |
| 指数点位 | 一般不可直接下单，策略用 ETF/期货要另规格 |

## 回测中的典型错误

| 错误 | 后果 | 修正 |
|---|---|---|
| 训练用前复权，实盘用现价算信号却参数照搬 | 信号偏移 | 研究与交易口径映射清楚 |
| 前复权序列中途重拉导致历史被改写 | 不可复现 | 落盘当时数据 + 记录版本 |
| 用复权价算「能否涨停买入」 | 逻辑错 | 可交易性用实际规则价 |
| 合并股票池时有的复权有的不复权 | 横截面乱序 | 统一预处理管道 |

## 前复权「会变」的问题

前复权以当前为锚，**未来新的分红会改写历史前复权序列**。因此：

- 科研复现：保存当时拉取的 CSV，不要假设永远网上一致；  
- 论文级：优先可追溯的因子与公司行动表；  
- 学习阶段：接受该限制，但要知道它存在。  

## 最小处理管道（本地）

```text
raw OHLC（注明口径）
  → 检查交易日历与缺失
  → 统一为研究口径（如 qfq）
  → 存 processed/
  → 所有策略只读 processed
  → 禁止在策略里临时改复权
```

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 跳空就是主力出货 | 先查是否除权除息 |
| 不复权更「真实」所以指标都用它 | 真实成交价 ≠ 适合做趋势比较 |
| 复权一次一劳永逸 | 数据更新与公司行动持续发生 |
| 红利无关紧要 | 长期复合差很远 |
