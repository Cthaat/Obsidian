# EXP-示例-时序动量-v0.1

> 写法示例；数字请用你本机 `reports/ts_mom_*.json` 替换。非投资建议。

## 1. 假设

- 过去 20 日收益为正时，短期延续概率高于随机（教学假设，需样本外检验）。  
- 空仓状态是策略的一部分，不是「没干活」。  

## 2. 数据

| 项 | 内容 |
|---|---|
| 源 | akshare 日线 qfq |
| 标的 | 000001 |
| 区间 | 与 processed 文件一致 |
| 路径 | `quant-lab/data/processed/000001_ohlcv.csv` |

## 3. 策略要点

| 项 | 内容 |
|---|---|
| 名称版本 | ts_momentum_v0.1 |
| 信号 | lookback=20 收益>0 则下一期做多 |
| 费用 | cost_bps=10 |
| 风控 | 无 max_dd（v0.1） |
| 唯一变化 | 相对 dual_ma 的新信号族 |

## 4. 预注册标准

| 类型 | 标准 |

| 失败 | 只能靠单年牛市；或成本 20bps 即崩且无解释 |
