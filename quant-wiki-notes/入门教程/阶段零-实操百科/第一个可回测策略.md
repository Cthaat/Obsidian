# 第一个可回测策略

> [!note] 核心问题
> 第一个策略的目标不是赚钱，而是完成**可检验的研究闭环**：清晰规则 → 可复现回测 → 成本与基准 → 失效条件 → 研究笔记。本篇给出推荐的第一策略规格、实现检查单与交付模板。

## 学习目标

读完这篇，你要能做到：

1. 写出一份他人能独立实现的策略说明书。
2. 在平台或本地完成双均线（或等价简单规则）回测。
3. 用分年、回撤、换手、基准对比解读结果。
4. 列出至少三条失效条件与下一步实验。
5. 拒绝「调参到完美再宣布成功」。

## 为什么第一策略推荐双均线

| 优点 | 说明 |
|---|---|
| 规则极简 | 焦点放在数据与成本，而非复杂信号 |
| 经典坑全集中 | 复权、未来函数、震荡市、成本 |
| 易对照教材 | 与技术分析、趋势跟踪概念衔接 |
| 易扩展 | 可加成交量过滤、股票池、风控 |

可选替代（同样要保持简单）：

| 策略 | 逻辑一句话 | 注意 |
|---|---|---|
| 动量（近 N 月收益排序） | 强者恒强假设 | 换手与回撤 |
| 价值粗筛（低 PE 等） | 便宜 | 财务 PIT、亏损股 |
| 均线 + 大盘过滤 | 降震荡市交易 | 过滤本身也需验证 |

第一策略**不要**上深度学习。

## 策略说明书模板（请复制填写）

### 1. 基本信息

| 字段 | 示例（假设，非推荐实盘） |
|---|---|
| 名称 | DEMO-DualMA-HS300-成分教学 |
| 版本 | v0.1 |
| 作者与日期 | 你 / 2026-07-13 |
| 市场 | A 股 |
| 频率 | 日频 |
| 方向 | 只做多 |
| 基准 | 沪深 300 全收益或价格基准（写明） |

### 2. 假设与逻辑

- **市场假设**（例）：中期价格趋势可延续一段时间。  
- **为何可能失效**（例）：震荡市反复止损；拥挤后冲击成本上升。  
- **不做的事**：不预测点位；不加杠杆。  

### 3. 股票池

| 规则 | 例 |
|---|---|
| 宇宙 | 某宽基指数成分 |
| 排除 | ST、*ST、停牌、上市不足 N 日 |
| 调仓时点 | 每日或信号日 |
| 最多持仓 | 1（单标的教学）或 5（等权） |

### 4. 信号

| 项目 | 定义 |
|---|---|
| 指标 | SMA(fast), SMA(slow) |
| 默认参数 | fast=5, slow=20（**仅教学默认**） |
| 买入 | 快线上穿慢线，且可交易 |
| 卖出 | 快线下穿慢线，或触及风控 |
| 信号时点 | 当日收盘确认，次日开盘/按引擎规则成交 |

### 5. 组合与下单

| 项目 | 定义 |
|---|---|
| 权重 | 单标的目标仓位 100% 或等权 |
| 最小交易单位 | 100 股（A 股常见） |
| 是否允许卖空 | 否 |
| 现金利息 | 忽略或按假设 |

### 6. 成本与约束（必须写）

| 项目 | 教学假设例 |
|---|---|
| 佣金 | 双边 万分之三，不足 5 元按 5 元（若引擎支持） |
| 印花税 | 卖出 0.05% 量级（以回测时点规则为准，需更新） |
| 滑点 | 0.1% 或固定 tick |
| 涨跌停 | 无法买入涨停、无法卖出跌停（若可建模） |

> [!warning]
> 税率与费率会变。正式研究以官方规则与券商实际为准；上表仅为**结构示例**。

### 7. 风险规则

| 规则 | 例 |
|---|---|
| 最大回撤熔断 | 组合回撤超 25% 暂停开新仓（假设） |
| 单日亏损暂停 | 可选 |
| 最长连续亏损次数 | 可选，防抖 |

### 8. 样本划分

| 集合 | 区间例（假设） | 用途 |
|---|---|---|
| 研究/训练 | 2016-01-01 ~ 2020-12-31 | 理解行为，**少**调参 |
| 测试 | 2021-01-01 ~ 2023-12-31 | 一次主评估 |
| 前视消毒 | 参数只在训练看，测试只跑一次主结果 | 防自欺 |

### 9. 失效条件

1. 测试集相对基准长期显著落后且回撤更大；  
2. 换手过高导致成本吞噬；  
3. 分年收益依赖单一年份；  
4. 换数据源或轻微延迟后信号翻转频繁。

### 解读提示

| 现象 | 可能原因 | 下一步 |
|---|---|---|
| 收益高、换手极高 | 成本低估 | 提高滑点再测 |
| 只在单边牛市好 | 趋势暴露 | 加大盘过滤或承认 beta |
| 训练好测试崩 | 过拟合/制度变化 | 简化参数，延长测试 |
| 与手动 Excel 不一致 | 成交时点/复权 | 对齐假设 |

## 最小参数实验纪律

允许的小实验：

| 实验 | 范围 | 纪律 |
|---|---|---|
| 均线周期 | 少量组合如 (5,20)(10,30)(20,60) | 先登记再跑，测试集不海选 |
| 加过滤 | 有/无成交量 | 一次只加一个模块 |
| 成本敏感性 | 滑点 0 / 5 / 10 / 20 bps | 看策略是否脆弱 |

禁止：在测试集上网格搜索数百组只汇报最好一组。

## 完成后的「毕业」标准（第一策略）

同时满足才算阶段零策略作业完成：

1. 说明书字段填全；  
2. 回测可复现；  
3. 费用与基准非空；  
4. 有测试集一次主结果；  
5. 失效条件 ≥ 3；  
6. 你能用 3 分钟向朋友讲清逻辑与风险。  

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 第一策略必须盈利 | 第一目标是闭环与诚实 |
| 参数越复杂越强 | 复杂提高过拟合表面空间 |
| 夏普 3 就结束 | 先怀疑数据与成本 |
| 只看总收益曲线 | 分年、回撤、换手 |
| 回测结束即实盘 | 先模拟，见下篇 |
