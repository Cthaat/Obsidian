# 从零开始的第一条实操路线

> [!note] 核心问题
> 「从哪里开始」最怕答案是一张无序链接清单。本篇给你**第一条可执行路线**：7 天最小闭环 + 4 周巩固，明确每天点开什么网站、装什么、写什么文件、怎样算完成。

## 学习目标

读完这篇，你要能做到：

1. 按清单完成第一次环境、取数、简单回测，不中途换赛道。
2. 在「云平台路径」和「本地路径」中选一条并坚持两周。
3. 用交付物定义进度，而不是用「学了多久」。
4. 知道卡关时优先查哪篇笔记、哪个官方文档。
5. 避免第一周就上实盘或追逐复杂机器学习。

## 开始前先做三个决定

| 决定 | 选项 | 本篇默认推荐 |
|---|---|---|
| 市场 | A 股为主 / 美股为主 / 都学 | **A 股为主**（国内学习者数据与社区 denser） |
| 路径 | 云研究平台 / 纯本地 | **云平台先跑通，本地并行补工程** |
| 第一策略 | 双均线 / 动量 / 价值打分 | **双均线**（逻辑浅、易对照坑） |

> [!important] 为什么双均线
> 不是因为它稳赚，而是因为它足够简单，能暴露**数据复权、交易成本、过拟合、未来函数**等真问题。复杂策略会把坑藏起来。

## 路径 A：云平台最快上手（推荐新手）

**主站**： [聚宽 JoinQuant](https://www.joinquant.com/)（研究 + 回测一体；具体套餐以官网为准）

### 7 天清单

| 天 | 动作 | 完成标准 |
|---:|---|---|
| 1 | 注册账号，浏览官方新手引导与社区「新手」帖 | 能打开研究环境（Notebook） |
| 2 | 官方文档：API、`get_price`、股票池 | 跑通示例：取沪深 300 成分或单票日线 |
| 3 | 画图：收盘价、20 日均线 | Notebook 里出图 |
| 4 | 写双均线策略骨架（短长均线交叉） | 回测能跑完不报错 |
| 5 | 加手续费、滑点、基准对比 | 结果表有收益、回撤、换手 |
| 6 | 改参数做**小**网格（如 5/20 vs 10/30），记录差异 | 有一页参数实验笔记 |
| 7 | 写策略一页纸：假设、规则、失效条件 | 见下方模板 |

**配套阅读**：[[国内量化平台对比与上手]]、[[第一个可回测策略]]、[[回测方法论]]。

### 聚宽上你要会的最小 API 概念

| 概念 | 用途 | 注意 |
|---|---|---|
| 研究环境 | 探索数据、画图 | 与回测引擎的数据接口可能略有差异，以文档为准 |
| 回测环境 | 按 bar 驱动策略 | 注意 `order` 成交假设 |
| 证券代码 | 标的标识 | 市场后缀、指数代码别混 |
| 股票池 / 过滤 | ST、停牌、次新 | 避免不可交易标的 |
| 费用设置 | 佣金、印花税、滑点 | 过低费用会美化结果 |

> [!tip]
> 第一周不要纠结「最优参数」。目标是：**完整跑通一次，并解释曲线为什么漂亮或难看**。

## 路径 B：本地工程路径（可与 A 并行）

适合想控制数据、写进 Git、以后接 vn.py 的人。

### 工具默认组合

| 层级 | 选择 | 原因 |
|---|---|---|
| 语言 | Python 3.11+ | 生态主流 |
| 数据 | [AKShare](https://akshare.akfamily.xyz/) | 免费、覆盖广、文档接口多 |
| 分析 | pandas / numpy / matplotlib | 最小集 |
| 回测 | backtrader 或 backtesting.py | 学习曲线相对平缓 |
| 版本 | git | 每次实验可回溯 |

安装细节见 [[Python量化环境搭建]]；数据见 [[数据源全景与选型]]；回测见 [[回测框架选型与最小示例]]。

### 7 天本地清单

| 天 | 动作 | 完成标准 |
|---:|---|---|
| 1 | 安装 Miniconda/venv，创建项目目录 `quant-lab` | `python -V` 正常 |
| 2 | `pip install akshare pandas matplotlib` | 脚本打印某股票最近 5 行行情 |
| 3 | 统一列名：date/open/high/low/close/volume，保存 CSV | 文件可重复生成 |
| 4 | 计算日收益、累计收益、简单均线 | 出图保存 PNG |
| 5 | 安装 backtrader，喂入 CSV | 空策略或打印 bar 成功 |
| 6 | 实现双均线买卖 | 有期末市值与交易次数 |
| 7 | 写 README：如何复现 + 成本假设 | 他人可按步骤跑 |

### 最小取数示例（教学用，接口以官网最新为准）

```python
# 假设示例：拉取 A 股日线（运行前请升级 akshare）
import akshare as ak

df = ak.stock_zh_a_hist(
    symbol="000001",
    period="daily",
    start_date="20200101",
    end_date="20231231",
    adjust="qfq",  # 前复权；研究时必须明确复权口径
)
print(df.tail())
```

> [!warning]
> AKShare 依赖公开网页源，接口会随源站变更。**每次使用前升级**，并在笔记中记录版本与拉取日期。商业用途请自行评估合规与稳定性。

## 第 2–4 周：巩固，不要换赛道

| 周 | 目标 | 具体任务 | 对接主线 |
|---:|---|---|---|
| 2 | 数据质量意识 | 对比复权前后；检查停牌/涨跌停；写数据检查表 | [[回测方法论]] |
| 3 | 策略规格化 | 把 Notebook 思路写成「规则说明书」；加基准 | [[常见量化策略]] |
| 4 | 模拟执行 | 每日/每周按信号模拟下单，记偏差 | [[风险管理框架]] [[交易心理与执行纪律]] |

第 4 周末应同时拥有：

1. 可复现的研究文件夹或平台策略；
2. 一页风控卡（最大回撤、单票上限）；
3. 至少 10 条模拟执行日志。

## 两个网站「从注册到第一张图」

### 1. 聚宽（路径 A）

1. 打开 [joinquant.com](https://www.joinquant.com/) 注册登录。  
2. 进入「研究」或官方标注的 Notebook 入口。  
3. 新建笔记本，运行官方「获取行情」示例。  
4. 用 `plot` 画收盘价。  
5. 另开「策略」回测，粘贴双均线模板并改标的。  
6. 导出或截图：收益曲线、回撤、交易详情。  

卡关时：官方帮助文档 → 社区按报错搜索 → 本库 [[国内量化平台对比与上手]]。

### 2. AKShare 文档（路径 B）

1. 打开 [akshare.akfamily.xyz](https://akshare.akfamily.xyz/)。  
2. 读「安装」与「快速入门」。  
3. 在「数据字典」找到股票日线接口。  
4. 复制示例，改 `symbol` 与日期。  
5. 确认 `adjust` 参数含义后再存盘。  

卡关时：Changelog、GitHub Issues、本库 [[AKShare概览]] 系列。

## 策略一页纸模板（第 7 天必填）

| 字段 | 示例（假设） |
|---|---|
| 名称 | 沪深 300 成分内双均线（教学） |
| 经济/行为逻辑 | 短期趋势跟随，牺牲震荡期 |
| 股票池 | 某指数成分，剔除 ST/停牌 |
| 信号 | 5 日均线上穿/下穿 20 日 |
| 仓位 | 等权 5 只或单标的满仓（教学用需写明） |
| 调仓频率 | 日频检查，信号触发交易 |
| 成本假设 | 佣金万三、印花税、滑点 0.1%（假设） |
| 样本 | 训练 2018–2021，测试 2022–2023（假设划分） |
| 风险限制 | 最大回撤超 25% 暂停（假设） |
| 失效条件 | 换手爆炸、长期横盘、成本敏感 |

把本表复制到你的 Vault，改成自己的版本。扩展见 [[第一个可回测策略]]。

## 明确不要做的事（第一月）

| 不要 | 原因 |
|---|---|
| 第一周实盘真钱 | 流程未验证，错误成本高 |
| 同时学 5 个框架 | 无法形成肌肉记忆 |
| 无成本回测就发朋友圈 | 自我欺骗 |
| 用未来数据调到完美再「样本外」 | 这是假样本外 |
| 高频/无证程序化实盘 | 合规与工程门槛远高于想象 |

## 卡关决策树

```text
报错在安装？ → Python量化环境搭建
报错在取数？ → 数据源全景与选型 + 官方文档升级
回测结果荒谬？ → 回测方法论（复权、未来函数、停牌）
不知道下一步学啥？ → 量化学习全景地图
想换更酷的 ML？ → 先完成双均线闭环与模拟 20 日
```

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 路线越长越认真 | 第一条路线要短、可完成 |
| 必须先读完所有书 | 书和主线并行，用项目拉动 |
| 云平台不够专业 | 专业的是你的研究规范，不是 IDE 皮肤 |
| 本地才是唯一正道 | 本地与云是阶段选择，不是道德问题 |
