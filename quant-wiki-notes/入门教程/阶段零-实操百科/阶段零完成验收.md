# 阶段零完成验收

> [!note] 核心问题
> 阶段零条目已经很多。本页用**验收清单**回答：怎样算走完实操百科？不必读完每一个链接，但必须拿到可展示的证据链。

## 学习目标

1. 对照清单自检完成度。  
2. 区分「必修 / 推荐 / 可选」。  
3. 知道验收通过后去阶段三/四/五作业如何衔接。  
4. 避免「收藏很多笔记却零运行」。  

## 必修（全部完成才算过关）

| # | 项 | 证据 |
|---:|---|---|
| 1 | 环境可复现 | `check_env.py` 或 `pip freeze` 片段 |
| 2 | 成功拉日线 | `processed/*_ohlcv.csv` + meta |
| 3 | 双均线回测 | `reports/dual_ma_*.json` |
| 4 | 策略说明书 | 含成本与 A 股未建模声明 |
| 5 | 至少 1 条 EXP | [[实验日志目录]] |
| 6 | 知道主数据源与复权口径 | 笔记 5 行即可 |
| 7 | 模拟或纸交易意识 | 日志 ≥5 日 **或** 书面说明为何暂缓 + 门槛 |

## 推荐（强烈建议）

| # | 项 | 证据 |
|---:|---|---|
| R1 | Backtrader 对照 | `bt_dual_ma_*.json` + 差异说明 |
| R2 | 时序动量 | `ts_mom_*.json` + EXP |
| R3 | 等权再平衡 | `port_equal_*.json` |
| R4 | 阶段二公司分析 | [[阶段二作业打通清单]] 打勾 |

## 一键自检命令（工程）

```powershell
cd ...\quant-lab
python check_env.py
# 若已有报告：
python scripts/summarize_report.py reports/你的主结果.json
```

## 能力口语化标准

验收时你应能不看笔记回答：

1. 我的数据从哪来、是否复权？  
2. 我的主策略一句话逻辑？  
3. 费用假设是多少？  
4. 最大回撤我能否承受？  
5. 为什么现在不上全仓实盘？  

答不出 → 未完成，回到对应笔记。

```text

缺回测 → 第一个可回测策略 + run_dual_ma
缺纪律 → 研究笔记与实验工作流 + EXP模板

缺风控语言 → 阶段四风控卡实操
```

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 读完目录=完成 | 要有 reports/日志证据 |
| 可选全做才毕业 | 必修满即可前进 |
| 收益高才算过 | 诚信闭环才算过 |
| 永远不验收一直加策略 | 用本页强制收敛 |
