# 组合层实操

> [!note] 核心问题
> 单标的策略教会信号；组合层回答：**多只股票如何加权、何时再平衡、单票上限与换手成本如何改变结果**。本篇对接 quant-lab 的等权/逆波动再平衡脚本，并链到阶段四概念。

## 学习目标

读完这篇，你要能做到：

1. 区分信号层与组合层。  
2. 跑通 `run_equal_weight_rebalance.py` 的 equal 与 inv_vol。  
3. 解释再平衡频率、max_weight、cost_bps 对换手的影响。  
4. 读懂 `avg_turnover` 与 `avg_gross_exposure`。  
5. 在说明书中写清未建模的 A 股约束。  

## 信号层 vs 组合层

| 层 | 问题 | 例子 |
|---|---|---|
| 信号 | 买谁/是否看多 | 动量、双均线 |
| 组合 | 各买多少、多久调一次 | 等权、风险平价近似、上限 |

理论：[[组合构建方法]]、[[风险预算与风险归因]]、[[资金管理与杠杆]]。

## 两种教学方案

| scheme | 规则 | 适合练习 |
|---|---|---|
| `equal` | 再平衡日对有价格的名字等权 | 理解再平衡与漂移 |
| `inv_vol` | 权重 ∝ 1/波动 | 理解「波动大少配」 |

权重在再平衡点计算 → 日度 ffill → `shift(1)` 次日生效。

## 命令

```powershell
cd ...\quant-lab
python scripts/pull_watchlist.py
python scripts/run_equal_weight_rebalance.py --scheme equal --rebalance ME --cost-bps 10
python scripts/run_equal_weight_rebalance.py --scheme inv_vol --rebalance ME --max-weight 0.4
```

输出：

- `reports/port_equal_ME.json`  
- `reports/port_*_last_weights.json`

| v0.5 | inv_vol |

## 如何读结果

| 字段 | 含义 |
|---|---|
| total_return / max_drawdown | 简化权益路径 |
| buy_hold_return | 池内近似等权买入持有 |
| avg_turnover | 平均权重变动（成本敏感度线索） |
| avg_gross_exposure | 权重和，<1 可能有现金 |

## 与动量/因子脚本的关系

| 脚本 | 组合思想 |
|---|---|
| `run_cross_section_momentum.py` | 排序后 top_k 等权（信号+组合一体） |
| `run_equal_weight_rebalance.py` | 不做 alpha 排序，只练权重与再平衡 |
| `run_factor_score.py` | 多因子分数 + top_k 组合 |

先跑 equal 再跑 factor，避免概念搅在一起。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 等权太简单没价值 | 它是基准组合层，先校准成本 |
| 再平衡越勤越好 | 换手与税/费会吃掉边沿 |
| 逆波动=真正风险平价 | 教学近似，忽略相关与约束优化 |
| 小池子结论可推广全 A | 不能 |
