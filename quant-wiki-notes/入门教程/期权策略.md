# 期权策略入门

> [!note] 核心问题
> 期权是一种把方向、波动率、时间和风险边界打包交易的工具。学习期权不能只看“杠杆高、收益大”，更要理解权利义务、到期收益、波动率、Greeks 和极端风险。

## 学习目标

读完这篇，你要能做到：

1. 理解看涨期权、看跌期权、买方、卖方的基本结构。
2. 看懂期权到期收益图。
3. 知道 Black-Scholes 定价公式背后的关键变量。
4. 理解 Delta、Gamma、Theta、Vega 等 Greeks 的含义。
5. 区分保护性看跌、备兑看涨、跨式、价差等基础策略。
6. 明白期权风险为什么可能远超表面权利金。

## 什么是期权

期权是一种合约，赋予买方在未来某个时间以约定价格买入或卖出标的资产的权利，但不是义务。

| 类型 | 买方权利 | 常见用途 |
|---|---|---|
| 看涨期权 Call | 以行权价买入标的 | 看多、杠杆、上行参与 |
| 看跌期权 Put | 以行权价卖出标的 | 看空、保护、下行保险 |

期权买方支付权利金，最大损失通常是权利金；期权卖方收取权利金，但可能承担更大的义务和风险。

## 期权的关键术语

| 术语 | 含义 |
|---|---|
| 标的资产 | 股票、ETF、指数、期货等 |
| 行权价 K | 未来可买入或卖出的价格 |
| 到期日 T | 期权合约失效日期 |
| 权利金 | 买方支付给卖方的价格 |
| 实值 | 立即行权有内在价值 |
| 平值 | 标的价格接近行权价 |
| 虚值 | 立即行权没有内在价值 |
| 内在价值 | 到期时可实现的价值 |
| 时间价值 | 对未来波动可能性的定价 |

## 到期收益结构

### 买入看涨期权

到期收益：

$$
\max(S_T - K, 0) - 权利金
$$

适合观点：看多，且希望用有限亏损参与上涨。

风险：如果到期价格没有超过行权价加权利金，可能亏损全部权利金。

### 买入看跌期权

到期收益：

$$
\max(K - S_T, 0) - 权利金
$$

适合观点：看空，或为持仓买保险。

风险：如果标的没有下跌，权利金会损失。

### 卖出期权

卖方收取权利金，但承担义务：

- 卖出看涨：标的上涨时可能亏损；
- 卖出看跌：标的下跌时可能亏损；
- 裸卖期权风险很高，必须谨慎。

## 期权价格由什么决定

| 变量 | 对期权价格的影响 |
|---|---|
| 标的价格 S | 看涨随 S 上升变贵，看跌随 S 上升变便宜 |
| 行权价 K | 越容易实值，期权越贵 |
| 到期时间 T | 时间越长，可能性越多，通常越贵 |
| 波动率 σ | 波动越大，期权越贵 |
| 无风险利率 r | 影响折现和远期价格 |
| 分红 | 影响持有标的和期权的相对价值 |

期权交易本质上不只是交易方向，也是在交易波动率和时间。

## Black-Scholes 定价直觉

Black-Scholes 模型假设标的价格服从几何布朗运动，并通过无套利和动态对冲推导期权理论价格。

欧式看涨期权公式：

$$
C = S_0 N(d_1) - K e^{-rT}N(d_2)
$$

其中：

$$
d_1 = \frac{\ln(S_0/K) + (r+\sigma^2/2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}
$$

你不需要一开始就推导公式，但要理解它告诉我们：

- 标的越高，看涨越贵；
- 波动率越高，期权越贵；
- 到期越久，期权通常越贵；
- 定价依赖假设，市场价格会反映供需和风险偏好。

## 隐含波动率

隐含波动率是把市场期权价格代入模型后，反推出的波动率。

它回答：

> 市场现在愿意为未来不确定性支付多高价格？

隐含波动率高，不代表一定会大涨或大跌，而是表示市场预期未来波动更大，或保险需求更强。

更多内容见 [[波动率]]。

## Greeks：期权风险的语言

| Greek | 含义 | 直觉 |
|---|---|---|
| Delta | 标的价格变化 1 单位，期权价格变化多少 | 方向暴露 |
| Gamma | Delta 对标的价格的敏感度 | 方向暴露变化速度 |
| Theta | 时间流逝对期权价格的影响 | 时间损耗 |
| Vega | 波动率变化对期权价格的影响 | 波动率暴露 |
| Rho | 利率变化对期权价格的影响 | 利率暴露 |

### Delta

Delta 接近 1 的看涨期权，走势更像股票；Delta 接近 0 的虚值期权，方向暴露较小但对波动更敏感。

### Gamma

Gamma 高时，Delta 会快速变化。临近到期的平值期权 Gamma 通常较高，风险也更尖锐。

### Theta

买期权的人通常支付时间价值，卖期权的人通常赚取时间流逝。但卖方承担尾部风险。

### Vega

如果你买入期权，你通常希望隐含波动率上升；如果你卖出期权，你通常害怕隐含波动率突然上升。

## 常见基础策略

### 1. 保护性看跌

持有标的资产，同时买入看跌期权。

作用：给持仓买保险，限制下跌风险。

代价：支付权利金，降低长期收益。

### 2. 备兑看涨

持有标的资产，同时卖出看涨期权。

作用：用卖出期权获得权利金收入。

风险：标的大涨时，上行收益被限制；标的下跌时仍会亏损。

### 3. 买入跨式

同时买入同一行权价、同一到期日的看涨和看跌。

适合观点：不确定方向，但预期大幅波动。

风险：如果波动不足，两个期权都会被时间价值侵蚀。

### 4. 垂直价差

同时买入和卖出不同行权价的同类型期权。

例子：

- 牛市看涨价差；
- 熊市看跌价差；
- 空头看涨价差。

作用：降低成本，同时限制最大收益和最大亏损。

### 5. 卖出看跌

卖出看跌期权，收取权利金。

适合观点：愿意以更低价格买入标的。

风险：标的暴跌时亏损巨大，不能把它简单当成“赚权利金”。

## 期权策略选择

| 市场观点 | 可能策略 | 主要风险 |
|---|---|---|
| 温和看多 | 备兑看涨、牛市价差 | 错过大涨 |
| 强烈看多 | 买入看涨 | 权利金归零 |
| 看空 | 买入看跌、熊市价差 | 时间损耗 |
| 持仓防守 | 保护性看跌 | 保险成本 |
| 预期大波动 | 买入跨式 | 波动不足 |
| 预期低波动 | 卖出跨式或卖期权 | 尾部风险巨大 |

新手应优先理解买方有限亏损策略和保护性策略，不建议直接裸卖期权。

## 期权风险

### 1. 杠杆风险

期权价格比标的便宜，但这不等于风险小。虚值期权可能归零。

### 2. 时间损耗

方向判断对了，但如果涨得太慢，买入期权仍可能亏钱。

### 3. 波动率风险

财报、宏观事件前隐含波动率可能很高。事件落地后，即使方向对，波动率下降也可能导致期权亏损。

### 4. 流动性风险

很多期权合约买卖价差大，成交量小。纸面价格不等于可交易价格。

### 5. 尾部风险

卖出期权时，小概率大亏可能吞掉长期权利金收益。

## 学习路径

1. 先理解看涨、看跌、买方、卖方。
2. 画出四种基础到期收益图。
3. 学 Delta、Theta、Vega。
4. 学保护性看跌和备兑看涨。
5. 再学价差和跨式。
6. 最后结合 [[风险管理框架]] 做仓位和压力测试。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 买期权亏损有限，所以很安全 | 权利金可能高频归零 |
| 卖期权胜率高，所以更安全 | 尾部亏损可能极大 |
| 看对方向就能赚钱 | 时间和波动率也会影响收益 |
| Black-Scholes 是真实价格 | 它是模型价格，依赖假设 |
| 期权适合新手重仓 | 新手应先小额学习和模拟 |

## 练习：画四张收益图

分别画出：

1. 买入看涨；
2. 卖出看涨；
3. 买入看跌；
4. 卖出看跌。

每张图标出：

- 最大收益；
- 最大亏损；
- 盈亏平衡点；
- 适合的市场观点；
- 最怕的风险。

如果画不出来，就还不适合实盘交易期权。
