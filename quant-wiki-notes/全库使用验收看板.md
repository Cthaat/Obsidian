## 第 1 周逐步命令（阶段零工程 · 复制即用）

> 在资源管理器进入：`quant-wiki-notes\入门教程\阶段零-实操百科\quant-lab`  
> 网站文档（对照）：[AKShare](https://akshare.akfamily.xyz/) · 可选 [聚宽](https://www.joinquant.com/)

### Day 1–2 环境

```powershell
cd "你的路径\quant-wiki-notes\入门教程\阶段零-实操百科\quant-lab"
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python check_env.py
python scripts/run_stage0_check.py --with-synthetic-backtest
```

- [ ] 输出含 `STAGE0_CHECK OK`  
- 笔记：[[Python量化环境搭建]] · [[quant-lab项目模板]]

### Day 3 拉数

```powershell
python scripts/pull_akshare_example.py --symbol 000001
# 若失败：升级 akshare，查官方数据字典；见 [[数据源全景与选型]] [[开源工具常见坑]]
```

- [ ] 存在 `data/processed/000001_ohlcv.csv` 与 `.meta.json`  
- 笔记：[[复权与公司行动实操]]

### Day 4–5 双均线 + 摘要

```powershell
python scripts/run_dual_ma.py --symbol 000001 --fast 5 --slow 20 --cost-bps 10
python scripts/summarize_report.py reports/dual_ma_000001_5_20.json
# 可选
python scripts/run_backtrader_dual_ma.py --symbol 000001
```

- [ ] `reports/dual_ma_*.json`  
- 笔记：[[第一个可回测策略]] · [[Backtrader对照实操]]

### Day 6 EXP

- 优先参考已跑通归档：[[EXP-2026-07-13-dualma-000001-v0.1]]  
- 或复制 [[EXP模板]] / [[EXP-示例-双均线-v0.1]] 填自己的数字  
- 粘贴 JSON 关键指标；写 3 条失效条件  
- [ ] EXP 笔记完成  

### Day 7 验收勾选

- [ ] [[阶段零完成验收]] 必修项尽量勾完
