# 技术分析与价格行为实操导航

> [!note] 核心问题
> 技术指标与 K 线内容极多，最容易陷入「指标崇拜」。本篇规定：**指标只做执行辅助与风险提示，不替代商业逻辑与风控**；并给出最小指标集、与双均线实验的接线、以及失效条件写法。

## 学习目标

1. 建立「趋势 / 震荡 / 极端」三态，再选指标。  
2. 用 2–3 个指标完成可回测或可日志化的规则。  
3. 会写每个信号的失效条件。  
4. 区分主观画线与可计算规则。  
5. 完成「指标策略一页纸」或明确「仅辅助看盘」。  

## 定位：技术分析在体系里的位置

| 层级 | 作用 |
|---|---|
| 研究主逻辑 | 因子、基本面、配置（优先） |
| 技术分析 | 入场节奏、滤掉极端、执行参考 |
| 禁止 | 无成本扫指标参数当圣杯 |

课程：[[技术分析入门]]。专题：[[技术指标/目录]] · [[K线形态/目录]]。

## 最小指标集（先精通再扩展）

| 用途 | 指标/概念 | 实验 |
|---|---|---|
| 趋势 | 均线、金叉死叉 | quant-lab 双均线 |
| 动量 | 收益/ROC、RSI（谨慎） | 时序动量脚本 |
| 波动 | 布林带/波动率 | 仓位与熔断 |
| 量价 | 成交量确认（辅助） | 日志观察 |
| 结构 | 支撑阻力（主观度高） | 必须规则化才回测 |

龙虎榜、游资、复杂形态：有兴趣再读，**不作为第一周主线**。

## 规则化三问（每个信号必答）

1. **状态**：现在偏趋势还是震荡？  
2. **信号**：可计算的触发条件是什么？  
3. **失效**：什么情况证明信号作废（时间止损、结构破坏、波动扩大）？

## 与量化回测的接线

| 主观说法 | 量化要求 |
|---|---|
| 「看起来要突破」 | 定义突破幅度与成交量阈值 |
| 「超买」 | 定义指标阈值与持有期 |
| 「支撑有效」 | 定义触及次数与失效跌破 |

无 [[回测与quant-lab对照清单]] 检查未来函数（指标是否用到当根未来）。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 指标越多越准 | 共线与过拟合 |
| 忽视成本的高频信号 | 手续费杀死边缘 |
| K 线故事替代仓位管理 | 风控优先 |
| 同权对待游资题材与系统规则 | 题材难复现 |
