# 合规与投资者保护实操导航

> [!note] 核心问题
> 收益研究若忽略规则、适当性、费用与资金安全，实盘第一步就可能踩雷。本篇把 [[合规、税务、运营与投资者保护]] 收成**可执行检查清单**，并挂上公开监管/投资者教育入口。

## 学习目标

1. 在说明书中写清「合规相关禁止事项」。  
2. 区分研究工具与实盘程序化报备要求。  
3. 列出账户与密钥安全最低标准。  
4. 识别高收益承诺与不可验证业绩。  
5. 完成合规自检表并链到风控卡。

## 运营安全最低标准

| 项 | 建议 |
|---|---|
| 密码 | 独立强密码 + 二次验证（若券商支持） |
| 权限 | 研究机与交易机分离更佳 |
| 日志 | 关键操作可追溯 |
| 备份 | 策略版本与配置可回滚 |

## 投资者保护：如何识破「坑」

| 红旗 | 应对 |
|---|---|
| 稳赚、保本高收益 | 离开 |
| 仅展示无成本曲线 | 要求成本与样本外 |
| 催促入金/转私下收款 | 停止 |
| 无法验证的「内部指标」 | 默认无效 |
| 要求关闭监管保护 | 拒绝 |

教育对照：[Investor.gov](https://www.investor.gov/introduction-investing) 关于费用、风险、研究投资等主题。

## 写入说明书的禁止事项（模板句）

1. 禁止无适当性评估购买高风险衍生品。  
2. 禁止未读程序化相关规定即自动实盘下单。  
3. 禁止向他人展示未注明假设的回测收益。  
4. 禁止使用来源不明的交易插件。  
5. 禁止账户出借与共享密钥。
