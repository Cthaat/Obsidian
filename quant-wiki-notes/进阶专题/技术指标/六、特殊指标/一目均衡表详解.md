# 一目均衡表详解

> [!note] 💡 概念解析
> 一目均衡表（Ichimoku Kinko Hyo）是日本技术分析家细田悟一发明的综合技术指标，由五条线和云区构成，能够同时判断趋势方向、支撑阻力和交易时机。

## 一、一目均衡表的起源

一目均衡表又称**云图指标**，由日本股票评论家细田悟一（笔名：一目千人）发明。该指标不仅在日本广泛使用，在海外也被众多交易者采用。

> [!tip] 核心思想
> "一目均衡"的意思是"一眼就能看出市场的均衡状态"，通过一张图表同时展示趋势、支撑阻力和动量信息。

## 二、五条线的构成与计算

### 2.1 转换线（Tenkan-sen）

$$\text{转换线} = \frac{\text{过去9天最高价} + \text{过去9天最低价}}{2}$$

- 用于分析**短期趋势**
- 转换线上行 → 上升力道大
- 转换线下行 → 下跌走势强

### 2.2 基准线（Kijun-sen）

$$\text{基准线} = \frac{\text{过去26天最高价} + \text{过去26天最低价}}{2}$$

- 用于分析**中期趋势**
- 基准线上行 → 上升力道大
- 基准线下行 → 下跌走势强

### 2.3 先行带A（Senkou Span A）

$$\text{先行带A} = \frac{\text{转换线} + \text{基准线}}{2}$$

- 预测**未来26天**的趋势
- 与先行带B共同构成云区

### 2.4 先行带B（Senkou Span B）

$$\text{先行带B} = \frac{\text{过去52天最高价} + \text{过去52天最低价}}{2}$$

- 预测**未来26天**的趋势
- 与先行带A共同构成云区

### 2.5 延迟线（Chikou Span）

- 显示**当天收盘价之前26天**的线
- 延迟线在K线上方 → 上升力道强
- 延迟线在K线下方 → 下跌走势强

## 三、云区（Kumo）

云区是先行带A与先行带B之间的区域，是一目均衡表的核心：

| 云区状态 | 含义 |
|---------|------|
| 价格在云区上方 | 上升趋势 |
| 价格在云区下方 | 下跌趋势 |
| 价格在云区内部 | 趋势不明，可能盘整 |
| 云区由薄变厚 | 趋势可能反转 |

## 四、三役好转与三役逆转

### 4.1 三役好转（买入信号）

同时满足以下三个条件：
1. 转换线超出基准线以上
2. 延迟线在价格之上
3. 价格超出云区以上

### 4.2 三役逆转（卖出信号）

同时满足以下三个条件：
1. 转换线低于基准线以下
2. 延迟线在价格之下
3. 价格低于云区以下

> [!important] 三役确认
> 三役好转/逆转是**最强的信号**，需要三个条件同时满足。单一条件的信号可靠性较低。

## 五、三大理论体系

### 5.1 时间论

分析时间与市场转换点的方法：
- 基本数值：**9、17、26**
- 分析在这期间市场是否容易出现转换

### 5.2 波动论

根据波动形状分析市场方向：
- **I型波动**：单纯上涨/下跌
- **V型波动**：上涨→下跌/下跌→上涨
- **N型波动**：上涨→下跌→上涨/下跌→上涨→下跌

> [!tip] N型波动是基本形态
> I型和V型波动会反复出现，最终形成N型波动。分析市场时要重点关注N型波动。

### 5.3 水平论（值幅观测论）

通过计算上涨/下跌幅度来预测价格目标：

| 计算方法 | 公式 |
|---------|------|
| N计算值 | 上涨3波幅度 = 上涨1波幅度 |
| E计算值 | 上涨3波幅度 = 上涨1波幅度 × 2 |
| V计算值 | 上涨3波幅度 = 调整2波幅度 × 2 |
| NT计算值 | 上涨3波幅度 = 上涨1波幅度 - 调整2波幅度 |

## 六、使用注意事项

> [!warning] 注意事项
> 1. 价格位于云区内部时，市场方向不明，可能进入箱形整理
> 2. 云区扭转时需特别注意，趋势可能反转
> 3. 在短时间周期图表（如5分钟、15分钟）中，云区信号可能频繁出现
> 4. 参数设定建议使用默认值（9、26、52），修改参数不会提高成功率

## 七、参数设置

在MT4/MT5中设置一目均衡表：
- **MT4**：插入 → 技术指标 → 趋势指标 → Ichimoku Kinko Hyo
- **MT5**：插入 → 指标 → 走向 → Ichimoku Kinko Hyo
- **默认参数**：9、26、52

## 实战掌握清单

> [!tip] 交易者视角
> 一目均衡表详解 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。技术指标是价格、成交量和波动率的二次加工，核心价值在于把主观观察变成稳定规则。

### 关键判断

- 先确认指标属于趋势、震荡、量能、波动率还是资金流。
- 判断当前市场是否适合该指标：趋势指标怕横盘，震荡指标怕单边。
- 把参数选择、信号延迟和交易频率写清楚。

### 落地动作

1. 用样本外数据检验信号，而不是只看历史图形好不好看。
2. 同时记录胜率、盈亏比、换手、滑点和回撤。
3. 把指标作为过滤器、触发器或退出规则，避免多个同源指标重复投票。

### 失效边界

- 参数过拟合。
- 忽略手续费和滑点。
- 在市场结构变化后继续迷信旧阈值。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
