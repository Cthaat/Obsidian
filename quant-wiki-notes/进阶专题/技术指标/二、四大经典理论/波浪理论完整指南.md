# 波浪理论完整指南

> [!note] 核心概述
> 艾略特波浪理论（Elliott Wave Theory）由拉尔夫·尼尔森·艾略特于1930年代提出，认为市场价格波动并非随机，而是呈现**有节奏、可预测的波浪结构**。一个完整周期由8个波浪组成：5浪推动+3浪调整。

## 一、理论起源

艾略特在研究数千张股市图表后发现，市场价格波动遵循固定的模式与周期，犹如自然界中的波浪。1938年出版《波浪原理》，数学基础来源于**斐波那契数列**。

## 二、5+3 核心结构

### 驱动浪（Impulse Waves）：1-2-3-4-5

| 浪 | 名称 | 特征 |
|----|------|------|
| **第1浪** | 希望的萌芽 | 少数敏锐投资者带动，涨势温和 |
| **第2浪** | 恐惧的回洗 | 对第1浪的回调，成交量萎缩 |
| **第3浪** | 群众的追捧 | 通常最长最强劲，成交量显著放大 |
| **第4浪** | 获利的喘息 | 复杂横向整理，部分投资者获利了结 |
| **第5浪** | 最后的疯狂 | 涨势不如第3浪，可能出现量价背离 |

### 调整波（Corrective Waves）：A-B-C

| 浪 | 名称 | 特征 |
|----|------|------|
| **A浪** | 下跌开始 | 多数人仍认为牛市未结束 |
| **B浪** | 牛市陷阱 | 向上反弹的"假象"，成交量不大 |
| **C浪** | 恐慌抛售 | 最具破坏力，跌幅通常超越A浪低点 |

## 三、三大铁律（不可违反）

| 铁律 | 内容 |
|------|------|
| **铁律一** | 第2浪的低点，永远不能低于第1浪的起点 |
| **铁律二** | 第3浪永远不能是1、3、5浪中最短的一浪 |
| **铁律三** | 第4浪的低点，永远不能进入第1浪的价格范围 |

## 四、两个指南（通常成立）

| 指南 | 内容 |
|------|------|
| **交替规律** | 所有波浪形态几乎交替轮流出现 |
| **扩延浪特性** | 三个推动浪中有且仅有一个浪出现延伸，形成9浪或13浪结构 |

## 五、斐波那契比率计算

### 回撤比率（用于调整波）

| 比率 | 应用 |
|------|------|
| 0.382 (38.2%) | 第4浪常见回撤 |
| 0.500 (50%) | 第2浪常见回撤 |
| 0.618 (61.8%) | 大部分第2浪的调整幅度 |

### 延伸比率（用于驱动浪）

| 比率 | 应用 |
|------|------|
| 1.618 | 第3浪 = 第1浪 × 1.618 |
| 2.618 | 第3浪极端延伸目标 |
| 0.618 | 第5浪 = 第1浪 × 0.618 |

### 计算公式

**第2浪目标**：
$$\text{目标} = \text{第1浪高点} - (\text{第1浪长度} \times \text{回撤比率})$$

**第3浪目标**：
$$\text{目标} = \text{第2浪低点} + (\text{第1浪长度} \times 1.618)$$

## 六、各浪特性详解

- **第1浪**：营造底部形态的一部分，相当于头肩底的底部
- **第2浪**：回调幅度可达第1浪的100%，经常出现超卖、背离
- **第3浪**：最具爆发性，成交量最大，出现突破信号（跳空缺口等）
- **第4浪**：与第2浪经常是交替形态关系（单复式交替）
- **第5浪**：在成熟股市中第3浪延伸，在新兴股市中第5浪延伸

## 七、调整波形态分类

| 类型 | 形态 | 特征 |
|------|------|------|
| **简单调整** | 之字型（N字型） | 5-3-5结构 |
| **复杂调整** | 平坦型 | 3-3-5结构 |
| **复杂调整** | 不规则型 | B浪超越A浪起点 |
| **复杂调整** | 三角形型 | 收敛三角 |

## 八、时间周期模型

- **循环周期**：两个顶/底之间的时间间隔
- **趋势周期**：市场升势/跌势的时间长度
- **完整趋势**运行时间：最短为第一波的1.618倍，最长为5.236倍

## 九、实战辅助工具

- **斐波那契回调线**：判断回调目标位
- **RSI指标**：超买超卖区域确认转折点
- **ZigZag指标**：自动标记转折点，加速波浪计数
- **通道线**：连接浪2和浪4终点，延伸预测浪5目标

## 课程化学习补充

> [!important] 学习定位
> 技术指标是价格与成交量的压缩表达，适合做信号过滤、风险控制和交易纪律，不适合孤立预测未来。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标参数是否符合交易周期
- 信号是否经过样本外验证
- 是否与趋势/量能/波动率共振
- 是否明确无效条件

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 指标共线导致虚假确认
- 震荡市和趋势市参数错配
- 过度优化
- 忽略滑点和交易成本

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
