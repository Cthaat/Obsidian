# 江恩时间法则

> [!note] 核心概述
> 江恩时间法则是江恩理论中最具特色的部分，通过数学模型与周期规律研究金融市场中**时间因素与价格波动的关联**。核心包含交易年分割体系、关键时间间隔及以8为基础的分割比率体系。"当时间到达时，价格就会跟随。"

## 一、核心理念：控制时间因素

江恩时间法则的核心方法论"控制时间因素"形成于1908年，并在1909年小麦期权预测中得到验证。

### 市场为什么遵循周期？

- 股市波动从来不是一条直线，而是**一轮又一轮的周期循环**
- 小周期嵌套在大周期之中
- 人心永远在波动，群体情绪在贪婪与恐惧之间来回摇摆
- 市场就像钟摆，永远在狂热的乐观和沮丧的悲观之间来回摆动

## 二、交易年分割体系

江恩将一个交易年按几何比例分割：

| 分割 | 时间间隔 | 应用 |
|------|---------|------|
| 1/2 年 | 6个月（26周） | 重要中期转折 |
| 1/4 年 | 3个月（13周） | 季度转折 |
| 1/8 年 | 6.5周 | 短期转折 |
| 1/3 年 | 4个月（17周） | 三分之一位 |

## 三、关键时间间隔

| 时间间隔 | 性质 | 说明 |
|---------|------|------|
| **30天** | 短期 | 一个月周期 |
| **49-52天** | 中短期 | 7-7.5周，重要变盘窗口 |
| **90天** | 中期 | 一季度，四分之一年 |
| **120天** | 中期 | 三分之一年 |
| **180天** | 中期 | 半年，最重要的中期周期 |
| **270天** | 中长期 | 四分之三年 |
| **360天** | 长期 | 完整一年 |

## 四、以8为基础的分割比率体系

江恩将价格波动以8为基础进行分割：

| 比率 | 百分比 | 支撑/阻力强度 |
|------|--------|-------------|
| 1/8 | 12.5% | 弱 |
| 2/8 | 25% | 中等 |
| 3/8 | 37.5% | 中等 |
| **4/8** | **50%** | **最强** |
| 5/8 | 62.5% | 中等 |
| **6/8** | **75%** | **强** |
| 7/8 | 87.5% | 弱 |
| 8/8 | 100% | 完全回撤 |

## 五、周期嵌套体系

| 周期层级 | 时间跨度 | 特征 |
|---------|---------|------|
| **长期周期** | 20年、30年、50年、100年 | 决定市场结构性大方向 |
| **中期周期** | 10年、15年 | 核心分析单元 |
| **短期周期** | 周线数据 | 研判短线转折 |

### 30年周期

30年是江恩长期分析的基础周期，因为：
- 30年 = 360个月，恰好是圆周360°的映射
- 30年包含多个完整的经济周期
- 历史上多次重大市场转折间隔约30年

### 10年周期

10年是中期核心单元：
- 市场往往在10年周期的相似位置出现类似走势
- 第5年（10年周期的一半）通常是重要的中期转折点

## 六、时间对称现象

### 转折点周年庆

历史上的重大起涨或起跌日，在未来年份的**同一天或相近日**，市场往往容易出现类似的反转现象。

### 时间对称

- 若前一段上升波段运行了30天，当下一次回档也接近30天时，就是需要高度警戒的"对称点"
- 这种对称现象在不同时间尺度上反复出现

## 七、拒绝线性思维，拥抱周期

> [!warning] 线性思维的陷阱
> - 行情连续涨→觉得会永远涨→无脑追涨
> - 行情连续跌→觉得要永远跌→恐慌割肉
>
> 绝大多数散户在用**线性思维**做投资，而市场运行的却是**周期规律**。

### 利用周期的方法

1. 判断当前市场处于周期的哪一段
2. 在低谷积累筹码，在高峰控制风险
3. 打开历史走势图，感受周期轮回
4. 在关键时间节点提高警觉

## 八、与斐波那契时间周期结合

以斐波那契数列（1, 2, 3, 5, 8, 13, 21, 34…）为间隔，计算潜在转折点。当江恩时间周期与斐波那契时间周期重合时，转折概率更高。

## 课程化学习补充

> [!important] 学习定位
> 技术指标是价格与成交量的压缩表达，适合做信号过滤、风险控制和交易纪律，不适合孤立预测未来。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标参数是否符合交易周期
- 信号是否经过样本外验证
- 是否与趋势/量能/波动率共振
- 是否明确无效条件

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 指标共线导致虚假确认
- 震荡市和趋势市参数错配
- 过度优化
- 忽略滑点和交易成本

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
