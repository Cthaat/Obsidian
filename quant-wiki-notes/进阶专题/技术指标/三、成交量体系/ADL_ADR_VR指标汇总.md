# ADL、ADR、OBV、VR指标汇总

> [!note] 💡 概念解析
> ADL（腾落指标）和ADR（涨跌比率）是大势类指标，专门用于研判市场整体趋势；OBV（能量潮）和VR（成交量比率）是成交量类指标，关注成交量与价格的关系。

## 一、大势类指标

> [!warning] 重要提醒
> 大势类指标是专门针对**大盘**设计的专用技术指标，主要描述整个证券市场的多空状况。这类指标**只能用于研判市场的整体趋势**，对个股的研判没有意义。

### 1.1 腾落指标（ADL）

**定义**：通过计算每个交易日价格上涨股票数和价格下跌股票数之间差值的累积结果，与综合指数对比，研判大盘未来趋势。

**特点**：
- 可以弥补股价指数受少数大盘权重股左右的程度
- 具有领先大盘出现上涨或下跌的趋势
- 不能发出明显的买卖信号

**应用法则**：

| 情境 | 信号 |
|------|------|
| ADL与股价指数同步上升创新高 | 确认上升趋势，继续看涨 |
| ADL与股价指数同步下跌创新低 | 确认下跌趋势，继续看跌 |
| ADL连续上升，股价指数持续下跌 | 底背离，买进 |
| ADL连续下跌，股价指数持续上涨 | 顶背离，卖出 |
| 股价指数高位，ADL开始下跌 | 顶背离，卖出 |
| 股价指数低位，ADL开始上升 | 底背离，买进 |

### 1.2 涨跌比率（ADR）

**定义**：通过计算连续多个交易日价格上涨股票累计数与价格下跌股票累计数之间的比值，研究多空两大阵营的比较优势地位。

**计算公式**：

$$\text{ADR}(N) = \frac{\sum \text{上涨股票数}(N日)}{\sum \text{下跌股票数}(N日)}$$

**参数设置**：
- 常用时间参数：**10天**
- 多空分界线：**1**
- 常态区域：**0.5-1.5**

**应用法则**：

| ADR数值 | 市场状态 | 操作建议 |
|--------|---------|---------|
| 0.5-1.5 | 多空势均力敌 | 观望 |
| < 0.5 | 超卖状态 | 伺机买进 |
| > 1.5 | 超买状态 | 逢高卖出 |

> [!tip] ADR使用技巧
> 使用ADR时周期的确定最好与**均线分析使用的周期一致**。

## 二、成交量类指标

### 2.1 能量潮指标（OBV）

以成交量的累积来反映多空双方力量对比的变化，理论基础是"**量先于价而行**"。

**应用法则**：
- OBV上升 → 股价将上涨，回档时买进
- OBV下降 → 股价将下跌，反弹时卖出
- OBV与股价变化趋势相同 → 确认行情
- OBV与股价背离 → 预示行情反转

### 2.2 成交量比率（VR）

**定义**：通过计算一定时期内上涨日成交量与下跌日成交量的比值，来衡量市场的买卖气势。

**计算公式**：

$$\text{VR}(N) = \frac{\sum \text{上涨日成交量}(N日)}{\sum \text{下跌日成交量}(N日)} \times 100$$

**应用法则**：

| VR数值 | 市场状态 | 操作建议 |
|--------|---------|---------|
| 80-150 | 正常波动 | 观望 |
| < 70 | 超卖 | 买进 |
| > 350 | 超买 | 卖出 |
| 450以上 | 极度超买 | 果断卖出 |

## 三、四大指标对比

| 指标 | 类型 | 适用范围 | 核心功能 |
|------|------|---------|---------|
| ADL | 大势指标 | 大盘 | 趋势确认与背离 |
| ADR | 大势指标 | 大盘 | 多空力量对比 |
| OBV | 成交量指标 | 个股/大盘 | 量价关系分析 |
| VR | 成交量指标 | 个股/大盘 | 买卖气势衡量 |

## 课程化学习补充

> [!important] 学习定位
> 技术指标是价格与成交量的压缩表达，适合做信号过滤、风险控制和交易纪律，不适合孤立预测未来。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标参数是否符合交易周期
- 信号是否经过样本外验证
- 是否与趋势/量能/波动率共振
- 是否明确无效条件

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 指标共线导致虚假确认
- 震荡市和趋势市参数错配
- 过度优化
- 忽略滑点和交易成本

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `ADL、ADR、OBV、VR指标汇总` 时，不要只把它当成孤立知识点。把指标当成信号过滤器和纪律工具，不能替代交易系统。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 趋势、动量、均值回归和波动率
- 成交量和资金流验证
- 多周期共振与冲突
- 成本、滑点和过度交易

### 进阶训练

1. 比较指标在趋势市和震荡市的表现
2. 给每个信号定义入场、退出、止损和暂停条件
3. 用样本外数据检查参数稳定性

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
