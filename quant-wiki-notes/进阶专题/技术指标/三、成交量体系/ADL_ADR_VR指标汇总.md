# ADL、ADR、OBV、VR指标汇总

> [!note] 💡 概念解析
> ADL（腾落指标）和ADR（涨跌比率）是大势类指标，专门用于研判市场整体趋势；OBV（能量潮）和VR（成交量比率）是成交量类指标，关注成交量与价格的关系。

## 一、大势类指标

> [!warning] 重要提醒
> 大势类指标是专门针对**大盘**设计的专用技术指标，主要描述整个证券市场的多空状况。这类指标**只能用于研判市场的整体趋势**，对个股的研判没有意义。

### 1.1 腾落指标（ADL）

**定义**：通过计算每个交易日价格上涨股票数和价格下跌股票数之间差值的累积结果，与综合指数对比，研判大盘未来趋势。

**特点**：
- 可以弥补股价指数受少数大盘权重股左右的程度
- 具有领先大盘出现上涨或下跌的趋势
- 不能发出明显的买卖信号

**应用法则**：

| 情境 | 信号 |
|------|------|
| ADL与股价指数同步上升创新高 | 确认上升趋势，继续看涨 |
| ADL与股价指数同步下跌创新低 | 确认下跌趋势，继续看跌 |
| ADL连续上升，股价指数持续下跌 | 底背离，买进 |
| ADL连续下跌，股价指数持续上涨 | 顶背离，卖出 |
| 股价指数高位，ADL开始下跌 | 顶背离，卖出 |
| 股价指数低位，ADL开始上升 | 底背离，买进 |

### 1.2 涨跌比率（ADR）

**定义**：通过计算连续多个交易日价格上涨股票累计数与价格下跌股票累计数之间的比值，研究多空两大阵营的比较优势地位。

**计算公式**：

$$\text{ADR}(N) = \frac{\sum \text{上涨股票数}(N日)}{\sum \text{下跌股票数}(N日)}$$

**参数设置**：
- 常用时间参数：**10天**
- 多空分界线：**1**
- 常态区域：**0.5-1.5**

**应用法则**：

| ADR数值 | 市场状态 | 操作建议 |
|--------|---------|---------|
| 0.5-1.5 | 多空势均力敌 | 观望 |
| < 0.5 | 超卖状态 | 伺机买进 |
| > 1.5 | 超买状态 | 逢高卖出 |

> [!tip] ADR使用技巧
> 使用ADR时周期的确定最好与**均线分析使用的周期一致**。

## 二、成交量类指标

### 2.1 能量潮指标（OBV）

以成交量的累积来反映多空双方力量对比的变化，理论基础是"**量先于价而行**"。

**应用法则**：
- OBV上升 → 股价将上涨，回档时买进
- OBV下降 → 股价将下跌，反弹时卖出
- OBV与股价变化趋势相同 → 确认行情
- OBV与股价背离 → 预示行情反转

### 2.2 成交量比率（VR）

**定义**：通过计算一定时期内上涨日成交量与下跌日成交量的比值，来衡量市场的买卖气势。

**计算公式**：

$$\text{VR}(N) = \frac{\sum \text{上涨日成交量}(N日)}{\sum \text{下跌日成交量}(N日)} \times 100$$

**应用法则**：

| VR数值 | 市场状态 | 操作建议 |
|--------|---------|---------|
| 80-150 | 正常波动 | 观望 |
| < 70 | 超卖 | 买进 |
| > 350 | 超买 | 卖出 |
| 450以上 | 极度超买 | 果断卖出 |

## 三、四大指标对比

| 指标 | 类型 | 适用范围 | 核心功能 |
|------|------|---------|---------|
| ADL | 大势指标 | 大盘 | 趋势确认与背离 |
| ADR | 大势指标 | 大盘 | 多空力量对比 |
| OBV | 成交量指标 | 个股/大盘 | 量价关系分析 |
| VR | 成交量指标 | 个股/大盘 | 买卖气势衡量 |

## 实战掌握清单

> [!tip] 交易者视角
> ADL、ADR、OBV、VR指标汇总 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。技术指标是价格、成交量和波动率的二次加工，核心价值在于把主观观察变成稳定规则。

### 关键判断

- 先确认指标属于趋势、震荡、量能、波动率还是资金流。
- 判断当前市场是否适合该指标：趋势指标怕横盘，震荡指标怕单边。
- 把参数选择、信号延迟和交易频率写清楚。

### 落地动作

1. 用样本外数据检验信号，而不是只看历史图形好不好看。
2. 同时记录胜率、盈亏比、换手、滑点和回撤。
3. 把指标作为过滤器、触发器或退出规则，避免多个同源指标重复投票。

### 失效边界

- 参数过拟合。
- 忽略手续费和滑点。
- 在市场结构变化后继续迷信旧阈值。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
