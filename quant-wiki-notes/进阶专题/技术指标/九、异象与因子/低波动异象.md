# 低波动异象

> [!note] 💡 概念解析
> 低波动异象（Low Risk Anomaly）是指长期来看，低波动率股票的期望收益高于高波动率股票的现象，这一在全球市场中均长期存在的异象主要由投资者非理性行为驱动。本文介绍了低风险异象的性质、成因及常见因子构造方法。

## 一、低波动异象的性质

### 1.1 波动率的分类

| 类型 | 定义 | 特点 |
|------|------|------|
| 历史波动率 | 过去一段时间收益率的标准差 | 最常用 |
| 已实现波动率 | 日内高频对数收益率的平方和 | 描述日内波动 |
| 隐含波动率 | 由期权价格通过Black-Scholes公式反推 | 对未来波动率的预期 |

### 1.2 核心特征

> [!important] 🔑 核心发现
> 股票波动率与未来收益率呈显著的负相关关系，即低波动率股票的期望收益高于高波动率股票。

### 1.3 收益特征

| 特征 | 说明 |
|------|------|
| 不对称性 | 空头收益远大于多头收益 |
| 逆周期性 | 牛市表现不佳，熊市绩效较好 |
| 回撤保护 | 在行情低迷时提供一定保护 |

**逆周期原因**：低风险异象主要是非理性行为驱动的，在投资者情绪高涨的牛市中，对高估值股票的错误定价难以修复，波动率因子会遭遇大幅回撤。

## 二、与其他因子的关系

### 2.1 与价值、市值因子的联系

| 关系 | 说明 |
|------|------|
| 低波动率股票 | 往往是低估值的大盘股 |
| 高波动率股票 | 往往是小市值、炒作行情更多的股票 |
| 与价值因子 | 部分可被质量因子解释 |

### 2.2 因子表现差异

低波动因子在以下个股中表现更好：
- 冲击成本较高的个股
- 无分析师覆盖的个股
- 非股指期货指数对应成分股
- 非融券的个股

### 2.3 与动量因子的关系

> [!tip] 💡 关键关系
> 如果用涨幅定义动量，波动率定义风险，那么动量因子和波动因子就是收益率一阶矩和二阶矩的关系。两者无法互相解释。

**非线性关系**：高波动输家组合收益远低于高波动赢家组合，高波动较差的表现主要是输家股票贡献的。

## 三、低波动异象的成因

### 3.1 个人投资者因素

| 行为偏差 | 影响 |
|----------|------|
| 博彩偏好 | 高估小概率事件，购买"彩票股" |
| 代表性偏差 | 对异常涨幅股票过度追捧 |
| 过度自信 | 对高波动率股票需求过度 |
| 需求错配 | 对低波动率股票需求不足 |

**结果**：高波动率股票当期价格高估，低波动率股票当期价格低估。

### 3.2 机构投资者因素

| 因素 | 说明 |
|------|------|
| 套利限制 | 缺乏低成本做空机制，高波动股票做空成本高 |
| 委托代理问题 | 超配低波动股票增加跟踪误差，影响基金经理考核 |
| 激励机制 | 基金经理通过挑选高波动股票最大化预期收入 |

> [!important] 🔑 核心结论
> 由于做空成本、杠杆受限、固定业绩比较基准等套利限制，机构投资者不能对低风险异常进行套利，甚至会进一步加剧由个人投资者行为偏差带来的低波动异象。

## 四、因子构造方法

### 4.1 日频波动率因子

| 因子 | 说明 |
|------|------|
| 简单波动率 std | 收益率标准差 |
| 日收益偏度 skew | 收益分布的偏度 |
| 真实波动率 | 考虑跳空的真实波动 |
| 振幅 | 日内最高价与最低价之比 |
| 特质波动率 | 剔除市场因素后的波动率 |
| 特质偏度 | 剔除市场因素后的偏度 |
| 特异度 | 个股特有风险 |
| MAX | 最大日收益率 |
| MIN | 最小日收益率 |

**振幅因子改进方向**：
- 方正证券：个股股价跳跃及其对振幅因子的改进
- 开源证券：振幅因子的隐藏结构

### 4.2 高频波动率因子

#### 高阶矩

| 因子 | 说明 |
|------|------|
| 已实现波动率 | 日内高频收益率的平方和 |
| 已实现偏度 | 日内收益率分布的偏度 |
| 已实现峰度 | 日内收益率分布的峰度 |
| 已实现超偏度 | 标准化后的偏度 |
| 已实现超峰度 | 标准化后的峰度 |

#### 尾部风险

| 因子 | 说明 |
|------|------|
| VaR | 在险价值 |
| cVaR | 条件在险价值 |
| VaR_RT | 基于已实现波动率的VaR |
| cVaR_RT | 基于已实现波动率的cVaR |

#### 风险不确定因子

| 因子 | 说明 |
|------|------|
| VoV | 波动率的波动率 |
| VoK | 波动率的峰度 |
| VoHT | 波动率的Hurst指数 |
| VoVaR/VocVaR | 波动率的VaR/cVaR |

#### 波动分解

| 因子 | 说明 |
|------|------|
| RJVP | 上行跳跃波动率 |
| SRJV | 上下行跳跃波动率不对称性 |
| RLJVP | 长程上行跳跃波动率 |
| SRLJV | 长程上下行跳跃波动率不对称性 |

> [!tip] 💡 因子计算
> 所有高频因子的计算中，收益率可以用特质收益率代替，即剔除市场因素后的个股收益率。

## 五、投资启示

| 启示 | 说明 |
|------|------|
| 低波动选股 | 长期配置低波动率股票可获得超额收益 |
| 逆周期配置 | 在牛市中低波动因子可能回撤，但熊市提供保护 |
| 因子组合 | 考虑动量与波动的非线性关系，避免高波动输家组合 |
| 适用范围 | 低波动因子在小市值、无分析师覆盖个股中效果更好 |

## 实战掌握清单

> [!tip] 交易者视角
> 低波动异象 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。技术指标是价格、成交量和波动率的二次加工，核心价值在于把主观观察变成稳定规则。

### 关键判断

- 先确认指标属于趋势、震荡、量能、波动率还是资金流。
- 判断当前市场是否适合该指标：趋势指标怕横盘，震荡指标怕单边。
- 把参数选择、信号延迟和交易频率写清楚。

### 落地动作

1. 用样本外数据检验信号，而不是只看历史图形好不好看。
2. 同时记录胜率、盈亏比、换手、滑点和回撤。
3. 把指标作为过滤器、触发器或退出规则，避免多个同源指标重复投票。

### 失效边界

- 参数过拟合。
- 忽略手续费和滑点。
- 在市场结构变化后继续迷信旧阈值。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
