# 五大核心技术指标指南

> [!note] 💡 概念解析
> 五大核心技术指标——MA（移动平均线）、MACD（指数平滑异同移动平均线）、RSI（相对强弱指数）、KDJ（随机指标）、BOLL（布林带），构成了技术分析的核心框架。

## 一、MA（移动平均线）

### 1.1 基本概念

$$\text{MA}(N) = \frac{P_1 + P_2 + \cdots + P_N}{N}$$

| 参数 | 说明 |
|------|------|
| 短期MA | 5日、10日、20日 |
| 中期MA | 30日、60日 |
| 长期MA | 120日、250日 |

### 1.2 核心信号

- **金叉**：短期MA上穿长期MA → 买入信号
- **死叉**：短期MA下穿长期MA → 卖出信号
- **多头排列**：短期MA > 中期MA > 长期MA → 上涨趋势
- **空头排列**：短期MA < 中期MA < 长期MA → 下跌趋势

## 二、MACD（指数平滑异同移动平均线）

### 2.1 计算公式

$$\text{DIF} = \text{EMA}(12) - \text{EMA}(26)$$
$$\text{DEA} = \text{EMA}(\text{DIF}, 9)$$
$$\text{MACD柱} = 2 \times (\text{DIF} - \text{DEA})$$

### 2.2 核心信号

| 信号 | 条件 | 含义 |
|------|------|------|
| 金叉 | DIF上穿DEA | 买入 |
| 死叉 | DIF下穿DEA | 卖出 |
| 顶背离 | 价格创新高，MACD未创新高 | 看跌 |
| 底背离 | 价格创新低，MACD未创新低 | 看涨 |

## 三、RSI（相对强弱指数）

### 3.1 计算公式

$$\text{RSI}(N) = 100 - \frac{100}{1 + \text{RS}}$$
$$\text{RS} = \frac{\text{N日平均涨幅}}{\text{N日平均跌幅}}$$

### 3.2 应用法则

| RSI数值 | 市场状态 | 操作建议 |
|--------|---------|---------|
| > 80 | 超买 | 卖出 |
| 50-80 | 正常 | 持有 |
| 20-50 | 正常 | 观望 |
| < 20 | 超卖 | 买入 |

## 四、KDJ（随机指标）

### 4.1 计算公式

$$\text{RSV} = \frac{C - L_9}{H_9 - L_9} \times 100$$
$$K = \frac{2}{3} K_{-1} + \frac{1}{3} \text{RSV}$$
$$D = \frac{2}{3} D_{-1} + \frac{1}{3} K$$
$$J = 3K - 2D$$

### 4.2 应用法则

- **金叉**：K线上穿D线 → 买入
- **死叉**：K线下穿D线 → 卖出
- **超买**：K、D > 80 → 卖出
- **超卖**：K、D < 20 → 买入

## 五、BOLL（布林带）

### 5.1 计算公式

$$\text{中轨} = \text{MA}(N)$$
$$\text{上轨} = \text{中轨} + k \times \sigma$$
$$\text{下轨} = \text{中轨} - k \times \sigma$$

其中 $\sigma$ 为标准差，$k$ 通常取2。

### 5.2 应用法则

- 价格触及**上轨** → 超买，可能回调
- 价格触及**下轨** → 超卖，可能反弹
- 布林带**收窄** → 变盘在即
- 布林带**扩张** → 趋势延续

## 六、五大指标的组合使用

> [!tip] 组合策略
> 1. **趋势确认**：MA + MACD
> 2. **超买超卖**：RSI + KDJ
> 3. **波动性分析**：BOLL
> 4. **多指标共振**：至少3个指标给出相同信号

## 实战掌握清单

> [!tip] 交易者视角
> 五大核心技术指标指南 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。技术指标是价格、成交量和波动率的二次加工，核心价值在于把主观观察变成稳定规则。

### 关键判断

- 先确认指标属于趋势、震荡、量能、波动率还是资金流。
- 判断当前市场是否适合该指标：趋势指标怕横盘，震荡指标怕单边。
- 把参数选择、信号延迟和交易频率写清楚。

### 落地动作

1. 用样本外数据检验信号，而不是只看历史图形好不好看。
2. 同时记录胜率、盈亏比、换手、滑点和回撤。
3. 把指标作为过滤器、触发器或退出规则，避免多个同源指标重复投票。

### 失效边界

- 参数过拟合。
- 忽略手续费和滑点。
- 在市场结构变化后继续迷信旧阈值。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
