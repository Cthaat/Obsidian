# MACD择时策略

> [!note] 💡 概念解析
> MACD择时策略教程介绍了如何利用MACD技术指标构建量化交易策略，包括MACD的组成原理、金叉死叉的实战用法、大盘择时概念，以及基于聚宽平台的完整策略实现。

## 一、MACD的组成

MACD（Moving Average Convergence and Divergence）即**指数平滑移动平均线**，由Gerald Appel于1970年提出，属于大势趋势类指标。

**"两线两柱一轴"组合**：

| 组成部分 | 说明 |
|----------|------|
| DIF（快线） | 短期快线，EMA12 - EMA26 |
| DEA（慢线） | 长期慢线，DIF的9日EMA |
| 红色能量柱 | 多头力量（DIF > DEA） |
| 绿色能量柱 | 空头力量（DIF < DEA） |
| 0轴 | 多空分界线 |

## 二、MACD核心思想

> [!important] 🔑 核心原则
> 由快、慢均线的离散、聚合来显示当前的多空状态和股价可能的发展变化趋势，并对买进、卖出时机作出研判。

**基本判断**：
- MACD从负数转向正数 → **买入信号**
- MACD从正数转向负数 → **卖出信号**
- MACD以大角度变化 → 代表市场大趋势的转变

## 三、DIF与DEA金叉的使用

### 3.1 零轴之下金叉（弱势金叉）

- 股价属于弱势市场
- 金叉为弱势金叉，股价极有可能短期弱势反弹或暂时止跌后再度夭折
- **一般没有做多价值**

### 3.2 零轴之上金叉（强势金叉）

- 股价属于强势市场
- 金叉后股价后市成功惯性上攻的概率较高

### 3.3 零上二次红金叉（极佳买点）

- 零轴之上出现二次金叉信号
- 属于强势中的强势
- 股价往往容易出现加速上涨
- **短线极佳买点**

## 四、DIF与DEA死叉的使用

### 4.1 零轴之上死叉

- 一般认为是股价上升趋势途中的短暂回调
- 后市股价仍有再次走强的可能

### 4.2 零轴之下死叉

- 股价下跌趋势途中的继续回调
- 后市股价惯性下跌的概率较高
- 零轴之下出现**二次死叉**时，股价更容易出现加速下跌
- **投资者应当空仓观望**

## 五、MACD择时策略设计

### 5.1 策略思路

**选股条件**：
- 10 < 市盈率（PE） < 40
- 市净率（PB） < 3
- 净利润环比增长率 > 0.3
- 净资产收益率（ROE） > 15
- 按销售毛利率降序排列，选取前50只

**交易规则**：
- **买入条件**：DIF上穿DEA（金叉），且DIF > 0（零轴之上）
- **卖出条件**：DIF下穿DEA（死叉），或DIF < 0

### 5.2 大盘择时概念

> [!tip] 💡 大盘择时
> 大盘择时是指通过技术指标判断大盘的整体走势，在大盘向好时积极参与，在大盘走弱时减少仓位或空仓。

**大盘择时的优势**：
- 避免系统性风险
- 提高策略整体收益
- 减少不必要的交易

### 5.3 策略实现要点

**使用talib计算MACD**：
```python
import talib
# 计算MACD
dif, dea, macd_hist = talib.MACD(
    close_prices,
    fastperiod=12,
    slowperiod=26,
    signalperiod=9
)
```

**调仓频率**：通常设置为每10个交易日调仓一次

## 六、策略优化方向

| 优化方向 | 说明 |
|----------|------|
| 多周期验证 | 日线MACD + 周线MACD共振 |
| 量价配合 | 金叉时成交量放大确认 |
| 基本面过滤 | 结合财务指标筛选优质标的 |
| 止损机制 | 设置最大亏损比例 |

> [!warning] ⚠️ 注意事项
> 1. 每天重新获取数据计算MACD，相当于滑动窗口计算，但300条数据产生的误差可忽略不计
> 2. 需要先过滤掉ST、停牌股票再进行筛选
> 3. 注意持仓限制和资金管理

## 课程化学习补充

> [!important] 学习定位
> 技术指标是价格与成交量的压缩表达，适合做信号过滤、风险控制和交易纪律，不适合孤立预测未来。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标参数是否符合交易周期
- 信号是否经过样本外验证
- 是否与趋势/量能/波动率共振
- 是否明确无效条件

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 指标共线导致虚假确认
- 震荡市和趋势市参数错配
- 过度优化
- 忽略滑点和交易成本

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `MACD择时策略` 时，不要只把它当成孤立知识点。把指标当成信号过滤器和纪律工具，不能替代交易系统。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 趋势、动量、均值回归和波动率
- 成交量和资金流验证
- 多周期共振与冲突
- 成本、滑点和过度交易

### 进阶训练

1. 比较指标在趋势市和震荡市的表现
2. 给每个信号定义入场、退出、止损和暂停条件
3. 用样本外数据检查参数稳定性

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
