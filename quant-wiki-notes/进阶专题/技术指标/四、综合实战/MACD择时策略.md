## 一、MACD的组成

MACD（Moving Average Convergence and Divergence）即**指数平滑移动平均线**，由Gerald Appel于1970年提出，属于大势趋势类指标。

**"两线两柱一轴"组合**：

| 组成部分 | 说明 |
|----------|------|
| DIF（快线） | 短期快线，EMA12 - EMA26 |
| DEA（慢线） | 长期慢线，DIF的9日EMA |
| 红色能量柱 | 多头力量（DIF > DEA） |
| 绿色能量柱 | 空头力量（DIF < DEA） |
| 0轴 | 多空分界线 |

## 二、MACD核心思想

> [!important] 🔑 核心原则
> 由快、慢均线的离散、聚合来显示当前的多空状态和股价可能的发展变化趋势，并对买进、卖出时机作出研判。

**基本判断**：
- MACD从负数转向正数 → **买入信号**
- MACD从正数转向负数 → **卖出信号**
- MACD以大角度变化 → 代表市场大趋势的转变

## 三、DIF与DEA金叉的使用

### 3.1 零轴之下金叉（弱势金叉）

- 股价属于弱势市场
- 金叉为弱势金叉，股价极有可能短期弱势反弹或暂时止跌后再度夭折
- **一般没有做多价值**

### 3.2 零轴之上金叉（强势金叉）

- 股价属于强势市场
- 金叉后股价后市成功惯性上攻的概率较高

### 3.3 零上二次红金叉（极佳买点）

- 零轴之上出现二次金叉信号
- 属于强势中的强势
- 股价往往容易出现加速上涨
- **短线极佳买点**

## 四、DIF与DEA死叉的使用

### 4.1 零轴之上死叉

- 一般认为是股价上升趋势途中的短暂回调
- 后市股价仍有再次走强的可能

### 4.2 零轴之下死叉

- 股价下跌趋势途中的继续回调
- 后市股价惯性下跌的概率较高
- 零轴之下出现**二次死叉**时，股价更容易出现加速下跌
- **投资者应当空仓观望**

## 五、MACD择时策略设计

### 5.1 策略思路

**选股条件**：
- 10 < 市盈率（PE） < 40
- 市净率（PB） < 3
- 净利润环比增长率 > 0.3
- 净资产收益率（ROE） > 15
- 按销售毛利率降序排列，选取前50只

**交易规则**：
- **买入条件**：DIF上穿DEA（金叉），且DIF > 0（零轴之上）
- **卖出条件**：DIF下穿DEA（死叉），或DIF < 0

### 5.2 大盘择时概念

> [!tip] 💡 大盘择时
> 大盘择时是指通过技术指标判断大盘的整体走势，在大盘向好时积极参与，在大盘走弱时减少仓位或空仓。

**大盘择时的优势**：
- 避免系统性风险
- 提高策略整体收益
- 减少不必要的交易

### 5.3 策略实现要点

**使用talib计算MACD**：
```python
import talib
# 计算MACD
dif, dea, macd_hist = talib.MACD(
    close_prices, 
    fastperiod=12, 
    slowperiod=26, 
    signalperiod=9
)
```

**调仓频率**：通常设置为每10个交易日调仓一次

## 六、策略优化方向

| 优化方向 | 说明 |
|----------|------|
| 多周期验证 | 日线MACD + 周线MACD共振 |
| 量价配合 | 金叉时成交量放大确认 |
| 基本面过滤 | 结合财务指标筛选优质标的 |
| 止损机制 | 设置最大亏损比例 |

> [!warning] ⚠️ 注意事项
> 1. 每天重新获取数据计算MACD，相当于滑动窗口计算，但300条数据产生的误差可忽略不计
> 2. 需要先过滤掉ST、停牌股票再进行筛选
> 3. 注意持仓限制和资金管理
