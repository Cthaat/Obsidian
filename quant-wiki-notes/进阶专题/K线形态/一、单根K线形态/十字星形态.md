# 十字星形态

> [!note] 十字星（Doji）
> 开盘价≈收盘价，实体被压成一条线：多空打了一整天，**打成平手**。它本身是**中性**的"犹豫信号"，方向完全取决于**出现的位置**和**下一根 K 线**。把十字星当"变盘提示灯"，而不是买卖指令。

## 一、四种十字星

| 类型 | 形态 | 含义 | 倾向 |
|---|---|---|---|
| 长腿十字 | 上下影都长 | 剧烈震荡、分歧极大 | 趋势末端易反转 |
| 蜻蜓十字 | 长下影、几乎无上影 | 砸下去被全部买回 | 偏看涨（低位） |
| 墓碑十字 | 长上影、几乎无下影 | 冲上去被全部打回 | 偏看跌（高位） |
| 一字线 | 无影无实体 | 涨跌停、完全停滞 | 极端单边 |

```
长腿十字   蜻蜓十字   墓碑十字
  │          ┼          ┃
  ┼          │          ┼
  │          │
```

## 二、为什么位置决定一切

> [!important] 同一个十字星，高低位含义相反
> - **上涨高位**出现（尤其墓碑十字）→ 多方推不动了，警惕见顶；
> - **下跌低位**出现（尤其蜻蜓十字）→ 空方砸不动了，可能见底；
> - **趋势中段**出现 → 多为暂歇/换手，意义有限。

## 三、确认与配合

| 维度 | 要点 |
|---|---|
| 确认 K 线 | 高位墓碑→等下一根阴线；低位蜻蜓→等下一根阳线 |
| 位置 | 关键支撑/阻力位附近的十字星更有意义 |
| 指标背离 | 与 MACD/RSI 背离同时出现，反转概率更高（[[技术指标]]相关） |
| 量能 | 放量十字星分歧更剧烈，变盘概率更大 |

## 四、实战要点

- 十字星是"等待"信号，不是"动手"信号——**先看下一根**；
- 在重要价位 + 指标背离 + 十字星三者共振时，才是高质量反转线索；
- 止损依托十字星当日的高/低点。

## 五、常见误区

| 误区 | 更好的理解 |
|---|---|
| 见十字星就反转 | 中性信号，需位置+确认 |
| 各类十字一个样 | 蜻蜓偏多、墓碑偏空，差别大 |
| 趋势中段的十字也重仓博反转 | 多为换手，意义弱 |
| 不等确认就动手 | 十字星本身不给方向 |

## 课程化学习补充

> [!important] 学习定位
> K线只描述价格行为，不单独构成交易系统；必须与趋势级别、成交量、波动率和止损规则一起使用。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 形态是否出现在关键位置
- 成交量是否确认
- 信号是否有统计样本支持
- 止损点是否先于入场确定

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 把形态当确定预测
- 忽略大级别趋势
- 假突破和诱多诱空
- 频繁交易导致成本吞噬

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `十字星形态` 时，不要只把它当成孤立知识点。把形态当成市场参与者行为的压缩表达，必须接受统计验证和止损规则约束。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 趋势级别和关键位置
- 成交量、波动率和盘口流动性
- 假突破和诱多诱空
- 交易成本和频率控制

### 进阶训练

1. 收集至少 50 个同类形态样本
2. 比较突破后 1/5/20 日收益分布
3. 为形态信号写出无效条件和止损点

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
