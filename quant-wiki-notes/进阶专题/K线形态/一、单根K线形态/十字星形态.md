# 十字星形态

> [!note] 十字星（Doji）
> 开盘价≈收盘价，实体被压成一条线：多空打了一整天，**打成平手**。它本身是**中性**的"犹豫信号"，方向完全取决于**出现的位置**和**下一根 K 线**。把十字星当"变盘提示灯"，而不是买卖指令。

## 一、四种十字星

| 类型 | 形态 | 含义 | 倾向 |
|---|---|---|---|
| 长腿十字 | 上下影都长 | 剧烈震荡、分歧极大 | 趋势末端易反转 |
| 蜻蜓十字 | 长下影、几乎无上影 | 砸下去被全部买回 | 偏看涨（低位） |
| 墓碑十字 | 长上影、几乎无下影 | 冲上去被全部打回 | 偏看跌（高位） |
| 一字线 | 无影无实体 | 涨跌停、完全停滞 | 极端单边 |

```

  │          ┼          ┃
  ┼          │          ┼
  │          │
```

## 二、为什么位置决定一切

> [!important] 同一个十字星，高低位含义相反
> - **上涨高位**出现（尤其墓碑十字）→ 多方推不动了，警惕见顶；
> - **下跌低位**出现（尤其蜻蜓十字）→ 空方砸不动了，可能见底；
> - **趋势中段**出现 → 多为暂歇/换手，意义有限。

## 三、确认与配合

| 维度 | 要点 |
|---|---|
| 确认 K 线 | 高位墓碑→等下一根阴线；低位蜻蜓→等下一根阳线 |
| 位置 | 关键支撑/阻力位附近的十字星更有意义 |
| 指标背离 | 与 MACD/RSI 背离同时出现，反转概率更高（[[技术指标]]相关） |
| 量能 | 放量十字星分歧更剧烈，变盘概率更大 |

## 四、实战要点

- 十字星是"等待"信号，不是"动手"信号——**先看下一根**；
- 在重要价位 + 指标背离 + 十字星三者共振时，才是高质量反转线索；
- 止损依托十字星当日的高/低点。

## 五、常见误区

| 误区 | 更好的理解 |
|---|---|
| 见十字星就反转 | 中性信号，需位置+确认 |
| 各类十字一个样 | 蜻蜓偏多、墓碑偏空，差别大 |
| 趋势中段的十字也重仓博反转 | 多为换手，意义弱 |
| 不等确认就动手 | 十字星本身不给方向 |

## 实战掌握清单

> [!tip] 交易者视角
> 十字星形态 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。K线形态只能描述价格行为，不能单独证明胜率；它必须和趋势、成交量、波动率、支撑阻力和交易计划结合。

### 关键判断

- 先判断大级别趋势和关键价位，再看形态本身。
- 观察形态出现时的成交量、波动率扩张和市场情绪。
- 用明确入场点、失效点和盈亏比约束主观解释。

### 落地动作

1. 把形态转成可量化条件，例如影线比例、实体大小、突破幅度和确认K线。
2. 用不同市场阶段分别统计胜率、平均盈亏和最大连续亏损。
3. 交易前写下如果形态失败，何时退出而不是临盘解释。

### 失效边界

- 震荡市频繁假突破。
- 低流动性品种容易被单笔交易扭曲形态。
- 只看图形名称，不看位置、量能和风险收益比。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 形态复盘

围绕 十字星形态 收集至少三十个案例，按上涨趋势、下跌趋势和震荡环境分类。每个案例记录位置、成交量、确认K线、入场点、止损点和实际盈亏。

### 交易化规则

- 把形态描述转成数值条件，减少主观解释。
- 只在风险收益比足够时交易，而不是看到形态就下单。
- 把失败样本单独归档，寻找假突破和诱多诱空特征。

### 失效边界

形态是概率工具，不是预测工具。

## 最小可执行项目

### 形态样本库

围绕 十字星形态 建一个形态样本库：每个样本记录位置、级别、趋势、成交量、确认信号、入场、止损、目标位和结果。

| 样本类型 | 学习重点 |
|---|---|
| 成功突破 | 量价和位置条件 |
| 假突破 | 诱多诱空特征 |
| 震荡失败 | 降低交易频率 |
| 趋势延续 | 持仓和移动止损 |

### 验收标准

能说出该形态的最佳使用环境和最危险环境。
