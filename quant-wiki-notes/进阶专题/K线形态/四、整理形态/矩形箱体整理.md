# 矩形箱体整理

> [!note] 矩形箱体整理（Rectangle / Box）
> 价格在一条**水平阻力**和一条**水平支撑**之间来回震荡，形成一个"箱子"。它代表多空在该区间内暂时平衡。箱体既可能是续势整理，也可能是反转区——**方向由突破决定**，突破前不预设。

## 一、形态特征

```
阻力 ───────────────  ← 多次顶到这里被压回
   ╱╲  ╱╲  ╱╲
   ╲╱  ╲╱  ╲╱
支撑 ───────────────  ← 多次跌到这里被托起
```

| 部件 | 含义 |
|---|---|
| 上边界 | 水平阻力（被反复测试） |
| 下边界 | 水平支撑（被反复测试） |
| 量能 | 箱体内通常逐步萎缩，突破时放量 |

## 二、箱体理论（Box Theory）

> [!tip] 箱体的核心思想
> 价格在"箱子"里运行，**有效突破箱子边界 = 新趋势可能开始**，箱体高度即最小目标。箱体整理**时间越长**，蓄势越久，突破后力度往往越大。

## 三、突破与目标

$$
\text{突破目标} \approx \text{突破点} \pm \text{箱体高度（阻力−支撑）}
$$

- 向上有效突破阻力 → 看涨，目标上移一个箱高；
- 向下有效跌破支撑 → 看跌，目标下移一个箱高。

## 四、续势 vs 反转

| 出现位置 | 倾向 |
|---|---|
| 上涨途中的箱体 | 多为续势，倾向向上突破 |
| 下跌途中的箱体 | 多为续势，倾向向下突破 |
| 长期顶部/底部的箱体 | 可能是反转区（顶部派发/底部吸筹） |

## 五、实战要点（含区间交易）

| 玩法 | 做法 |
|---|---|
| 区间交易 | 近支撑买、近阻力卖（箱体足够宽时） |
| 突破交易 | 放量破边界后顺势入场 |
| 止损 | 区间交易破边界即止损；突破交易回到箱内即止损 |

> [!warning] 假突破是箱体最大陷阱
> 主力常制造"假突破"诱多/诱空：缩量突破边界后迅速拉回箱内。**真突破要放量 + 站稳（如 3 日不回箱）**。假突破后价格会回到箱体内运行。

## 六、常见误区

| 误区 | 更好的理解 |
|---|---|
| 提前猜突破方向 | 突破前不预设，跟随突破 |
| 缩量突破也追 | 真突破要放量 |
| 箱体内重仓赌方向 | 区间内应高抛低吸或观望 |
| 不设止损 | 假突破频繁，止损必须 |

## 课程化学习补充

> [!important] 学习定位
> K线只描述价格行为，不单独构成交易系统；必须与趋势级别、成交量、波动率和止损规则一起使用。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 形态是否出现在关键位置
- 成交量是否确认
- 信号是否有统计样本支持
- 止损点是否先于入场确定

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 把形态当确定预测
- 忽略大级别趋势
- 假突破和诱多诱空
- 频繁交易导致成本吞噬

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
