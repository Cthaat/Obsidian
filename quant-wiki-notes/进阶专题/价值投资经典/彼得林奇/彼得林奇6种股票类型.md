# 彼得林奇的六种股票分类法

> [!note] 核心框架
> 彼得林奇在麦哲伦基金13年年化29%的传奇业绩背后，是对不同类型股票采取差异化策略。他把所有股票划分为六类，每类的投资逻辑和估值方法各不相同。

## 六类股票

| 类型 | 特征 | 策略 | 代表 |
| --- | --- | --- | --- |
| **缓慢增长型** | 增速2-4%，成熟大企业 | 高股息策略 | 电力、公用事业 |
| **稳定增长型** | 增速10-12%，大市值 | 低位买入+波段操作 | 可口可乐、宝洁 |
| **快速增长型** | 增速20-25%，规模小 | 核心持仓 | 早期星巴克 |
| **周期型** | 与宏观经济波动同步 | 逆周期买入 | 钢铁、化工、汽车 |
| **困境反转型** | 处于困境但有望反转 | 高赔率博弈 | 克莱斯勒 |
| **隐蔽资产型** | 资产价值未被股价反映 | 深度价值挖掘 | 土地/矿产公司 |

## 核心原则

### 1. PEG指标
- PEG = PE / 盈利增长率
- PEG < 1 意味着被低估
- PEG = 0.5 是林奇最理想的买入点

### 2. 选股经验法则

| 法则 | 说明 |
| --- | --- |
| 从生活中发现 | 投资你了解的公司 |
| 避开热门股 | 热门行业的热门股最容易被高估 |
| 公司名枯燥更好 | 好名字不一定是好股票 |
| 关注机构低配 | 被华尔街忽视的股票更有机会 |
| 现金头寸 | 高现金比例提供安全垫 |

## 六类股票的买卖时机

| 类型 | 买入时机 | 卖出时机 |
| --- | --- | --- |
| 缓慢增长 | 大跌后 | 大涨30-50% |
| 稳定增长 | PE处于历史低位 | PE创新高 |
| 快速增长 | 估值合理的成长期 | 增速放缓信号 |
| 周期型 | 市盈率最高时（利润谷底） | 库存堆积 |
| 困境反转 | 困境已充分反映 | 反转成功后 |
| 隐蔽资产 | 资产打大折 | 收购要约出现 |
