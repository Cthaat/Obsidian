# 《穷查理宝典：查理·芒格的智慧箴言录》摘要

> [!note] 查理·芒格
> 巴菲特的长期合伙人、伯克希尔的副董事长。他对巴菲特最大的影响，是把投资理念从"捡便宜的烟蒂股"升级为"**以合理价格买伟大公司**"。本书核心可归纳为四块：多元思维模型、反向思维、完备性分析、人生哲学。

## 一、多元思维模型

芒格认为现实问题很少正好落在某个学科边界内，必须**跨学科**借用模型，组成一张"格栅"。

| 学科 | 核心模型 |
|---|---|
| 数学 | 复利、排列组合、概率 |
| 物理学 | 临界点、杠杆、倾覆力矩 |
| 化学 | 自我催化（正反馈） |
| 生物学 | 进化论、适者生存 |
| 工程学 | 冗余备份、断裂点 |
| 心理学 | 认知误判倾向 |

> [!important] Lollapalooza 效应
> 当几种力量同时朝一个方向作用，效果远超简单叠加。无论是超级牛股还是惊天暴雷，背后往往是多因素共振。详见 [[芒格6大思维模型]]。

## 二、反向思维（Invert, always invert）

- 核心方法："要是知道我会死在哪里就好了，那我将永远不去那个地方。"
- **达尔文式自我否定**：主动寻找、认真对待反对自己的证据，避免确认偏误（[[投资心理偏误]]）。
- 经典反向应用："如何确保人生痛苦"——列出让人失败/痛苦的做法，然后逐一避开。

## 三、完备性分析

### 投资决策清单（十大模块）
风险 → 独立 → 准备 → 谦逊 → 严格分析 → 资本配置 → 耐心 → 决心 → 适应变化 → 专注。

把它当成投资前的 checklist，逐项过一遍，避免漏掉关键风险。

### 双轨分析
- **理性轨道**：用多元思维模型做客观分析；
- **潜意识轨道**：识别自己正在被哪些**心理误判倾向**影响（嫉妒、过度乐观、社会认同、激励偏差……）。

> [!tip] 芒格的"人类误判心理学"
> 芒格列举了约二十多种心理倾向（如激励导致的偏差、避免不一致倾向、社会认同、剥夺超级反应等）。它们与 [[投资心理偏误]]、[[行为金融学基础]] 高度呼应——投资亏损往往不是因为不懂，而是被这些倾向劫持。

## 四、人生哲学

| 原则 | 内容 |
|---|---|
| 终身学习 | "没有不每天阅读的聪明人" |
| 避开陷阱 | 反向思维，先求不败 |
| 能力圈 | 不做能力圈之外的事，知道边界 |
| 诚实自律 | "保持诚实，就不必记住谎言" |
| 极度耐心 | "一生只打 20 个孔"——少做、做对、重仓看得懂的 |
| 与值得的人同行 | 别为你不尊敬的人工作 |

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 思维模型是空谈 | 是用来做决策和排雷的实用工具 |
| 多读书=会投资 | 还要会跨学科组合、对抗自己的心理偏误 |
| 反向思维消极 | "先求不败"恰恰是进攻的前提 |
| 集中重仓人人可学 | 建立在极深研究与耐心之上 |

## 课程化学习补充

> [!important] 学习定位
> 经典投资思想的价值在于建立决策原则：能力圈、安全边际、长期复利、反身性和风险控制，而不是照搬大师持仓。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 企业是否在能力圈内
- 安全边际来自估值还是质量
- 持有逻辑是否可被证伪
- 仓位是否匹配不确定性

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 把名人语录当交易信号
- 长期主义掩盖错误
- 低估值陷阱
- 忽视组合层面的回撤

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
