# 《穷查理宝典：查理芒格的智慧箴言录》摘要

> [!note] 查理芒格
> 巴菲特的长期合伙人。微软创始人比尔盖茨称赞："查理芒格是他遇到过的最渊博的思想家。"

## 一、多元思维模型

芒格认为现实问题很少正好落在某一个学科的界限内，需要跨学科知识。

### 核心学科与模型

| 学科 | 核心模型 |
| --- | --- |
| 数学 | 复利原理、排列组合 |
| 物理学 | 临界点、倾覆力矩 |
| 化学 | 自我催化模型 |
| 生物学 | 现代达尔文模型 |
| 工程学 | 冗余备份模型 |
| 心理学 | 认知误判模型 |

> **Lollapalooza 效应**：当几种力量共同作用于同一方向时，效果远超简单叠加。

## 二、反向思维

- 核心方法："要是知道我会死在哪里就好了，那我将永远不去那个地方。"
- 达尔文榜样：发表进化论前反复自我否定
- 应用：确保人生痛苦的四个方法（反向建议）

## 三、完备性分析

### 投资清单（十大模块）
风险 → 独立 → 准备 → 谦虚 → 严格分析 → 配置 → 耐心 → 决心 → 改变 → 专注

### 双轨分析
- 理性轨道：应用多元思维模型，客观理性分析
- 潜意识轨道：识别25种常见的心理误判倾向
