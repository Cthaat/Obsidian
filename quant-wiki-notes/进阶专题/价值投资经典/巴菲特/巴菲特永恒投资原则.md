# 巴菲特永恒投资原则

> [!note] 20条铁律
> 以下是巴菲特历经数十年市场考验的20条核心投资原则，每条都经过了时间的检验。

## 投资哲学篇

1. **不要亏损** — 第一条规则：不要赔钱。第二条规则：不要忘记第一条。
2. **安全边际** — 永远以远低于内在价值的价格购买
3. **市场先生** — 市场是你的仆人，不是你的向导
4. **能力圈** — 只投资你真正理解的业务
5. **独立思考** — 别人贪婪时恐惧，别人恐惧时贪婪

## 选股原则篇

| 原则 | 说明 |
| --- | --- |
| 护城河 | 只投资具有持久竞争优势的企业 |
| 优秀管理 | 管理层必须诚实能干 |
| 确定性 | 宁可要模糊的正确，不要精确的错误 |
| 价格合理 | 以合理的价格买入伟大的公司 |
| 现金流为王 | 关注自由现金流而非会计利润 |

## 持有原则篇

| 原则 | 说明 |
| --- | --- |
| 长期持有 | 最喜欢的持有期是永远 |
| 集中投资 | 把鸡蛋放在一个篮子里——然后看好它 |
| 不要频繁交易 | 摩擦成本是复利最大的敌人 |
| 忽略噪音 | 不要被短期市场波动吓倒 |
| 反向思维 | 在别人恐惧时买入 |

## 风险控制篇

| 原则 | 说明 |
| --- | --- |
| 不借钱投资 | 杠杆是唯一能让聪明人破产的东西 |
| 储备现金 | 永远保持充足的流动性 |
| 避免永久损失 | 波动不是风险，永久性资本损失才是 |
| 不懂不做 | 错过总比犯错好 |
| 持续学习 | 每天阅读500页——知识像复利一样积累 |
