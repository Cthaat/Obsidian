# 巴菲特价值投资思想的核心原则

> [!note] 六大核心原则
> 安全边际、长期投资、企业内在价值、优秀管理团队、经济护城河、简单透明的业务模式。

## 一、安全边际

> 以40美分的价格买入价值1美元的资产。

- 核心：购买价格远低于内在价值的资产
- 作用：为投资提供"安全垫"，即使判断有偏差也不至于亏损
- 格雷厄姆的"烟蒂股"策略即是安全边际的极端应用

## 二、内在价值

- 定义：企业未来现金流折现的总和
- 与市场价格的区别：市场先生情绪波动 → 价格偏离价值 → 投资机会
- 关键：估算内在价值而非精确计算 → 大致正确胜过精确错误

## 三、经济护城河

| 护城河类型 | 示例 |
| --- | --- |
| 品牌壁垒 | 可口可乐 |
| 转换成本 | 银行、软件 |
| 网络效应 | 微信、Visa |
| 成本优势 | 沃尔玛 |
| 规模效应 | 台积电 |

## 四、优秀管理团队

- 诚实正直（巴菲特最看重）
- 能力圈内的专业能力
- 股东利益导向（而非追求规模）

## 五、长期投资

- "我们最喜欢的持有期是永远"
- 复利的力量需要时间发酵
- 减少交易成本（摩擦成本）和税务负担

## 六、简单透明的业务

> 只投资自己能理解的企业。

- 回避复杂金融工程和看不懂的商业模式
- 典型案例：长期回避科技股，直到完全理解苹果的消费属性
