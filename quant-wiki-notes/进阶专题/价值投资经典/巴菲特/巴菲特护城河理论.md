# 巴菲特护城河理论

> [!note] 核心概念
> "护城河"是巴菲特对持久竞争优势的形象比喻。拥有宽广护城河的企业能够持续创造超额回报，抵御竞争侵蚀。

## 五类护城河

| 护城河类型 | 定义 | 典型案例 |
| --- | --- | --- |
| **无形资产** | 品牌、专利、监管许可 | 可口可乐品牌、辉瑞专利 |
| **转换成本** | 客户更换供应商的成本高昂 | 银行账户、企业软件 |
| **网络效应** | 越多用户越有价值 | 微信、Visa、阿里 |
| **成本优势** | 持续低于竞争对手的成本 | 沃尔玛、UPS |
| **规模效应** | 规模带来的天然壁垒 | 台积电、亚马逊 |

## 护城河分析框架

### 四问法

1. **企业是否有护城河？** — 高利润率+高资本回报率的持续性
2. **护城河的来源是什么？** — 属于上述五类的哪一种？
3. **护城河的宽度有多宽？** — 能维持多少年的竞争优势？
4. **护城河在加强还是削弱？** — 竞争格局的演变趋势

## 护城河的量化指标

| 指标 | 强护城河特征 |
| --- | --- |
| ROIC（投入资本回报率） | 持续 > 15% |
| 毛利率 | 稳定且高于行业平均 |
| 市场份额 | 行业前三且份额稳定 |
| 定价权 | 能够转嫁成本上升给客户 |

## 护城河在减弱的企业

- 技术变迁威胁（如传统零售 vs 电商）
- 监管变化削弱壁垒
- 管理层战略失误导致品牌受损
- 消费者习惯变迁
