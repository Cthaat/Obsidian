# 达利欧：大周期秩序

> [!note] 核心观点
> 达利欧认为经济的运行遵循某种永恒的"秩序"，由三大力量驱动：长期债务周期、短期债务周期、生产率增长。理解这三者的互动，是看懂宏观经济的关键。

## 一、三大驱动力

| 力量 | 周期长度 | 特征 |
| --- | --- | --- |
| 生产率增长 | 长期（数十年） | 缓慢而稳定，决定经济长期趋势 |
| 长期债务周期 | 50-75年 | 债务积累→去杠杆的超级周期 |
| 短期债务周期 | 5-8年 | 商业周期，央行政策调节 |

## 二、长期债务周期

### 周期阶段

| 阶段 | 特征 |
| --- | --- |
| 早期 | 债务水平低，借款人信誉良好 |
| 中期 | 债务加速积累，泡沫形成 |
| 顶部 | 债务不可持续，泡沫破裂 |
| 萧条期 | 去杠杆，资产价格暴跌 |
| 再平衡 | 债务重组，央行干预，回归正常 |

### 去杠杆的四种方式
1. 债务削减/违约
2. 紧缩支出（通缩性）
3. 财富转移（税收/再分配）
4. 印钞（通胀性）

> 成功的去杠杆需要四种方式的美妙平衡。

## 三、对投资者的启示

- 关注全球债务水平变化
- 不同周期阶段配置不同资产
- 理解央行政策的底层逻辑
- 历史会重复但不会完全一样
