# 桥水基金创始人Ray Dalio：全天候投资组合

> [!note] 核心理念
> 全天候策略追求在任何经济环境下都能获得稳定收益，核心是通过**风险平价**而非资本分配来实现真正的分散投资。

## 一、策略起源

- 达利欧受困于：如何构建一个无论发生什么都不至于崩盘的投资组合？
- 目标：穿越繁荣、衰退、通胀、通缩四种经济环境
- 核心思想：**风险平价**（Risk Parity）而非资金等分

## 二、四种经济环境

| 环境 | 表现好的资产 |
| --- | --- |
| 增长超预期 + 通胀超预期 | 商品、黄金、新兴市场股票 |
| 增长超预期 + 通胀低于预期 | 股票、公司债 |
| 增长低于预期 + 通胀超预期 | 通胀挂钩债券、商品、黄金 |
| 增长低于预期 + 通胀低于预期 | 国债、投资级债券 |

## 三、经典全天候配置

| 资产类别 | 配置比例 |
| --- | --- |
| 美国长期国债 | 40% |
| 美国中期国债 | 15% |
| 股票 | 30% |
| 黄金 | 7.5% |
| 大宗商品 | 7.5% |

> 债券比例高达 55%，因为用风险衡量时股票的波动远大于债券。

## 四、全天候策略特点

| 特征 | 说明 |
| --- | --- |
| 超越牛熊 | 不试图择时或预测市场方向 |
| 波动较低 | 最大回撤显著低于纯股票组合 |
| 复数收益来源 | 不同环境有不同资产支撑 |
| 被动管理 | 季度或年度再平衡即可 |
| 长期复利 | 虽不追求最高年化，但稳定性极佳 |
