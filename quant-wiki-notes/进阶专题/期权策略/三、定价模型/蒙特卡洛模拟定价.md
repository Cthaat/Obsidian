# 蒙特卡洛模拟定价

> [!note] 核心要点
> 蒙特卡洛模拟通过随机抽样模拟标的资产价格路径，计算期权到期收益的期望值并折现，得到期权理论价格。特别适用于欧式期权和复杂路径依赖期权。

---

## 一、基本原理

蒙特卡洛模拟定价基于**风险中性定价原则**：
1. 标的资产的预期收益率 = 无风险利率
2. 对未来的现金流进行无风险利率折现
3. 仅是概率空间的变换，不需要对投资者风险喜好进行假设

### 价格路径模拟

在风险中性世界中，标的资产价格服从：

$$S_T = S_0 \exp\left[\left(r - \frac{\sigma^2}{2}\right)T + \sigma\sqrt{T} \cdot Z\right]$$

其中 $Z \sim N(0,1)$ 为标准正态随机变量。

---

## 二、定价步骤

### Step 1：生成随机路径

从标准正态分布中抽取 $M$ 个随机数 $Z_i$，计算对应的到期价格：

$$S_T^{(i)} = S_0 \exp\left[\left(r - \frac{\sigma^2}{2}\right)T + \sigma\sqrt{T} \cdot Z_i\right]$$

### Step 2：计算到期收益

$$\text{Payoff}^{(i)} = \max(S_T^{(i)} - K, 0) \quad \text{(Call)}$$

### Step 3：折现求平均

$$C = e^{-rT} \cdot \frac{1}{M}\sum_{i=1}^{M} \text{Payoff}^{(i)}$$

---

## 三、Python 实现

```python
import numpy as np

def monte_carlo_option(S, K, T, r, sigma, M=100000, option_type='call'):
    """
    蒙特卡洛模拟期权定价

    Parameters
    ----------
    S : 标的价格
    K : 行权价
    T : 到期时间（年）
    r : 无风险利率
    sigma : 波动率
    M : 模拟路径数量
    option_type : 'call' 或 'put'
    """
    np.random.seed(42)
    Z = np.random.standard_normal(M)
    ST = S * np.exp((r - 0.5*sigma**2)*T + sigma*np.sqrt(T)*Z)

    if option_type == 'call':
        payoff = np.maximum(ST - K, 0)
    else:
        payoff = np.maximum(K - ST, 0)

    price = np.exp(-r*T) * np.mean(payoff)
    std_error = np.exp(-r*T) * np.std(payoff) / np.sqrt(M)

    return price, std_error

# 示例
price, se = monte_carlo_option(S=100, K=105, T=0.5, r=0.05, sigma=0.30)
print(f"期权价格: {price:.4f}")
print(f"标准误差: {se:.4f}")
```

## 四、方差缩减技术

### 4.1 对偶变量法（Antithetic Variates）

同时生成 $Z$ 和 $-Z$，利用负相关性减少方差：

```python
def mc_antithetic(S, K, T, r, sigma, M=50000):
    Z = np.random.standard_normal(M)
    # 正向路径
    ST1 = S * np.exp((r - 0.5*sigma**2)*T + sigma*np.sqrt(T)*Z)
    # 反向路径
    ST2 = S * np.exp((r - 0.5*sigma**2)*T + sigma*np.sqrt(T)*(-Z))

    payoff1 = np.maximum(ST1 - K, 0)
    payoff2 = np.maximum(ST2 - K, 0)
    payoff = (payoff1 + payoff2) / 2

    return np.exp(-r*T) * np.mean(payoff)
```

### 4.2 控制变量法（Control Variates）

使用已知解析解的期权（如标准 BS 期权）作为控制变量，减少估计误差。

## 五、适用场景

| 场景 | 适用性 |
|------|--------|
| 欧式期权 | ✅ 适用，但 BS 解析解更快 |
| 美式期权 | ⚠️ 需要特殊处理（Longstaff-Schwartz 方法） |
| 路径依赖期权 | ✅ 最佳选择（亚式、障碍期权等） |
| 多标的期权 | ✅ 适用（篮子期权、彩虹期权） |
| 奇异期权 | ✅ 灵活性高 |

> [!warning] 局限性
> - 计算速度较慢（相比解析解和二叉树）
> - 需要大量模拟路径才能达到足够精度
> - 美式期权定价需要特殊处理
> - 每次运行结果略有不同（随机性）

---

## 六、三种定价方法精度比较

对于欧式期权，三种方法的结果应趋于一致：
- **BS 解析解**：精确值（最快）
- **二叉树**：随步数增加收敛于 BS 值
- **蒙特卡洛**：随模拟路径增加收敛于 BS 值（有标准误差）

## 实战掌握清单

> [!tip] 交易者视角
> 蒙特卡洛模拟定价 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。期权交易的本质是方向、波动率、时间价值和尾部风险的组合表达。

### 关键判断

- 先画收益结构，确认最大盈利、最大亏损和盈亏平衡点。
- 检查Delta、Gamma、Theta、Vega和保证金。
- 比较隐含波动率、历史波动率、期限结构和偏度。

### 落地动作

1. 每个组合都要做压力测试：大涨、大跌、横盘、波动率跳升和流动性收缩。
2. 把开仓、调仓、止损、到期处理和行权风险写进计划。
3. 避免用卖权利金掩盖尾部亏损。

### 失效边界

- 低估Gamma和跳空风险。
- 忽略保证金追加。
- 把高胜率卖方策略误认为低风险。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
