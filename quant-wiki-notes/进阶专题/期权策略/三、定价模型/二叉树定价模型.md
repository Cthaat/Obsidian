# 二叉树定价模型

> [!note] 核心要点
> 二叉树模型由 Cox、Ross 和 Rubinstein 于 1979 年提出，通过将连续价格变化离散化为"上涨/下跌"路径，构建树状结构来模拟价格演变并估算期权价值。可处理美式期权等复杂情况。

---

## 一、核心原理

二叉树模型将连续时间的随机过程离散化为有限数量的时间步长和状态。每个时间节点只有两个可能的价格变化：

- **向上变动**：资产价格以比例 $u$ ($u > 1$) 上升
- **向下变动**：资产价格以比例 $d$ ($0 < d < 1$) 下降

利用风险中性定价理论，结合无风险套利原则，逐步回溯计算出期权的理论价值。

---

## 二、关键参数

### 价格运动因子

$$u = e^{\sigma\sqrt{\Delta t}}, \quad d = e^{-\sigma\sqrt{\Delta t}} = \frac{1}{u}$$

### 风险中性概率

$$p = \frac{e^{(r-y)\Delta t} - d}{u - d}, \quad q = 1 - p$$

其中 $r$ 为无风险利率，$y$ 为股息收益率，$\Delta t$ 为时间步长。

> [!important] 风险中性概率
> $p$ 并非真实世界的上涨概率，而是风险中性世界中为了使资产预期收益等于无风险利率而设定的等价概率，其核心是**消除套利机会**。

---

## 三、模型构建步骤

### Step 1：构建二叉价格树

将期权有效期 $T$ 划分为 $n$ 个等长时间步（$\Delta t = T/n$）。第 $t$ 步第 $j$ 个节点的股票价格为：

$$S_{t,j} = S_0 \cdot u^{t-j} \cdot d^j$$

### Step 2：计算终端节点的期权价值

$$\text{看涨期权：} C = \max(S - K, 0)$$
$$\text{看跌期权：} P = \max(K - S, 0)$$

### Step 3：反向推导各节点的期权价值

**欧式期权**：

$$C = e^{-(r-y)\Delta t} \cdot (p \cdot C_u + q \cdot C_d)$$

**美式期权**（需考虑提前行权）：

$$C = \max(S - K, \; e^{-(r-y)\Delta t} \cdot (p \cdot C_u + q \cdot C_d))$$

$$P = \max(K - S, \; e^{-(r-y)\Delta t} \cdot (p \cdot P_u + q \cdot P_d))$$

---

## 四、美式期权 vs 欧式期权

| 维度 | 欧式期权 | 美式期权 |
|------|---------|---------|
| 行权时间 | 仅到期日 | 到期日前任意时间 |
| 定价方式 | 仅考虑继续持有价值 | 取 max(立即行权, 继续持有) |
| 价格关系 | $C_{\text{欧式}} \leq C_{\text{美式}}$ | 美式 ≥ 欧式 |
| 提前行权 | 不可能 | 可能（特别是 Put） |

---

## 五、与 Black-Scholes 的关系

二叉树模型可以被视为 Black-Scholes 模型所基于的连续过程的**离散时间近似**。随着时间步数增加，二叉树计算出的欧式期权价值会**收敛于 BSM 公式**给出的价值。

从数值分析角度看，二叉树方法可以被视为 BSM 偏微分方程的**显式有限差分法**的一个特例。

### 手动实现

```python
import numpy as np

def binomial_tree(S, K, T, r, sigma, N, option_type='call', american=False):
    """二叉树期权定价"""
    dt = T / N
    u = np.exp(sigma * np.sqrt(dt))
    d = 1 / u
    p = (np.exp(r*dt) - d) / (u - d)

    # 构建价格树
    stock = np.zeros((N+1, N+1))
    for i in range(N+1):
        for j in range(i+1):
            stock[i,j] = S * u**(i-j) * d**j

    # 计算到期收益
    option = np.zeros((N+1, N+1))
    for j in range(N+1):
        if option_type == 'call':
            option[N,j] = max(stock[N,j] - K, 0)
        else:
            option[N,j] = max(K - stock[N,j], 0)

    # 反向推导
    for i in range(N-1, -1, -1):
        for j in range(i+1):
            hold = np.exp(-r*dt) * (p*option[i+1,j] + (1-p)*option[i+1,j+1])
            if american:
                if option_type == 'call':
                    exercise = max(stock[i,j] - K, 0)
                else:
                    exercise = max(K - stock[i,j], 0)
                option[i,j] = max(hold, exercise)
            else:
                option[i,j] = hold

    return option[0,0]
```

## 七、优缺点

| 优点 | 缺点 |
|------|------|
| 可处理美式期权 | 步数多时计算密集 |
| 直观的树状结构 | 需要选择合适的步数 |
| 可处理路径依赖期权 | 对奇异期权可能不够高效 |
| 收敛于 BSM 结果 | 参数假设与 BSM 相同 |

## 实战掌握清单

> [!tip] 交易者视角
> 二叉树定价模型 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。期权交易的本质是方向、波动率、时间价值和尾部风险的组合表达。

### 关键判断

- 先画收益结构，确认最大盈利、最大亏损和盈亏平衡点。
- 检查Delta、Gamma、Theta、Vega和保证金。
- 比较隐含波动率、历史波动率、期限结构和偏度。

### 落地动作

1. 每个组合都要做压力测试：大涨、大跌、横盘、波动率跳升和流动性收缩。
2. 把开仓、调仓、止损、到期处理和行权风险写进计划。
3. 避免用卖权利金掩盖尾部亏损。

### 失效边界

- 低估Gamma和跳空风险。
- 忽略保证金追加。
- 把高胜率卖方策略误认为低风险。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
