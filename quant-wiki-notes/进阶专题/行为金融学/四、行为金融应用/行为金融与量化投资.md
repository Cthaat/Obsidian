# 行为金融与量化投资

> [!note] 核心观点
> 行为金融学与量化投资的结合是一个新兴交叉领域。量化策略用代码代替人脑决策，试图摆脱情绪干扰，但程序的设计依然离不开人的参与——行为偏差可能"悄悄"嵌入模型之中。理解行为偏差既是策略设计的机会，也是风险管理的必需。

## 行为金融学在量化投资中的应用

### 1. 因子投资与行为因子

#### 传统因子

| 因子 | 定义 | 传统解释 |
|------|------|----------|
| 价值（Value） | 低P/E、低P/B | 风险补偿 |
| 动量（Momentum） | 过去收益趋势 | 信息扩散 |
| 质量（Quality） | 高ROE、低杠杆 | 基本面优势 |
| 规模（Size） | 小市值 | 流动性溢价 |
| 低波动（Low Vol） | 低波动率 | 波动率之谜 |

#### 行为因子

| 因子 | 数据来源 | 行为偏差基础 |
|------|----------|-------------|
| 情绪因子 | NLP文本分析 | 情绪偏差 |
| 注意力因子 | 媒体报道、搜索热度 | 有限注意力 |
| Salience因子 | 价格偏离度 | 显著性偏差 |
| 宏观认知因子 | 调查数据 | 宏观预期偏差 |

### 2. 动量与反转策略的行为基础

```
新信息 → 投资者反应不足 → 价格趋势延续（动量）
    → 过度关注后 → 过度反应 → 价格反转
```

- 短期（数日-数周）：高Salience股票动量更强
- 中期（1-3个月）：高Salience股票出现显著反转
- 长期（3-5年）：过度反应导致长期反转

### 3. 价值因子的行为解释

- 投资者注意力被热门股吸引
- 对"冷门股"缺乏兴趣 → 被低估
- 形成长期价值溢价
- 行为型价值因子：关注度指标 + 传统估值指标

## 程序化交易中的行为偏差

### 过滤器功能

- 算法严格执行预设策略
- 不受市场短期波动或情绪扰动
- 有效规避损失厌恶、过度自信等偏差

### 隐性放大器

| 风险来源 | 说明 |
|----------|------|
| 数据挖掘偏差 | 对历史规律过度拟合 |
| 事后归因 | 模型设计受结果影响 |
| 策略趋同 | 多机构类似策略→集体羊群 |
| 流动性枯竭 | 订单集中涌入→价格剧烈波动 |

### 情绪信号嵌入

- 分析新闻情绪指数
- 监测社交媒体评论情绪
- 极端负面时自动降低仓位/减少交易频率
- 将非理性情绪转化为可量化信号

## 散户行为与市场微观结构

### 注意力偏差的微观表现

1. **订单流异常聚集**：平日清淡的股票出现海量小额买单
2. **价差扩大**：情绪推动下买卖价差异常波动
3. **价格冲顶与反转**：关注度减弱后价格快速回调

### 高频交易中的行为捕捉

- 捕捉社交媒体讨论量、搜索热度
- 识别异常交易信号
- 在情绪推动初期快速跟随订单流
- 在非理性高位利用散户订单滞后性套利

## 风险管理中的行为金融

### 投资者情绪与风险

熊建龙研究发现：
- 个体投资受前景理论主导
- 羊群效应加强下形成一致预期
- 情绪与股市风险形成螺旋加强
- 不同市场间风险蔓延

### Sent-EGARCH-M 模型

- 含投资者情绪变量的波动率模型
- 结合极值理论估计VaR
- 更准确捕捉行为偏差对风险的影响

## 未来发展方向

1. **跨市场行为分析**：不同市场间的行为偏差传导
2. **宏观行为因子**：将宏观认知偏差纳入因子模型
3. **AI与行为金融**：深度学习在行为数据挖掘中的应用
4. **ESG与行为金融**：环境、社会因素对投资者行为的影响

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `行为金融与量化投资` 时，不要只把它当成孤立知识点。把心理偏差转成流程约束，而不是停留在自我提醒。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 损失厌恶、确认偏差和过度自信
- 交易日志和事前计划
- 市场情绪和拥挤交易
- 自动化纪律和风控约束

### 进阶训练

1. 在交易日志中记录入场理由和情绪状态
2. 为每种偏差设计一个流程拦截点
3. 复盘亏损是否来自策略失效还是执行偏差

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
