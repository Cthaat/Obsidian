# 后悔理论

> [!note] 核心定义
> 后悔理论（Regret Theory）由 Loomes、Sugden 和 Bell 分别提出，将后悔和欣喜因素纳入传统效用函数。核心思想是投资者在投资过程中常出现后悔的心理状态，并据此做出决策。

## 典型后悔场景

| 市场环境 | 后悔情境 |
|----------|----------|
| 大牛市 | 没有及时介入看好的股票 |
| 大牛市 | 过早卖出获利的股票 |
| 熊市 | 没能及时止损出局 |
| 熊市 | 获小利未兑现，又被套牢 |
| 平衡市 | 持有的不涨，别人推荐的涨了 |
| 任意 | 卖出不涨的→买入推荐的→原来持有的涨了，推荐的跌了 |

## 核心机制

### 害怕后悔与追求自豪

Hersh Shefrin 和 Meir Statman 发现投资者同时受两种动机驱动：

1. **害怕后悔**：避免做出错误决策带来的痛苦
2. **追求自豪**：渴望做出正确决策带来的满足感

### 处置效应的产生

这两种动机导致：
- 持有**获利**股票的时间太**短**（想体验自豪感）
- 持有**亏损**股票的时间太**长**（避免确认失败的后悔）

> 当持有A（获利20%）和B（亏损20%）时，多数投资者选择卖出A而非B。因为卖出B确认了失败决策的后悔，而卖出A产生了正确投资的自豪感。

### 从众行为的驱动

- 买大家都看好的股票 → 即使亏损也"大家都错了"
- 自作主张买冷门股下跌 → 无法合理解释买入理由
- 基金经理偏好名气大的股票 → 下跌时被解雇可能性较小

## 理论修正

### 刘晓东和刘力臻的修正

将**追求欣喜**和**避免后悔**作为二次评估的决策动机引入后悔理论模型：
- 成功解释了Kahneman和Tversky 1979年提出的悖论问题
- 增加"情绪动机影响投资决策"作为行为金融学基础假设

### 现实期望效用理论

陆剑清提出：面对收益时遵循"收益最大化"，面对损失时遵循"后悔最小化"。

## 对投资行为的影响

1. **优柔寡断**：害怕后悔导致决策拖延
2. **从众心理**：跟随大众以分散决策责任
3. **确认偏差**：寻找支持自己持股的信息
4. **过早获利了结**：急于体验自豪感
5. **过久持有亏损**：不愿面对后悔情绪

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
