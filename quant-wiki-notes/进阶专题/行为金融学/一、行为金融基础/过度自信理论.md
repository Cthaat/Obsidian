# 过度自信理论

> [!note] 核心定义
> 过度自信（Overconfidence）是指投资者对自己的能力和知识的高估、对风险和不确定性的低估，以及对自己的决策和判断结果的过分自信。它是行为金融学四大理论之一。

## 表现形式

- 高估自己的判断力——当人们称对某事有90%把握时，成功概率仅约70%
- 低估风险和不确定性
- 对决策结果过分自信
- 将盈利归因于自己，亏损归因于外部因素（[[确认偏差|自我归因偏差]]）

## 心理根源

| 来源 | 说明 |
|------|------|
| 自我偏见 | 天生倾向于高估自身能力 |
| 信息收集偏见 | 更容易记住与自己观点一致的信息 |
| 确认偏见 | 忽略与自己观点不一致的信息 |
| 直觉依赖 | 过于依赖主观判断，忽略客观数据 |

## 真过度自信 vs 伪过度自信

朱玥琳研究指出：

- **真过度自信**：对增强自信心的信息给予重视，对损害自信心的信息给予回避
- **伪过度自信**：对正确的私人信息给予重视

## 对市场的影响

### 对股票价格的影响

过度自信投资者比例与股票均衡价格的关系：
- 过度自信投资者比例越大 → 股票均衡价格越高 → 股票收益越低
- 过度自信程度升高 → 市场风险溢价下降

### 市场循环机制

```
过度自信 → 股价偏离 → 需要新高强度信息修正 → 继续过度自信 → 股价循环往复
```

理性投资者可从中进行套利。

### 过度交易效应

- 过度自信导致投资者频繁交易（Excessive Trading）
- 对市场噪声反应过激
- 忽视潜在风险和损失
- 推高整体市场波动性

## 在资产定价中的体现

Daniel, Hirshleifer 和 Subrahmanyam（1998）将过度自信引入资产定价模型：

1. **第1阶段**：投资者对信息过度自信 → 价格偏离基本面 → 短期快速上涨
2. **第2阶段**：市场情绪回落 → 价格修正
3. **自我归因偏差**：早期成功强化信心 → 价格进一步偏离 → 更大幅度修正

## 实证证据

- 中国A股市场投资者过度自信现象显著
- 过度自信与羊群行为共同影响股价崩盘风险
- 程序化交易中的过度自信：策略设计者对模型过度拟合（Data Mining Bias）

## 克服方法

1. 建立客观的投资评估体系
2. 记录投资决策和结果，定期复盘
3. 主动寻找反面证据
4. 设定止损规则，减少情绪干扰
5. 使用量化工具辅助决策
