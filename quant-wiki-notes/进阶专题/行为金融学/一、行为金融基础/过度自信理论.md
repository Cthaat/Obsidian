# 过度自信理论

> [!note] 核心定义
> 过度自信（Overconfidence）是指投资者对自己的能力和知识的高估、对风险和不确定性的低估，以及对自己的决策和判断结果的过分自信。它是行为金融学四大理论之一。

## 表现形式

- 高估自己的判断力——当人们称对某事有90%把握时，成功概率仅约70%
- 低估风险和不确定性
- 对决策结果过分自信
- 将盈利归因于自己，亏损归因于外部因素（[[确认偏差|自我归因偏差]]）

## 心理根源

| 来源 | 说明 |
|------|------|
| 自我偏见 | 天生倾向于高估自身能力 |
| 信息收集偏见 | 更容易记住与自己观点一致的信息 |
| 确认偏见 | 忽略与自己观点不一致的信息 |
| 直觉依赖 | 过于依赖主观判断，忽略客观数据 |

## 真过度自信 vs 伪过度自信

朱玥琳研究指出：

- **真过度自信**：对增强自信心的信息给予重视，对损害自信心的信息给予回避
- **伪过度自信**：对正确的私人信息给予重视

## 对市场的影响

### 对股票价格的影响

过度自信投资者比例与股票均衡价格的关系：
- 过度自信投资者比例越大 → 股票均衡价格越高 → 股票收益越低
- 过度自信程度升高 → 市场风险溢价下降

### 市场循环机制

```
过度自信 → 股价偏离 → 需要新高强度信息修正 → 继续过度自信 → 股价循环往复
```

理性投资者可从中进行套利。

### 过度交易效应

- 过度自信导致投资者频繁交易（Excessive Trading）
- 对市场噪声反应过激
- 忽视潜在风险和损失
- 推高整体市场波动性

## 在资产定价中的体现

Daniel, Hirshleifer 和 Subrahmanyam（1998）将过度自信引入资产定价模型：

1. **第1阶段**：投资者对信息过度自信 → 价格偏离基本面 → 短期快速上涨
2. **第2阶段**：市场情绪回落 → 价格修正
3. **自我归因偏差**：早期成功强化信心 → 价格进一步偏离 → 更大幅度修正

## 实证证据

- 中国A股市场投资者过度自信现象显著
- 过度自信与羊群行为共同影响股价崩盘风险
- 程序化交易中的过度自信：策略设计者对模型过度拟合（Data Mining Bias）

## 克服方法

1. 建立客观的投资评估体系
2. 记录投资决策和结果，定期复盘
3. 主动寻找反面证据
4. 设定止损规则，减少情绪干扰
5. 使用量化工具辅助决策

## 实战掌握清单

> [!tip] 交易者视角
> 过度自信理论 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。行为金融的价值在于识别自己和市场参与者的系统性偏差，并把偏差转成可执行的纪律。

### 关键判断

- 识别偏差发生在信息收集、判断、下单、持有还是止损环节。
- 区分个人心理错误和群体行为造成的市场机会。
- 用规则约束情绪最强的时刻。

### 落地动作

1. 交易前写下反方观点和失效点。
2. 交易后记录情绪、冲动、犹豫和执行偏差。
3. 把常犯错误转成硬性限制，例如冷静期、仓位上限和复盘后再交易。

### 失效边界

- 用心理学解释一切行情。
- 知道偏差但没有流程约束。
- 把短期盈利当作决策正确的证据。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 行为日志

围绕 过度自信理论 建立交易心理日志：记录交易前的信心、恐惧、冲动、信息来源和是否偏离计划。连续记录比单次反思更有价值。

### 纠偏规则

- 每笔交易前写反方观点。
- 盈利后限制立刻加仓，亏损后限制翻本交易。
- 对止损拖延、过早止盈和冲动追涨单独统计。

### 验收标准

能否减少重复错误，比能否解释心理偏差更重要。

## 最小可执行项目

### 交易错误库

围绕 过度自信理论 建一个交易错误库：每次错误记录触发事件、情绪、原计划、实际动作、损失、偏差类型和下一次约束。

| 错误 | 约束例子 |
|---|---|
| 追涨 | 突破后等待回踩或缩小仓位 |
| 扛亏 | 失效点触发强制减仓 |
| 过早止盈 | 分批退出而不是一次清仓 |
| 翻本交易 | 当日亏损到线停止交易 |

### 验收标准

同类错误出现频率下降，才说明行为训练有效。
