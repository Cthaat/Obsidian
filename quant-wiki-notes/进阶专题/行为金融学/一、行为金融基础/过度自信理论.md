# 过度自信理论

> [!note] 核心定义
> 过度自信（Overconfidence）是指投资者对自己的能力和知识的高估、对风险和不确定性的低估，以及对自己的决策和判断结果的过分自信。它是行为金融学四大理论之一。

## 表现形式

- 高估自己的判断力——当人们称对某事有90%把握时，成功概率仅约70%
- 低估风险和不确定性
- 对决策结果过分自信
- 将盈利归因于自己，亏损归因于外部因素（[[确认偏差|自我归因偏差]]）

## 心理根源

| 来源 | 说明 |
|------|------|
| 自我偏见 | 天生倾向于高估自身能力 |
| 信息收集偏见 | 更容易记住与自己观点一致的信息 |
| 确认偏见 | 忽略与自己观点不一致的信息 |
| 直觉依赖 | 过于依赖主观判断，忽略客观数据 |

## 真过度自信 vs 伪过度自信

朱玥琳研究指出：

- **真过度自信**：对增强自信心的信息给予重视，对损害自信心的信息给予回避
- **伪过度自信**：对正确的私人信息给予重视

## 对市场的影响

### 对股票价格的影响

过度自信投资者比例与股票均衡价格的关系：
- 过度自信投资者比例越大 → 股票均衡价格越高 → 股票收益越低
- 过度自信程度升高 → 市场风险溢价下降

### 市场循环机制

```
过度自信 → 股价偏离 → 需要新高强度信息修正 → 继续过度自信 → 股价循环往复
```

理性投资者可从中进行套利。

### 过度交易效应

- 过度自信导致投资者频繁交易（Excessive Trading）
- 对市场噪声反应过激
- 忽视潜在风险和损失
- 推高整体市场波动性

## 在资产定价中的体现

Daniel, Hirshleifer 和 Subrahmanyam（1998）将过度自信引入资产定价模型：

1. **第1阶段**：投资者对信息过度自信 → 价格偏离基本面 → 短期快速上涨
2. **第2阶段**：市场情绪回落 → 价格修正
3. **自我归因偏差**：早期成功强化信心 → 价格进一步偏离 → 更大幅度修正

## 实证证据

- 中国A股市场投资者过度自信现象显著
- 过度自信与羊群行为共同影响股价崩盘风险
- 程序化交易中的过度自信：策略设计者对模型过度拟合（Data Mining Bias）

## 克服方法

1. 建立客观的投资评估体系
2. 记录投资决策和结果，定期复盘
3. 主动寻找反面证据
4. 设定止损规则，减少情绪干扰
5. 使用量化工具辅助决策

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `过度自信理论` 时，不要只把它当成孤立知识点。把心理偏差转成流程约束，而不是停留在自我提醒。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 损失厌恶、确认偏差和过度自信
- 交易日志和事前计划
- 市场情绪和拥挤交易
- 自动化纪律和风控约束

### 进阶训练

1. 在交易日志中记录入场理由和情绪状态
2. 为每种偏差设计一个流程拦截点
3. 复盘亏损是否来自策略失效还是执行偏差

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
