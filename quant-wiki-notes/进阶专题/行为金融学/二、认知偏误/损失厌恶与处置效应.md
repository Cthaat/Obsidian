# 损失厌恶与处置效应

> [!note] 核心概念
> **损失厌恶**（Loss Aversion）指投资者对损失的敏感度远高于对收益的满足感——亏损1美元的痛苦约是盈利1美元快乐的2倍。**处置效应**（Disposition Effect）是损失厌恶的直接后果：投资者过早卖出盈利股票，过久持有亏损股票。

## 损失厌恶

### 理论基础

- Kahneman 和 Tversky 在[[期望理论|期望理论]]中首次系统描述
- 是人类进化中形成的生存机制——对威胁的敏感性高于机会

### 在投资中的表现

| 状态 | 心理反应 | 行为倾向 |
|------|----------|----------|
| 盈利 | 风险厌恶 | 过早获利了结 |
| 亏损 | 风险偏好 | 持有甚至加仓 |

**典型场景**：
- 股价上涨 → 急于兑现利润，害怕利润化为乌有
- 股价下跌 → 不愿卖出，寻找有利信息增强持股信心
- "不卖就不算亏"——这是最典型的损失厌恶表现

### 对市场的影响

杨仁美研究发现：
- 股票收益上涨时 → 投资者反应不足
- 价格下跌时 → 投资者反应过度
- 损失厌恶心理偏差对股市波动造成显著影响

### 两倍效应

> 人们在从事金融交易中，赋予"避害"因素的考虑权重是"趋利"因素的**两倍**。

## 处置效应

### 定义

由 Shefrin 和 Statman 首次命名，Odean (1998) 通过个人账户数据验证：
- 投资者卖出赢利股票的概率是卖出亏损股票概率的 **1.5 倍**（国外）
- 中国股市中这一比例为 **2 倍**（赵学军、王永宏, 1998-2000）

### 心理机制

```
盈利股票 → 卖出体验自豪感 → 过早卖出
亏损股票 → 卖出确认后悔 → 过久持有
```

### 处置效应的后果

1. **资产价格偏差**：大量未实现收益的股票 → 低估；大量未实现亏损的股票 → 高估
2. **收益率差异**：有大量未实现亏损的股票最终预期回报更高
3. **动量交易机会**：利用处置效应带来的价格偏差获利

## 投资策略启示

### 克服损失厌恶

1. 设定明确的止损规则并严格执行
2. 将投资视为概率游戏，接受单笔亏损
3. 关注投资组合整体收益而非单笔盈亏
4. 建立交易日志，客观记录和评估

### 利用处置效应

- 买入有大量未实现亏损的股票（被过度压低）
- 卖出有大量未实现收益的股票（可能被过度追捧）
- 动量交易策略：基于价格走势差幅获利

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
