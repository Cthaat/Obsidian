# 行为金融学的认知偏差全景

> [!note] 分类体系
> 行为金融学中的偏差可分为两大类：**认知偏差**（Cognitive Bias，可被纠正/mitigate）和**情绪偏差**（Emotional Bias，只能适应/accommodate）。前者源于信息处理错误，后者源于情感驱动。

## 一、认知偏差（Cognitive Bias）

### 1. 信念固执（Belief Perseverance）

| 偏差 | 英文 | 说明 |
|------|------|------|
| 代表性偏差 | Representativeness | 以过去经验直接推断未来 |
| 控制幻觉 | Illusion of Control | 感觉能控制投资过程，实际不能 |
| 保守偏差 | Conservatism | 对现有信息赋予过大权重，新信息权重过小 |
| 后见之明 | Hindsight | 事后诸葛亮，根据结果调整记忆 |
| [[目录|确认偏差]] | Confirmation | 只关注支持自己观点的信息 |

### 2. 信息处理偏差（Information Processing）

| 偏差 | 英文 | 说明 |
|------|------|------|
| 框架依赖 | Framing | 表达方式影响决策判断 |
| [[目录|锚定效应]] | Anchoring | 初始值影响后续定量判断 |
| [[目录|心理账户]] | Mental Accounting | 对不同来源资金区别对待 |
| 易得依赖 | Availability | 依赖容易获得的信息，而非正确信息 |

## 二、情绪偏差（Emotional Bias）

记忆口诀：**LOSSER**

| 偏差 | 英文 | 说明 |
|------|------|------|
| [[目录|损失厌恶]] | Loss Aversion | 面对损失风险偏好，面对收益风险厌恶 |
| [[目录|过度自信]] | Overconfidence | 高估自身判断力，喜欢"对归自己，错归别人" |
| 自我控制 | Self-Control | 缺乏自我控制，无法达到投资目的 |
| 维持现状 | Status Quo | 只愿维持现状，不愿改变（懒惰） |
| 禀赋效应 | Endowment | 对持有时间长的资产产生感情，高估价值 |
| [[目录|后悔厌恶]] | Regret Aversion | 害怕独立决策，体现随大流、羊群效应 |

## 偏差的相互作用

```
锚定效应 → 设定参照点（买入价）
    ↓
损失厌恶 → 围绕参照点评估盈亏
    ↓
确认偏差 → 寻找支持自己判断的信息
    ↓
过度自信 → 高估判断的正确性
    ↓
处置效应 → 过早卖出盈利，过久持有亏损
    ↓
羊群行为 → 跟随大众确认自己的决策
```

## 偏差对市场的系统性影响

| 市场现象 | 主要偏差来源 |
|----------|-------------|
| 价格泡沫 | 过度乐观、羊群行为、代表性偏差 |
| 市场崩盘 | 损失厌恶、恐慌性羊群 |
| 动量效应 | 反应不足、锚定效应 |
| 长期反转 | 过度反应、代表性偏差 |
| 波动率之谜 | 损失厌恶、框架依赖 |
| 交易量过大 | 过度自信、控制幻觉 |

## 克服偏差的系统方法

### 认知偏差的纠正

1. 建立客观的投资评估体系
2. 主动寻找反面证据
3. 使用标准化决策清单
4. 多元化信息来源
5. 定期复盘和校准

### 情绪偏差的适应

1. 设定自动化投资规则（如定投）
2. 明确的止损止盈纪律
3. 减少查看投资组合的频率
4. 建立投资委员会分散个人情绪
5. 使用量化工具辅助决策

## 实战掌握清单

> [!tip] 交易者视角
> 行为金融学的认知偏差全景 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。行为金融的价值在于识别自己和市场参与者的系统性偏差，并把偏差转成可执行的纪律。

### 关键判断

- 识别偏差发生在信息收集、判断、下单、持有还是止损环节。
- 区分个人心理错误和群体行为造成的市场机会。
- 用规则约束情绪最强的时刻。

### 落地动作

1. 交易前写下反方观点和失效点。
2. 交易后记录情绪、冲动、犹豫和执行偏差。
3. 把常犯错误转成硬性限制，例如冷静期、仓位上限和复盘后再交易。

### 失效边界

- 用心理学解释一切行情。
- 知道偏差但没有流程约束。
- 把短期盈利当作决策正确的证据。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
