# 行为金融学的认知偏差全景

> [!note] 分类体系
> 行为金融学中的偏差可分为两大类：**认知偏差**（Cognitive Bias，可被纠正/mitigate）和**情绪偏差**（Emotional Bias，只能适应/accommodate）。前者源于信息处理错误，后者源于情感驱动。

## 一、认知偏差（Cognitive Bias）

### 1. 信念固执（Belief Perseverance）

| 偏差 | 英文 | 说明 |
|------|------|------|
| 代表性偏差 | Representativeness | 以过去经验直接推断未来 |
| 控制幻觉 | Illusion of Control | 感觉能控制投资过程，实际不能 |
| 保守偏差 | Conservatism | 对现有信息赋予过大权重，新信息权重过小 |
| 后见之明 | Hindsight | 事后诸葛亮，根据结果调整记忆 |
| [[目录|确认偏差]] | Confirmation | 只关注支持自己观点的信息 |

### 2. 信息处理偏差（Information Processing）

| 偏差 | 英文 | 说明 |
|------|------|------|
| 框架依赖 | Framing | 表达方式影响决策判断 |
| [[目录|锚定效应]] | Anchoring | 初始值影响后续定量判断 |
| [[目录|心理账户]] | Mental Accounting | 对不同来源资金区别对待 |
| 易得依赖 | Availability | 依赖容易获得的信息，而非正确信息 |

## 二、情绪偏差（Emotional Bias）

记忆口诀：**LOSSER**

| 偏差 | 英文 | 说明 |
|------|------|------|
| [[目录|损失厌恶]] | Loss Aversion | 面对损失风险偏好，面对收益风险厌恶 |
| [[目录|过度自信]] | Overconfidence | 高估自身判断力，喜欢"对归自己，错归别人" |
| 自我控制 | Self-Control | 缺乏自我控制，无法达到投资目的 |
| 维持现状 | Status Quo | 只愿维持现状，不愿改变（懒惰） |
| 禀赋效应 | Endowment | 对持有时间长的资产产生感情，高估价值 |
| [[目录|后悔厌恶]] | Regret Aversion | 害怕独立决策，体现随大流、羊群效应 |

## 偏差的相互作用

```
锚定效应 → 设定参照点（买入价）
    ↓
损失厌恶 → 围绕参照点评估盈亏
    ↓
确认偏差 → 寻找支持自己判断的信息
    ↓
过度自信 → 高估判断的正确性
    ↓
处置效应 → 过早卖出盈利，过久持有亏损
    ↓
羊群行为 → 跟随大众确认自己的决策
```

## 偏差对市场的系统性影响

| 市场现象 | 主要偏差来源 |
|----------|-------------|
| 价格泡沫 | 过度乐观、羊群行为、代表性偏差 |
| 市场崩盘 | 损失厌恶、恐慌性羊群 |
| 动量效应 | 反应不足、锚定效应 |
| 长期反转 | 过度反应、代表性偏差 |
| 波动率之谜 | 损失厌恶、框架依赖 |
| 交易量过大 | 过度自信、控制幻觉 |

## 克服偏差的系统方法

### 认知偏差的纠正

1. 建立客观的投资评估体系
2. 主动寻找反面证据
3. 使用标准化决策清单
4. 多元化信息来源
5. 定期复盘和校准

### 情绪偏差的适应

1. 设定自动化投资规则（如定投）
2. 明确的止损止盈纪律
3. 减少查看投资组合的频率
4. 建立投资委员会分散个人情绪
5. 使用量化工具辅助决策

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `行为金融学的认知偏差全景` 时，不要只把它当成孤立知识点。把心理偏差转成流程约束，而不是停留在自我提醒。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 损失厌恶、确认偏差和过度自信
- 交易日志和事前计划
- 市场情绪和拥挤交易
- 自动化纪律和风控约束

### 进阶训练

1. 在交易日志中记录入场理由和情绪状态
2. 为每种偏差设计一个流程拦截点
3. 复盘亏损是否来自策略失效还是执行偏差

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
