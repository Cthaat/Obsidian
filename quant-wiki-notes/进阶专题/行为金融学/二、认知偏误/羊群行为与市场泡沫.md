# 羊群行为与市场泡沫

> [!note] 核心概念
> **羊群行为**（Herd Behavior）指投资者在面对不确定性时，倾向于跟随大众的行为，放弃独立判断。这种行为可导致市场价格波动加剧，形成市场泡沫和崩溃。Keynes 称之为"动物精神"。

## 羊群行为的成因

### 信息瀑布理论

Banerjee (1992) 提出：
- 早期投资者根据有限信息采取行动
- 后续投资者倾向于模仿而非独立分析
- "复制"行为层层叠加 → 价格脱离真实价值

### 心理根源

| 驱动因素 | 说明 |
|----------|------|
| 信息不对称 | 缺乏专业判断，选择跟随"聪明钱" |
| 后悔厌恶 | 害怕独立决策失败的后悔 |
| 社会认同 | 与群体站在一起的心理诉求 |
| 有限注意力 | 关注热点，跟随大众关注焦点 |
| 追求时尚 | 对相对经济地位的追求 |

### 社交媒体时代的放大效应

- Reddit、Twitter 等平台降低信息传递成本
- 模因（Meme）、流行语强化群体认同
- 投资决策兼具"与群体站在一起"的心理诉求
- **GameStop事件**是典型案例

## 羊群行为的市场影响

### 泡沫形成机制

```
热点信息/事件 → 注意力聚焦 → 群体跟风买入 → 价格持续上涨
→ 更多人关注 → 正反馈循环 → 价格严重偏离基本面 → 泡沫
→ 情绪逆转 → 集中抛售 → 崩盘
```

### 实证表现

- 羊群效应在A股市场表现尤为明显
- "跟风""跟庄"是中国股市的典型特征
- 个体投资者的羊群行为导致市场预测出现系统性偏差

### 与情绪的螺旋加强

熊建龙研究发现：
- 个体投资受[[期望理论|前景理论]]主导
- 在羊群效应加强下形成一致预期的投资者情绪
- 情绪与股市风险形成**螺旋加强**
- 并在不同市场相互蔓延

## 量化投资中的羊群行为

### 识别信号

| 信号 | 数据来源 |
|------|----------|
| 订单流集中 | 高频交易数据 |
| 社交媒体热度 | Reddit、Twitter讨论量 |
| 搜索指数 | Google Trends |
| 成交量异常 | 短期交易量激增 |
| 价差扩大 | 做市商报价异常 |

### 程序化交易中的"集体羊群"

- 大量机构使用类似量化策略
- 策略同时触发卖出信号 → 订单集中涌入
- 价格剧烈波动甚至流动性枯竭
- 这是行为偏差在市场层面的**放大**

## 经典案例

### GameStop (2021)

1. Reddit论坛散户形成合力
2. 集中买入GME、AMC等标的
3. 挑战传统机构空头头寸
4. 价格短期大幅推高后迅速回调
5. 反映市场情绪极端爆发和社会认同偏差

### 中国股市的"政策市"

- 个人投资者对政策面消息反应尤为强烈
- 普通投资者信息不完全，表现出过度反应
- 机构投资者可采取反向投资策略

## 巴菲特的智慧

> "别人恐惧我贪婪，别人贪婪我恐惧。"

这是对羊群行为最精辟的逆向策略。
