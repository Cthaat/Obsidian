# 锚定效应与心理账户

> [!note] 核心概念
> **锚定效应**（Anchoring）指投资决策中，某变量的前特定值会影响决策人的定量判断。**心理账户**（Mental Accounting）指投资人对投资目标进行分层处理，通过不同的资产组合达成不同的目标，导致非最优的整体资产配置。

## 锚定效应

### 定义与机制

- 由 Tversky 和 Kahneman 提出
- 人们在做判断时，会过度依赖最先获得的信息（"锚"）
- 后续调整通常不充分

### 在投资中的表现

| 锚定对象 | 表现 |
|----------|------|
| 买入价格 | 以成本价判断盈亏，而非关注当前价值 |
| 历史高点 | 认为股票"便宜"因为远低于历史高点 |
| 分析师目标价 | 以某个分析师预测作为判断基准 |
| 整数关口 | 对50、100等整数价格有心理锚定 |
| 近期价格 | 以最近的价格作为判断"正常"的基准 |

### 过度自信与锚定的叠加

- 投资者对增强自信心的信息给予重视
- 对损害自信心的信息给予回避
- 新的、高强度的信息才能修正过度自信

### 在量化投资中的应用

- 监测价格偏离锚定值的程度
- 利用锚定效应导致的反应不足设计动量策略
- 识别投资者对特定价格水平的心理预期

## 心理账户

### 定义与机制

Richard Thaler 提出的概念：
- 投资者不会将所有财富视为一个整体
- 而是将资金分配到不同的"心理账户"中
- 每个账户有独立的目标、风险承受度和决策规则

### 典型表现

| 心理账户 | 行为特征 |
|----------|----------|
| 退休账户 | 保守投资，长期持有 |
| 投机账户 | 激进操作，追求高收益 |
| 意外之财 | 更容易冒险或挥霍 |
| 辛苦所得 | 谨慎使用，不愿冒险 |

### 非理性后果

1. **分散化不足**：每个账户独立决策，忽略整体资产配置
2. **资金非同质性**：同样100元，来源不同→风险态度不同
3. **沉没成本**：对已投入的资金产生特殊对待
4. **损失分散化**：将损失分散到不同账户以减轻心理痛苦

### 与传统理论的矛盾

- 传统经济学：资金是可替代的（fungible）
- 心理账户：资金不可完全替代，存在"标签效应"

## 两者的关系

```
锚定效应 → 设定参照点（如买入价）
    ↓
心理账户 → 根据参照点将投资分类
    ↓
决策偏差 → 在不同账户中采取不同的非最优策略
```

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `锚定效应与心理账户` 时，不要只把它当成孤立知识点。把心理偏差转成流程约束，而不是停留在自我提醒。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 损失厌恶、确认偏差和过度自信
- 交易日志和事前计划
- 市场情绪和拥挤交易
- 自动化纪律和风控约束

### 进阶训练

1. 在交易日志中记录入场理由和情绪状态
2. 为每种偏差设计一个流程拦截点
3. 复盘亏损是否来自策略失效还是执行偏差

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
