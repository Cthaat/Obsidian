# 锚定效应与心理账户

> [!note] 核心概念
> **锚定效应**（Anchoring）指投资决策中，某变量的前特定值会影响决策人的定量判断。**心理账户**（Mental Accounting）指投资人对投资目标进行分层处理，通过不同的资产组合达成不同的目标，导致非最优的整体资产配置。

## 锚定效应

### 定义与机制

- 由 Tversky 和 Kahneman 提出
- 人们在做判断时，会过度依赖最先获得的信息（"锚"）
- 后续调整通常不充分

### 在投资中的表现

| 锚定对象 | 表现 |
|----------|------|
| 买入价格 | 以成本价判断盈亏，而非关注当前价值 |
| 历史高点 | 认为股票"便宜"因为远低于历史高点 |
| 分析师目标价 | 以某个分析师预测作为判断基准 |
| 整数关口 | 对50、100等整数价格有心理锚定 |
| 近期价格 | 以最近的价格作为判断"正常"的基准 |

### 过度自信与锚定的叠加

- 投资者对增强自信心的信息给予重视
- 对损害自信心的信息给予回避
- 新的、高强度的信息才能修正过度自信

### 在量化投资中的应用

- 监测价格偏离锚定值的程度
- 利用锚定效应导致的反应不足设计动量策略
- 识别投资者对特定价格水平的心理预期

## 心理账户

### 定义与机制

Richard Thaler 提出的概念：
- 投资者不会将所有财富视为一个整体
- 而是将资金分配到不同的"心理账户"中
- 每个账户有独立的目标、风险承受度和决策规则

### 典型表现

| 心理账户 | 行为特征 |
|----------|----------|
| 退休账户 | 保守投资，长期持有 |
| 投机账户 | 激进操作，追求高收益 |
| 意外之财 | 更容易冒险或挥霍 |
| 辛苦所得 | 谨慎使用，不愿冒险 |

### 非理性后果

1. **分散化不足**：每个账户独立决策，忽略整体资产配置
2. **资金非同质性**：同样100元，来源不同→风险态度不同
3. **沉没成本**：对已投入的资金产生特殊对待
4. **损失分散化**：将损失分散到不同账户以减轻心理痛苦

### 与传统理论的矛盾

- 传统经济学：资金是可替代的（fungible）
- 心理账户：资金不可完全替代，存在"标签效应"

## 两者的关系

```
锚定效应 → 设定参照点（如买入价）
    ↓
心理账户 → 根据参照点将投资分类
    ↓
决策偏差 → 在不同账户中采取不同的非最优策略
```

## 实战掌握清单

> [!tip] 交易者视角
> 锚定效应与心理账户 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。行为金融的价值在于识别自己和市场参与者的系统性偏差，并把偏差转成可执行的纪律。

### 关键判断

- 识别偏差发生在信息收集、判断、下单、持有还是止损环节。
- 区分个人心理错误和群体行为造成的市场机会。
- 用规则约束情绪最强的时刻。

### 落地动作

1. 交易前写下反方观点和失效点。
2. 交易后记录情绪、冲动、犹豫和执行偏差。
3. 把常犯错误转成硬性限制，例如冷静期、仓位上限和复盘后再交易。

### 失效边界

- 用心理学解释一切行情。
- 知道偏差但没有流程约束。
- 把短期盈利当作决策正确的证据。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 行为日志

围绕 锚定效应与心理账户 建立交易心理日志：记录交易前的信心、恐惧、冲动、信息来源和是否偏离计划。连续记录比单次反思更有价值。

### 纠偏规则

- 每笔交易前写反方观点。
- 盈利后限制立刻加仓，亏损后限制翻本交易。
- 对止损拖延、过早止盈和冲动追涨单独统计。

### 验收标准

能否减少重复错误，比能否解释心理偏差更重要。

## 最小可执行项目

### 交易错误库

围绕 锚定效应与心理账户 建一个交易错误库：每次错误记录触发事件、情绪、原计划、实际动作、损失、偏差类型和下一次约束。

| 错误 | 约束例子 |
|---|---|
| 追涨 | 突破后等待回踩或缩小仓位 |
| 扛亏 | 失效点触发强制减仓 |
| 过早止盈 | 分批退出而不是一次清仓 |
| 翻本交易 | 当日亏损到线停止交易 |

### 验收标准

同类错误出现频率下降，才说明行为训练有效。
