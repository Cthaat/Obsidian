# 确认偏差与代表性偏差

> [!note] 核心概念
> **确认偏差**（Confirmation Bias）指投资者只关注和接受与自己已有观点一致的信息，忽略相反证据。**代表性偏差**（Representativeness）指以过去的经验直接推断未来，忽视基础概率。两者同属**信念固执**（Belief Perseverance）类认知偏差。

## 确认偏差

### 表现形式

- 只看支持自己观点的分析报告
- 在新闻中只关注利好/利空（取决于持仓方向）
- 将模糊信息解读为支持自己判断
- 忽视或贬低反面证据

### 心理机制

```
已有信念 → 选择性信息收集 → 信念强化 → 更强的选择性 → 极端化判断
```

### 在投资中的危害

| 阶段 | 危害 |
|------|------|
| 研究阶段 | 只寻找支持买入/卖出的信息 |
| 持仓阶段 | 忽视基本面恶化信号 |
| 止损阶段 | 寻找"还会涨回来"的理由 |
| 卖出后 | 只关注证明卖出正确的信息 |

### 克服方法

1. **主动寻找反面证据**：每次做决策前列出3个反对理由
2. **魔鬼代言人**：指定团队成员扮演反对角色
3. **预设检查点**：提前设定需要重新评估的条件
4. **多元化信息源**：订阅不同立场的分析

## 代表性偏差

### 定义

- 人们倾向于根据事物的表面相似性做判断
- 忽视基础概率（base rate）
- 以过去经验直接推断未来

### 典型案例

| 场景 | 代表性偏差表现 |
|------|---------------|
| 连续上涨 | "这只股票趋势好，还会涨" |
| 连续下跌 | "这只股票不行，还会跌" |
| 过去业绩好 | "基金经理能力强，未来也会好" |
| 某行业热门 | "这个行业未来也会热门" |

### 导致的市场现象

- **短期动量效应**：连续上涨后继续买入
- **过度反应**：对近期表现过度外推
- **均值回归忽视**：忽视极端表现会回归平均的趋势

## 保守偏差

与代表性偏差相对：

- 做投资决策时，倾向于给现有信息更大权重
- 新的信息被赋予过小权重
- 导致对新信息的**反应不足**

### 与代表性偏差的互动

```
信息出现 →
  代表性偏差：过度反应（短期趋势外推）
  保守偏差：反应不足（忽视新信息）
  → 两者交替作用 → 价格的过度反应与反应不足并存
```

## 其他信念固执类偏差

### 控制幻觉（Illusion of Control）

- 感觉自己可以控制投资的某个部分或整个过程
- 选择号码、盯盘、频繁操作 → 增加"控制感"
- 实际上无法控制市场走向

### 后见之明（Hindsight Bias）

- 事后诸葛亮
- 结果发生后声称自己有先见之明
- 根据结果调整自己的记忆和意见

## 实战掌握清单

> [!tip] 交易者视角
> 确认偏差与代表性偏差 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。行为金融的价值在于识别自己和市场参与者的系统性偏差，并把偏差转成可执行的纪律。

### 关键判断

- 识别偏差发生在信息收集、判断、下单、持有还是止损环节。
- 区分个人心理错误和群体行为造成的市场机会。
- 用规则约束情绪最强的时刻。

### 落地动作

1. 交易前写下反方观点和失效点。
2. 交易后记录情绪、冲动、犹豫和执行偏差。
3. 把常犯错误转成硬性限制，例如冷静期、仓位上限和复盘后再交易。

### 失效边界

- 用心理学解释一切行情。
- 知道偏差但没有流程约束。
- 把短期盈利当作决策正确的证据。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 行为日志

围绕 确认偏差与代表性偏差 建立交易心理日志：记录交易前的信心、恐惧、冲动、信息来源和是否偏离计划。连续记录比单次反思更有价值。

### 纠偏规则

- 每笔交易前写反方观点。
- 盈利后限制立刻加仓，亏损后限制翻本交易。
- 对止损拖延、过早止盈和冲动追涨单独统计。

### 验收标准

能否减少重复错误，比能否解释心理偏差更重要。
