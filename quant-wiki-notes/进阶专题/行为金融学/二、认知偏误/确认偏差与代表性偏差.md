# 确认偏差与代表性偏差

> [!note] 核心概念
> **确认偏差**（Confirmation Bias）指投资者只关注和接受与自己已有观点一致的信息，忽略相反证据。**代表性偏差**（Representativeness）指以过去的经验直接推断未来，忽视基础概率。两者同属**信念固执**（Belief Perseverance）类认知偏差。

## 确认偏差

### 表现形式

- 只看支持自己观点的分析报告
- 在新闻中只关注利好/利空（取决于持仓方向）
- 将模糊信息解读为支持自己判断
- 忽视或贬低反面证据

### 心理机制

```
已有信念 → 选择性信息收集 → 信念强化 → 更强的选择性 → 极端化判断
```

### 在投资中的危害

| 阶段 | 危害 |
|------|------|
| 研究阶段 | 只寻找支持买入/卖出的信息 |
| 持仓阶段 | 忽视基本面恶化信号 |
| 止损阶段 | 寻找"还会涨回来"的理由 |
| 卖出后 | 只关注证明卖出正确的信息 |

### 克服方法

1. **主动寻找反面证据**：每次做决策前列出3个反对理由
2. **魔鬼代言人**：指定团队成员扮演反对角色
3. **预设检查点**：提前设定需要重新评估的条件
4. **多元化信息源**：订阅不同立场的分析

## 代表性偏差

### 定义

- 人们倾向于根据事物的表面相似性做判断
- 忽视基础概率（base rate）
- 以过去经验直接推断未来

### 典型案例

| 场景 | 代表性偏差表现 |
|------|---------------|
| 连续上涨 | "这只股票趋势好，还会涨" |
| 连续下跌 | "这只股票不行，还会跌" |
| 过去业绩好 | "基金经理能力强，未来也会好" |
| 某行业热门 | "这个行业未来也会热门" |

### 导致的市场现象

- **短期动量效应**：连续上涨后继续买入
- **过度反应**：对近期表现过度外推
- **均值回归忽视**：忽视极端表现会回归平均的趋势

## 保守偏差

与代表性偏差相对：

- 做投资决策时，倾向于给现有信息更大权重
- 新的信息被赋予过小权重
- 导致对新信息的**反应不足**

### 与代表性偏差的互动

```
信息出现 →
  代表性偏差：过度反应（短期趋势外推）
  保守偏差：反应不足（忽视新信息）
  → 两者交替作用 → 价格的过度反应与反应不足并存
```

## 其他信念固执类偏差

### 控制幻觉（Illusion of Control）

- 感觉自己可以控制投资的某个部分或整个过程
- 选择号码、盯盘、频繁操作 → 增加"控制感"
- 实际上无法控制市场走向

### 后见之明（Hindsight Bias）

- 事后诸葛亮
- 结果发生后声称自己有先见之明
- 根据结果调整自己的记忆和意见

## 课程化学习补充

> [!important] 学习定位
> 行为金融学解释人为什么会系统性犯错；交易中的价值在于把偏误变成流程约束和反向观察指标。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 当前决策受哪种偏误影响
- 是否有预设退出规则
- 是否用数据替代情绪判断
- 复盘是否记录当时心理状态

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 知道偏误但不改变流程
- 逆向交易过早
- 把市场情绪当必然反转
- 亏损后加仓摊平

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `确认偏差与代表性偏差` 时，不要只把它当成孤立知识点。把心理偏差转成流程约束，而不是停留在自我提醒。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 损失厌恶、确认偏差和过度自信
- 交易日志和事前计划
- 市场情绪和拥挤交易
- 自动化纪律和风控约束

### 进阶训练

1. 在交易日志中记录入场理由和情绪状态
2. 为每种偏差设计一个流程拦截点
3. 复盘亏损是否来自策略失效还是执行偏差

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
