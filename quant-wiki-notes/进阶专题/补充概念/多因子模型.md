# 多因子模型

> [!note] 核心定义
> 多因子模型把一只资产的收益拆解为若干**共同收益来源（因子）**的线性组合加上个体残差。它是量化选股与组合管理的核心框架——理解它，才能系统化地回答"我赚的是哪几种钱、各占多少、还能不能持续"。

## 一、定义与直觉

单因子模型（CAPM）认为股票收益只由"市场"一个因子驱动，但现实里，决定一只股票长期相对表现的因素显然不止市场涨跌：估值贵不贵、市值大不大、过去涨势如何、盈利质量好不好……这些可观察特征都被证明与未来收益存在系统性关系。**多因子模型就是把这些"共同收益来源"显式建模出来。**

核心思想：**成千上万只股票的收益波动，并不是相互独立的，而是被少数几个共同因子牵引。** 比如所有低估值股票会因为"价值风格"集体起落，所有小盘股会因为"规模风格"同涨同跌。把这些共同部分（因子）剥离出来，剩下的才是每只股票的个体特异部分。

一个类比：因子之于股票，就像"基本面 + 风格 + 行业"之于一个人的工资。两个人收入差异，一部分来自共同的大环境（行业景气、地域、学历溢价 = 因子），一部分来自个人独有的能力运气（残差）。多因子模型就是把收益做这样一次"成分拆解"。

它解决三件事：① **解释**——横截面上谁涨谁跌由哪些因子决定；② **预测/选股**——用因子打分给股票排序，构建组合（alpha 模型）；③ **风险管理**——度量组合在各因子上的暴露，控制风险（风险模型）。

## 二、原理与公式

### 2.1 一般形式

对资产 $i$，其收益由 $K$ 个因子线性解释：

$$r_i=\alpha_i+\sum_{k=1}^{K}\beta_{ik}\,f_k+\epsilon_i$$

符号含义：

- $r_i$：资产 $i$ 的收益（实务中常用超额收益，即扣掉无风险利率）。
- $f_k$：第 $k$ 个**因子收益**（共同收益来源，如价值、动量、规模等的收益）。
- $\beta_{ik}$：资产 $i$ 对因子 $k$ 的**载荷/暴露**（敏感度）。
- $\alpha_i$：因子无法解释的部分（截距），是主动管理追求的目标。
- $\epsilon_i$：残差（特异风险），均值为 0，理论上与各因子不相关、不同资产间相互独立。

CAPM 是 $K=1$ 的特例；Fama-French 三因子是 $K=3$；[[Carhart四因子模型]] 是 $K=4$。多因子模型把这个框架一般化到任意多个因子。

### 2.2 因子的两种来源

- **基本面/特征因子**：来自财务与价格特征，如 B/M（价值）、市值（规模）、过去收益（动量）、ROE/毛利（质量）、波动率（低波）。这是选股最常用的一类。
- **宏观因子**：如利率、通胀、信用利差、汇率、油价等，更多用于资产配置和风险归因。

### 2.3 组合层面的暴露与收益

组合的因子暴露是成分股暴露的加权和。设权重为 $w_i$，则组合在因子 $k$ 上的暴露：

$$\beta_{p,k}=\sum_i w_i\,\beta_{ik}$$

组合预期超额收益可近似分解为"各因子暴露 × 因子预期收益"之和。这正是多因子模型既能**主动获取收益**（提高想要的因子暴露），又能**控制风险**（约束不想要的因子暴露）的数学根基。

## 三、构建流程与评价指标

### 3.1 标准构建流程

一套可落地的多因子选股，通常按下面的流水线推进：

| 步骤 | 做什么 | 关键点 |
|---|---|---|
| 1. 因子选择 | 提出候选因子并给出**经济解释** | 先有逻辑再有数据，避免纯挖掘出的伪因子 |
| 2. 数据清洗 | 处理缺失、停牌、退市、复权、财报对齐 | 严防**前视偏差**（用了当时还没公布的数据） |
| 3. 单因子检验 | 算 IC/IR、做分层回测，看因子是否有效 | 同时看方向、稳定性、单调性 |
| 4. 数据预处理 | **去极值 → 标准化 → 中性化** | 见下方，三步是因子可比与正交的前提 |
| 5. 因子合成 | 把多个有效因子合成综合得分 | 等权、IC 加权或最大化 IR 加权 |
| 6. 组合优化 | 在风险/换手/行业等约束下定权重 | 兼顾预期收益、风险、交易成本与容量 |

**第 4 步的三件套**，是新手最易忽略却最关键的工程细节：

- **去极值（winsorize）**：把极端离群值（如 PE 为负、超大值）截断到合理分位（如 1%/99% 分位或均值 ± 3 倍标准差），避免少数异常值主导排序。
- **标准化（z-score）**：把每个因子转成均值 0、标准差 1，使不同量纲的因子（PE 和动量）可以加总比较。
- **中性化（neutralization）**：用回归剔除因子里夹带的**行业**和**市值**成分，避免"价值因子其实在赌某个行业/小盘"。常用做法是把因子值对行业哑变量和市值回归，取残差作为中性化后的因子。

### 3.2 因子正交化与多重共线性

实务里候选因子彼此高度相关（如多个估值类因子），直接放进模型会出现**多重共线性**：回归系数不稳定、符号乱跳、难以解释每个因子的边际贡献。应对方式是 [[因子正交化]]——通过 Gram-Schmidt 正交化、对称正交化或主成分等方法，把相关因子转成互不相关的成分，让每个因子贡献"干净"的独立信息。中性化本质上也是正交化的一种（让因子正交于行业和市值）。

### 3.3 因子拥挤与衰减

因子不是免费午餐：一个被广泛知晓并大量资金涌入的因子，会发生 [[因子拥挤与衰减]]。

- **拥挤**：太多人同向交易同一因子，推高估值、压缩未来收益，并在去杠杆时引发同向踩踏（拥挤反转）。
- **衰减（decay）**：因子被发表、被套利后，超额收益随时间下降甚至消失。

这意味着多因子模型需要**持续监控**：跟踪因子拥挤度、IC 的滚动变化，对衰减的因子及时降权或剔除。

### 3.4 评价指标

衡量一个因子/模型好坏，核心看这几项：

- **IC（信息系数）**：因子值与下期收益的横截面相关系数（常用 Rank IC，即秩相关）。绝对值越大、方向越稳，因子越有效。经验上月度 Rank IC 均值能稳定在 0.03–0.05 已属不错（**经验区间，非保证**）。
- **IR（信息比率）**：$IR=\dfrac{\overline{IC}}{\sigma_{IC}}$，即 IC 均值除以其波动。衡量因子有效性的**稳定性**，IR 高说明因子不仅有效而且可靠。
- **分层回测**：按因子值把股票分成 5–10 组，看各组未来收益是否**单调**排列。单调性好说明因子区分度强。
- **多空收益**：做多最高分组、做空最低分组的收益，是因子"纯收益"的直接体现；同时要看其波动、回撤与换手。

> [!example] IC 与 IR 的直觉
> 因子 A 月度 Rank IC 均值 0.06、波动 0.20，IR = 0.30；因子 B 均值 0.04、波动 0.06，IR ≈ 0.67（**均为假设**）。虽然 A 的"平均有效性"更高，但 B 更稳定、更可靠，组合配置中往往给 B 更高权重。**有效不等于稳定，二者都要看。**

## 四、Alpha 模型 vs 风险模型

多因子框架在实务里有两条不同目的的分支，**初学者极易混淆**：

| 维度 | Alpha 模型（选股） | 风险模型（如 Barra） |
|---|---|---|
| 目的 | **预测**未来收益、获取超额回报 | **解释/度量**风险，做归因与优化约束 |
| 因子取舍 | 只要能赚钱，因子越独特越好（怕拥挤） | 求**全面**，覆盖所有主要风险来源 |
| 因子是否公开 | 越私密越好，公开就拥挤衰减 | 行业标准化、公开透明（如 Barra 风格因子） |
| 典型输出 | 股票预期收益排序 / 打分 | 因子暴露、因子协方差矩阵、风险归因 |
| 用在哪一步 | 流程中的"因子合成、选股" | 流程中的"组合优化、风险约束" |

一句话区分：**Alpha 模型回答"该买谁"，风险模型回答"我现在扛着哪些风险、组合该怎么配权重"。** 成熟的量化流程两者并用——用 alpha 模型给出收益预期，用风险模型（协方差矩阵 + 约束）做 [[组合优化]]，在控制风险和交易成本的前提下落地权重。

### A股 落地的制度约束

- **T+1**：当日买入不能当日卖出，调仓节奏与换手测算都要按此设定。
- **涨跌停**：涨停买不进、跌停卖不出，会让模型信号无法成交，回测须做涨跌停成交过滤。
- **印花税与佣金**：卖出单边印花税叠加佣金，高换手因子（如动量）净收益受损严重，回测必须扣费。
- **流动性与容量**：小盘股因子收益高但容量小，资金规模一大就冲击成本飙升、收益消失，**容量是策略能管多少钱的硬约束**。
- **数据口径**：财报披露有时滞，因子对齐必须用"可获得时点"的数据，严防前视偏差。

## 五、常见误区与失效条件

| 误区 | 正确理解 |
|---|---|
| 因子越多模型越好 | 因子多易**过拟合**与共线，关键是独立、有逻辑、互补，而非数量 |
| 单因子 IC 高就能直接用 | 还要看 **IR（稳定性）**、单调性、换手与容量；高 IC 高换手可能不赚钱 |
| 不做中性化也无所谓 | 不中性化的因子常**暗含行业/市值赌注**，暴露漂移、归因失真 |
| 历史回测好就能上线 | 因子会**拥挤与衰减**，须样本外验证并持续监控滚动 IC |
| Alpha 模型和风险模型是一回事 | 一个求**预测收益**（怕公开），一个求**度量风险**（要全面透明），目的相反 |
| 回测忽略交易成本和涨跌停 | A股 高换手策略扣费后可能由盈转亏，**成本与制度必须前置进回测** |
| 因子合成等权最简单所以最好 | 等权忽略各因子稳定性差异；IC/IR 加权常更稳健，但也要防过拟合 |

> [!warning] 多因子模型的根本前提
> 它假设"历史上有效的共同收益来源在未来继续有效"。一旦市场结构、制度或资金行为改变（因子拥挤、风格切换、监管变化），模型就会失效。多因子不是一次性建好就完事，而是**需要持续检验、监控和迭代**的活系统。
