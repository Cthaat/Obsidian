# PS

> [!note] 核心定义
> 市销率（PS）= 总市值 ÷ 营业收入，衡量"市场愿意为公司每一元营收付多少钱"。当公司亏损、PE 失效时，PS 是少数仍能用的估值锚——因为再差的公司也有营收。它常用于成长股、亏损期的周期股，以及利润被一次性因素扭曲的情形。

## 一、定义与直觉

$$PS = \frac{总市值}{营业收入} = \frac{股价}{每股营收(SPS)}$$

PE 看的是"赚了多少钱"，PS 看的是"做了多大生意"。为什么需要 PS？因为分母换成了营收：

- 营收几乎总是正的，**亏损公司也能算 PS**，而 PE 此时为负、失效；
- 营收比利润**更难被会计手段操纵**（利润受折旧、减值、一次性损益等大量调节，营收相对干净）；
- 营收波动远小于利润，**周期低谷期**用 PS 比用 PE 稳定。

这让 PS 成为给"还没赚钱、但在快速做大"的公司估值的主力工具，典型如早期 SaaS、互联网平台、生物医药、新能源等。

> [!tip] PS 与净利率的桥梁
> 一个关键关系：$PS = PE \times 净利率$。推导：$PE \times 净利率 = \dfrac{市值}{净利润}\times\dfrac{净利润}{营收}=\dfrac{市值}{营收}=PS$。它说明同样的 PS，净利率越高的公司其实越便宜（隐含 PE 越低）。**跨净利率水平比较 PS 是错的**——10% 净利率和 30% 净利率的公司，PS 不可直接比。

## 二、适用场景

| 场景 | 为何用 PS |
|---|---|
| 尚未盈利的成长股 | EPS 为负 PE 失效，但有高速增长的营收，PS 衡量市场对营收规模的定价 |
| SaaS / 互联网平台 | 早期重投入扩张、刻意不盈利，营收（尤其经常性收入 ARR）是核心 |
| 周期股低谷期 | 景气底部利润微薄甚至亏损，PS 比 PE 稳定，可用历史 PS 区间判断底部 |
| 利润被一次性项目扭曲 | 大额减值、资产处置、补贴使当期利润失真时，营收更能反映真实经营规模 |
| 困境反转标的 | 业务还在（有营收）但暂时亏损，PS 衡量"恢复盈利后的潜在价值" |

## 三、计算案例（假设）

| 项目 | 成长股 A（亏损） | 成熟股 B |
|---|---|---|
| 总市值 | 100 亿 | 100 亿 |
| 营业收入 | 20 亿 | 50 亿 |
| 净利润 | −2 亿（亏损） | 10 亿 |
| PS | 5.0 | 2.0 |
| PE | 无意义（负） | 10 |
| 净利率 | −10% | 20% |

A 公司亏损，PE 无法使用，但 PS=5；B 公司 PS=2 看起来更便宜。然而 B 的净利率 20%、A 还在亏钱——B 的低 PS 由其高盈利能力支撑。对 A 的判断要落到："5 倍 PS 是否过度透支了它未来转正后的盈利？"如果 A 未来净利率能做到 15%，则其隐含 PE = PS/净利率 = 5/0.15 ≈ 33 倍。（数字均为假设）

## 四、优点与缺点

**优点**：

- 适用范围广（亏损、周期底部都能用）；
- 营收难操纵，稳定性高；
- 适合衡量市场对"规模/份额"的定价。

**缺点**：

- **完全忽略盈利能力和成本结构**：高营收不等于会赚钱，亏损扩大的公司 PS 可能很低却毫无价值；
- **不同商业模式不可比**：自营（高营收低毛利，如零售、贸易）和平台抽佣（低营收高毛利）的 PS 天然差几个量级；
- **忽略负债**：两家 PS 相同但一家高负债一家净现金，风险完全不同（此时 EV/Sales 更合理）。

> [!warning] 高营收的幻觉
> PS 最大的危险是把"营收大"误当"价值高"。一家靠烧钱补贴换营收、毛利为负、永远不可能盈利的公司，营收越大亏得越多，低 PS 反而是警报。用 PS 必须同时盯住毛利率、净利率趋势和盈利路径——PS 只回答"生意多大"，不回答"生意好不好"。

## 五、常见误区与失效条件

| 误区 | 正确理解 |
|---|---|
| PS 低就是便宜 | PS 忽略盈利能力，低 PS 可能是低毛利或永不盈利的生意 |
| 跨行业/跨商业模式比 PS | 自营与平台、不同净利率的公司 PS 不可直接比，需用"PS=PE×净利率"换算 |
| 用 PS 给成熟盈利公司估值 | 成熟公司有稳定利润，应优先用 PE，PS 只是辅助 |
| 只看营收增长不看毛利 | 烧钱换来的营收增长若毛利为负，是价值毁灭 |
| 忽略负债 | 高负债公司应改用 EV/Sales（含债务的企业价值/营收） |

> [!note] 失效条件小结
> PS 在以下情形会误导：① 公司商业模式决定了天然高营收低毛利；② 营收靠不可持续的补贴/烧钱堆出；③ 负债结构差异大；④ 把它当成成熟盈利公司的主估值。记住：PS 是"亏损期和高成长期"的过渡工具，公司一旦稳定盈利就应回到 PE / 现金流估值。
