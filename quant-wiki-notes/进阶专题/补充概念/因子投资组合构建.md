# 因子投资组合构建

> [!note] 核心定位
> 因子投资组合构建 属于因子投资和系统化选股框架。它的核心不是“指标看起来有效”，而是确认收益来源、数据口径、检验方法、成本、容量和衰减风险。

## 一、从单因子到组合的关键变化

单因子研究回答“这个特征是否有预测能力”，组合构建回答“如何把多个预测能力变成一条可交易、可承受、可持续的资金曲线”。两者不是一回事。

| 问题 | 单因子研究 | 组合构建 |
|---|---|---|
| 目标 | 验证一个信号是否有效 | 把多个信号合成可投资组合 |
| 关注点 | IC、分层收益、方向稳定性 | 权重、风险、相关性、成本、容量 |
| 主要风险 | 数据挖掘、样本内过拟合 | 因子重复、行业集中、换手过高 |
| 输出 | 因子分数或排序 | 持仓列表、权重、调仓计划 |

## 二、组合构建流程

1. **确定股票池**：指数成分、全市场、行业池，还是满足流动性和财务质量门槛的可投资池。
2. **计算因子分数**：对每个因子做去极值、标准化、中性化，确保方向一致。
3. **合成多因子分数**：等权、IC加权、风险调整加权、机器学习模型输出都可以，但必须可解释。
4. **加入约束**：行业权重、单票权重、换手率、跟踪误差、流动性、黑名单。
5. **生成目标权重**：从分数到权重，可以用分层、多空、最大化预期收益/风险、风险平价等方式。
6. **交易成本优化**：控制调仓幅度，避免为了微小分数变化频繁交易。
7. **上线监控**：跟踪因子贡献、行业暴露、回撤、换手、成交质量和容量。

## 三、权重方法对比

| 方法 | 做法 | 适合场景 | 风险 |
|---|---|---|---|
| 等权持有 | 选择高分组股票等权买入 | 简单、稳健、易解释 | 忽略风险差异和流动性 |
| 分数加权 | 分数越高权重越大 | 因子强弱有明显差异 | 极端分数可能导致集中 |
| 风险约束优化 | 在风险模型下最大化预期收益 | 机构组合、指数增强 | 对风险模型和收益预测敏感 |
| 分层组合 | 每层观察收益差异，高分层建仓 | 因子验证和教学 | 不是完整实盘组合 |
| 多空组合 | 买高分、卖低分 | 研究因子纯收益 | A股做空受限，实盘难复制 |

## 四、组合构建的常见坑

- **因子重复**：价值因子里的 PE、PB、PCF 高度相关，不代表三倍信号。
- **隐性行业暴露**：质量因子可能偏消费，价值因子可能偏银行周期。
- **小市值偏置**：回测收益可能主要来自小盘风格，不是因子本身。
- **换手过高**：每次调仓都消耗成本，纸面 alpha 被交易摩擦吃掉。
- **容量不足**：小票策略在小资金时有效，资金变大后成交冲击巨大。
- **过度优化**：复杂优化器会把历史噪音当成精确答案。

因子组合的最终目标不是让回测曲线最漂亮，而是让每个收益来源都能解释、每个风险暴露都能监控、每次调仓都能复盘。

## 六、研究 `因子投资组合构建` 时必须验证什么

因子类主题最容易出现“看起来专业但不能交易”的问题。真正的研究必须把指标、经济解释、统计检验和实盘约束放在一起。

| 维度 | 检查问题 |
|---|---|
| 定义 | 指标公式是否唯一，字段和时间戳是否清楚 |
| 解释 | 收益来自风险补偿、行为偏差、流动性，还是数据挖掘 |
| 稳定性 | IC、Rank IC、分层收益是否跨时期稳定 |
| 风险 | 是否隐藏行业、市值、流动性或beta暴露 |
| 成本 | 换手、滑点、冲击成本是否会吞噬收益 |
| 容量 | 资金规模扩大后是否还能成交 |
| 衰减 | 因子公开或拥挤后收益是否下降 |

因子研究的结果不应只是“有效/无效”，而应说明它适合进入哪类组合、承担什么风险、在什么环境下需要降权。

## 八、从研究报告到组合权重

`因子投资组合构建` 进入组合前，需要从“研究有效”转成“仓位合理”。一个因子即使长期有效，也可能因为波动大、换手高、容量小或与已有持仓相关性高而不适合重仓。

| 决策 | 需要的信息 |
|---|---|
| 是否使用 | 经济解释、IC、分层收益、样本外表现 |
| 用多大权重 | 因子波动、回撤、相关性、拥挤度 |
| 多久调仓 | 信号半衰期、交易成本、税费和滑点 |
| 何时降权 | IC下降、分层失效、成本上升、拥挤加剧 |
