# 龙虎榜分析

> [!note] 核心定义
> 龙虎榜是交易所在个股出现价格、换手或振幅等"异常波动"达标后，于收盘后公开披露的**买入与卖出金额前五名的营业部（席位）名单**。它是 A 股少数能看到真实大额成交去向的公开数据，常用于追踪游资动向、识别机构参与，但绝不能简单照搬"跟着龙虎榜买"。

读完这篇，你要能做到：

1. 说出龙虎榜的常见触发情形（涨跌幅、换手、振幅、连续涨跌）；
2. 区分机构专用、知名游资、沪深股通、普通营业部四类席位的含义；
3. 按五步法读一份龙虎榜，并算出上榜净买入与净买入率；
4. 理解披露滞后 + T+1 带来的跟单结构性劣势，识别接力、对倒、一日游陷阱。

## 一、定义与直觉

普通行情只告诉你"今天成交了多少",看不到"是谁在买卖"。龙虎榜则在满足触发条件后,把当日买入额最大的前五个席位、卖出额最大的前五个席位及其金额公开出来。

它能回答三个普通盘口回答不了的问题：

- 今天的暴涨/暴跌，是机构在动还是游资在炒？
- 买方是知名游资接力，还是北向/机构进场？
- 卖方前五是不是就是昨天买进来的那批人在出货？

直觉类比：龙虎榜像一场牌局结束后，亮出**下注最大的五个玩家**的名字和筹码。你看不到全部玩家，也看不到他们手里还剩多少牌，但能从"谁押得最重"推测牌桌的主导力量。

## 二、触发规则与披露内容

### 2.1 常见触发情形（以交易所规则为准，阈值会调整）

A 股龙虎榜由沪深交易所按"异常波动/严重异常波动"规则触发，常见情形包括（具体数值以交易所最新规定为准，下列为习惯性概括）：

| 触发维度 | 习惯性描述 |
|---|---|
| 日涨跌幅偏离 | 当日收盘价格涨跌幅达到异动标准（不同板块阈值不同） |
| 日换手率 | 换手率异常偏高 |
| 日振幅 | 当日振幅达到异动标准 |
| 连续涨跌 | 连续数日累计涨跌幅达标 |
| 特别关注 | ST、新股、退市整理及交易所重点监控品种另有更严披露 |

> [!warning] 板块差异很大
> 主板与创业板/科创板的涨跌停（±10% vs ±20%）和异动阈值不同，注册制板块的披露口径也更细。学习时以**交易所当期发布的实施细则**为准，不要把某一年的具体数字当永恒标准。

### 2.2 披露内容

- 买入前五席位名称及买入金额；
- 卖出前五席位名称及卖出金额；
- 由此可算出**上榜席位合计净买入** = 买五合计 − 卖五合计。

### 2.3 三种值得重点区分的"上榜结构"

同样是上榜，结构不同，含义天差地别：

| 上榜结构 | 特征 | 含义倾向 |
|---|---|---|
| 机构净买入 | 买方出现多个"机构专用" | 相对中长线认可，持续性可能更好 |
| 游资接力 | 买方为多个知名游资营业部 | 短线情绪博弈，波动大、周期短 |
| 买卖对倒 | 同类席位既在买一也在卖一 | 分歧大或自我做盘，需高度警惕 |

> [!tip] 先看"谁"，再看"多少"
> 龙虎榜分析的第一步永远是读席位性质（机构 / 游资 / 沪深股通 / 普通营业部），第二步才是看金额和净买入。脱离席位身份只看净买入数字，等于丢掉了龙虎榜最有价值的信息。

## 三、席位解读与案例

席位名称是龙虎榜的灵魂，不同席位含金量天差地别。

| 席位类型 | 名称特征 | 常见含义 |
|---|---|---|
| 知名游资营业部 | 某些常年活跃的特定营业部 | 短线炒作资金，擅长打板、接力，持股周期短 |
| 机构专用席位 | 显示为"机构专用" | 公募/险资/大型机构，通常资金体量大、周期相对长 |
| 沪股通/深股通席位 | 显示为"沪股通专用/深股通专用" | 经港股通通道的外资成交，被视作外资动向参考 |
| 普通营业部 | 一般券商营业部名称 | 信息含量较低，可能是分散散户或不愿暴露的资金 |

### 3.1 一个上榜解读算例（假设）

某股涨停上榜，席位数据简化如下（假设）：

| 方向 | 席位 | 金额（万元） |
|---|---|---|
| 买一 | 某知名游资营业部 | 8000 |
| 买二 | 机构专用 | 5000 |
| 买三~买五 | 普通营业部合计 | 4000 |
| 卖一 | 机构专用 | 6000 |
| 卖二~卖五 | 普通营业部合计 | 5000 |

- 买五合计 = 8000 + 5000 + 4000 = 17000 万（假设）
- 卖五合计 = 6000 + 5000 = 11000 万（假设）
- **上榜净买入 = +6000 万（假设）**

解读要点：买方有游资+机构进场，卖方却也有机构在出，呈现"机构对倒/分歧"格局。净买入为正只说明上榜席位偏买方，**不代表全场净买入为正**（前五之外的成交未知）。

### 3.2 净买入与次日表现的经验关系

经验上人们观察到一些**倾向**（非定律、无精确胜率）：

- 知名游资大额净买入、且当日封板较强的票，次日**情绪溢价**往往更高，但波动同样更大；
- 出现"机构专用"席位大额净买入，市场常解读为相对中长线认可，持续性可能更好；
- 若买一就是某游资、卖一也是同类游资，常被视为"一日游/接力博弈"，次日不确定性很高。

> [!warning] 不编造胜率
> 坊间流传"某游资上榜次日上涨概率 70%"之类数字大多缺乏稳健样本支撑，且会随市场环境、监管力度快速失效。本文不提供任何精确胜率，只描述方向性经验。

### 3.3 净买入率：把金额标准化

直接比较净买入金额对大小盘不公平，常把净买入对成交额做标准化：

$$
\text{上榜净买入率} = \frac{\text{买五合计} - \text{卖五合计}}{\text{当日总成交额}}
$$

净买入率越高，说明上榜大资金在当日成交中占比越重、控盘/参与程度越高。但同样地，**高净买入率也可能来自对倒放量**，需结合席位身份判断，不能只看比率。

### 3.4 一段游资接力的典型节奏（假设）

题材股的"接力"往往沿着这样的节奏推进（纯假设示意，用来理解结构而非预测）：

| 阶段 | 龙虎榜典型特征 | 风险位置 |
|---|---|---|
| 启动 | 某游资买一首板，卖方分散 | 相对低，但不确定能否连板 |
| 发酵 | 多家知名游资同时上榜接力 | 中，情绪溢价快速抬升 |
| 高潮 | 买卖前五大量重叠、净买入率仍高 | 高，分歧加剧、随时见顶 |
| 退潮 | 买一变卖一，前排游资转为净卖出 | 极高，最后接力者被埋 |

关键认知：**散户因为披露滞后+T+1，最早能跟进的时点通常已在"发酵—高潮"之间**，恰恰是风险快速上升的区域；而启动阶段的低风险位你看不到、进不去。这就是龙虎榜跟单的结构性劣势。

## 四、实战应用与交易规则

龙虎榜更适合做**事后验证与风格识别**，而不是机械跟单。

读一份龙虎榜，建议按固定五步走，避免一眼只盯净买入数字：

1. **看触发原因**：涨幅、换手还是振幅？是连板还是单日异动？
2. **看买方席位性质**：机构专用 / 知名游资 / 沪深股通 / 普通营业部，先定身份。
3. **看卖方席位**：卖一是不是昨天的买方？机构在不在出货？
4. **算净买入与净买入率**：占当日成交多少，参与度高不高。
5. **判结构**：机构净买入、游资接力，还是买卖对倒/分歧？

**用法一：识别资金性质。** 同样是涨停，机构专用进场与纯游资接力，含义完全不同。前者可关注基本面催化，后者更要按情绪周期对待。

**用法二：观察接力与分歧。** 买卖前五若高度重叠（同类游资互为对手盘），说明分歧大、博弈重；买方集中而卖方分散，筹码相对锁定。

**用法三：配合资金流向与两融。** 龙虎榜（真实席位）+ [[资金流向]]（估算情绪）+ [[融资融券数据]]（杠杆温度）三者交叉，比单看龙虎榜更可靠。

结合 A 股制度，跟庄必须正视这些约束：

- **T+1**：龙虎榜收盘后才出，你看到时已无法当日跟进，最早次日开盘介入，而次日常高开，**你的成本天然高于上榜资金**。
- **涨跌停 + 打板**：游资多在涨停板买入，散户次日若追高开很容易买在情绪高点；一旦次日不及预期，T+1 让你无法当天止损。
- **印花税与佣金**：高频跟龙虎榜做短线，卖出 0.05% 印花税加佣金，摩擦成本侵蚀收益。
- **流动性**：小盘题材股是游资主战场，进出对价格冲击大，散户难以按理想价格成交。

> [!example] 把龙虎榜写成一条规则（示例）
> "只把龙虎榜用于判断资金性质：若上榜为机构专用净买入且公司有明确基本面催化，纳入研究池观察；若为纯游资接力的一日游格局，不参与追板。" —— 强调用它筛选，而非用它发出买入指令。

## 五、常见误区与失效条件

| 误区 | 正确理解 |
|---|---|
| 跟着龙虎榜买入就能赚 | 你最早次日才能进，成本高于上榜资金，且 T+1 无法当日止损 |
| 上榜净买入为正=全场在买 | 只反映前五席位，前五之外的真实净额未知 |
| 知名游资买了就一定涨 | 游资擅长短线博弈与对倒，可能正在借势出货 |
| 机构专用一定是长线利好 | 机构也会做波段、打新解禁、对倒，需结合方向与持续性 |
| 沪深股通席位=聪明钱必赢 | 通道资金构成复杂，含被动指数与短线，不等于稳赚 |
| 龙虎榜数字可机械量化套用 | 触发阈值、监管力度、风格会变，历史规律样本外易失效 |

> [!warning] 三个典型陷阱
> **游资接力博弈**：上一棒拉板吸引下一棒，链条一旦断裂，最后接力者深套。
> **对倒做盘**：同一资金多席位自买自卖，制造"大资金抢筹"假象诱多。
> **一日游**：当天涨停上榜，次日高开后快速派发，跟风盘高位被埋。

## 六、复盘要点

把龙虎榜纳入复盘时，固定核对几个问题：

- [ ] 上榜是因为涨幅、换手还是振幅触发？触发原因不同，含义不同。
- [ ] 买卖前五各是什么性质的席位？机构、游资、沪深股通还是普通营业部？
- [ ] 上榜净买入为正，但它只占总成交多少？是否只是前五的局部结论？
- [ ] 买卖席位是否高度重叠（对倒/分歧）？是不是典型一日游结构？
- [ ] 如果我次日才能进场，开盘价相对上榜资金成本高出多少？这笔"迟到成本"我承受得起吗？

记住：龙虎榜是 A 股少有的**真实大额成交去向**数据，价值在于识别资金性质、辅助理解一段行情**是谁在主导**；但因为披露滞后 + T+1，它几乎不能用来"照单跟买"。把它当显微镜，不要当发令枪。
