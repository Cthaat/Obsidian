# 情绪因子

> [!note] 核心定位
> 情绪因子 属于因子投资和系统化选股框架。它的核心不是“指标看起来有效”，而是确认收益来源、数据口径、检验方法、成本、容量和衰减风险。

## 一、核心定义

情绪因子 需要被看作一个可检验的收益来源。它必须回答：指标如何计算、为什么应获得收益、在什么股票池有效、是否需要行业/市值中性化、扣除成本后是否还能落地。

## 二、研究流程

标准流程是：定义因子 -> 数据清洗 -> 去极值和标准化 -> 中性化 -> IC/Rank IC 检验 -> 分层收益 -> 换手和成本 -> 样本外 -> 组合权重 -> 上线监控。任何跳过成本和容量的因子研究都不完整。

## 三、风险与失效

因子可能失效于拥挤交易、风格轮动、市场制度变化、数据泄漏、行业暴露伪装成alpha、参数过拟合。特别要警惕多个相似因子叠加后实际押注同一个风险。

## 四、实战检查清单

- 是否有经济解释
- 是否能在交易时点获得数据
- 是否做了行业和市值中性化
- IC和分层收益是否稳定
- 换手和容量是否可承受
- 因子回撤时如何降权或暂停

## 六、研究 `情绪因子` 时必须验证什么

因子类主题最容易出现“看起来专业但不能交易”的问题。真正的研究必须把指标、经济解释、统计检验和实盘约束放在一起。

| 维度 | 检查问题 |
|---|---|
| 定义 | 指标公式是否唯一，字段和时间戳是否清楚 |
| 解释 | 收益来自风险补偿、行为偏差、流动性，还是数据挖掘 |
| 稳定性 | IC、Rank IC、分层收益是否跨时期稳定 |
| 风险 | 是否隐藏行业、市值、流动性或beta暴露 |
| 成本 | 换手、滑点、冲击成本是否会吞噬收益 |
| 容量 | 资金规模扩大后是否还能成交 |
| 衰减 | 因子公开或拥挤后收益是否下降 |


因子研究的结果不应只是“有效/无效”，而应说明它适合进入哪类组合、承担什么风险、在什么环境下需要降权。

## 八、从研究报告到组合权重

`情绪因子` 进入组合前，需要从“研究有效”转成“仓位合理”。一个因子即使长期有效，也可能因为波动大、换手高、容量小或与已有持仓相关性高而不适合重仓。

| 决策 | 需要的信息 |
|---|---|
| 是否使用 | 经济解释、IC、分层收益、样本外表现 |
| 用多大权重 | 因子波动、回撤、相关性、拥挤度 |
| 多久调仓 | 信号半衰期、交易成本、税费和滑点 |
| 何时降权 | IC下降、分层失效、成本上升、拥挤加剧 |

## 九、复盘维度

```text
本期 `情绪因子` 的收益贡献：
IC / Rank IC 是否稳定：
高分组是否仍跑赢低分组：
换手是否升高：
```

因子投资不是找一次答案，而是持续监控收益来源是否还存在。

## 深度案例与训练

### 因子研究项目

围绕 情绪因子 做一个最小研究项目：定义因子公式，确定股票池，处理极值和缺失值，做行业市值中性化，然后观察 IC、Rank IC、分层收益、换手率和最大回撤。

### 质量检查

- 因子方向必须有经济解释，不能只因为历史收益好看。
- 分层收益要有单调性，且不能只靠极端一组贡献。
- 组合收益要扣除交易成本，并检查容量和拥挤度。

### 失效跟踪

每月记录因子收益、波动、相关性、行业暴露和持仓重合度。如果因子收益下降，要区分是经济逻辑失效、交易拥挤、市场风格切换，还是数据处理错误。

## 最小可执行项目

### 单因子研究卡

围绕 情绪因子 完成一张单因子研究卡：公式、方向、股票池、数据频率、去极值方法、标准化方法、中性化变量、IC、分层收益、换手、成本、容量和失效条件。

| 环节 | 验收标准 |
|---|---|
| 定义 | 公式清晰，可复现 |
| 解释 | 有风险补偿、行为偏差或制度逻辑 |
| 检验 | 样本外仍有稳定性 |
| 交易 | 扣费后仍有收益空间 |
| 监控 | 能观察拥挤和衰减 |

### 常见失败

因子看似有效，实际只是在复制行业、市值或流动性暴露；或者换手太高，实盘成本吃掉全部收益。
