# 锚定效应

> [!note] 核心定义
> 锚定效应是指人在做判断时，会过度依赖最先接触到的某个数值（"锚"），并以它为基准做不充分的调整。在投资中，买入成本价、历史高点、整数关口等都会成为锚，悄悄扭曲你的买卖决策——这是亏损死扛、回本就跑等行为的深层心理根源之一。

## 一、定义与直觉

Tversky 与 Kahneman 在 1970 年代的经典实验中发现：先让人看一个随机数字（比如转盘转出的数），再问"非洲国家占联合国比例是多少"，看到大数字的人估得明显更高——尽管那个数字与问题毫无关系。这说明**一旦有了参照锚，后续判断就会被它"拽住"，调整往往不足。**

投资里这一点尤其危险，因为我们身边到处是现成的锚：

- 它给你一个虚假的"参照点"，让你觉得有了依据；
- 而真正该依据的是公司的**内在价值**，与你的成本、历史价格毫无关系——市场不知道也不在乎你的成本价。

> [!tip] 一句话记忆
> 你的买入成本价是全市场最不重要的数字之一——只有你自己在乎它。但恰恰是它，最容易变成左右你卖不卖的锚。

## 二、投资中常见的"锚"

| 锚 | 表现 |
|---|---|
| 买入成本价 | "等回本了就卖""跌破成本不卖"——以成本而非价值决定买卖 |
| 历史高/低点 | "从 100 跌到 50，便宜一半"——但 50 可能仍然高估 |
| 52 周高/低 | 接近 52 周低点就觉得"跌够了"，接近高点就不敢买 |
| 整数关口 | 3000 点、100 元等心理整数被当成天然支撑/阻力 |
| 分析师目标价 | 把别人给的目标价当锚，忽视假设是否成立 |
| IPO 发行价 / 增发价 | "破发了就便宜"——发行价本身可能定得过高 |

## 三、锚定如何扭曲决策

1. **以成本价定买卖**：最普遍。盈利时一接近成本上方就急于"落袋为安"（怕利润缩回去），亏损时死守成本价不止损（不愿把账面亏损变成实亏）。这与 [[损失厌恶]] 和 [[处置效应]] 互相强化，导致典型的"卖盈持亏"。
2. **估值被锚带偏**：先看到某个 PE 或目标价，后续分析不自觉向它靠拢，丧失独立判断。
3. **止损位被扭曲**：止损本应设在"逻辑被证伪"或"技术结构破坏"处，却被设在成本价附近这种无意义的锚上。
4. **对新信息反应不足**：基本面已经恶化，但因为锚定在过去的"好印象"或高价，调整太慢，迟迟不肯下调预期。

> [!warning] 锚定 + 损失厌恶 = 死扛
> 单独的锚定只是判断偏差，但叠加损失厌恶就会变成实质性亏损：成本价这个锚 + "不愿认亏"的痛苦，共同让人把"等回本"当成策略。市场从不保证回本，许多退市股、长期下跌股就是这样被一路扛到深套。

## 四、应对方法

- **用内在价值替代成本价**：每次做卖出决策时强迫自己回答——"如果我今天没有持仓、以现价重新评估，我会买入吗？"答案为否就该卖，与成本无关。
- **多锚交叉验证**：不依赖单一参照，同时看历史分位、同业对比、绝对估值（DCF）、安全边际，让任何单一锚的影响被稀释。
- **贝叶斯式更新**：把基本面新信息当作更新先验的证据，主动问"这条新消息应该让我把估值调高还是调低、调多少"，对抗"调整不足"。
- **刻意"无锚"复核**：定期假装自己第一次看这只股票，用一张空白纸重新估值，再和当前判断对比，看锚把你带偏了多少。
- **预设规则前置**：在买入前（情绪中性时）就写好止损、止盈、加减仓规则，避免事后被成本锚牵着走。

## 五、常见误区与失效条件

| 误区 | 正确理解 |
|---|---|
| "跌破成本不卖，等回本" | 市场不认你的成本；该不该卖只取决于现价相对价值是否仍高估 |
| "从高点腰斩了，很便宜" | 历史高点是无意义的锚，腰斩后可能仍然贵，要看绝对估值 |
| "分析师目标价 X 元，还有空间" | 目标价依赖一串假设，应自己验证假设而非锚定结论 |
| "整数关口必然支撑/阻力" | 整数关口只是心理锚，没有基本面意义，常被轻易击穿 |
| 以为自己理性、不受锚影响 | 锚定是无意识的，越自信不受影响越容易中招，需用规则强制对冲 |

> [!note] 失效条件小结
> 锚定在以下情形危害最大：① 持仓深套时用成本价做决策；② 高波动、信息快速变化的市场里调整不足；③ 重大利空已出但仍锚定过去的好印象。破解的核心只有一句：决策依据是"现价 vs 价值"，永远不是"现价 vs 你的成本或某个历史数字"。
