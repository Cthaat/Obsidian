# 低波动策略

> [!note] 核心定位
> 低波动策略 属于因子投资和系统化选股框架。它的核心不是“指标看起来有效”，而是确认收益来源、数据口径、检验方法、成本、容量和衰减风险。

## 一、核心定义

低波动策略 需要被看作一个可检验的收益来源。它必须回答：指标如何计算、为什么应获得收益、在什么股票池有效、是否需要行业/市值中性化、扣除成本后是否还能落地。

## 二、研究流程

标准流程是：定义因子 -> 数据清洗 -> 去极值和标准化 -> 中性化 -> IC/Rank IC 检验 -> 分层收益 -> 换手和成本 -> 样本外 -> 组合权重 -> 上线监控。任何跳过成本和容量的因子研究都不完整。

## 三、风险与失效

因子可能失效于拥挤交易、风格轮动、市场制度变化、数据泄漏、行业暴露伪装成alpha、参数过拟合。特别要警惕多个相似因子叠加后实际押注同一个风险。

## 四、实战检查清单

- 是否有经济解释
- 是否能在交易时点获得数据
- 是否做了行业和市值中性化
- IC和分层收益是否稳定
- 换手和容量是否可承受
- 因子回撤时如何降权或暂停

## 六、研究 `低波动策略` 时必须验证什么

因子类主题最容易出现“看起来专业但不能交易”的问题。真正的研究必须把指标、经济解释、统计检验和实盘约束放在一起。

| 维度 | 检查问题 |
|---|---|
| 定义 | 指标公式是否唯一，字段和时间戳是否清楚 |
| 解释 | 收益来自风险补偿、行为偏差、流动性，还是数据挖掘 |
| 稳定性 | IC、Rank IC、分层收益是否跨时期稳定 |
| 风险 | 是否隐藏行业、市值、流动性或beta暴露 |
| 成本 | 换手、滑点、冲击成本是否会吞噬收益 |
| 容量 | 资金规模扩大后是否还能成交 |
| 衰减 | 因子公开或拥挤后收益是否下降 |

## 七、实战落地

```text
因子名称：低波动策略
预期方向：

股票池：
中性化变量：行业 / 市值 / beta / 其他

成本假设：
上线权重：
降权或停用条件：
```

因子研究的结果不应只是“有效/无效”，而应说明它适合进入哪类组合、承担什么风险、在什么环境下需要降权。

## 八、从研究报告到组合权重

`低波动策略` 进入组合前，需要从“研究有效”转成“仓位合理”。一个因子即使长期有效，也可能因为波动大、换手高、容量小或与已有持仓相关性高而不适合重仓。

| 决策 | 需要的信息 |
|---|---|
| 是否使用 | 经济解释、IC、分层收益、样本外表现 |
| 用多大权重 | 因子波动、回撤、相关性、拥挤度 |
| 多久调仓 | 信号半衰期、交易成本、税费和滑点 |
| 何时降权 | IC下降、分层失效、成本上升、拥挤加剧 |

## 九、复盘维度

```text
本期 `低波动策略` 的收益贡献：
IC / Rank IC 是否稳定：
高分组是否仍跑赢低分组：
换手是否升高：
行业或市值暴露是否异常：
是否需要降权、暂停或重新研究：
```

因子投资不是找一次答案，而是持续监控收益来源是否还存在。

## 深度案例与训练

### 策略拆解

学习 低波动策略 时，写清楚策略的六个零件：信号、过滤器、入场、退出、仓位和风控。任何说不清这六项的策略，都还只是想法。

### 验证路径

1. 用最简单规则建立基准收益。
2. 加入手续费、滑点、冲击成本和容量约束。
3. 做样本外、滚动窗口和参数扰动。
4. 统计收益来自哪些时期、哪些品种和哪些市场状态。

### 实盘监控

上线后关注信号命中率、成交质量、回撤、换手、资金占用和异常订单。策略失效时优先降仓或暂停，而不是继续调参寻找安慰。

## 最小可执行项目

### 策略一页纸

围绕 低波动策略 写一张策略一页纸：收益来源、适用品种、信号、过滤器、入场、退出、仓位、风控、成本、容量、暂停条件和复盘指标。

| 检验 | 合格标准 |
|---|---|
| 逻辑 | 能解释为什么会有收益 |
| 回测 | 有样本外和参数扰动 |
| 成本 | 扣除手续费、滑点和冲击成本 |
| 风险 | 有最大回撤和连续亏损预案 |
| 实盘 | 有监控、降仓和暂停规则 |

### 常见失败

策略在报告中赚钱，却在实盘中输给交易成本、容量限制、信号衰减和执行错误。
