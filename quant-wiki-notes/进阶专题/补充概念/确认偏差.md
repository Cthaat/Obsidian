# 确认偏差

> [!note] 核心定义
> 确认偏差（Confirmation Bias）指人倾向于选择性地寻找、解读和记忆那些**支持已有观点**的信息，同时忽视、淡化或扭曲反对它的证据。对交易者而言，它让你只看见自己想看见的，把一笔已经被证伪的交易死死攥在手里。

## 一、定义与直觉

人脑天生不是为了"求真"，而是为了"省力地维护一个连贯的世界观"。一旦你形成某个判断——"这只票要涨""这个行业是未来"——大脑就会进入**辩护模式**而非**检验模式**：

- 在搜集信息时，你下意识只去找支持自己的材料（只翻看多的研报、只关注利好新闻）。
- 在解读信息时，对支持证据照单全收，对反对证据吹毛求疵（"这个数据有水分""空头别有用心"）。
- 在记忆信息时，记住了应验的预测，忘掉了落空的那些。

通俗类比：你买了某品牌的车，之后会感觉满大街都是这款车——其实数量没变，是你的注意力被"已有选择"过滤了。投资里同样如此：持有一只票之后，你会觉得"利好"越来越多，因为你只在看利好。

确认偏差与过度自信互为表里：[[过度自信]] 让你坚信自己的判断，确认偏差则负责不断给这份坚信"喂料"、屏蔽反驳，于是错误判断也能维持很久。

## 二、投资中的典型表现

| 表现 | 具体行为 | 后果 |
|---|---|---|
| 选择性看多/看空 | 只读支持自己方向的研报、雪球帖、大 V 观点 | 信息输入从一开始就偏 |
| 双重标准 | 利好证据零门槛采信，利空证据百般挑刺 | 证据天平被人为压偏 |
| 回音室 / 抱团取暖 | 只待在观点一致的群和社区，互相强化 | 听不到任何有效反方 |
| 选择性记忆 | 记住"我当初看对了"，忘记看错的次数 | 高估自己的判断力 |
| 屁股决定脑袋 | 一旦持仓，立场从"判断者"变成"辩护人" | 客观性随仓位一起消失 |

### "屁股决定脑袋"与沉没成本

这是确认偏差在持仓后最危险的变形。**建仓前**你或许还能两面权衡；**建仓后**，承认看错等于承认自己愚蠢且要兑现亏损，于是大脑自动切换成持仓的辩护律师：所有利空都被解释成"短期扰动""错杀""黄金坑"。

它常与**沉没成本谬误**叠加：已经亏了 20%、已经研究了三个月、已经公开喊过单——这些"投入"本应与"现在是否该持有"无关，却让人为了证明先前的投入正确而继续加码。两者合流，就是散户最常见的深套路径：**越跌越确信，越确信越补仓，越补仓越下不来。**

> [!warning] 公开表态会锁死你
> 一旦你在群里、社交媒体公开喊了某只票，确认偏差会因为"维护面子和一致性"急剧增强。从此你不再是为账户做决策，而是为"证明当初没说错"做决策。建仓可以低调，复盘必须诚实。

## 三、为什么它在市场里格外致命

- **市场会奖励纠错、惩罚固执**：价格是无数人的合力，不在乎你的立场。证据变了却不改判断，亏损会持续累积。
- **它制造"虚假印证"**：在上涨趋势里，只看多的人会觉得"我每次都对"，自信被喂大（连回 [[过度自信]]）；趋势一旦反转，毫无防备。
- **它让止损失效**：价格止损可以机械执行，但**逻辑止损**（买入理由被证伪就走）依赖你诚实承认证据变了——而这正是确认偏差最擅长阻止的事。
- **它助推群体行为**：当一群人各自屏蔽反方、互相印证，就形成 [[羊群行为]] 和抱团，为泡沫提供心理燃料。

## 四、应对：主动给自己找麻烦

确认偏差的解药不是"提醒自己客观"（你会觉得自己已经很客观），而是**把证伪做成强制流程**。

### 1. 主动写空头论点（红队）

买入前，强制自己写出**至少三条"我为什么可能是错的"**：最强的看空逻辑是什么？做空的人在想什么？如果写不出有力的反方，说明你研究得还不够，或者已经陷在自己的叙事里。

### 2. 预设证伪条件

建仓时就白纸黑字写下：**"出现什么情况，我就承认看错并离场。"** 例如——某季度营收增速跌破 X%、毛利率连续两季下滑、核心客户流失、关键政策反转。把抽象的"逻辑被破坏"翻译成**具体、可观测的触发器**。这样卖出就不再依赖当时的心理斗争，而是事先约定的纪律。

### 3. 决策前写下理由，日后复盘

建仓当天写下完整买入逻辑和当时掌握的信息，**封存**。几个月后拿出来对照：当初的关键假设兑现了吗？哪些证据其实当时就有、却被你忽略了？这能直接对抗选择性记忆——白纸黑字让你无法事后篡改"我当初的想法"。

### 4. 找最聪明的反方，而不是最弱的

辩论时人爱挑对手最差的论点来驳（稻草人）。要刻意去找**最有说服力的反对意见**：去读专门做空这家公司的深度报告，去和水平最高、立场相反的人聊。能正面回应最强反方而判断依然成立，结论才扎实。

### 5. 结构化对抗信息偏食

- 给每个重仓建一个"**反方证据清单**"，主动收集利空，与利好并列。
- 离开回音室：刻意关注与你立场相反的优质账户。
- 用 [[综合金融知识体系/投资政策书IPS与交易计划模板|交易计划清单]] 把"我看了哪些反方证据"做成必填项。

> [!tip] 区分"立场"和"判断"
> 健康的做法是把仓位和自我价值剥离开：看错一笔交易，不代表你这个人差，只代表这条假设被证伪了。承认错误越廉价，纠错越快，确认偏差的破坏力就越小。

## 五、常见误区与失效条件

| 误区 | 正确理解 |
|---|---|
| 我看了很多资料，所以判断很客观 | 关键不是看了多少，而是看的是不是**只有一边**；信息偏食越多越自信也越偏 |
| 利空都是主力洗盘/空头唱衰，不必理会 | 这正是双重标准的典型；利空和利好要用**同一把尺子**检验真伪 |
| 反正都套了，跌了再补摊低成本 | 这是确认偏差叠加沉没成本；该问的是"以今天的信息，我还会买吗"，而非历史成本 |
| 找个反方意见走个过场就行 | 走过场只是自我安慰；要找**最强**反方并能正面回应，才算真检验 |
| 我已经公开看多，改口太丢人 | 为面子持仓是拿账户补贴自尊；市场不在乎你的面子 |
| 知道有确认偏差，我就能避开 | 知道≠免疫；它对"自认客观"者照样起作用，只能靠预设证伪条件、封存理由、红队等流程对冲 |

> [!example] 证伪条件模板（假设）
> 持有某股，建仓时写明触发离场的三条硬指标：① 下季度营收同比增速 < 10%；② 毛利率较当前下滑超过 3 个百分点；③ 第一大客户订单占比明显下降。任一条出现即按计划减仓或清仓，不再临场找理由。数字为假设，目的是把"逻辑止损"变成可执行规则。
