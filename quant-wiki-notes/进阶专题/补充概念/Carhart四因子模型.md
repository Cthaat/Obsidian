# Carhart四因子模型

> [!note] 核心定义
> Carhart 四因子模型是在 Fama-French 三因子（市场、规模、价值）基础上，再加入一个**动量因子（UMD/WML）**的资产定价模型。它把一只股票或基金的超额收益拆成四个系统性来源加上一个真正的 α；交易者懂它，才能分清一个组合赚的是"承担了某种风险/暴露"，还是基金经理"真本事"。

## 一、定义与直觉

CAPM 只用一个市场因子解释股票收益，但实证里它解释力有限：小盘股、低估值股长期跑赢市场，CAPM 解释不了。Fama 和 French 在 1990 年代提出**三因子模型**，加入规模因子（SMB）和价值因子（HML），大幅提升了横截面解释力。

但三因子还留了一个明显的"窟窿"：**动量**。Jegadeesh 与 Titman 发现，过去 6–12 个月涨得多的股票，未来一段时间倾向于继续跑赢过去跌得多的股票。这个现象三因子模型解释不了——很多被三因子判为"有 α"的基金，其实只是踩中了动量。

Carhart 在 1997 年研究共同基金业绩时，把动量因子（记作 **UMD**，Up-minus-Down；也常写作 **WML**，Winners-minus-Losers，或 **MOM**）补进模型，形成四因子框架。它的核心用途不是"预测某只股票涨跌"，而是**业绩归因**：把一个组合的收益分解到四条公认的收益来源上，剩下解释不了的那部分，才是真正意义上的 α。

一句类比：四因子就像给基金收益做"成分体检"。市场、规模、价值、动量是四种已知"营养成分"，先把它们扣掉，剩下的才是基金经理独有的贡献。如果一个"明星基金"的收益几乎全被四个因子吃掉，那它的高收益只是因子暴露，换成低费率的因子 ETF 也能复制。

## 二、原理与公式

### 2.1 四因子回归式

对资产或组合 $i$，用时间序列回归估计它对四个因子的敏感度：

$$R_i-R_f=\alpha_i+\beta_{MKT}\,MKT+\beta_{SMB}\,SMB+\beta_{HML}\,HML+\beta_{UMD}\,UMD+\epsilon_i$$

符号含义：

- $R_i-R_f$：资产 $i$ 的**超额收益**（收益减去无风险利率 $R_f$，如国债收益率）。
- $\alpha_i$：截距，即四个因子都解释不了的部分，是衡量"真本事"的核心。
- $MKT=R_m-R_f$：市场因子，整个市场组合相对无风险利率的超额收益。
- $SMB$（Small Minus Big）：规模因子，小市值股票组合收益减去大市值股票组合收益。
- $HML$（High Minus Low）：价值因子，高账面市值比（高 B/M，即"便宜"）股票收益减去低账面市值比（"贵"）股票收益。
- $UMD$（Up Minus Down）：动量因子，过去赢家组合收益减去过去输家组合收益。
- $\beta_{MKT},\beta_{SMB},\beta_{HML},\beta_{UMD}$：四个因子的载荷（敏感度/暴露），由回归估计。
- $\epsilon_i$：残差，均值为 0 的随机扰动。

### 2.2 各因子怎么构造（多空组合直觉）

四个因子里，**除市场因子外，其余三个都是"多空对冲组合"的收益**：做多一类股票、做空对立的一类，理论上不花本金、剥离市场涨跌，只留下该特征带来的收益差。

| 因子 | 全称 | 多头（买） | 空头（卖） | 想捕捉的收益来源 |
|---|---|---|---|---|
| MKT | 市场超额收益 | 全市场组合 | 无风险资产 | 承担系统性市场风险的补偿 |
| SMB | 规模 | 小市值股票 | 大市值股票 | 小盘相对大盘的长期溢价 |
| HML | 价值 | 高 B/M（便宜） | 低 B/M（贵） | 价值股相对成长股的溢价 |
| UMD | 动量 | 过去 12 个月赢家 | 过去 12 个月输家 | 价格趋势延续 |

> [!tip] 动量因子的标准构造（经典做法）
> 排序窗口常取**过去 12 个月、跳过最近 1 个月**（写作 "12-1"，跳过最近 1 个月是为了避开短期反转效应），按累计收益分成赢家组和输家组，做多赢家、做空输家，通常**每月再平衡**一次。正因为月度调仓，动量因子换手率很高，交易成本是它落地时绕不开的难题。

### 2.3 为什么动量要单独列一个因子

把动量单列，根本原因是它**和其他三个因子相关性低、且无法被它们解释**：

- 动量与价值常常**负相关**——价值买的是已经跌下来的便宜货，动量买的是正在上涨的赢家，两者方向时常相反。
- 三因子回归基金收益时，残差里仍能看到明显的动量痕迹：很多基金的"超额收益"其实来自持续持有上涨股票。
- 不单列动量，就会把"踩中趋势"错记成基金经理的 α，归因失真。

### 2.4 怎么读懂四个因子载荷

回归出来的 β 不是抽象数字，每一个都对应一种可解释的风格画像：

| 载荷 | 取值 | 含义 |
|---|---|---|
| $\beta_{MKT}$ | > 1 / < 1 | 比市场更激进 / 更防御 |
| $\beta_{SMB}$ | > 0 / < 0 | 偏小盘 / 偏大盘 |
| $\beta_{HML}$ | > 0 / < 0 | 偏价值（买便宜） / 偏成长（买贵） |
| $\beta_{UMD}$ | > 0 / < 0 | 顺势追涨（动量） / 逆势抄底（反转） |

把四个 β 连起来读，就得到一只基金的"风格指纹"。例如"$\beta_{SMB}>0,\ \beta_{HML}<0,\ \beta_{UMD}>0$"画出的是一只**偏小盘、偏成长、强动量**的成长型基金；而"$\beta_{SMB}<0,\ \beta_{HML}>0,\ \beta_{UMD}<0$"则是**偏大盘、偏价值、带反转**的深度价值风格。这正是四因子模型最实用的地方——不必听基金的自我宣传，回归系数会说真话。

## 三、计算 / 参数 / 案例

### 3.1 业绩归因算例

假设对某只主动股票基金做四因子回归，得到下表结果（**全部为假设数据，仅作演示**）：

| 项目 | 估计值 | 解读 |
|---|---:|---|
| $\alpha_i$（月度） | +0.05% | 月均约 +0.6%/年，统计上不显著 |
| $\beta_{MKT}$ | 1.05 | 略高于市场，进攻性略强 |
| $\beta_{SMB}$ | 0.40 | 明显偏向小盘 |
| $\beta_{HML}$ | -0.30 | 偏成长（做空了价值） |
| $\beta_{UMD}$ | 0.55 | 重度暴露动量，明显追涨 |
| $R^2$ | 0.93 | 四因子已解释 93% 的收益波动 |

读法：这只基金过去的好业绩，**主要由"偏小盘 + 偏成长 + 强动量"三重因子暴露贡献**，扣掉因子后 α 几乎为零且不显著。结论：它的超额收益更像"风格暴露"而非选股能力，用低费率的小盘/动量 ETF 组合大概率能复制，没必要为它支付高额主动管理费。

### 3.2 用四因子预期收益做粗略分解

把因子载荷乘以各因子的长期平均收益，可估算"由因子暴露贡献的预期超额收益"。设某组合 $\beta_{MKT}=1.0,\ \beta_{SMB}=0.3,\ \beta_{HML}=0.2,\ \beta_{UMD}=0.4$，假设各因子年化平均收益为（**均为假设**）市场 6%、SMB 2%、HML 3%、UMD 5%：

$$E[R-R_f]\approx 1.0\times6\%+0.3\times2\%+0.2\times3\%+0.4\times5\%=6\%+0.6\%+0.6\%+2.0\%=9.2\%$$

也就是说，光靠因子暴露就预期能拿到约 9.2% 的年化超额收益。基金若实际只跑出 9% 左右，α 基本为零。

> [!warning] 因子收益不是常数
> 上面的"因子长期平均收益"只是历史均值，**会随时段大幅波动甚至转负**。价值因子在 2010 年代很长一段时间表现疲弱，动量因子则会在市场急转时遭遇"动量崩溃"。用历史均值外推未来，必须保留充分的不确定性。

## 四、实战应用与使用规则

四因子模型在实务中**主要是分析与归因工具，而非直接下单的择时信号**。常见用法：

1. **基金/策略尽职调查**：跑四因子回归，看 α 是否显著为正、$R^2$ 有多高。$R^2$ 越高、α 越接近 0，说明该产品越像"因子贴现包"，越该用费率而非"明星光环"来评判。
2. **识别风格漂移**：定期滚动回归，观察 $\beta_{SMB}/\beta_{HML}/\beta_{UMD}$ 是否漂移。一只号称"价值"的基金若 $\beta_{HML}$ 转负、$\beta_{UMD}$ 飙升，说明它偷偷在追涨。
3. **组合风险管理**：把组合的四个 β 加总，判断整体暴露是否符合预期，避免无意中在某个因子上过度集中。
4. **构造因子组合**：若想主动获取动量收益，可按 UMD 构造法选股，但必须把交易成本前置。

落到 **A股**，有几处制度特征必须考虑：

- **T+1 与动量换手**：动量因子需月度调仓、换手率高，A股 T+1 无法日内回转，调仓只能次日生效，实际成交价与信号价存在滑点。
- **涨跌停**：动量组合的"赢家"常处于强势甚至涨停状态，涨停时**买不进**、跌停时**卖不出**，会系统性削弱动量策略可实现收益，这是 A股 动量落地的核心摩擦。
- **印花税与佣金**：A股 卖出单边征收印花税，叠加佣金，高换手的动量策略对成本极其敏感，回测必须扣费。
- **小盘流动性与"壳"逻辑**：SMB 在 A股 长期显著，但小盘股容量有限、流动性差，且历史上受"炒小炒壳""退市新规"等政策扰动，因子稳定性不及成熟市场，需谨慎对待。
- **因子在地化**：直接套用海外的 SMB/HML/UMD 序列并不严谨，A股 应使用本地股票池、本地财务与价格数据自行构造因子。

> [!example] 一条可复盘的研究规则
> "评估任何主动股票基金前，先做四因子回归：若 α 不显著且 $R^2>0.9$，默认它是因子暴露的打包产品，按因子 ETF 的费率水平给它定价，不为历史排名支付溢价。"

## 五、局限与常见误区

四因子模型的主要局限：

- **动量崩溃（momentum crash）**：在市场由跌转涨的剧烈反转期（如熊市见底反弹），过去的"输家"暴力反弹，动量空头端巨亏，UMD 因子可能在短期内出现极端回撤。动量收益呈现"平时小赚、偶尔巨亏"的负偏态。
- **交易成本高**：动量月度调仓，换手率远高于价值，扣除交易成本后的净收益会大幅缩水，在 A股 涨跌停与印花税环境下尤甚。
- **因子并非永远有效**：价值、规模、动量都经历过多年低迷甚至失效的时段，模型基于历史，不保证未来。
- **缺失的维度**：四因子之后，学界又补充了盈利能力、投资风格等因子（Fama-French 五因子），说明四因子并非"终极模型"，对某些资产仍有解释缺口。

| 误区 | 正确理解 |
|---|---|
| 四因子模型能预测个股涨跌 | 它主要是**归因/解释**工具，回答"收益来自哪里"，不是择时信号 |
| α 为正就一定是基金经理有本事 | 必须看**统计显著性**和样本长度；样本短、运气成分大时正 α 可能只是噪音 |
| 动量和价值可以无脑叠加 | 两者常**负相关**，简单叠加可能互相抵消，需在组合层面统筹 |
| 动量因子收益高所以风险低 | 动量是**负偏态**收益，存在动量崩溃尾部风险，高均值掩盖了极端回撤 |
| 海外的因子序列能直接用在 A股 | 因子需**用本地数据重构**，制度（T+1、涨跌停、印花税）会改变可实现收益 |
| $R^2$ 高是模型的优点、与基金好坏无关 | 对基金而言，$R^2$ 高恰恰说明它**像因子打包产品**，越该用低费率替代 |
