# 保守偏差

> [!note] 核心定位
> 保守偏差 是行为金融中的典型心理偏差。它的重要性不在于给错误贴标签，而在于帮助交易者提前设计阻断规则，避免在持仓、加仓、止损和复盘时被情绪接管。

## 一、核心含义

保守偏差 会让交易者在不知不觉中扭曲信息处理：该承认错误时拖延，该客观看待证据时选择性解释，该控制仓位时让情绪扩大风险。它通常不会单独出现，而是和过度自信、损失厌恶、确认偏差、羊群行为一起影响交易决策。

## 二、交易中的典型表现

常见表现包括：只看支持持仓的信息；亏损后把短线交易改成长期投资；因为已经投入研究时间而不愿离场；对已经拥有的资产给出过高价值；在群体共识中放弃独立判断。

## 三、阻断规则

有效方法不是提醒自己“要理性”，而是写入流程：买入前写反方证据，建仓时写证伪条件，持仓中定期检查事实是否变化，亏损后按预案减仓而不是临场辩解。

## 四、复盘问题

复盘时问：我是否因为持仓而改变了判断标准？是否忽视了最强反方证据？是否把历史投入当成继续持有的理由？是否为了维护面子而拒绝认错？

## 六、把偏差变成交易规则

学习 `保守偏差` 的重点，是把心理问题提前写进流程。交易者在平静时往往知道自己应该客观，但在持仓亏损、盈利回吐、公开表达观点之后，很容易开始维护自尊而不是维护账户。

| 场景 | 偏差如何出现 | 规则化处理 |
|---|---|---|
| 买入前 | 只寻找支持观点的信息 | 强制写三条反方证据 |
| 持仓中 | 把不利信息解释成短期扰动 | 每周检查原始假设是否被证伪 |
| 亏损后 | 因为不愿认错而补仓 | 补仓必须重新写独立买入理由 |
| 盈利后 | 过度相信自己的判断 | 按风险预算而不是情绪加仓 |
| 复盘时 | 只记住成功案例 | 记录所有偏离流程的行为 |

## 七、实战案例写法

```text
触发偏差的交易：
当时的情绪：
我选择性忽略了什么信息：
如果没有持仓，我会如何判断：
下次阻断规则：
```

优秀交易者不是没有偏差，而是知道自己一定会有偏差，所以提前用清单、仓位上限、证伪条件和交易日志把偏差关进流程里。

## 八、把 `保守偏差` 放入交易日志

行为偏差最怕“事后合理化”。建议每次交易日志都加入一个心理字段，记录当时是否受到 `保守偏差` 的影响。

| 日志字段 | 记录方式 |
|---|---|
| 决策前情绪 | 兴奋、恐惧、焦虑、想证明自己、害怕错过 |
| 反方证据 | 至少写出最强的一条反对意见 |
| 仓位理由 | 仓位是否来自风险预算，而不是情绪强度 |
| 证伪条件 | 什么事实出现后必须承认判断错了 |
| 复盘结论 | 错误来自研究、组合、执行还是心理 |

## 九、训练方法

1. 回看最近三笔亏损交易，标出是否存在 `保守偏差`。
2. 把“我当时为什么不愿执行规则”写成一句话。
3. 给这个偏差设计一个可执行阻断动作，例如冷静期、减仓阈值、红队复核或禁止加仓。

真正的心理训练不是追求冷血，而是让情绪无法绕过流程。

## 深度案例与训练

### 情景演练

把 保守偏差 放到一笔真实交易里观察：你因为前期盈利而提高仓位，或者因为前期亏损而急于翻本，此时判断很容易从“证据驱动”滑向“情绪驱动”。训练时不要只写结果，要记录信息来源、当时情绪、仓位变化和是否偏离原计划。

### 纠偏动作

- 交易前写下反方证据，强迫自己看到与持仓相反的信息。
- 对加仓、补仓、止损和止盈设置硬规则，避免临场解释。
- 将每次冲动交易标记出来，单独统计其收益、回撤和后悔程度。

### 复盘训练

连续二十笔交易后，统计哪些错误重复出现：追涨、扛亏、过早止盈、过度交易、忽视风险提示。行为金融的目标不是给错误起名字，而是把常犯错误变成下一次交易前的检查项。

## 最小可执行项目

### 偏差识别练习

围绕 保守偏差 复盘最近二十次买卖，给每次交易标注触发情绪、信息来源、是否偏离计划、是否存在追涨杀跌、是否因为成本价或短期盈亏改变判断。不要只看盈亏，要看决策质量。

| 输入 | 操作 | 输出 |
|---|---|---|
| 交易流水 | 标注情绪和动作 | 偏差频率表 |
| 持仓记录 | 对比原计划 | 执行偏差清单 |
| 复盘日志 | 找重复错误 | 下次交易前规则 |

### 验收标准

能说出自己最常见的三类偏差，并为每类偏差设置一个硬性约束，例如冷静期、仓位上限、反方证据检查或禁止当日翻本交易。


## 进一步掌握

学习 `保守偏差` 时，要把它当作交易前的自检工具，而不是心理学名词。行为偏差最危险的地方在于它常常披着“经验”“直觉”“我很了解这只股票”的外衣出现，让人把情绪当成证据，把成本价当成价值，把短期结果当成能力。

| 检查维度 | 应用方法 | 常见错误 |
|---|---|---|
| 触发场景 | 记录在哪些亏损、盈利或舆论环境下出现 | 事后才承认自己受影响 |
| 纠偏规则 | 写成冷静期、降仓、反方证据或复盘问题 | 只说下次注意 |
| 结果追踪 | 统计同类错误频率是否下降 | 只复盘亏钱交易 |

### 训练任务

围绕 `保守偏差` 记录 20 次真实决策：当时情绪、原计划、实际动作、是否偏离、事后证据和结果。若同类偏差重复出现，就把它改写成硬规则，例如降低仓位、延迟下单、必须写反方理由或暂停交易。
