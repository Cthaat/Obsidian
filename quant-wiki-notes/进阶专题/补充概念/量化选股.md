# 量化选股

> [!note] 核心定位
> 量化选股 属于策略研究和组合落地主题。重点是先定义收益来源，再验证数据、成本、风险、容量和失效条件，最后写成可监控的交易流程。

## 一、量化选股的核心逻辑

量化选股是把“什么股票更值得买”转化为可计算、可排序、可复盘的规则。它不等于让模型替你思考，而是把投资假设系统化。

常见收益来源包括：

- 估值修复：低估值股票被重新定价。
- 盈利质量：高现金流、高 ROE、低杠杆公司更稳健。
- 动量延续：强势股票在一段时间内继续强势。
- 反转修复：短期过度下跌后价格回归。
- 行业轮动：不同宏观和政策阶段行业表现不同。
- 行为偏差：市场过度悲观、过度乐观或反应不足。

## 二、选股系统流程

| 阶段 | 要做什么 |
|---|---|
| 股票池 | 明确全市场、指数成分、行业池，剔除停牌、ST、低流动性标的 |
| 因子定义 | 价值、质量、成长、动量、波动率、流动性、情绪等 |
| 数据处理 | 去极值、标准化、缺失值、公告日、复权、行业分类 |
| 打分排序 | 单因子、多因子等权、IC加权、模型预测 |
| 风险约束 | 行业、市值、单票权重、换手、流动性、跟踪误差 |
| 回测验证 | 分层收益、IC、组合收益、回撤、成本、样本外 |
| 实盘执行 | 调仓、下单、成交监控、异常处理和复盘 |

## 三、常见选股因子

| 因子 | 指标示例 | 关注点 |
|---|---|---|
| 价值 | PE、PB、PS、股息率、EV/EBITDA | 便宜是否有原因 |
| 质量 | ROE、ROIC、毛利率、现金流、应计 | 利润是否真实且可持续 |
| 成长 | 收入增长、利润增长、PEG | 增长是否已被高估值透支 |
| 动量 | 3/6/12个月收益、52周高点 | 趋势是否拥挤和反转 |
| 低波动 | 波动率、Beta、最大回撤 | 防御性和牛市跑输风险 |
| 流动性 | 换手率、成交额、Amihud | 容量和退出成本 |
| 情绪 | 分析师预期、资金流、新闻情绪 | 数据噪音和时效性 |

## 四、量化选股的风控

量化选股最容易“看起来很分散，实际很集中”。必须检查：

- 行业暴露是否过度集中。
- 市值暴露是否只是小盘风格。
- 因子之间是否高度相关。
- 换手率是否过高。
- 回撤是否集中发生在某类市场环境。
- 实盘成交是否会明显偏离回测价格。

## 八、策略上线前的最后一问

`量化选股` 上线前，要从“能赚钱”转向“亏钱时怎么办”。策略在历史上赚钱并不稀奇，能解释亏损、限制亏损、复盘亏损，才是真正可用。

| 问题 | 最低答案 |
|---|---|
| 为什么赚钱 | 清晰收益来源，不依赖玄学参数 |
| 为什么会亏 | 明确失效环境和风险来源 |
| 亏多少停 | 有回撤、波动、连续亏损阈值 |
| 如何降级 | 降仓、暂停、回到基准策略 |
| 如何复盘 | 数据、信号、执行、组合、心理分开记录 |

## 深度案例与训练

### 策略拆解

学习 量化选股 时，写清楚策略的六个零件：信号、过滤器、入场、退出、仓位和风控。任何说不清这六项的策略，都还只是想法。

### 验证路径

1. 用最简单规则建立基准收益。
2. 加入手续费、滑点、冲击成本和容量约束。
3. 做样本外、滚动窗口和参数扰动。
4. 统计收益来自哪些时期、哪些品种和哪些市场状态。

### 实盘监控

上线后关注信号命中率、成交质量、回撤、换手、资金占用和异常订单。策略失效时优先降仓或暂停，而不是继续调参寻找安慰。

## 最小可执行项目

### 策略一页纸

围绕 量化选股 写一张策略一页纸：收益来源、适用品种、信号、过滤器、入场、退出、仓位、风控、成本、容量、暂停条件和复盘指标。

| 检验 | 合格标准 |
|---|---|
| 逻辑 | 能解释为什么会有收益 |
| 回测 | 有样本外和参数扰动 |
| 成本 | 扣除手续费、滑点和冲击成本 |
| 风险 | 有最大回撤和连续亏损预案 |
| 实盘 | 有监控、降仓和暂停规则 |

### 常见失败

策略在报告中赚钱，却在实盘中输给交易成本、容量限制、信号衰减和执行错误。
