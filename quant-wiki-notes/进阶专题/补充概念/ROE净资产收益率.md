# ROE净资产收益率

> [!note] 核心定位
> ROE净资产收益率 属于估值或财务分析工具。使用时要同时看计算口径、盈利质量、行业差异、周期位置和现金流验证，不能把单一指标当成买卖结论。

## 一、核心定义

ROE净资产收益率 是把企业盈利、现金流、资产质量或增长预期转成可比较指标的工具。它不能单独证明便宜或昂贵，必须结合行业、周期、商业模式、会计质量和资本开支。

## 二、使用顺序

先确认口径，再比较同业和历史分位，然后检查利润是否变成现金流，最后做悲观、基准、乐观三情景。对周期行业，要用周期中枢利润；对成长企业，要检查增长是否已经被估值充分计入。

## 三、常见误区

低估值可能是价值陷阱，高增长可能是估值陷阱。会计利润可能来自一次性收益或激进确认，股息率可能来自不可持续分红，ROE可能由高杠杆堆出来。

## 四、实战检查清单

- 口径是否清楚
- 是否和可比公司比较
- 是否检查现金流
- 是否考虑资本开支和营运资本
- 是否做三情景和敏感性
- 估值错误时仓位是否可承受

## 六、估值指标的交叉验证

`ROE净资产收益率` 不能单独决定买卖。估值判断至少要和现金流、资产负债表、行业周期、增长质量一起看。

| 检查项 | 问题 |
|---|---|
| 分子 | 市值、企业价值、价格是否取对 |
| 分母 | 净利润、EBITDA、ROE、股息是否可持续 |
| 口径 | 是否扣除一次性损益，是否使用TTM或预测值 |
| 行业 | 同业可比性是否成立 |
| 周期 | 当前利润是在周期顶部还是底部 |
| 现金流 | 会计利润是否转化为经营现金流 |
| 安全边际 | 悲观情景下是否仍有保护 |

## 八、估值结论如何写进交易计划

`ROE净资产收益率` 的结论必须连接到价格、仓位和退出，而不是停留在“便宜/贵”。

| 估值判断 | 交易动作 |
|---|---|
| 明显低估但质量一般 | 只能小仓位或等待催化剂 |
| 合理低估且质量较好 | 可进入观察仓或分批建仓 |
| 估值合理但缺少安全边际 | 等待价格或基本面变化 |
| 高估且预期拥挤 | 检查减仓、对冲或不参与 |

## 深度案例与训练

### 估值案例

学习 ROE净资产收益率 时，选择三类公司练习：稳定现金流企业、高成长企业和周期企业。分别写出适用估值方法、关键假设、估值区间和最容易出错的地方。

### 敏感性分析

- 改变增长率、利润率、折现率、资本开支和周转率，观察估值区间如何变化。
- 用绝对估值和相对估值交叉验证，避免单一指标误导。
- 检查利润是否能转化为现金流，防止会计利润掩盖经营质量。

### 风险提示

估值低不等于便宜，估值高也不必然泡沫。真正需要判断的是：市场低估了什么、高估了什么、你的假设错在哪里、错了之后仓位如何处理。

## 最小可执行项目

### 估值备忘录

围绕 ROE净资产收益率 写一页估值备忘录：公司业务、利润来源、现金流质量、估值方法、关键假设、估值区间、安全边际和退出条件。估值结论必须给出区间，而不是单点价格。

| 场景 | 需要改变的假设 |
|---|---|
| 保守 | 增速下降、利润率回落、折现率上升 |
| 基准 | 经营保持当前趋势 |
| 乐观 | 份额提升、费用率改善、资本回报提高 |

### 验收标准

能解释估值对哪一个变量最敏感，并说明如果该变量判断错误，仓位如何处理。
