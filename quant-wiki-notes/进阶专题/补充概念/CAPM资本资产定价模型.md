# CAPM资本资产定价模型

> [!note] 核心定义
> CAPM（资本资产定价模型）给出单个资产的预期收益与其系统性风险之间的线性关系：资产承担的市场风险越高（β 越大），市场要求它提供的预期收益就越高。交易者要懂它，是因为它定义了"该资产凭风险应该赚多少"这个基准——你的真实收益减去这个基准，才是值得追问的超额收益 α。

## 一、定义与直觉

CAPM 由 Sharpe、Lintner 等人在 1960 年代提出，是现代金融把"风险"和"收益"用一个公式连起来的第一次成功尝试。它要回答的问题非常朴素：**在一个有大量分散化投资者的市场里，一项资产的"合理"预期收益应该由什么决定？**

它的答案分两步：

第一步，**风险可以拆成两部分**。一只股票的总风险里，一部分来自整个市场的共同涨跌（利率、经济周期、流动性、风险偏好），这叫**系统性风险**；另一部分来自公司自己的事（某条产品线失败、管理层变动、一次诉讼），这叫**非系统性风险**。

第二步，**只有躲不掉的风险才配拿补偿**。当你持有几十只股票时，各家公司的"自己的事"有好有坏、互相抵消，非系统性风险被分散掉了；但市场整体下跌时所有股票一起跌，这部分谁也躲不开。既然非系统性风险可以靠免费的分散化消除，理性市场就不会为它付钱——**只有系统性风险被定价**。

一个通俗类比：一艘船上有很多乘客。某个乘客晕船、行李丢了，是他个人的倒霉事，跟别人无关（非系统性）；但整条船遇到风暴，所有人一起颠簸，这是系统性的。保险公司愿意为"全船风暴"收保费、付赔偿，却不会为"某个人晕船"单独定价——因为后者在一船人里被平均掉了。资产的预期收益，就是市场为"陪着大盘一起颠簸"付给你的报酬。

## 二、原理与公式

### 2.1 核心定价公式

$$E(R_i)=R_f+\beta_i\,\big(E(R_m)-R_f\big)$$

其中：

- $E(R_i)$ = 资产 $i$ 的预期收益率；
- $R_f$ = 无风险利率（常用国债收益率近似）；
- $E(R_m)$ = 市场组合的预期收益率（实务中用宽基指数代理，如沪深300）；
- $E(R_m)-R_f$ = **市场风险溢价**，即承担一单位市场风险整体能多赚多少；
- $\beta_i$ = 资产 $i$ 对市场风险的敏感系数。

公式可以读成一句话：**资产收益 = 无风险收益 + β 倍的市场风险溢价**。无风险利率是"时间的报酬"（钱借出去总要有利息），后一项是"承担系统性风险的报酬"，β 决定了你领多少倍。

### 2.2 β 的定义与含义

$$\beta_i=\frac{\mathrm{Cov}(R_i,R_m)}{\mathrm{Var}(R_m)}$$

其中 $\mathrm{Cov}(R_i,R_m)$ 是资产收益与市场收益的协方差，$\mathrm{Var}(R_m)$ 是市场收益的方差。

β 衡量的是**资产收益对市场波动的敏感程度**，可以直观理解为"市场涨跌 1%，这只股票平均跟着涨跌几个百分点"：

| β 取值 | 含义 | 典型例子（示意） |
|---:|---|---|
| β = 1 | 与市场同步 | 宽基指数本身 |
| β > 1 | 比市场更激进，放大涨跌 | 高弹性的券商、科技成长股 |
| 0 < β < 1 | 比市场更防御，波动更小 | 公用事业、必需消费 |
| β = 0 | 与市场不相关 | 理论上的无风险资产 |
| β < 0 | 与市场反向 | 极少数对冲性资产 |

注意分母是市场的方差而不是资产自己的方差——这正是 CAPM 的精髓：**它只关心资产和市场"共同波动"的那部分，不关心资产自己有多波动**。一只股票哪怕日内剧烈震荡，只要这些震荡与大盘无关（高非系统性风险、低 β），CAPM 也不会因此要求它提供更高收益。

### 2.3 证券市场线（SML）

把上面的定价公式画到"β—预期收益"坐标系里，就得到一条直线，叫**证券市场线（Security Market Line, SML）**：

- 横轴是 β（系统性风险），纵轴是预期收益；
- 直线截距 = $R_f$，斜率 = 市场风险溢价 $E(R_m)-R_f$；
- β = 1 处对应的就是市场组合的预期收益。

SML 给出了"每一档系统性风险应该配多少预期收益"的公平定价线。它的用法是把现实资产点上去：

- 点落在 SML **上方**：在同样风险下预期收益更高，相对被低估（值得买）；
- 点落在 SML **下方**：风险不变但预期收益偏低，相对被高估。

> [!tip] SML 与 CML 不要混
> 证券市场线（SML）横轴是 β，描述单个资产；资本市场线（CML）横轴是总波动率 σ，只描述"无风险资产 + 市场组合"的有效组合。单只股票通常落在 CML 下方（因为它带着没被补偿的非系统性风险），但只要定价公平，它一定落在 SML 上。

### 2.4 α：超额收益

把资产的**实际平均收益**减去 CAPM 给出的"应得收益"，差值就是 **α（阿尔法）**：

$$\alpha_i = \bar{R_i} - \big[R_f+\beta_i(\bar{R_m}-R_f)\big]$$

- $\alpha>0$：跑赢了它的风险所"应得"的收益，是真正的超额本事（或运气）；
- $\alpha=0$：不多不少，收益完全由承担的市场风险解释；
- $\alpha<0$：承担了风险却没拿到应得回报。

α 是主动投资追求的目标：**β 是市场免费给的（买指数就有），α 才是你的能力**。后文的多因子模型，本质就是在质疑"被 CAPM 算成 α 的那部分，是不是其实只是别的风险暴露"。

## 三、计算 / 参数 / 案例

下面用一个**假设**的算例走一遍。设：

- 无风险利率 $R_f = 2.5\%$（假设）；
- 市场预期收益 $E(R_m)=9.5\%$（假设），则市场风险溢价 = $9.5\%-2.5\%=7.0\%$；
- 三只股票的 β 分别为 0.6、1.0、1.6（假设）。

代入 $E(R_i)=R_f+\beta_i\,(E(R_m)-R_f)$：

| 股票 | β（假设） | 风险溢价部分 = β × 7.0% | CAPM 预期收益 $E(R_i)$ |
|---|---:|---:|---:|
| A（防御） | 0.6 | 4.2% | 2.5% + 4.2% = **6.7%** |
| B（与市场同步） | 1.0 | 7.0% | 2.5% + 7.0% = **9.5%** |
| C（激进） | 1.6 | 11.2% | 2.5% + 11.2% = **13.7%** |

可以看到 β 每增加 1，预期收益就抬高一个市场风险溢价（7.0%）。

再看 α：假设股票 C 过去一段时间的实际年化收益是 16%（假设），而 CAPM 认为它"应得" 13.7%，则

$$\alpha_C = 16\% - 13.7\% = +2.3\%$$

这 2.3% 就是它相对自身风险的超额表现。但要警惕：这个 α 可能只是因为 C 还暗含了别的风险暴露（比如它同时是只小盘股、价值股），而 CAPM 的单一市场因子没能捕捉到——这正是引出多因子模型的入口。

> [!warning] β 是估计值，不是常数
> 表里的 β 都当成已知，但现实中 β 要用历史收益回归估计，且会随回归窗口（1 年 / 3 年）、数据频率（日 / 周 / 月）和市场状态变化。换个窗口，同一只股票的 β 可能从 0.9 跳到 1.3，CAPM 预期收益也跟着变。所以 CAPM 的"精确数字"从来不精确，它更像一把标尺而非答案。

## 四、实战应用与使用方式

CAPM 在实务里很少被当成"算出股价"的工具（它太粗糙），更多是当**基准和分析框架**：

1. **算必要收益率，做估值折现**：DCF 里股权成本 $r_e$ 通常就用 CAPM 求得（$r_e=R_f+\beta(R_m-R_f)$），再并入 WACC 作为折现率。β 越高、风险越大，折现率越高、估值越低。
2. **业绩归因**：用 CAPM 回归一个基金或策略的收益，分离出 β（市场暴露）和 α（选股/择时能力）。如果一只"明星基金"的超额收益其实都来自高 β（牛市加杠杆式上涨），那它没有真本事，熊市会原形毕露。
3. **判断相对高低估**：把个股点到 SML 上，看在上方还是下方，作为相对定价的粗筛。
4. **仓位与风格控制**：用组合的加权 β 表达对市场的总暴露。看多就把组合 β 抬到 1 以上，谨慎就压到 1 以下，这是 Beta 择时的语言。

> [!tip] 结合 A 股制度的提醒
> CAPM 假设可以无摩擦地随时买卖、自由做空、无税费。A 股的现实约束会让它进一步偏离：T+1 限制日内调仓、涨跌停会让 β 在极端行情中失真（停在跌停板上时价格不动，回归出来的 β 被低估）、个股做空成本高使得"卖出高估资产把点拉回 SML"的套利难以实现。因此在 A 股，CAPM 的结论要当方向性参考，不要当精确定价。

### 假设与现实的偏离：为什么要走向多因子

CAPM 的优雅建立在一串强假设上：投资者理性且只看均值—方差、可无成本借贷与做空、信息对称、所有人持有相同的市场组合。现实里这些都不完全成立，于是出现了 CAPM **无法解释的系统性规律（异象）**：

- **规模效应**：长期看小盘股的平均收益高于 CAPM 用 β 预测的水平——多出来的收益不该存在，却长期存在。
- **价值效应**：低估值（高账面市值比）的股票，收益也系统性高于 CAPM 预测。
- **低 β 异象（低波动异象）**：理论上高 β 该有高收益，但实证中**低 β / 低波动股票的风险调整后收益反而更好**，高 β 股票并没有兑现它"应得"的高回报。这与 CAPM 的核心结论直接矛盾，常被归因于杠杆受限、彩票偏好等行为因素。

这些规律意味着：β 之外还有别的、能被定价的系统性风险来源。于是 Fama 和 French 把规模、价值补成三因子，Carhart 再补上动量成四因子，**多因子模型**由此诞生——它们都是在 CAPM 的骨架上，承认"市场不是唯一的定价因子"。

## 五、常见误区与失效条件

| 误区 | 正确理解 |
|---|---|
| β 就是一只股票的总风险 | β 只衡量**系统性风险**（与市场共动的部分）；总风险还含可被分散掉的非系统性风险，后者不进入 CAPM 定价 |
| 历史 β 能稳定预测未来 β | β 随估计窗口、数据频率和市场状态漂移，并不稳定；用历史 β 直接当未来值会有明显误差 |
| 高 β 一定带来高收益 | CAPM 说的是高 β 有更高的**预期**收益；实证中存在"低 β 异象"，高 β 常常没兑现应得回报 |
| 落在 SML 上方就闭眼买 | SML 偏离基于 β 和市场溢价的估计，估计误差大；且 A 股做空难、有 T+1 与涨跌停，套利回归未必发生 |
| α 永远是真本事 | 用 CAPM 算出的 α 可能只是规模、价值、动量等未被纳入的因子暴露；换更全的因子模型后 α 常常缩水甚至消失 |
| CAPM 能算出准确的目标价 | CAPM 是风险—收益的基准框架，不是精确定价器；它最适合做相对比较和成本估计，而非给出"内在价值数字" |

> [!warning] 何时最不可靠
> 当资产现金流极不稳定（周期股、初创公司）、市场处于极端状态（流动性枯竭、单边涨跌停）、或做空与借贷受限时，CAPM 的假设破坏得最严重，结论最不可信。这些场景下应优先用多因子框架或情景分析交叉验证。
