# PE

> [!note] 核心定位
> PE 属于估值或财务分析工具。使用时要同时看计算口径、盈利质量、行业差异、周期位置和现金流验证，不能把单一指标当成买卖结论。

## 一、PE 到底衡量什么

PE（Price to Earnings Ratio，市盈率）是“市场愿意为每一单位利润支付多少价格”。最基本公式是：

$$PE = \frac{\text{公司市值}}{\text{归母净利润}} = \frac{\text{股价}}{\text{每股收益}}$$

它本质上不是“便宜/贵”的绝对标签，而是把价格、利润、增长、风险和利率压缩到一个倍数里。低 PE 可能代表低估，也可能代表利润即将下滑；高 PE 可能代表泡沫，也可能代表高质量增长、低资本开支和强竞争优势。

## 二、常见 PE 口径

| 口径 | 含义 | 适用场景 | 风险 |
|---|---|---|---|
| 静态 PE | 当前市值 / 上一年度净利润 | 粗略查看历史估值 | 不能反映最新业绩变化 |
| TTM PE | 当前市值 / 最近四个季度净利润 | 常用横向比较 | 仍可能受一次性损益影响 |
| 动态 PE | 当前市值 / 预测未来一年净利润 | 成长股、业绩拐点 | 依赖预测质量 |
| 扣非 PE | 市值 / 扣非归母净利润 | 排除非经常损益 | 仍需检查会计调节 |
| 远期 PE | 市值 / 未来数年预测利润 | 高成长企业 | 容易把乐观假设当现实 |

看 PE 前必须问：分母是真实经营利润，还是一次性收益、资产处置、补贴、投资收益或会计调节？

## 三、PE 与增长、利率和质量

一个简化理解是：PE 由增长率、折现率和盈利质量共同决定。

| 变量 | 对 PE 的影响 |
|---|---|
| 增长率更高 | 市场愿意给更高 PE |
| 利率/折现率更高 | PE 通常下降 |
| 盈利更稳定 | PE 通常更高 |
| 资本开支更低 | 同样利润更有价值 |
| 护城河更强 | 未来现金流更可靠 |
| 周期波动更大 | PE 需要谨慎解释 |

所以比较 PE 时，不能只比数字。银行、周期、消费、软件、医药、地产、资源股的 PE 含义完全不同。跨行业直接比较 PE，往往比不看还危险。

## 四、PE 的典型陷阱

- **周期顶部低 PE**：周期行业在景气高点利润暴增，PE 看起来很低，但下一轮利润下滑会让估值迅速变贵。
- **一次性利润低 PE**：利润来自资产处置或投资收益，不代表主营业务价值。
- **价值陷阱**：公司长期低 PE，是因为治理差、行业衰退、现金流差或增长停滞。
- **高 PE 合理化**：用“未来空间很大”解释任何价格，却没有现金流和竞争优势支撑。
- **忽略股本稀释**：利润增长但不断增发，单股价值未必提升。

## 五、实战使用方法

| 步骤 | 检查问题 |
|---|---|
| 1. 明确口径 | 静态、TTM、扣非、动态还是远期 PE |
| 2. 看行业位置 | 同行业公司、历史分位、商业模式是否可比 |
| 3. 看利润质量 | 现金流、应收账款、毛利率、费用资本化是否异常 |
| 4. 看增长假设 | 当前 PE 是否已经计入过高增长 |
| 5. 做情景估值 | 悲观、基准、乐观利润下 PE 分别是多少 |
| 6. 接到仓位 | 估值错误时最大亏损是否可承受 |

PE 最适合用于“初筛和交叉验证”，不适合单独作为买入理由。真正的结论应写成：“在某利润口径、某增长假设、某行业比较下，这个估值区间是否提供足够安全边际。”

## 八、估值结论如何写进交易计划

`PE` 的结论必须连接到价格、仓位和退出，而不是停留在“便宜/贵”。

| 估值判断 | 交易动作 |
|---|---|
| 明显低估但质量一般 | 只能小仓位或等待催化剂 |
| 合理低估且质量较好 | 可进入观察仓或分批建仓 |
| 估值合理但缺少安全边际 | 等待价格或基本面变化 |
| 高估且预期拥挤 | 检查减仓、对冲或不参与 |

## 深度案例与训练

### 估值案例

学习 PE 时，选择三类公司练习：稳定现金流企业、高成长企业和周期企业。分别写出适用估值方法、关键假设、估值区间和最容易出错的地方。

### 敏感性分析

- 改变增长率、利润率、折现率、资本开支和周转率，观察估值区间如何变化。
- 用绝对估值和相对估值交叉验证，避免单一指标误导。
- 检查利润是否能转化为现金流，防止会计利润掩盖经营质量。

### 风险提示

估值低不等于便宜，估值高也不必然泡沫。真正需要判断的是：市场低估了什么、高估了什么、你的假设错在哪里、错了之后仓位如何处理。
