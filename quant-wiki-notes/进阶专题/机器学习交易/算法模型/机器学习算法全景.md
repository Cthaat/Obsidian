# 机器学习算法全景

> [!note] 概述
> 系统介绍量化交易中常用的机器学习算法，从传统ML到深度学习，覆盖每类算法的原理、适用场景和交易应用。

## 一、传统机器学习

### 支持向量机（SVM）

**原理**：寻找最优超平面将不同类别分开。

**量化应用**：
- 股票涨跌二分类预测
- 异常市场状态识别
- 结合多个因子做信号生成

### 随机森林（Random Forest）

**原理**：多棵决策树的集成，通过投票/平均提高稳定性。

**量化应用**：
- 多因子选股（非线性关系捕捉）
- 特征重要性分析
- 抵抗过拟合（集成天然正则化）

> 适合处理因子之间的非线性交互关系，比线性回归更具表达力。

### XGBoost / LightGBM

**原理**：梯度提升树，逐棵添加弱学习器。

**量化应用**：
- 因子挖掘竞赛的主力模型
- 中低频选股策略
- 特征工程自动化

## 二、深度学习模型

### LSTM（长短期记忆网络）

```python
# LSTM核心：三个门控机制
# 遗忘门 → 决定丢弃哪些信息
# 输入门 → 决定存储哪些新信息
# 输出门 → 决定输出什么
```

**量化应用**：
- 时序价格预测（日线/分钟级）
- 技术指标模式识别
- 订单流异常检测

**关键考虑**：
- 时间序列必须按时间顺序划分训练/测试集（不可随机shuffle）
- 需要大量数据，避免参数过多
- 金融数据信噪比低，需合理设计标签

### GRU（门控循环单元）

LSTM的简化版，减少一个门控，参数更少。

| 对比 | LSTM | GRU |
|------|------|-----|
| 参数量 | 较多 | 较少（约少25%） |
| 训练速度 | 较慢 | 较快 |
| 长期记忆 | 更强 | 略弱 |
| 适合场景 | 长序列 | 中等序列 |

### Transformer

**原理**：基于自注意力机制，捕捉全局依赖关系。

**量化优势**：
- 并行计算，训练效率高
- 多头注意力捕捉多时间尺度关系
- 对长序列表现优异
- 适合多资产联合建模

### CNN在金融的应用

- 将价量数据转化为"图像"进行模式识别
- K线形态识别
- 多时间维度卷积

## 三、生成模型

### GAN（生成对抗网络）

**量化应用**：
- **数据增强**：生成合成市场数据训练策略
- **特征生成**：GAN生成新时序特征增强选股
- **风险模拟**：生成极端市场场景压力测试

> 西南证券研报：GAN_GRU因子在全A市场RankIC达7.03%，多空年化46.64%

### AutoEncoder（自编码器）

**量化应用**：
- 因子降维：将高维因子压缩到低维
- 去噪：去除市场数据中的噪声
- 特征提取：从原始数据学习有效表征

## 四、强化学习（RL）

| 算法 | 特点 | 交易应用 |
|------|------|---------|
| DQN | 离散动作空间 | 买/卖/持有决策 |
| PPO | 连续动作空间 | 仓位比例控制 |
| A2C/A3C | Actor-Critic架构 | 多目标优化 |

**核心优势**：直接学习"怎么做"而非"预测什么"，适合交易决策。

## 五、模型训练注意事项

### 时间序列的特殊性

```
❌ 错误：随机划分训练/测试集
   (用未来数据训练 → 预测过去 → 泄漏)

✅ 正确：保持时间顺序
   训练集: [2015-2019]
   验证集: [2020-2021]
   测试集: [2022-2023]
```

### 防止过拟合

- 使用时间序交叉验证（非随机K折）
- 加入正则化（L1/L2/Dropout）
- 早停（Early Stopping）
- 简化模型（参数量适配数据量）

### 标签设计

| 回归标签 | 分类标签 |
|---------|---------|
| 未来N天收益率 | 上涨/下跌 |
| 超额收益（vs指数） | 排名分位数 |
| 实现波动率 | 趋势延续/反转 |

## 实战掌握清单

> [!tip] 交易者视角
> 机器学习算法全景 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。机器学习在交易中的价值是处理非线性和高维特征，但最大的敌人是泄漏、过拟合和不可交易。

### 关键判断

- 明确标签定义、预测周期、样本切分和特征可得时间。
- 用时间序列交叉验证而不是随机打乱。
- 比较简单线性模型、树模型和深度模型的真实增益。

### 落地动作

1. 先建立可解释基准，再增加复杂度。
2. 检查特征重要性、模型稳定性、换手和容量。
3. 上线后监控数据漂移、预测分布漂移和收益衰减。

### 失效边界

- 标签泄漏。
- 样本外表现坍塌。
- 模型信号无法低成本成交。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
