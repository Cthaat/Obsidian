# 🌱 成长因子 (Growth Factor)

> [!note] 核心逻辑
> 成长因子衡量公司的**扩张速度**。高成长公司享有估值溢价，因为今天的增长意味着未来更大的现金流。但需要注意区分"真成长"和"周期性增长"，以及估值过高带来的风险。

## 一、因子列表

本类别包含 **14个因子**，涵盖盈利增长、营收增长和成长估值权衡：

| # | 因子名称 | 构建方法 |
|---|---------|---------|
| 1 | PEG (市盈率增长比) | PE(TTM) / 净利润增速 |
| 2 | 单季净利润同比 | 单季净利润 YoY |
| 3 | 稀释EPS同比增长 | 稀释每股收益 YoY |
| 4 | 营收环比增长 | 季度营收 QoQ |
| 5 | 净利润率变化 | 资产净利润率变动 |
| 6 | 市盈率调整因子 | PE经过增长率调整 |
| 7 | 质量增长 | 结合质量的增长因子 |
| 8 | 创新投入比率 | 研发支出 / 营收 |
| 9 | 异常毛利润 | 毛利润超出行业均值的部分 |
| 10 | 净资产每股同比增长 | 每股净资产 YoY |
| 11 | 员工人数增长率 | 员工数量 YoY |
| 12 | 持续PEG一致性 | PEG稳定性指标 |
| 13 | PE增长率+股息率 | 综合增长与回报 |
| 14 | 60日利润市值比 | 短期利润/市值变化 |

## 二、核心因子详解

### 2.1 PEG (市盈率增长比) ★★★★★

$$PEG = \frac{PE(TTM)}{\text{净利润增长率}}$$

- PEG < 1 → 成长性被低估，可能是投资机会
- PEG > 2 → 估值已经透支了增长
- 解决了PE单一指标**只看估值不看成长**的缺陷

> 详见：[[成长因子详解-PEG]]

### 2.2 单季净利润同比增长 ★★★★

$$YoY\_NI = \frac{NI_{t}^{Q}}{NI_{t-4}^{Q}} - 1$$

- 最直接的盈利增长衡量
- 需要剔除季节性因素和一次性损益
- 结合多季度趋势避免单季噪声

### 2.3 营收环比增长 ★★★★

$$QoQ\_Revenue = \frac{Revenue_t}{Revenue_{t-1}} - 1$$

- 营收增长是利润增长的先行指标
- 体现了业务量的真实扩张

### 2.4 创新投入比率 ★★★

$$R\&D\_Ratio = \frac{\text{研发支出}}{\text{营业收入}}$$

- 高研发投入 → 创新能力强 → 未来成长潜力大
- 科技和医药行业该因子尤其重要
- 需区分"高投入"和"高效投入"

## 三、成长的陷阱

| 风险 | 说明 |
|-----|------|
| 均值回归 | 高增长难以维持，增长率随时间衰减 |
| 估值过高 | 高成长搭配高估值，一旦增速放缓遭遇戴维斯双杀 |
| 并购式成长 | 通过收购实现的增长不持久 |
| 基数效应 | 小基数高增长率可能是统计幻象 |

## 四、A股成长因子特征

| 特点 | 说明 |
|-----|------|
| PEG 有效 | A股 PEG 因子年化 IC 约 0.04-0.06 |
| 小盘成长 | 小盘成长股在牛市中弹性最大 |
| 行业分化 | 科技/医药成长因子更有效 |
| 周期风险 | 周期股的"成长"可能是周期高点 |

## 实战掌握清单

> [!tip] 交易者视角
> 🌱 成长因子 (Growth Factor) 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。因子研究要证明信号背后的经济逻辑、统计稳定性和可交易性，而不是堆砌指标。

### 关键判断

- 明确因子定义、方向、覆盖范围和缺失值处理。
- 检验IC、Rank IC、分层收益、换手和行业市值暴露。
- 判断收益来自风险补偿、行为偏差、制度结构还是数据噪音。

### 落地动作

1. 做去极值、标准化、中性化和样本外检验。
2. 用组合层面的归因确认因子是否真的贡献alpha。
3. 持续监控拥挤、衰减、容量和相关性变化。

### 失效边界

- 因子动物园。
- 把行业或市值暴露误认为alpha。
- 忽略交易成本导致纸面有效、实盘无效。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 因子实验

围绕 🌱 成长因子 (Growth Factor) 做一次完整实验：定义信号、确定股票池、处理极值、做标准化和中性化，再输出 IC、Rank IC、分层收益、换手率、最大回撤和行业暴露。

### 组合验证

- 检查因子是否只在某个行业、市值段或市场阶段有效。
- 用样本外和滚动窗口观察稳定性。
- 将因子放入多因子组合，看新增收益是否被相关性抵消。

### 复盘标准

因子有效不等于可交易。必须确认容量、成本、拥挤、衰减和实盘成交质量。
