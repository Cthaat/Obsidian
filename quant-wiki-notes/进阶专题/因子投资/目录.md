> [!note] 主题概述

## 🧭 从因子概念到可交易组合

因子投资的学习顺序不应停留在“认识很多指标”。更好的路径是：

1. **先理解因子是什么**：从 [[什么是因子|什么是因子]] 开始，分清风险补偿、行为偏差和数据挖掘。
2. **再学习因子分类**：用 [[因子分类体系|因子分类体系]] 把价值、动量、质量、规模、波动率、流动性、情绪和基本面因子分组。
3. **然后做单因子检验**：每个因子都要看 IC、Rank IC、分层收益、换手、成本和样本外稳定性。
4. **再进入多因子组合**：处理行业/市值中性化、因子相关性、正交化、权重上限和风险贡献。
5. **最后看实盘衰减**：跟踪因子拥挤、容量、交易冲击、风格轮动和失效条件。

## ✅ 因子研究检查清单

| 问题 | 为什么重要 |
|---|---|
| 因子是否有经济解释？ | 没有解释的统计关系更容易是偶然噪音 |
| 数据在交易时点是否真实可得？ | 财报、公告、停牌和复权口径会制造前视偏差 |
| 是否做行业和市值中性化？ | 避免把行业 beta 或小盘暴露误认为 alpha |
| 分层收益是否单调？ | 好因子通常高分组和低分组应有清晰差异 |
| IC 是否稳定？ | 长期稳定的小 IC 往往比短期高 IC 更可用 |
| 换手和成本是否可承受？ | 高频换仓会让纸面 alpha 变成实盘亏损 |
| 因子之间是否高度相关？ | 高相关因子叠加会造成重复下注 |
| 是否监控拥挤和衰减？ | 公开因子会随资金涌入而收益下降 |

## ⚠️ 常见误区

- 把“回测收益高”当成因子有效，而不检查数据泄漏和成本。
- 把单因子排名直接当买卖指令，而不做组合风险约束。
- 把行业、市值、流动性暴露误认为选股能力。
- 用过多相似指标堆模型，造成因子重复和过拟合。
- 因子失效后不断调参，而不是回到经济解释和样本外验证。
