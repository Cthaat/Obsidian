# 🔧 因子构建方法

> [!note] 概述
> 因子从原始数据到可用于投资的信号，需要经过**数据清洗 → 计算 → 标准化 → 中性化** 四步标准流程。每一步的处理质量直接影响因子有效性。

## 一、构建流程总览

```mermaid
graph LR
    A[原始数据] --> B[数据清洗]
    B --> C[因子值计算]
    C --> D[缺失值处理]
    D --> E[极端值处理]
    E --> F[标准化]
    F --> G[行业中性化]
    G --> H[市值中性化]
    H --> I[最终因子信号]
```

## 二、数据清洗

| 处理项 | 方法 | 说明 |
|-------|------|------|
| 缺失值 | 剔除 / 均值填充 / 回归填充 | 缺失比例>50%应剔除因子 |
| 异常值 | MAD法 / 百分位截断 | 通常对1%和99%分位进行截断 |
| 停牌处理 | 剔除或前值填充 | 停牌期间数据不可靠 |
| ST/退市 | 剔除 | 风险警示股票数据异常 |

### MAD 极端值处理

$$x'_i = \begin{cases} \text{Median} + 3 \times \text{MAD}, & x_i > \text{Median} + 3 \times \text{MAD} \\ \text{Median} - 3 \times \text{MAD}, & x_i < \text{Median} - 3 \times \text{MAD} \\ x_i, & \text{otherwise} \end{cases}$$

其中 MAD = Median(|$x_i$ - Median(x)|)

## 三、因子值计算

### 3.1 基本面因子计算

| 因子 | 公式示例 | 数据来源 |
|-----|---------|---------|
| B/P (账面市值比) | 净资产 / 总市值 | 季报 + 日行情 |
| ROE | 净利润(TTM) / 净资产 | 季报滚动 |
| PEG | PE(TTM) / 净利润增速 | 季报 + 分析师预期 |

> [!warning] 注意
> 使用基本面数据时需注意**披露延迟**（季报发布后约1-2月才在数据库更新），防止前视偏差。

### 3.2 量价因子计算

| 因子 | 公式示例 | 计算周期 |
|-----|---------|---------|
| 动量 | Σ 日收益 (最近K个月) | 1-12个月 |
| 特质波动率 | CAPM/FF3残差标准差 | 最近N个交易日 |
| Amihud非流动性 | 1/N × Σ(|R|/成交量) | 最近N日平均 |
| RSI | 100 - 100/(1 + 平均上涨/平均下跌) | 14日 |

## 四、标准化方法

| 方法 | 公式 | 特点 |
|-----|------|------|
| Z-Score | $(x - μ) / σ$ | 均值0，标准差1 |
| 排名标准化 | $rank(x) / N$ | [0,1]均匀分布 |
| 极差标准化 | $(x - min)/(max - min)$ | [0,1]映射 |

> [!tip] 推荐
> 横截面**排名标准化**最稳健，不受极端值影响。Z-Score在存在极端值时会失真。

## 五、中性化处理

### 5.1 行业中性化

$$Y_{adj} = Y_{raw} - \sum_j X_{industry_j}·\beta_j$$

- 对每个行业的因子值去均值，或
- 用因子值对行业哑变量做回归，取残差

### 5.2 市值中性化

$$Y_{adj} = Y_{raw} - \text{ln(MarketCap)}·\beta_{size}$$

用原始因子值对 ln(市值) 做横截面回归，取残差。消除小盘股对因子信号的系统性偏差。

### 5.3 多变量中性化

$$Y_{adj} = Y_{raw} - X·\beta$$

其中 X 包括：行业哑变量 + ln(市值) + 其他需中性化的因子暴露。

## 六、因子合成

当多个子因子构建大类因子时：

| 合成方法 | 说明 |
|---------|------|
| **等权** | 各子因子标准化后等权平均 |
| **IC加权** | 按历史IC值加权 |
| **最大化IR** | 基于协方差矩阵求解最优权重 |
| **机器学习** | 使用GBDT/神经网络非线性合成 |

$$F_{final} = \sum_{i} w_i · f_i$$

> [!note] 构建原则
> 因子构建没有"最优"方案，需根据**投资目标**（收益驱动 vs 风险驱动）、**市场环境** 和**实施约束** 选择。关键是**一致性**——同一策略中所有因子应采用相同的构建标准。

## 课程化学习补充

> [!important] 学习定位
> 因子投资把可解释的收益来源系统化，关键是因子定义、数据口径、检验方法、组合构建和衰减监控。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 因子方向是否有经济解释
- 是否做去极值/中性化/标准化
- IC 和分层收益是否稳定
- 换手和容量是否可交易

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 因子动物园
- 样本内挖掘
- 拥挤交易
- 行业和市值暴露伪装成 alpha

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `🔧 因子构建方法` 时，不要只把它当成孤立知识点。把因子当成可解释、可检验、可组合的收益来源。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 因子定义、IC 和分层收益
- 行业/市值中性化和风险暴露
- 换手、容量和拥挤
- 组合构建和衰减监控

### 进阶训练

1. 做单因子分层回测
2. 检查因子与行业市值暴露的关系
3. 建立因子失效和拥挤监控

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
