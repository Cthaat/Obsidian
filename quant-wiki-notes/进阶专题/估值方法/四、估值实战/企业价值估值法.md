# 企业价值估值法

> [!note] 💡 概念解析
> 除主流的DCF和相对估值法外，还有一系列特殊估值方法适用于特定情境，包括重置成本法、清算价值法、RNAV法、创业投资估值法等，为不同类型企业提供了多元化的估值视角。

## 一、重置成本法

**定义**：重建一个与目前企业模式、产能、效率相同的企业需要花费的成本。

**公式**：
$$\text{待评估资产} = \text{重置全价} - \text{综合贬值}$$

或：
$$\text{待评估资产价值} = \text{重置全价} \times \text{综合成新率}$$

**适用**：资产密集型行业（基建、能源），不适用于轻资产科技公司。

## 二、清算价值法

**定义**：根据企业目前所有资产的变卖价值确定企业价值。

**适用场景**：
- 破产清算
- 解散清算
- 企业进入解散或破产时最为理想的估值模式

> [!warning] 清算估值
> 只要资产负债表上资产的评估是准确的，对股东而言，所有资产甩卖清偿后所剩的结果就是账面价值（股东权益）。

## 三、博克斯估值法

由美国人博克斯创立，适用于初创企业：

| 项目 | 估值 |
|------|------|
| 一个好的创意 | 100万元 |
| 一个好的盈利模式 | 100万元 |
| 优秀的管理团队 | 100-200万元 |
| 优秀的董事会 | 100万元 |
| 巨大的产品前景 | 100万元 |
| **合计** | **100-600万元** |

## 四、NAV净资产价值法

**定义**：地产企业当前储备项目的现金流折现价值剔除负债后的净资产价值。

$$NAV = \frac{\text{物业面积} \times \text{市场均价} - \text{净负债}}{\text{总股本}}$$

**优势**：为企业价值设定估值底线，对"地产项目公司"尤为适用。

**局限**：度量的是有形资产价值，不考虑品牌、管理能力和经营模式差异。

## 五、创业投资估值法

通过评估目标公司退出时的股权价值，基于目标回报倍数倒推出当前价值：

$$\text{当前股权价值} = \frac{\text{退出时股权价值}}{\text{目标回报倍数}} = \frac{\text{退出时股权价值}}{(1+\text{目标收益率})^N}$$

**适用**：互联网、区块链、大数据、信息化、云计算等新兴轻资产行业。

## 六、格雷厄姆估值公式

$$V = EPS \times (8.5 + 2G \times 100)$$

其中 $8.5$ 为业绩稳定增长公司的合理PE，$G$ 为预期年增长率。

## 七、互联网平台估值模式

$$\text{平台价值} = \text{粉丝量} \times \text{活跃程度} \times \text{转化率} \times \text{客单价} \times \text{复购率}$$

再根据数据在过去几年的趋势进行动态估值。

## 八、方法选择建议

| 企业特征 | 推荐方法 |
|---------|---------|
| 现金流大但利润低 | 市现率（PCF） |
| 轻资产、利润成长空间大 | 市盈率（PE） |
| 老国企、盈利乏力 | 市净率（PB） |
| 独角兽、亏损经营 | 创业投资估值法 / PS |
| 地产项目公司 | NAV |
| 待清算企业 | 清算价值法 |

## 课程化学习补充

> [!important] 学习定位
> 估值是给未来现金流定价，不是寻找精确数字；优秀交易者关注区间、假设弹性和安全边际。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 核心假设是否可解释
- 折现率/增长率是否做敏感性分析
- 是否与行业可比公司交叉验证
- 是否考虑资本周期和会计质量

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用单点估值制造虚假确定性
- 忽略周期高点利润
- 把低估值误认为低风险
- 现金流口径前后不一致

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
