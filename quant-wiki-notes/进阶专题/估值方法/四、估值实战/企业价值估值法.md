# 企业价值估值法

> [!note] 💡 概念解析
> 除主流的DCF和相对估值法外，还有一系列特殊估值方法适用于特定情境，包括重置成本法、清算价值法、RNAV法、创业投资估值法等，为不同类型企业提供了多元化的估值视角。

## 一、重置成本法

**定义**：重建一个与目前企业模式、产能、效率相同的企业需要花费的成本。

**公式**：
$$\text{待评估资产} = \text{重置全价} - \text{综合贬值}$$

或：
$$\text{待评估资产价值} = \text{重置全价} \times \text{综合成新率}$$

**适用**：资产密集型行业（基建、能源），不适用于轻资产科技公司。

## 二、清算价值法

**定义**：根据企业目前所有资产的变卖价值确定企业价值。

**适用场景**：
- 破产清算
- 解散清算
- 企业进入解散或破产时最为理想的估值模式

> [!warning] 清算估值
> 只要资产负债表上资产的评估是准确的，对股东而言，所有资产甩卖清偿后所剩的结果就是账面价值（股东权益）。

## 三、博克斯估值法

由美国人博克斯创立，适用于初创企业：

| 项目 | 估值 |
|------|------|
| 一个好的创意 | 100万元 |
| 一个好的盈利模式 | 100万元 |
| 优秀的管理团队 | 100-200万元 |
| 优秀的董事会 | 100万元 |
| 巨大的产品前景 | 100万元 |
| **合计** | **100-600万元** |

## 四、NAV净资产价值法

**定义**：地产企业当前储备项目的现金流折现价值剔除负债后的净资产价值。

$$NAV = \frac{\text{物业面积} \times \text{市场均价} - \text{净负债}}{\text{总股本}}$$

**优势**：为企业价值设定估值底线，对"地产项目公司"尤为适用。

**局限**：度量的是有形资产价值，不考虑品牌、管理能力和经营模式差异。

## 五、创业投资估值法

通过评估目标公司退出时的股权价值，基于目标回报倍数倒推出当前价值：

$$\text{当前股权价值} = \frac{\text{退出时股权价值}}{\text{目标回报倍数}} = \frac{\text{退出时股权价值}}{(1+\text{目标收益率})^N}$$

**适用**：互联网、区块链、大数据、信息化、云计算等新兴轻资产行业。

## 六、格雷厄姆估值公式

$$V = EPS \times (8.5 + 2G \times 100)$$

其中 $8.5$ 为业绩稳定增长公司的合理PE，$G$ 为预期年增长率。

## 七、互联网平台估值模式

$$\text{平台价值} = \text{粉丝量} \times \text{活跃程度} \times \text{转化率} \times \text{客单价} \times \text{复购率}$$

再根据数据在过去几年的趋势进行动态估值。

## 八、方法选择建议

| 企业特征 | 推荐方法 |
|---------|---------|
| 现金流大但利润低 | 市现率（PCF） |
| 轻资产、利润成长空间大 | 市盈率（PE） |
| 老国企、盈利乏力 | 市净率（PB） |
| 独角兽、亏损经营 | 创业投资估值法 / PS |
| 地产项目公司 | NAV |
| 待清算企业 | 清算价值法 |

## 实战掌握清单

> [!tip] 交易者视角
> 企业价值估值法 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。估值不是算出一个精确数字，而是在现金流、增长、折现率、资本结构和市场情绪之间建立合理区间。

### 关键判断

- 确认企业处于成长期、成熟期、周期底部还是困境修复。
- 选择与商业模式匹配的估值方法，而不是机械套PE或DCF。
- 拆分利润质量、现金流质量、资本开支和负债约束。

### 落地动作

1. 建立保守、基准、乐观三种情景，并做敏感性分析。
2. 把绝对估值与相对估值交叉验证。
3. 估值结论必须连接到仓位、买入区间、退出区间和安全边际。

### 失效边界

- 用高增长假设掩盖商业模式问题。
- 忽略周期顶部的低估值陷阱。
- 把会计利润当作可分配现金流。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
