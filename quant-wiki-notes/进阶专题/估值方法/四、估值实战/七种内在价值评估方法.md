# 七种内在价值评估方法

> [!note] 💡 概念解析
> 本文系统解析量化投资中7种核心内在价值评估方法，包括绝对估值法（DCF、DDM、NAV）和相对估值法（PE、PB、PS、EV/EBITDA），构建从理论到实操的完整评估体系。

## 一、估值方法分类体系

```
内在价值评估方法
├── 绝对估值法
│   ├── DCF（现金流折现模型）
│   ├── DDM（股利贴现模型）
│   └── NAV（净资产价值法）
└── 相对估值法
    ├── PE（市盈率）
    ├── PB（市净率）
    ├── PS（市销率）
    └── EV/EBITDA（企业价值倍数）
```

## 二、绝对估值法详解

### 2.1 DCF模型

$$V = \sum_{t=1}^{n} \frac{FCFF_t}{(1+WACC)^t} + \frac{TV}{(1+WACC)^n}$$

核心假设：企业价值等于未来自由现金流（FCFF）的现值总和。

### 2.2 DDM模型

三种形态：
- **零增长**：$V = D/r$
- **固定增长**（戈登模型）：$V = D_1/(r-g)$
- **三阶段**：高速增长 + 过渡增长 + 稳定增长

### 2.3 NAV模型

$$NAV = \text{资产公允价值} - \text{负债公允价值}$$

适用于资产主导型企业（房地产、资源股），需调整表外资产（专利、品牌价值）。

## 三、相对估值法详解

### 3.1 PE（市盈率）

$$PE = \frac{\text{股价}}{\text{EPS}}$$

分动态PE（TTM）、静态PE、前瞻PE。

### 3.2 PB（市净率）

$$PB = \frac{\text{股价}}{\text{BPS}}$$

低于1倍可能存在破净机会。

### 3.3 PS（市销率）

$$PS = \frac{\text{股价}}{\text{SPS}}$$

适合亏损或盈利不稳定企业。

### 3.4 EV/EBITDA

$$EV/EBITDA = \frac{EV}{EBITDA}$$

剔除资本结构和折旧政策影响，适合跨行业对比。

## 四、Python实现示例

### 4.1 FCFF计算

```python
def calculate_fcff(ebit, tax_rate, depreciation, amortization, capex, nwc_change):
    return ebit * (1 - tax_rate) + depreciation + amortization - capex - nwc_change
```

### 4.2 WACC计算

```python
def calculate_wacc(equity_value, debt_value, cost_equity, cost_debt, tax_rate):
    total = equity_value + debt_value
    we = equity_value / total
    wd = debt_value / total
    return we * cost_equity + wd * cost_debt * (1 - tax_rate)
```

### 4.3 戈登增长模型

```python
def gordon_model(dividend_next_year, cost_equity, growth_rate):
    if growth_rate >= cost_equity:
        raise ValueError("增长率必须小于股权成本")
    return dividend_next_year / (cost_equity - growth_rate)
```

## 五、WACC公式展开

$$WACC = w_e \cdot r_e + w_d \cdot r_d \cdot (1-T_c)$$

其中 $r_e$ 用CAPM模型计算：

$$r_e = r_f + \beta \cdot (r_m - r_f)$$

## 六、终端价值合理性检验

要求永续增长率 $g < WACC$。

**示例**：FCFF第10年为1000万，WACC=12%，g=3%

$$TV = \frac{1000 \times 1.03}{0.12 - 0.03} = 11444.44 \text{万}$$

## 七、估值方法选择树

估值最常见的错误，不是公式不会用，而是把不适合的公式套到错误的企业上。估值前先判断企业类型，再选择主模型和交叉验证模型。

| 企业类型 | 优先方法 | 辅助验证 | 关键关注点 |
|---|---|---|---|
| 稳定现金流企业 | DCF、DDM | PE、EV/EBITDA | 分红可持续性、自由现金流质量、折现率 |
| 高成长但尚未成熟企业 | DCF三阶段、PS | 单位经济模型、可比公司 | 增长率衰减、获客成本、盈利拐点 |
| 周期行业 | NAV、PB、EV/EBITDA | 周期底部/顶部利润复原 | 不要用景气高点利润估“便宜” |
| 金融企业 | PB、ROE、DDM | 资本充足率、拨备质量 | 资产质量、杠杆、监管资本 |
| 资源与地产 | NAV、重估价值 | PB、现金流覆盖 | 资产公允价值、负债期限、流动性 |
| 平台型企业 | DCF、PS、EV/EBITDA | 留存率、网络效应指标 | 用户质量、变现率、竞争格局 |
| 并购或重组标的 | EV/EBITDA、NAV | 协同价值、清算价值 | 交易确定性、债务、少数股东权益 |

一个实用顺序是：

1. 先判断企业赚的是经营现金流、资产重估、周期弹性、金融杠杆还是平台网络效应。
2. 再选一个主模型，只让它回答一个核心问题。
3. 用另一类模型做交叉验证，例如 DCF 给价值区间，PE/PB 看市场定价分位。
4. 把最乐观、基准、悲观三种假设全部写出来，不允许只保留自己喜欢的结果。
5. 最后把估值区间连接到[[综合金融知识体系/投资政策书IPS与交易计划模板|交易计划]]：买入条件、仓位上限、退出条件和复盘触发器。

## 八、三情景估值与安全边际

优秀投资者很少说“这家公司值100元”，更常说“在这些假设下，它大概值80到120元；如果关键假设错了，价值可能只有55元”。估值的目标是给决策提供区间，而不是制造精确感。

| 情景 | 需要回答的问题 | 典型处理 |
|---|---|---|
| 悲观情景 | 如果行业竞争恶化、利润率下降、融资变贵，价值还剩多少？ | 降低增长率、提高折现率、压低利润率 |
| 基准情景 | 管理层正常执行、行业没有重大意外时，合理价值是多少？ | 使用可解释的行业中位假设 |
| 乐观情景 | 如果竞争优势扩大、现金流改善，价值上限在哪里？ | 谨慎提高增长率和利润率，但保持终端假设保守 |

安全边际不是“估值低就买”，而是“即使部分假设错了，组合仍有足够缓冲”。实际操作中可以把安全边际拆成四层：

- 价格安全边际：买入价低于基准价值区间。
- 质量安全边际：企业有真实现金流、稳健资产负债表和可持续竞争优势。
- 组合安全边际：单一标的错误不会伤害整体账户。
- 行为安全边际：你能在回撤和噪音中继续执行原计划。

## 九、从估值到交易决策

估值不能直接替代交易决策。估值回答“值多少钱”，交易还要回答“什么时候买、买多少、错了怎么办”。

| 估值结论 | 交易含义 | 仍需确认 |
|---|---|---|
| 明显低估 | 可能进入研究池或观察仓 | 是否存在价值陷阱、催化剂和流动性问题 |
| 合理偏低 | 可以等待更好价格或分批建仓 | 安全边际是否足够覆盖判断错误 |
| 合理偏高 | 不急于买入，重点观察基本面变化 | 是否有质量溢价或长期复利能力 |
| 明显高估 | 谨慎追高，已有仓位需检查仓位风险 | 高估是否由稀缺性、政策或流动性推高 |

估值复盘时至少记录以下问题：

- 最终收益来自估值修复、业绩增长、分红回购、行业贝塔，还是市场情绪？
- 买入时最大的反对意见是什么，后来是否发生？
- 估值模型中哪个假设最敏感，它是否被后续数据证伪？
- 如果重新做一次，仓位、买入节奏和退出规则会如何改变？

常见估值陷阱：

- 用单点估值代替估值区间。
- 把周期顶点利润当成常态利润。
- 忽略股权稀释、资本开支和营运资本占用。
- 把低 PE、低 PB 直接等同于低风险。
- 终端增长率过高，导致绝大部分价值来自不可验证的远期假设。
- 用“市场会重估”替代清晰催化剂和持有期限。

## 实战掌握清单

> [!tip] 交易者视角
> 七种内在价值评估方法 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。估值不是算出一个精确数字，而是在现金流、增长、折现率、资本结构和市场情绪之间建立合理区间。

### 关键判断

- 确认企业处于成长期、成熟期、周期底部还是困境修复。
- 选择与商业模式匹配的估值方法，而不是机械套PE或DCF。
- 拆分利润质量、现金流质量、资本开支和负债约束。

### 落地动作

1. 建立保守、基准、乐观三种情景，并做敏感性分析。
2. 把绝对估值与相对估值交叉验证。
3. 估值结论必须连接到仓位、买入区间、退出区间和安全边际。

### 失效边界

- 用高增长假设掩盖商业模式问题。
- 忽略周期顶部的低估值陷阱。
- 把会计利润当作可分配现金流。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
