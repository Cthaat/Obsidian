# 七种内在价值评估方法

> [!note] 💡 概念解析
> 本文系统解析量化投资中7种核心内在价值评估方法，包括绝对估值法（DCF、DDM、NAV）和相对估值法（PE、PB、PS、EV/EBITDA），构建从理论到实操的完整评估体系。

## 一、估值方法分类体系

```
内在价值评估方法
├── 绝对估值法
│   ├── DCF（现金流折现模型）
│   ├── DDM（股利贴现模型）
│   └── NAV（净资产价值法）
└── 相对估值法
    ├── PE（市盈率）
    ├── PB（市净率）
    ├── PS（市销率）
    └── EV/EBITDA（企业价值倍数）
```

## 二、绝对估值法详解

### 2.1 DCF模型

$$V = \sum_{t=1}^{n} \frac{FCFF_t}{(1+WACC)^t} + \frac{TV}{(1+WACC)^n}$$

核心假设：企业价值等于未来自由现金流（FCFF）的现值总和。

### 2.2 DDM模型

三种形态：
- **零增长**：$V = D/r$
- **固定增长**（戈登模型）：$V = D_1/(r-g)$
- **三阶段**：高速增长 + 过渡增长 + 稳定增长

### 2.3 NAV模型

$$NAV = \text{资产公允价值} - \text{负债公允价值}$$

适用于资产主导型企业（房地产、资源股），需调整表外资产（专利、品牌价值）。

## 三、相对估值法详解

### 3.1 PE（市盈率）

$$PE = \frac{\text{股价}}{\text{EPS}}$$

分动态PE（TTM）、静态PE、前瞻PE。

### 3.2 PB（市净率）

$$PB = \frac{\text{股价}}{\text{BPS}}$$

低于1倍可能存在破净机会。

### 3.3 PS（市销率）

$$PS = \frac{\text{股价}}{\text{SPS}}$$

适合亏损或盈利不稳定企业。

### 3.4 EV/EBITDA

$$EV/EBITDA = \frac{EV}{EBITDA}$$

剔除资本结构和折旧政策影响，适合跨行业对比。

## 四、Python实现示例

### 4.1 FCFF计算

```python
def calculate_fcff(ebit, tax_rate, depreciation, amortization, capex, nwc_change):
    return ebit * (1 - tax_rate) + depreciation + amortization - capex - nwc_change
```

### 4.2 WACC计算

```python
def calculate_wacc(equity_value, debt_value, cost_equity, cost_debt, tax_rate):
    total = equity_value + debt_value
    we = equity_value / total
    wd = debt_value / total
    return we * cost_equity + wd * cost_debt * (1 - tax_rate)
```

### 4.3 戈登增长模型

```python
def gordon_model(dividend_next_year, cost_equity, growth_rate):
    if growth_rate >= cost_equity:
        raise ValueError("增长率必须小于股权成本")
    return dividend_next_year / (cost_equity - growth_rate)
```

## 五、WACC公式展开

$$WACC = w_e \cdot r_e + w_d \cdot r_d \cdot (1-T_c)$$

其中 $r_e$ 用CAPM模型计算：

$$r_e = r_f + \beta \cdot (r_m - r_f)$$

## 六、终端价值合理性检验

要求永续增长率 $g < WACC$。

**示例**：FCFF第10年为1000万，WACC=12%，g=3%

$$TV = \frac{1000 \times 1.03}{0.12 - 0.03} = 11444.44 \text{万}$$

## 课程化学习补充

> [!important] 学习定位
> 估值是给未来现金流定价，不是寻找精确数字；优秀交易者关注区间、假设弹性和安全边际。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 核心假设是否可解释
- 折现率/增长率是否做敏感性分析
- 是否与行业可比公司交叉验证
- 是否考虑资本周期和会计质量

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用单点估值制造虚假确定性
- 忽略周期高点利润
- 把低估值误认为低风险
- 现金流口径前后不一致

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `七种内在价值评估方法` 时，不要只把它当成孤立知识点。把估值当成假设压力测试，不把任何单一模型当成真实价值本身。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 财报质量和现金流口径
- 利率、折现率和资本周期
- 行业可比公司与历史分位
- 安全边际、仓位和催化剂

### 进阶训练

1. 给同一家公司做 DCF、PE/PB 和情景估值三张表
2. 把增长率、毛利率、WACC 做敏感性矩阵
3. 写出估值错误时的退出规则

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
