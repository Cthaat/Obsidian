# EV/EBITDA估值法

> [!note] 💡 概念解析
> EV/EBITDA是企业价值倍数估值法，排除融资决策（利息、负债）和会计政策（折旧、摊销、税率）的影响，纯粹从运营效率角度衡量公司价值，是跨行业比较的"净化器"。

## 一、计算公式

$$EV/EBITDA = \frac{\text{企业价值(EV)}}{\text{息税折旧摊销前利润(EBITDA)}}$$

### 企业价值（EV）

$$EV = \text{市值} + \text{总负债} - \text{现金及现金等价物}$$

EV代表收购整个公司需要支付的总价格，比市值更全面地反映公司价值。

### EBITDA

$$EBITDA = \text{净利润} + \text{利息} + \text{所得税} + \text{折旧} + \text{摊销}$$

EBITDA大致代表主营业务的现金流，剔除了资本结构和折旧政策的影响。

## 二、与PE的对比

| 比较维度 | PE | EV/EBITDA |
|---------|-----|-----------|
| 分子 | 股权价值（市值） | 企业价值（EV） |
| 分母 | 净利润 | EBITDA |
| 资本结构影响 | 受影响 | 不受影响 |
| 折旧政策影响 | 受影响 | 不受影响 |
| 税率影响 | 受影响 | 不受影响 |
| 跨国可比性 | 较差 | 较好 |

## 三、适用场景

### 最适合
- **资本密集型行业**：航空、电信、能源、制造业
- **高负债行业**：剔除了资本结构差异
- **折旧摊销重的行业**：核电、酒店、物业出租
- **跨行业/跨国比较**：剔除了税率、资本结构等干扰因素
- **并购定价**：最早用作收购兼并的定价标准

### 不适用
- 轻资产、低资本支出的公司
- 需要大量资本开支维持运营的公司（可能过于"美化"盈利）

> [!warning] 注意事项
> - EBITDA**不等于**真正的现金流，忽略了营运资本变动
> - 计算相对复杂
> - 与PE一样不能体现企业未来的成长性
> - 需结合其他估值法使用

## 四、操作方法

$$\text{公司股权价值} = \text{可比公司EV/EBITDA} \times \text{公司EBITDA} - \text{债务价值}$$

> [!example] 计算示例
> 公司D：市值2000万，净债务500万，现金100万，EBITDA 500万
> - EV = 2000 + 500 - 100 = 2400万
> - EV/EBITDA = 2400/500 = 4.8
> - 若同行业平均EV/EBITDA为8，则公司D可能被低估

## 课程化学习补充

> [!important] 学习定位
> 估值是给未来现金流定价，不是寻找精确数字；优秀交易者关注区间、假设弹性和安全边际。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 核心假设是否可解释
- 折现率/增长率是否做敏感性分析
- 是否与行业可比公司交叉验证
- 是否考虑资本周期和会计质量

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用单点估值制造虚假确定性
- 忽略周期高点利润
- 把低估值误认为低风险
- 现金流口径前后不一致

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
