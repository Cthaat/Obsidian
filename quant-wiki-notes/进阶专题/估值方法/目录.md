## 核心公式速查

| 方法 | 公式 |
|------|------|
| DCF | $V = \sum \frac{FCF_t}{(1+WACC)^t} + \frac{TV}{(1+WACC)^n}$ |
| DDM | $V = \frac{D_1}{r-g}$ |
| PE | $PE = \frac{\text{股价}}{\text{EPS}}$ |
| PB | $PB = \frac{\text{股价}}{\text{BPS}}$ |
| PEG | $PEG = \frac{PE}{G}$ |
| EV/EBITDA | $\frac{EV}{EBITDA}$ |
| WACC | $WACC = w_e r_e + w_d r_d(1-T_c)$ |
