# 估值方法对比与选择指南

> [!note] 💡 概念解析
> 估值方法主要分为绝对估值法和相对估值法两大类。没有一把钥匙能打开所有的锁，聪明的投资者会根据不同的公司灵活选用甚至组合使用多种估值工具。

## 一、估值方法全景图

### 1.1 绝对估值法

基于公司自身未来现金回报来估算内在价值：

| 方法 | 核心逻辑 | 适用场景 |
|------|---------|---------|
| DCF | 未来自由现金流折现 | 现金流稳定的成熟企业 |
| DDM | 未来股利折现 | 分红稳定的公用事业/金融 |
| NAV | 净资产公允价值 | 房地产、资源股 |

### 1.2 相对估值法

通过可比公司比率分析来估值：

| 方法 | 公式 | 适用场景 |
|------|------|---------|
| PE | 市值/净利润 | 盈利稳定的企业 |
| PB | 市值/净资产 | 重资产行业 |
| PEG | PE/增长率 | 高成长企业 |
| PS | 市值/销售收入 | 亏损但营收增长的企业 |
| EV/EBITDA | 企业价值/EBITDA | 跨行业/跨国比较 |
| PCF | 市值/经营现金流 | 现金流充裕的企业 |

## 二、绝对估值法 vs 相对估值法

| 比较维度 | 绝对估值法 | 相对估值法 |
|---------|-----------|-----------|
| 思路 | 计算独立价值 | 与可比公司比较 |
| 精确度 | 理论上更精确 | 快速但依赖市场有效性 |
| 数据需求 | 大量预测数据 | 历史/当前数据 |
| 主观性 | 高（假设敏感） | 中（依赖可比选择） |
| 操作难度 | 高 | 低 |
| 核心优势 | 揭示内在价值 | 简单快速 |
| 核心劣势 | "Garbage In, Garbage Out" | "泡沫传染"风险 |

> [!tip] 三角测量法
> 专业分析师使用多种方法（三角测量）来估计合理的价值范围，而非依赖单一模型。建议至少用2种方法交叉验证。

## 三、估值方法选择决策树

```
公司是否盈利稳定？
├── 是 → 是否重资产？
│   ├── 是 → PB + DCF
│   └── 否 → PE + DCF
├── 否 → 是否有营收？
│   ├── 是 → PS + PEG（如有增长）
│   └── 否 → 创业投资估值法 / 用户指标
└── 周期性？→ PB（周期底部）+ PE（周期位置）
```

## 四、比较估值法的底层逻辑

比较估值法基于金融学的**一价定律**——相同的资产应该有相同的价格。现实中没有完全相同的企业，因此该方法是一价定律的应用变体：**相似的资产应该有相似的估值**。

> [!warning] 泡沫传染风险
> 比较估值法依赖于假设市场是有效的。如果整个市场都被高估了，那么比较估值法得出的目标公司估值也同样被高估——这就是"泡沫传染"。

## 五、格雷厄姆估值公式

本杰明·格雷厄姆在《聪明的投资者》中的估值公式：

$$V = EPS \times (8.5 + 2G \times 100)$$

其中：
- $V$ = 股票价值
- $EPS$ = 当前每股预期收益
- $G$ = 预期年增长率
- $8.5$ = 业绩稳定增长公司的合理PE

## 六、特殊估值方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 重置成本法 | 重建相同企业需要的成本 | 资产密集型行业 |
| 清算价值法 | 所有资产变卖后的价值 | 破产/解散企业 |
| 博克斯估值法 | 对管理团队、盈利模式等逐项估值 | 初创企业 |
| RNAV法 | 重估净资产（物业面积×均价-净负债）/总股本 | 房地产 |
| 创业投资估值法 | 基于退出时价值倒推当前价值 | 新兴轻资产行业 |

## 实战掌握清单

> [!tip] 交易者视角
> 估值方法对比与选择指南 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。估值不是算出一个精确数字，而是在现金流、增长、折现率、资本结构和市场情绪之间建立合理区间。

### 关键判断

- 确认企业处于成长期、成熟期、周期底部还是困境修复。
- 选择与商业模式匹配的估值方法，而不是机械套PE或DCF。
- 拆分利润质量、现金流质量、资本开支和负债约束。

### 落地动作

1. 建立保守、基准、乐观三种情景，并做敏感性分析。
2. 把绝对估值与相对估值交叉验证。
3. 估值结论必须连接到仓位、买入区间、退出区间和安全边际。

### 失效边界

- 用高增长假设掩盖商业模式问题。
- 忽略周期顶部的低估值陷阱。
- 把会计利润当作可分配现金流。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
