# 估值方法对比与选择指南

> [!note] 💡 概念解析
> 估值方法主要分为绝对估值法和相对估值法两大类。没有一把钥匙能打开所有的锁，聪明的投资者会根据不同的公司灵活选用甚至组合使用多种估值工具。

## 一、估值方法全景图

### 1.1 绝对估值法

基于公司自身未来现金回报来估算内在价值：

| 方法 | 核心逻辑 | 适用场景 |
|------|---------|---------|
| DCF | 未来自由现金流折现 | 现金流稳定的成熟企业 |
| DDM | 未来股利折现 | 分红稳定的公用事业/金融 |
| NAV | 净资产公允价值 | 房地产、资源股 |

### 1.2 相对估值法

通过可比公司比率分析来估值：

| 方法 | 公式 | 适用场景 |
|------|------|---------|
| PE | 市值/净利润 | 盈利稳定的企业 |
| PB | 市值/净资产 | 重资产行业 |
| PEG | PE/增长率 | 高成长企业 |
| PS | 市值/销售收入 | 亏损但营收增长的企业 |
| EV/EBITDA | 企业价值/EBITDA | 跨行业/跨国比较 |
| PCF | 市值/经营现金流 | 现金流充裕的企业 |

## 二、绝对估值法 vs 相对估值法

| 比较维度 | 绝对估值法 | 相对估值法 |
|---------|-----------|-----------|
| 思路 | 计算独立价值 | 与可比公司比较 |
| 精确度 | 理论上更精确 | 快速但依赖市场有效性 |
| 数据需求 | 大量预测数据 | 历史/当前数据 |
| 主观性 | 高（假设敏感） | 中（依赖可比选择） |
| 操作难度 | 高 | 低 |
| 核心优势 | 揭示内在价值 | 简单快速 |
| 核心劣势 | "Garbage In, Garbage Out" | "泡沫传染"风险 |

> [!tip] 三角测量法
> 专业分析师使用多种方法（三角测量）来估计合理的价值范围，而非依赖单一模型。建议至少用2种方法交叉验证。

## 三、估值方法选择决策树

```
公司是否盈利稳定？
├── 是 → 是否重资产？
│   ├── 是 → PB + DCF
│   └── 否 → PE + DCF
├── 否 → 是否有营收？
│   ├── 是 → PS + PEG（如有增长）
│   └── 否 → 创业投资估值法 / 用户指标
└── 周期性？→ PB（周期底部）+ PE（周期位置）
```

## 四、比较估值法的底层逻辑

比较估值法基于金融学的**一价定律**——相同的资产应该有相同的价格。现实中没有完全相同的企业，因此该方法是一价定律的应用变体：**相似的资产应该有相似的估值**。

> [!warning] 泡沫传染风险
> 比较估值法依赖于假设市场是有效的。如果整个市场都被高估了，那么比较估值法得出的目标公司估值也同样被高估——这就是"泡沫传染"。

## 五、格雷厄姆估值公式

本杰明·格雷厄姆在《聪明的投资者》中的估值公式：

$$V = EPS \times (8.5 + 2G \times 100)$$

其中：
- $V$ = 股票价值
- $EPS$ = 当前每股预期收益
- $G$ = 预期年增长率
- $8.5$ = 业绩稳定增长公司的合理PE

## 六、特殊估值方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 重置成本法 | 重建相同企业需要的成本 | 资产密集型行业 |
| 清算价值法 | 所有资产变卖后的价值 | 破产/解散企业 |
| 博克斯估值法 | 对管理团队、盈利模式等逐项估值 | 初创企业 |
| RNAV法 | 重估净资产（物业面积×均价-净负债）/总股本 | 房地产 |
| 创业投资估值法 | 基于退出时价值倒推当前价值 | 新兴轻资产行业 |

## 课程化学习补充

> [!important] 学习定位
> 估值是给未来现金流定价，不是寻找精确数字；优秀交易者关注区间、假设弹性和安全边际。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 核心假设是否可解释
- 折现率/增长率是否做敏感性分析
- 是否与行业可比公司交叉验证
- 是否考虑资本周期和会计质量

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用单点估值制造虚假确定性
- 忽略周期高点利润
- 把低估值误认为低风险
- 现金流口径前后不一致

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
