# 可转债量化选债系统

> [!note] 简介
> 开源可转债量化选债系统，通过获取实时数据、计算关键指标、实施筛选策略来辅助投资决策。使用Akshare免费数据源。

## 功能特点

- 获取全市场可转债实时数据（Akshare）
- 计算关键指标：转股溢价率、到期收益率、转股价值、纯债价值等
- 多种选债策略
- 风险过滤和流动性筛选
- 汇总报告生成（Markdown格式）
- 结果导出为CSV文件

## 环境要求

- Python 3.8+
- 依赖库：pandas, numpy, akshare, requests, scipy

## 六大策略

| 策略 | 命令 | 适用场景 |
| --- | --- | --- |
| 双低策略 | `double_low_filter` | 震荡市，攻守兼备 |
| 高YTM策略 | `high_ytm_filter` | 熊市防守 |
| 下修博弈 | `xiaxiu` | 博弈下修机会 |
| 卡叔2026 | `kashu2026` | 多维度筛选 |
| 盛唐估值 | `shengtang` | BS模型估值选低估 |
| 综合评分 | `composite` | 默认策略 |

### 卡叔2026策略四大类
1. **小市值高溢价**：博弈下修
2. **绝对防御临期债**：低期权价值、质地安全
3. **大股东未减持**：可能先下修再减持
4. **存续期2年附近**：上市公司开始重视促转股

### 盛唐估值模型
```
转债估值 = max(债底,回售价值) + 转股期权价值 + 下修期权价值 - 强赎损失
```
