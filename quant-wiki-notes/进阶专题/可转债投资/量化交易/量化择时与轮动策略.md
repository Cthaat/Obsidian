# 可转债量化择时、轮动、选券策略

> [!note] 策略框架
> 从宏观转债配置择时、中观转债行业轮动、微观量化择券三个维度构建6个量化小策略，年化收益最高达 **26%**。

## 一、低价策略优化

### 传统低价策略困境
- 信用风险冲击：2024年最大回撤达14%
- 策略容量有限：常低于5亿元

### 改进方案

| 策略 | 年化收益 | 夏普比率 | 策略容量 |
| --- | --- | --- | --- |
| 缝补型（次低价） | 17.35% | 1.2 | 25亿+ |
| 平替型（期权策略） | 14% | 1.5 | 20亿+ |

> 缝补型策略主要通过转债池优化规避风险，收益较传统低价策略提升2倍。

## 二、双低策略优化

### 双低五因子增强策略

除低价、溢价率外，新增三个因子：
- 双低因子历史分位数
- 隐含波动率
- 转债价格动量

| 指标 | 表现 |
| --- | --- |
| 年化收益 | **24%** |
| 较经典双低提升 | +10% |
| 超额最大回撤 | < 9% |

### 四因子中观行业轮动

| 指标 | 表现 |
| --- | --- |
| 年化收益 | **26%** |
| 最大回撤 | < 11% |
| 夏普比率 | > 2 |
| 关注行业 | 传媒、食品饮料 |

## 三、因子说明

| 因子 | 作用 |
| --- | --- |
| 转债动量 | 捕获前期超跌行业均值回归空间 |
| 收益成交因子 | 关注成交活跃但未在价格体现的左侧机会 |
