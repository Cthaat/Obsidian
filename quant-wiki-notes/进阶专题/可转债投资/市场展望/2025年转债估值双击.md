# 2025年可转债期待估值双击

> [!note] 核心观点
> 无风险收益率迈入"1字头"，可转债可能是广义固收类资产中为数不多的"高息资产"。中证转债指数未来一年潜在收益约 **10.8%**。有望复制2021年平价和转股溢价率双击。

## 一、机构一致看多

| 机构 | 观点 |
| --- | --- |
| 信达证券 | 潜在收益率约 10.8% |
| 国海证券 | 有望复制2021年估值双击 |
| 华西证券 | 类权益资产势必得到重视 |
| 招银理财 | 转债估值仍处历史中低水平 |
| 国泰君安 | 供需偏紧趋势持续 |

## 二、供给收缩逻辑

- 80亿元成银转债提前赎回
- 500亿元浦发转债 + 100亿元中信转债即将到期
- 2025年预计有1228亿元转债到期或强赎退市
- 发行规模预计300-400亿元
- 市场总规模可能降至6500亿元

| 变量 | 预测 |
| --- | --- |
| 2025年净供给收缩 | 约 394亿元 |
| 2024年净供给收缩 | 约 1145亿元（斜率放缓） |

## 三、下修博弈支撑估值

- 大量转债即将触发下修条件
- 下修后转股价值提升 → 估值修复
- 配合权益市场回暖 → 平价和溢价率双重提升

## 四、资金面支撑

- 银行理财直投可转债规模较小但增长空间大
- 转债ETF资金持续流入
- 潜在入市资金体量可观

> [!tip] 配置建议
> 当前时点的可转债资产配置价值很高，估值处于历史中低水平，债券机会成本降低，权益结构性机会不断涌现。
