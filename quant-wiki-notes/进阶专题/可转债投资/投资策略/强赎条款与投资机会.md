# 转债强赎条款与投资机会

> [!note] 核心发现
> 2025年权益市场震荡走牛，强赎转股规模显著回升。精准判断强赎机会、提前埋伏（公告前15交易日）或有超七成胜率、8%以上涨幅。

## 强赎条款概述

- 触发条件：正股价在连续30个交易日内累计15个交易日 > 转股价的130%
- 赎回价格：通常约101元（面值附近）
- 2022-2024年转股/强赎退出比例从96%降至82%，到期赎回升至近20%

## 强赎标的特征

| 维度 | 特征 |
| --- | --- |
| 行业 | 电力设备、电子、计算机占比近1/4 |
| 评级 | AA-、AA、A+ 合计占比近7成 |
| 规模 | 余额3亿元以下占比过半 |
| 期限 | 2024-2025年四成强赎券于2年内到期 |

## 强赎博弈策略

### 判断强赎概率的关键指标

| 因素 | 强赎组特征 | 不强赎组特征 |
| --- | --- | --- |
| 未转股比例 | 44.1%（较低） | 64.5% |
| 大股东持债比例 | 28.8%（较低） | 31.9% |
| 评级 | A-以下 100% 强赎 | AA以上 48% |
| 平均余额 | 6.6亿 | 4.6亿 |
| 剩余期限 | 2.6年 | 3.4年 |
| 正股市值 | <50亿小市值偏高 | >100亿偏低 |

### 策略应用

> [!tip] 重点关注
> - 未转股比例 < 50%
> - 股权稀释率 < 8%
> - 大股东及管理层未减持完毕
> - 余额较小（< 3亿）
> - 即将到期（2年内）
> - 正股公司市值 < 50亿

> [!warning] 风险提示
> 强赎公告次一交易日转债平均跌幅 **2%-3%**，需注意把握退出时机。
