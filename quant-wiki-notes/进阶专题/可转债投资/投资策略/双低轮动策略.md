# 双低轮动策略

> [!note] 策略定义
> 双低轮动策略是基于低价格和低转股溢价率的量化投资方法，通过定期筛选并轮动持有符合条件的转债，平衡风险与收益。

## 双低指标计算

- **双低值** = 可转债价格 + 转股溢价率 × 100
- 转股溢价率 = (转债价格 / 转股价值 - 1) × 100%
- 转股价值 = 100元 / 当前转股价 × 股价

## 操作步骤

### 1. 筛选标的
- 数据来源：集思录或宁稳网
- 按"双低值"排序

**排除条件**：

| 排除项 | 原因 |
| --- | --- |
| 已公告强赎的转债 | 带感叹号标识 |
| 剩余期限 < 1年 | 期权价值低 |
| PB < 1 的转债 | 无下修空间（净资产限制） |
| 可交换债（EB） | 不同品种 |

### 2. 构建组合

| 市场转债数量 | 持仓数 | 仓位分配 |
| --- | --- | --- |
| < 100只 | 5只 | 每只20% |
| 100-200只 | 10只 | 每只10% |
| > 300只 | 20只 | 每只5% |

### 3. 定期轮动
- 周期：通常每月一次
- 波动大时可缩短至半月
- 卖旧买新：卖出不再属于双低排名的持仓，买入新标的

## 风控措施

- 单只债持仓 ≤ 100万（防流动性风险）
- 分散行业，避免过度集中
- 关注信用评级变化
