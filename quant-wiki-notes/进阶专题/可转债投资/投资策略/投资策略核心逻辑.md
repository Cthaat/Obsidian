# 可转债投资策略核心逻辑与实操要点

> [!note] 六大策略全景
> 可转债投资可从六大维度切入：下修博弈、强赎防守、热门题材、惰性债、双低策略、小规模妖债，构建攻守兼备的投资组合。

## 一、下修博弈策略 📉

**核心逻辑**：买价格≤110元、债底扎实的转债，博公司宣布下修转股价。

- 目的：公司下修后，转股价值跳升至100元 + 20%溢价率 → 价格涨至120元
- **关键筛选条件**：
  - 大股东持债比例高
  - 有减持动机（上市1年内 + 大股东持债 > 50%）
  - 债底无评级下调风险

> [!warning] 风险警示
> 2023年利元转债下修失败，价格从117元暴跌至103元。

## 二、强赎防守策略 🔐

**核心逻辑**：高负债率、曾下修过的转债强赎意愿强。YTM > 10% 时机构会当高息债持有。

| 指标 | 数据 |
| --- | --- |
| 历史强赎退市比例 | 80% |
| 历史违约比例 | 1%-2% |

- 避开临期债（剩余 < 1年）
- 关注强赎价极低（如100.38元）的转债

## 三、热门板块策略 🚀

**逻辑**：半导体/AI/人形机器人等热门板块转债，YTM > 1%即有「防守+进攻」属性。

- 2025年案例：AI芯片相关转债涨幅超50%
- 风口：杭州千亿基金押注AI，华为昇腾芯片推动人形机器人成本降40%

## 四、惰性债策略 💤

**适用场景**：利率下行期，固收资金急需高息资产。

- 条件：基本面安全 + 纯债YTM > 3%-4%
- 风险：避开评级B/C级的弱资质债
- 彩蛋：突然下修即为意外收益

## 五、双低策略 ⚖️

| 条件 | 值 |
| --- | --- |
| 价格 | ≤ 110元 |
| 溢价率 | ≤ 25% |
| 2024年收益 | 17.6% |

操作技巧：网格条件单自动高抛（涨1%卖，换入跌1%的新标的）

## 六、小规模妖债策略 🎭

- 条件：剩余规模 2-4亿
- 逻辑：容易被游资炒作成"妖债"
- 案例：东时转债（规模<1亿）因炒作潜力大，价格长期抗跌
