# 转债市场信用冲击风险可控

> [!note] 核心结论
> 转债正股退市风险整体可控，当前平价支撑较强，2025年内到期转债偿债压力有限，预计2025年转债市场信用冲击大概率不会重演。

## 一、2024年信用冲击复盘

**评级下调只是诱因**，深层次原因：
- 权益市场2022-2024年持续下跌
- 转债平价整体处于低位
- 退市规则趋严 + 个别信用风险暴露
- 正股上涨预期趋弱
- 大量转债即将进入回售期或到期
- 当平价和债底均无法支撑价格 → 定价锚失效

## 二、2025年为何不会重演

| 理由 | 说明 |
| --- | --- |
| 退市风险可控 | 正股退市风险整体可控 |
| 平价支撑较强 | 权益市场好转 |
| 偿债压力有限 | 2025年内到期压力不大 |
| 投资者认知充分 | 市场对信用冲击有更充分认知 |

## 三、信用风险评估维度

1. 偿债能力指标（流动比率、利息保障倍数）
2. 资本结构（资产负债率、短期债务占比）
3. 外部评级和评级展望
4. 外部支持（大股东实力、政府支持）
5. 是否有条件下修

## 四、风险提示

> [!warning] 关注
> - 理财赎回风险
> - 转股溢价率压缩风险
> - 正股表现不及预期
