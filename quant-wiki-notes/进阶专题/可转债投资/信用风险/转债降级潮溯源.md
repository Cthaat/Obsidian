# 转债降级潮之下：溯源、风险特征与信用展望

> [!note] 核心观点
> 可转债市场经历了"个案化→规模化→常态化"的降级演进路径。2024年搜特转债成为首只实质性违约的转债，打破近30年"零违约"记录。

## 一、历史演进三阶段

| 阶段 | 时间 | 特征 |
| --- | --- | --- |
| 早期 | 2008-2016 | 仅3起评级下调，AA+→AA，个案化微调 |
| 扩容 | 2017-2022 | 政策松绑，AA及以下占比提升，信用分层 |
| 爆发 | 2023至今 | 零违约打破，搜特/鸿达/蓝盾/岭南先后违约 |

## 二、风险特征：数说评级下调

| 维度 | 数据 |
| --- | --- |
| 中低等级集中 | AA-及以下占比 74.85% |
| 民企重灾区 | 民营企业占比 93.20% |
| 多次降级 | 28.83% 发生2次及以上下调 |
| 高发行业 | 基础化工、计算机、医药生物、电力装备 |

### 光伏行业为何未出现大范围降级？
- 头部企业海外市场可分流经营压力
- 技术优势较强、资金储备相对充裕
- 转债剩余转股期限充足

## 三、四类核心风险

| 风险类型 | 主要表现 |
| --- | --- |
| 宏观经济/行业 | 景气度走弱、供给过剩 |
| 经营风险 | 盈利能力恶化、业务集中度过高 |
| 财务风险 | 高负债、短债集中、现金流紧张 |
| 公司治理 | 信披违规、审计非标、股权质押高 |

## 四、市场信用展望

- **整体温和、局部紧张** 的格局
- 高评级转债仍是"稀有资产"（AA+及以上仅占16%只数）
- 2026年进入兑付压力上升期，2027年达到巅峰
- 2025年强赎加速：截至9月已有101只选择强赎
