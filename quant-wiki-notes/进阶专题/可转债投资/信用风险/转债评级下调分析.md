# 转债评级下调怎么看？

> [!note] 核心结论
> 2021-2024年转债市场共发生193次主体评级下调，超61%集中在6月。评级下调后转债价格普遍"先抑后扬"，约1个月后估值触底反弹。

## 一、评级调整特征

| 维度 | 特征 |
| --- | --- |
| 频次 | 2021-2024年共193次下调，占全部调整 4.51% |
| 季节性 | 超61%集中在6月 |
| 行业 | 工业、材料、信息技术、可选消费占比超80% |
| 企业性质 | 民营企业占比超 80% |
| 评级分布 | AA和AA- 合计占比 79.76% |
| 地区 | 广东、江苏、上海、浙江、北京合计 66.67% |

## 二、下调触发因素

1. **公司业绩亏损**：如岭南转债 EBITDA 利润率跌至 -119%
2. **偿债能力不足**：流动比率、利息保障倍数过低
3. **行业景气下行**：地产、钢铁、电力设备承压
4. **公司治理风险**：信息披露违规、财务造假等
5. **流动性压力**：短期债务占比过高、现金净流量过低

## 三、行业分化格局

| 低风险行业 | 驱动因素 |
| --- | --- |
| 计算机 | 净利润同比激增 471.86% |
| 电子 | ROE 提升至 6.27% |
| 有色金属 | 大宗商品周期上行，ROE 达 11.3% |

| 高风险行业 | 风险因素 |
| --- | --- |
| 房地产 | 净利润骤降 274% |
| 钢铁 | ROE 为负，在建工程收缩 42% |
| 电力设备 | ROE 仅 3.6%，资本开支连续收缩 |

## 四、应对策略

### 三重防御体系

| 策略 | 操作 | 示例 |
| --- | --- | --- |
| 偏股型对冲 | 正股弹性对冲信用风险 | 豪24转债 |
| 低价错杀修复 | 把握现金流稳健但被过度压估值的标的 | 晶能转债 |
| 短久期高YTM防御 | 回售条款锁定债底 | 万顺转2 |

> [!tip] 配置建议
> 关注三类机会：景气进攻 + 错杀修复 + 债底防御，形成攻守兼备策略。
