# Alpha因子与量化交易入门

> [!note] 核心概念
> Alpha因子是量化投资中最核心的概念，代表着一个交易策略能够战胜市场平均回报的那部分能力。它源于经典的资本资产定价模型（CAPM）。

## 一、Alpha的由来：从CAPM到量化因子

Alpha源于经典的**资本资产定价模型（CAPM）**：

```
E(R_i) = R_f + β_i × (E(R_m) - R_f) + α_i
```

- `R_f`：无风险收益率（如国债）
- `β_i`：资产对市场风险的敏感度
- `R_m`：市场整体平均收益
- `β_i × (E(R_m) - R_f)`：承担市场风险的补偿
- `α_i`：无法由市场波动解释的"额外回报"

## 二、从"阿尔法"到"因子"

在量化投资中，我们说"找到Alpha因子"而非"赚Alpha"。**因子**是能系统性解释资产收益差异的变量：

| 因子类型 | 示例 | 逻辑 |
|---------|------|------|
| 估值类 | PE、PB | 买入便宜股票 |
| 动量类 | 过去12个月收益率 | 延续趋势 |
| 成长类 | 营收增长率、利润增长率 | 投资高增长公司 |
| 质量类 | ROE、毛利率、负债率 | 买入好公司 |
| 情绪类 | 散户关注度、新闻情绪 | 捕捉市场情绪 |

## 三、Alpha因子的构建流程

1. **定义假设** — 例如：低估值股票未来可能跑赢高估值股票
2. **提取数据** — 从财报、行情、舆情、新闻中清洗数据
3. **构建指标** — 用数学方式量化，如 PE = 股价/每股收益
4. **测试有效性** — 在历史数据上检验因子的选股能力
5. **组合与中性化** — 对行业、规模、市值进行中性化

## 四、因子评估指标

- **IC（信息系数）**：因子值与未来收益的相关系数
- **IR（信息比率）**：IC的均值/IC的标准差
- **分层收益**：按因子值分组后的收益差异
- **换手率**：因子组合的交易频率
