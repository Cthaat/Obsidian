# 配对交易协整理论

> [!note] 协整理论
> 协整理论是配对交易的数学基础，用于检验两个非平稳时间序列之间是否存在长期均衡关系。

## 什么是协整

### 单整与平稳
- **平稳序列**：均值和方差不随时间变化
- **I(1)序列**：一阶差分后平稳
- **I(0)序列**：本身就是平稳的

### 协整定义
如果两个I(1)序列的线性组合是I(0)的，则称这两个序列协整。

```
若 X_t ~ I(1), Y_t ~ I(1)
且 Z_t = X_t - β*Y_t ~ I(0)
则 X_t 和 Y_t 协整
```

## 协整检验方法

### 1. Engle-Granger两步法
```python
from statsmodels.tsa.stattools import coint

# 第一步：OLS回归
# Y_t = α + β*X_t + ε_t

# 第二步：对残差进行ADF检验
score, pvalue, _ = coint(Y, X)
```

### 2. Johansen检验
```python
from statsmodels.tsa.vector_ar.vecm import coint_johansen

# Johansen检验可以检验多个变量之间的协整关系
result = coint_johansen(data, det_order=0, k_ar_diff=1)
```

## 协整与配对交易

### 配对选择标准
1. 经济逻辑：同行业、产业链关系
2. 统计检验：协整检验p值<0.05
3. 平稳性：价差序列平稳
4. 半衰期：回归速度适中
