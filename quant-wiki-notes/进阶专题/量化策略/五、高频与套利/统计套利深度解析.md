# 统计套利深度解析

> [!note] 统计套利
> 统计套利（Statistical Arbitrage）是一种量化投资策略，核心是"利用价格的短期偏离历史统计关系，在预期回归时获利"。它基于大数据、概率统计和均值回归。

## 核心原理：协整 + 价差回归

### 基本假设

| 假设 | 说明 |
|-----|------|
| 均值回归 | 相关股票对的价差不会无限偏离历史均值 |
| 协整关系 | 两资产长期存在稳定统计关系 |
| 短期噪音 | 市场情绪、流动性导致短期偏离 |

### 经典模型：配对交易

步骤拆解（以A、B两只股票为例）：

| 步骤 | 操作 | 数学表达 |
|-----|------|---------|
| 1. 选对 | 选择同行业、强相关股票 | 相关系数 > 0.8 |
| 2. 建模 | OLS回归 | P_A = α + β·P_B + ε |
| 3. 价差 | 计算残差 | Spread = P_A - (α + β·P_B) |
| 4. 标准化 | Z-Score | Z = (Spread - μ) / σ |
| 5. 信号 | 阈值触发 | Z > +2 卖A买B，Z < -2 买A卖B |
| 6. 平仓 | 回归均值 | Z回归0时对冲平仓 |

## 三大数学支柱

| 支柱 | 工具 | 作用 |
|-----|------|------|
| 协整检验 | Engle-Granger、Johansen | 确认长期均衡关系 |
| 平稳性检验 | ADF检验 | 确保Spread是I(0) |
| 风险建模 | GARCH、OU过程 | 预测回归速度与半衰期 |

## OU过程建模

```
dS = κ(μ - S)dt + σdW
```

- κ：回归速度
- μ：长期均值
- σ：波动率
- 半衰期 = ln(2)/κ
