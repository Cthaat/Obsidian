# 均值回归配对交易

> [!note] 配对交易
> 配对交易是均值回归策略的经典应用，通过交易两只高度相关股票的价差来获利。核心是找到价差的均值回归特性。

## 配对交易原理

1. 找到两只历史走势高度相关的股票（如茅台和五粮液）
2. 当它们的价格差偏离正常范围时
3. 买入被低估的，卖空被高估的
4. 等待价差回归均值

## Python实现

### 1. 选择配对
```python
import statsmodels.api as sm

# 协整检验
from statsmodels.tsa.stattools import coint
score, pvalue, _ = coint(stock_a, stock_b)
print(f'协整检验p值: {pvalue}')  # p值<0.05表示存在协整关系
```

### 2. 计算价差
```python
# OLS回归确定对冲比例
model = sm.OLS(stock_a, sm.add_constant(stock_b)).fit()
hedge_ratio = model.params[1]

# 计算价差
spread = stock_a - hedge_ratio * stock_b
```

### 3. 生成信号
```python
# Z-Score标准化
zscore = (spread - spread.mean()) / spread.std()

# 交易信号
longs = zscore < -2   # 价差过小，做多A做空B
shorts = zscore > 2   # 价差过大，做空A做多B
exits = abs(zscore) < 0.5  # 平仓
```

## 关键指标

| 指标 | 说明 | 标准 |
|-----|------|------|
| 相关系数 | 价格相关性 | >0.8 |
| 协整p值 | 协整关系显著性 | <0.05 |
| 半衰期 | 价差回归速度 | 越短越好 |
| ADF检验 | 价差平稳性 | p<0.05 |
