# 量化交易全景图

> [!note] 全景图
> 先建立全局视野，再深入细节。知道量化世界有哪些玩家、哪些策略、为什么需要多智能体架构。

## 量化交易全景

### 量化交易的核心

量化交易核心思想很简单：**将投资理念固化为具体的规则，用历史数据验证其有效性，然后让计算机严格执行，克服人性的弱点。**

### 五大经典策略框架

1. **趋势跟踪** — 右侧交易者
2. **均值回归** — 价格终将回归价值
3. **因子投资** — 寻找股票的"优秀基因"
4. **统计套利** — 捕捉微小的定价偏差
5. **波动率交易** — 在恐惧和贪婪之间套利

## 量化交易的参与者

| 类型 | 代表 | 特点 |
|-----|------|------|
| 量化对冲基金 | Citadel、Two Sigma | 多策略、大规模 |
| 高频交易公司 | Virtu、Jump Trading | 低延迟、高换手 |
| 量化私募 | 九坤、幻方 | A股为主 |
| 公募量化 | 各大基金公司 | 指数增强 |
| 个人量化 | 散户投资者 | 小规模 |

## 量化交易的演进

- **第一代**：技术指标策略
- **第二代**：多因子模型
- **第三代**：机器学习
- **第四代**：AI Agent
