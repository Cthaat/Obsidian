# 量化投资完全指南

> [!note] 完全指南
> 量化投资完全指南，涵盖从入门到进阶的系统性知识，帮助你建立完整的量化投资知识框架。

## 什么是量化投资

量化投资是利用数学模型、统计方法和计算机程序来进行投资决策的方法。核心是将投资理念固化为规则，用数据验证，让计算机执行。

## 量化投资的优势

- **纪律性**：克服人性弱点
- **系统性**：全面覆盖市场机会
- **可验证**：历史数据回测
- **可扩展**：程序化批量处理

## 量化投资的主要策略

| 策略类型 | 核心逻辑 | 代表方法 |
|---------|---------|---------|
| 趋势跟踪 | 追涨杀跌 | MA、布林带 |
| 均值回归 | 物极必反 | RSI、配对交易 |
| 因子投资 | 优中选优 | 多因子模型 |
| 统计套利 | 捕捉偏差 | 协整、价差交易 |
| 高频交易 | 微观结构 | 做市、订单簿 |

## 量化投资流程

```
数据获取 → 因子构建 → 策略设计 → 回测验证 → 风险控制 → 实盘执行
```

## 入门路径

1. 学习金融基础知识
2. 掌握Python编程
3. 理解统计学原理
4. 学习经典策略
5. 实践回测和模拟
6. 逐步进入实盘
