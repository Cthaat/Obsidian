# 量化策略案例分析

> [!note] 本篇定位
> 与其罗列"某机构年化多少"，不如把**几个具体策略从头到尾走一遍**：假设是什么、规则怎么写、回测怎么看、收益归因到哪、什么情况下会失效。下面三个案例（趋势、多因子、配对）覆盖了三大主流逻辑，方法论比结论更重要。所有数字均为**示例**，用于说明分析框架，不代表真实业绩。

## 统一分析框架

```mermaid
graph LR
    A[1 假设] --> B[2 规则]
    B --> C[3 回测]
    C --> D[4 归因]
    D --> E[5 失效场景]
    E --> A
```

| 环节 | 要回答的问题 |
|---|---|
| 假设 | 这个规律为什么可能存在？（经济/行为解释） |
| 规则 | 何时买、何时卖、买多少？ |
| 回测 | 含成本、防未来函数、样本外是否仍有效？ |
| 归因 | 收益来自 beta、因子还是真 alpha？ |
| 失效 | 什么市场环境/结构变化会让它失灵？ |

## 案例一：双均线趋势（趋势跟踪）

- **假设**：价格形成趋势后会延续一段时间（动量效应）。
- **规则**：MA20 上穿 MA60 满仓持有，下穿则空仓；次日开盘执行（`shift(1)`）。
- **回测要点**：在趋势品种（如部分商品/指数）上夏普尚可，但**震荡市反复假突破、来回挨打**。
- **归因**：收益主要来自少数几段大趋势，多数时间小亏磨损（趋势策略的典型"胜率低、盈亏比高"）。
- **失效场景**：长期横盘震荡；参数对均线周期敏感（过拟合风险）。

> [!example] 关键观察
> 趋势策略的体验是"大部分时间难受、靠少数几次大行情赚钱"。扛不住连续小亏的人会在黎明前离场。

## 案例二：低估值多因子选股

- **假设**：便宜（低 PE/PB）+ 质量好（高 ROE）+ 有动量的股票，长期跑赢。
- **规则**：在股票池里对价值、质量、动量因子打分合成，选 Top-N 等权持有，月度调仓。
- **回测要点**：长期有超额，但有明显的**风格周期**（价值因子可能多年跑输）。
- **归因**：用 [[业绩评估与归因]] 的回归拆出 alpha 与因子 beta——很多"超额"其实是因子暴露，不是选股能力。
- **失效场景**：因子拥挤、风格切换、价值陷阱（便宜是因为基本面在恶化）。方法见 [[多因子策略实战]]、[[因子投资体系]]。

## 案例三：配对交易（市场中性）

- **假设**：两只长期协整的同业股票，价差偏离后会回归。
- **规则**：用回归求对冲比率构建价差，z-score > 2 反向开仓，回到 0 附近平仓。
- **回测要点**：市场中性、与大盘相关性低，但**对协整关系的稳定性高度依赖**。
- **归因**：收益来自价差均值回归，而非市场方向；需扣除做空成本与冲击。
- **失效场景**：协整关系破裂（并购、基本面分化、退市）、交易拥挤。详见 [[配对交易策略]]、[[统计套利深度解析]]。

## 三个案例的横向对比

| 维度 | 双均线趋势 | 低估值多因子 | 配对交易 |
|---|---|---|---|
| 逻辑 | 动量延续 | 因子溢价 | 均值回归 |
| 与大盘相关 | 高 | 中高 | 低（中性） |
| 胜率 | 低 | 中 | 较高 |
| 盈亏比 | 高 | 中 | 低 |
| 最怕 | 震荡市 | 风格切换 | 关系破裂 |

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 只看回测收益曲线 | 必须做归因，分清运气、beta 与 alpha |
| 一个策略全天候有效 | 每个策略都有它的"最怕环境" |
| 参数调到最优就上实盘 | 最优参数往往最过拟合，要样本外验证 |
| 高换手不算成本 | 成本能吃掉大半收益 |
| 不写失效条件 | 不知道何时该停，等于裸奔 |

## 课程化学习补充

> [!important] 学习定位
> 量化策略是投资假设、数据工程、回测验证、风险预算和执行系统的组合，不是单一公式。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 假设是否可证伪
- 数据是否 point-in-time
- 绩效是否扣除真实成本
- 上线后是否监控衰减

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 数据挖掘偏差
- 因子拥挤
- 换手过高
- 实盘偏离回测

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
