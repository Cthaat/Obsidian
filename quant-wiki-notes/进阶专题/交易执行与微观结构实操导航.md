# 交易执行与微观结构实操导航

> [!note] 核心问题
> 纸面 alpha 常死在成交价。本篇连接课程 [[市场微观结构与交易执行]] 与系统文 [[交易系统、执行与市场微观结构]]，给出**个人可执行的下单纪律、成本拆解与日志字段**，不要求你做机构算法单。

## 学习目标

1. 区分市价/限价的「成交确定性 vs 价格确定性」。  
2. 拆解成本：佣金税费、价差、滑点、冲击、机会成本。  
3. 在说明书中写清成交假设（回测 vs 实盘）。  
4. 用模拟日志记录计划价与成交价偏差。  
5. 知道 TWAP/VWAP 等算法的适用场景（认知级）。

## 个人投资者最小执行纪律

| 规则 | 说明 |
|---|---|
| 小单优先限价 | 控制价格；接受不成交 |
| 急用市价要接受滑点 | 记录 deviation |
| 避开集合竞价乱下大单（除非策略如此） | 开盘价形成复杂 |
| 低流动性标的降权 | 冲击与无法成交 |
| 回测成本 ≥ 保守 bps | 见研究包敏感性 |
| 涨跌停按不可成交处理 | 假成交是最大谎言 |

## 订单类型决策树（简化）

```text
需要立刻成交？
  是 → 市价/对手价（接受滑点）
  否 → 限价（可能不成交）
能否接受部分成交？
  否 → 了解 FOK/IOC 等（以券商支持为准）
A 股是否涨跌停/停牌？
  是 → 按不可交易处理
```

## 与回测的接线

| 回测假设 | 实盘检查 |
|---|---|
| 收盘价成交 | 是否改为次日开盘/均价并前后一致 |
| 无限量 | 成交量是否支撑 |
| 固定 bps | 小票是否应提高 |
| 忽略排队 | 开盘/尾盘是否特殊 |

填 [[回测与quant-lab对照清单]] 成本/成交节。

## 算法单认知（不必会写代码）

| 算法 | 直觉 | 个人何时关心 |
|---|---|---|
| TWAP | 时间均匀切 | 较大单、怕选时 |
| VWAP | 贴近量加权均价 | 跟踪基准 |
| POV | 占市场成交比例 | 更大单 |
| IS | 最小化实施缺口 | 机构执行台 |

个人小单：先把限价纪律与日志做好，再谈算法。
