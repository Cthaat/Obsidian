# 2026年宏观经济展望

> [!note] 核心要点
> 2025年中央经济工作会议标志中国宏观政策历史性转向：货币政策从"稳健"转向"**适度宽松**"，财政政策"更加积极"。2026年将是中国资产重估的关键之年，主要矛盾从"防御缩表"转向"进攻复苏"。

## 宏观政策新范式

### 货币政策：历史性的转向

**"适度宽松"的信号意义**：
- 2008年应对金融危机以来首次重提
- 政策优先级重置："稳增长"+"对抗通缩"权重大幅提升
- 预期管理工具：传递央行呵护流动性的决心

**2026年工具箱展望**：

| 工具 | 预期操作 |
|------|----------|
| 降准 | 上半年和下半年各1-2次 |
| 降息 | LPR和政策利率显著下调 |
| SFISF扩容 | 支持非银机构获取流动性 |
| 科技再贷款 | 支持"AI+"和设备更新 |
| 住房再贷款 | 支持房地产去库存 |

### 财政政策：突破约束的"更加给力"

**赤字率突破3%常态化**：
- 预计2026年赤字率按**4%**安排
- 赤字规模约5.66-5.9万亿元
- 超长期特别国债发行规模不低于**2万亿元**

**支出结构性转向：从"物"到"人"**：
- 消费端补贴（消费券、以旧换新）
- 民生保障（养老金、医疗保险补助）

### 政策协同："组合拳"的乘数效应

- 消除合成谬误：收缩性政策暂停
- 跨周期与逆周期结合
- 经济政策与非经济性政策统一纳入宏观政策取向一致性评估

## 三大核心战略支柱

### 支柱一：新质生产力与数字经济

#### "人工智能+"行动

| 层面 | 投资机会 |
|------|----------|
| 基础设施层 | 算力中心、国产GPU/NPU、液冷技术 |
| 应用层 | 垂类模型、AI Agent |
| 自主可控 | 信创（数据库、操作系统、办公软件） |

#### 数据要素：第五生产要素

- 数据确权与交易
- 公共数据授权运营
- **数据资产入表**改善资产负债表

#### 信创产业

- 2+8+N体系深化
- 金融、电信行业国产化替代率显著提升

### 支柱二：低空经济

**市场规模**：2025年1.5万亿元 → 2035年3.5万亿元

**产业链投资地图**：

| 环节 | 机会 |
|------|------|
| 基础设施 | 垂直起降场、低空空管系统 |
| 飞行器制造 | eVTOL整机、电池、碳纤维复材 |
| 应用场景 | 物流配送、文旅观光 |

**试点城市**：合肥、杭州、深圳、苏州、成都、重庆

### 支柱三：银发经济

**市场规模**：2035年达19-30万亿元

**细分赛道**：

| 赛道 | 机会 |
|------|------|
| 居家适老化改造 | 智能家居、卫浴建材 |
| 智慧养老 | 穿戴设备、陪伴机器人 |
| 抗衰老医疗 | 慢病管理、消费医疗 |
| 银发旅游 | 康养旅居 |

- **工业属性**：光伏+AI硬件需求

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `2026年宏观经济展望` 时，不要只把它当成孤立知识点。把宏观看作资产配置的背景概率，而不是单笔交易的确定答案。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 增长、通胀、利率和信用周期
- 央行反应函数与财政政策
- 汇率、商品和全球资金流
- 预期差和市场定价

### 进阶训练

1. 把一个宏观观点映射到股票、债券、商品和汇率
2. 区分领先、同步和滞后指标
3. 写出观点被证伪的三个数据条件

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
