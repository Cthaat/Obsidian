# 宏观研究指导投资

> [!note] 核心要点
> 宏观经济研究是投资决策的顶层导航，核心传导链条：**经济周期运行→政策逆周期调控→资产价格重定价**。通过指标跟踪预判周期位置，在市场形成一致预期前完成资产布局。

## 核心传导链条

```
经济增长+通胀+政策 → 盈利预期+贴现率 → 资产价格 → 大类资产轮动
```

## 指标体系（机构实操版）

### 领先指标（前瞻预判）

| 指标 | 投资含义 |
|------|----------|
| 社融增速 | 实体融资需求 |
| M1-M2剪刀差 | 企业活期存款活跃度 |
| 企业中长期贷款 | 投资意愿 |
| 制造业PMI | 产业景气度 |

### 同步指标（验证判断）

| 指标 | 投资含义 |
|------|----------|
| 工业增加值 | 生产端 |
| 用电量 | 经济活跃度 |

### 滞后指标（确认拐点）

| 指标 | 投资含义 |
|------|----------|
| CPI/PPI | 通胀水平 |
| GDP | 经济总量 |
| 失业率 | 就业状况 |

## 大类资产传导机制

### 利率债

**核心指标**：GDP增速、CPI、PPI、DR007、MLF利率、M2-社融剪刀差、PMI

**传导机制**：
> 经济下行+通胀回落 → 货币政策宽松（降准/降息） → 银行间利率下行 → 债券收益率下行 → **债券价格上涨**

**布局信号**：
- PMI连续2-3个月低于50
- CPI同比低于1%，PPI同比负增长
- MLF利率下调、DR007低于政策利率中枢

### 信用债

**核心指标**：社融增速、工业企业利润、PMI、信用利差

**传导机制**：
> 经济企稳复苏 → 企业盈利改善 → 违约风险下降 → 风险偏好提升 → **信用利差收窄 → 信用债上涨**

### A股权益市场

**核心指标**：社融增速、M1-M2剪刀差、工业企业利润、10年期国债收益率、PMI、CPI、社零增速

**传导机制**：
> 社融增速企稳回升 → 实体融资需求回暖 → 企业盈利预期上行 + 流动性宽松 → **A股估值修复**

**行业轮动规则**：

| 宏观环境 | 优势板块 | 逻辑 |
|----------|----------|------|
| 经济复苏初期 | 金融、地产 | 利率下行+政策宽松 |
| 经济过热期 | 周期、资源 | 需求旺盛+通胀上行 |
| 滞胀期 | 消费、医药 | 防御性+确定性 |
| 衰退期 | 债券、公用事业 | 避险+稳定现金流 |

### 大宗商品

#### 黄金

**核心指标**：美国实际利率、美元指数、全球央行购金量

**传导机制**：
> 美国实际利率下行 → 持有成本下降 → **黄金价格上涨**

**案例**：2024年实际利率从2.5%降至1.2%，伦敦金从2000涨至2700美元/盎司（+35%）

#### 原油/工业金属

**核心指标**：中美欧PMI、OPEC+产量政策、美元指数

**传导机制**：
> 全球经济复苏 → 工业品需求提升 + 供给端收缩 → **大宗商品价格上涨**

#### 外汇（人民币汇率）

**核心指标**：中美利差、贸易顺差、美元指数

**传导机制**：
> 中美利差走阔 → 外资流入 → **人民币升值**

## 四阶段周期定位框架

| 阶段 | 经济特征 | 政策取向 | 优势资产 |
|------|----------|----------|----------|
| 复苏期 | GDP回升、通胀低 | 宽松 | 股票、信用债 |
| 过热期 | GDP高、通胀上行 | 收紧 | 商品、通胀债券 |
| 滞胀期 | GDP回落、通胀高 | 两难 | 现金、黄金 |
| 衰退期 | GDP低、通胀回落 | 宽松 | 债券、成长股 |

## 投资心法

1. **边际变化比绝对值更重要** — PMI从48回升至49比绝对值更重要
2. **预期差是超额收益来源** — 判断是否超预期
3. **政策是中国核心变量** — 政策边际变化影响远大于经济数据
4. **领先指标优先** — 绝不用滞后数据做决策

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `宏观研究指导投资` 时，不要只把它当成孤立知识点。把宏观看作资产配置的背景概率，而不是单笔交易的确定答案。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 增长、通胀、利率和信用周期
- 央行反应函数与财政政策
- 汇率、商品和全球资金流
- 预期差和市场定价

### 进阶训练

1. 把一个宏观观点映射到股票、债券、商品和汇率
2. 区分领先、同步和滞后指标
3. 写出观点被证伪的三个数据条件

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
