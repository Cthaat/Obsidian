# 行为金融学理论

> [!note] 核心要点
> 行为金融学从心理学角度研究投资者决策，认为投资者并非完全理性。四大理论支柱：**期望理论、过度自信、后悔理论、过度反应**，对解释股票价格波动、市场异象及风险管理具有显著作用。

## 行为金融学的兴起

传统金融理论假设投资者理性（均值方差模型、CAPM、有效市场假说），但现实中存在大量**金融异象**无法解释。行为金融学将心理学引入金融决策，假设投资者为**非理性**的自然人。

> [!quote] Keynes
> "心理因素是决定个人投资者行为的主要因素。投资者总是非理性的，他们的交易行为充满了'动物精神'。"

## 四大理论

### 1. 期望理论（Prospect Theory）

**提出者**：Kahneman & Tversky（1979）

核心发现：
- 投资者决策基于**价值函数**而非预期效用最大化
- 收益区间：价值函数向下凹 → **风险厌恶**
- 亏损区间：价值函数向上凸 → **风险偏好**

> [!important] 关键洞察
> 在亏损时投资者更加厌恶风险，在盈利时对风险的容忍度增加。面对收益遵循"收益最大化"原则，面对损失遵循"后悔最小化"原则。

### 2. 过度自信（Overconfidence）

**表现**：
- 高估自己的能力和知识
- 低估风险和不确定性
- 过分自信于自己的决策

**来源**：自我偏见、信息收集偏见、确认偏见

**影响**：
- 过度自信投资者比例越大 → 股票均衡价格越高 → 收益越低
- 需要新的高强度信息才能修正过度自信

### 3. 后悔理论（Regret Theory）

**提出者**：Loomes、Sugden、Bell（1982）

核心思想：投资者在投资过程中常出现后悔心理
- 大牛市没及时介入 → 后悔
- 过早卖出获利股票 → 后悔
- 熊市没及时止损 → 后悔
- 听别人推荐换股后原股上涨 → 更后悔

**修正**：追求欣喜和避免后悔作为决策动机

### 4. 过度反应（Overreaction Theory）

**提出者**：De Bond & Richard（1985）

核心发现：
- 投资者过度重视**近期信息**，对长期趋势判断偏差
- 导致市场短期暴涨暴跌
- 中国市场对"黑天鹅"事件存在过度反应，但调整迅速

## 实证研究方向

### 股票价格波动

2008年金融危机前后投资者心理变化：
1. **快速上涨阶段**：过度自信，对泡沫反应不足
2. **快速下跌阶段**：敏感性递减，越亏越不放手
3. **缓慢回升阶段**：未正确匹配股价与经营情况

### 市场异象

- **股权溢价之谜** — 传统CAPM无法解释
- **日历效应** — 有效市场假说无法解释
- 投资者情绪指数可修正CAPM模型的错误定价

### 风险管理

- 投资者情绪与股市风险形成**螺旋加强**
- 羊群效应下形成一致预期
- 情绪在不同市场相互蔓延

## 行为金融学的局限

| 局限 | 说明 |
|------|------|
| 缺乏现实普适性 | 模型假设与现实存在差距 |
| 模型构建困难 | 指标选取存在主观性 |
| 缺乏环境分析 | 较少考虑心理因素的影响因素 |

## 投资启示

1. **认识自身心理偏差** — 过度自信、锚定效应、羊群效应
2. **控制情绪** — 以理性态度对待市场波动
3. **筛选信息** — 了解信息质量、来源和时效性
4. **持续学习** — 了解最新研究成果和市场动态
