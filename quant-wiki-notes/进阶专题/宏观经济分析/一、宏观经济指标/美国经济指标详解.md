# 美国经济指标详解

> [!note] 核心要点
> 美国是全球最大经济体，美元是全球最重要的国际货币，美国宏观经济指标长期被视为全球经济发展的风向标。核心增长指标包括GDP、PMI、零售销售、非农就业等。

## 指标总览

| 指标 | 说明 | 发布频率 |
|------|------|----------|
| GDP（国内生产总值） | 总体经济指标 | 每季度（初值→修正值→终值） |
| 零售销售 | 消费指标（"恐怖数据"） | 每月中旬 |
| PMI（采购经理人指数） | 产业先行指标 | 每月第一个工作日 |
| 非农就业人数（NFP） | 就业指标 | 每月第一个周五 |
| 耐用品订单 | 制造业领先指标 | 月中公布上月数据 |
| 营建许可&新屋开工 | 房地产领先指标 | 每月第三个周二 |

## GDP（国内生产总值）

### 统计方法

美国采用**支出法**，GDP构成：
- **消费支出**：约70%（核心驱动）
- **企业投资**：约19%
- **政府支出**
- **净出口**

### GDP发布节奏

1. **初值**：季度结束后第一个月底（时效性最强，准确性最弱）
2. **第二估计值**：初值后一个月
3. **最终估计值**：再后一个月
4. **年度修订**：每年7-8月修订前5年数据

> [!tip] 投资影响
> GDP正增长 → 企业盈利改善 → 消费信心增加 → 但需警惕联准会加息抑制通胀

## PMI（采购经理人指数）

### 先行指标特性

PMI被视为先行指标的原因：
1. 统计生产、新订单、员工数等**前置环节**
2. **及时性强**，下月首个工作日即可发布

### 两大发布机构

| 机构 | 样本 | 特点 |
|------|------|------|
| ISM（美国供应管理协会） | 400+企业采购主管 | 较主观，权重一致 |
| S&P Global（Markit） | 800+企业 | 实际业务调查，不同权重 |

### Markit PMI权重

- 新订单指数：30%
- 生产指数：25%
- 从业人员指数：20%
- 供应商配送时间：15%
- 原材料库存：10%

### 荣枯线判断

| PMI值 | 经济状态 |
|-------|----------|
| > 50 | 经济扩张 |
| = 50 | 经济平稳 |
| < 50 | 经济收缩 |

> [!warning] 注意
> 美国服务业主导地位增强，制造业PMI长期低于50不一定代表经济衰退。2022年以来制造业PMI长期低于荣枯线，但经济和股市仍持续增长。

## 零售销售（"恐怖数据"）

### 重要性

- 美国消费占GDP的**70%**
- 零售销售占消费支出的**40%**
- **零售强劲，经济就不会差**

### 为何被称为"恐怖数据"

1. 经济晴雨表敏感性高
2. 数据共振效应（影响消费者信心、贸易等）
3. 引发行情巨震（2008年连续12月萎缩，道指单月跌20%）
4. 受汽油价格波动扭曲

### 构成（13个产业）

- 汽车零件（约20%）
- 非商店零售（>10%）
- 日用百货（>10%）
- 食品饮料（>10%）
- 加油站（10%）
- 其他：建筑园艺、医疗保健、服饰等

### 分析维度

| 维度 | 方法 |
|------|------|
| 总体趋势 | 绝对值看规模，环比/同比看变化 |
| 核心数据 | 剔除汽车、汽油、建筑材料 |
| 预期差异 | 实际值 vs 市场预期 |
| 消费结构 | 必需品稳定性 vs 非必需品信心 |

## 非农就业人数（NFP）

- 每月第一个周五发布
- 非农就业人口占美国就业**90%以上**
- 被称为"非农日"，是市场最关注的就业数据
