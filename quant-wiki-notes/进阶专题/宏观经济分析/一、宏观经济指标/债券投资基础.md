# 债券投资基础

> [!note] 核心要点
> 债券是固定收益证券，发行方承诺在未来特定日期支付利息和本金。按风险属性分为**利率债**和**信用债**两大类，是资产配置的重要工具。

## 债券分类

### 按发行主体

| 类型 | 发行方 | 信用等级 |
|------|--------|----------|
| 国债 | 中央政府 | 最高 |
| 地方政府债 | 地方政府 | 高 |
| 政策性金融债 | 政策性银行 | 高 |
| 企业债/公司债 | 企业 | 中等 |
| 可转债 | 上市公司 | 中等（含股性） |

### 按信用等级

| 类型 | 特点 | 主要风险 |
|------|------|----------|
| **利率债** | 国债、政策性金融债等 | 利率风险（不存在信用违约） |
| **信用债** | 企业债、公司债等 | 信用风险（可能违约） |
| **高收益债** | 信用评级较低 | 高信用风险 |

## 核心概念

| 概念 | 说明 |
|------|------|
| 票面利率 | 发行时约定的年利率 |
| 到期收益率（YTM） | 持有至到期的年化收益率 |
| 久期（Duration） | 衡量价格对利率变动的敏感度 |
| 凸性（Convexity） | 久期随利率变化的程度 |

### 收益率曲线

| 形态 | 特征 | 经济含义 |
|------|------|----------|
| 正常曲线 | 长期利率 > 短期利率 | 经济正常 |
| 平坦曲线 | 长短期利率接近 | 经济转折 |
| **倒挂曲线** | 长期利率 < 短期利率 | **常预示经济衰退** |

## 债券风险

1. **利率风险** — 利率上升，债券价格下跌
2. **信用风险** — 发行方违约
3. **流动性风险** — 无法及时卖出
4. **通胀风险** — 通货膨胀侵蚀实际收益
5. **汇率风险** — 外币债券的汇率波动

> [!important] 利率与债券价格的关系
> 市场利率与债券价格**成反比**：
> - 市场利率走高 → 债券价格走低
> - 市场利率走低 → 债券价格上涨
>
> 持有到期的投资人不受中途价格波动影响

## 信用评级

| 等级 | 含义 |
|------|------|
| AAA | 最高信用等级 |
| AA | 高质量 |
| A | 中上等级 |
| BBB | 中等（投资级最低） |
| BB及以下 | 投机级（垃圾债） |

## 可转债

### 特点
- 具有债券和股票**双重属性**
- 下有保底（债性），上不封顶（股性）

### 关键指标
- 转股价格、转股溢价率
- 纯债价值
- 赎回条款、回售条款

## 债券基金历史表现

| 指标 | 短债基金 | 中长债基金 |
|------|----------|------------|
| 历史最大回撤 | -1.32% | -4.01% |
| 长期年化收益 | 约3.23% | 约4.13% |
| 最好年份收益 | 6.13% | — |

> [!tip] 投资要点
> 债券市场违约率整体处于较低水平（2023年约0.11%，低于银行损失类贷款0.38%）。只要融资主体不违约，持有到期终获本金和利息。

## 投资策略

1. **利率预期策略** — 预期降息时增加久期
2. **信用策略** — 精选高信用评级债券
3. **骑乘策略** — 利用收益率曲线形状
4. **免疫策略** — 匹配资产负债久期
5. **可转债套利**

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `债券投资基础` 时，不要只把它当成孤立知识点。把宏观看作资产配置的背景概率，而不是单笔交易的确定答案。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 增长、通胀、利率和信用周期
- 央行反应函数与财政政策
- 汇率、商品和全球资金流
- 预期差和市场定价

### 进阶训练

1. 把一个宏观观点映射到股票、债券、商品和汇率
2. 区分领先、同步和滞后指标
3. 写出观点被证伪的三个数据条件

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
