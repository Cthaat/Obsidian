# 债券投资基础

> [!note] 核心要点
> 债券是固定收益证券，发行方承诺在未来特定日期支付利息和本金。按风险属性分为**利率债**和**信用债**两大类，是资产配置的重要工具。

## 债券分类

### 按发行主体

| 类型 | 发行方 | 信用等级 |
|------|--------|----------|
| 国债 | 中央政府 | 最高 |
| 地方政府债 | 地方政府 | 高 |
| 政策性金融债 | 政策性银行 | 高 |
| 企业债/公司债 | 企业 | 中等 |
| 可转债 | 上市公司 | 中等（含股性） |

### 按信用等级

| 类型 | 特点 | 主要风险 |
|------|------|----------|
| **利率债** | 国债、政策性金融债等 | 利率风险（不存在信用违约） |
| **信用债** | 企业债、公司债等 | 信用风险（可能违约） |
| **高收益债** | 信用评级较低 | 高信用风险 |

## 核心概念

| 概念 | 说明 |
|------|------|
| 票面利率 | 发行时约定的年利率 |
| 到期收益率（YTM） | 持有至到期的年化收益率 |
| 久期（Duration） | 衡量价格对利率变动的敏感度 |
| 凸性（Convexity） | 久期随利率变化的程度 |

### 收益率曲线

| 形态 | 特征 | 经济含义 |
|------|------|----------|
| 正常曲线 | 长期利率 > 短期利率 | 经济正常 |
| 平坦曲线 | 长短期利率接近 | 经济转折 |
| **倒挂曲线** | 长期利率 < 短期利率 | **常预示经济衰退** |

## 债券风险

1. **利率风险** — 利率上升，债券价格下跌
2. **信用风险** — 发行方违约
3. **流动性风险** — 无法及时卖出
4. **通胀风险** — 通货膨胀侵蚀实际收益
5. **汇率风险** — 外币债券的汇率波动

> [!important] 利率与债券价格的关系
> 市场利率与债券价格**成反比**：
> - 市场利率走高 → 债券价格走低
> - 市场利率走低 → 债券价格上涨
>
> 持有到期的投资人不受中途价格波动影响

## 信用评级

| 等级 | 含义 |
|------|------|
| AAA | 最高信用等级 |
| AA | 高质量 |
| A | 中上等级 |
| BBB | 中等（投资级最低） |
| BB及以下 | 投机级（垃圾债） |

## 可转债

### 特点
- 具有债券和股票**双重属性**
- 下有保底（债性），上不封顶（股性）

### 关键指标
- 转股价格、转股溢价率
- 纯债价值
- 赎回条款、回售条款

## 债券基金历史表现

| 指标 | 短债基金 | 中长债基金 |
|------|----------|------------|
| 历史最大回撤 | -1.32% | -4.01% |
| 长期年化收益 | 约3.23% | 约4.13% |
| 最好年份收益 | 6.13% | — |

> [!tip] 投资要点
> 债券市场违约率整体处于较低水平（2023年约0.11%，低于银行损失类贷款0.38%）。只要融资主体不违约，持有到期终获本金和利息。

## 投资策略

1. **利率预期策略** — 预期降息时增加久期
2. **信用策略** — 精选高信用评级债券
3. **骑乘策略** — 利用收益率曲线形状
4. **免疫策略** — 匹配资产负债久期
5. **可转债套利**

## 实战掌握清单

> [!tip] 交易者视角
> 债券投资基础 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。宏观分析不是预测新闻，而是理解增长、通胀、利率、信用、汇率和政策反应函数如何影响资产价格。

### 关键判断

- 区分领先、同步和滞后指标。
- 观察政策目标、约束和市场预期差。
- 把宏观结论映射到股票、债券、商品、外汇和现金。

### 落地动作

1. 建立基准情景、上行情景和下行情景。
2. 跟踪利率曲线、信用利差、通胀预期和资金面。
3. 用仓位和对冲表达观点，而不是孤立预测。

### 失效边界

- 把宏观叙事当作交易信号。
- 忽略政策滞后和市场提前定价。
- 用单一指标解释复杂周期。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 宏观到资产

围绕 债券投资基础 建立一个三情景框架：基准情景、上行情景和下行情景。每个情景都要写出增长、通胀、利率、信用、汇率和政策反应，再映射到股票、债券、商品和外汇。

### 训练动作

- 每周更新一张宏观仪表盘，区分领先、同步和滞后指标。
- 记录市场最关心的变量是否发生切换。
- 检查自己的交易观点是否已经被价格提前反映。

### 风险边界

宏观分析常见问题是看对大方向却做错资产、期限或仓位。
