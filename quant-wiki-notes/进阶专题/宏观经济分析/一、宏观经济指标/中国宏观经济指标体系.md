# 中国宏观经济指标体系

> [!note] 核心要点
> 分析判断经济形势围绕四大宏观目标进行：**经济增长、充分就业、物价稳定、国际收支平衡**。这四大目标相互联系、相互影响、相互制约，宏观调控的目的在于寻求最佳平衡点。

## 一、经济增长

经济增长通常指一个国家经济总产出或人均产出水平的持续增加，狭义上主要指**国内生产总值（GDP）增长**。

### 供给端指标

| 指标 | 说明 |
|------|------|
| 第一产业增加值 | 农林牧渔业 |
| 第二产业增加值 | 工业增加值 + 建筑业增加值 |
| 服务业增加值 | 第三产业 |

### 需求端指标（三驾马车）

| 指标 | 说明 |
|------|------|
| 社会消费品零售总额 | 消费 |
| 固定资产投资 | 投资 |
| 进出口总额 | 净出口 |

> [!tip] 2023年数据
> - GDP达1260582亿元，增长5.2%
> - 人均GDP达89358元（约1.27万美元）
> - 经济增量约6万亿元，相当于一个中等国家一年的经济总量

## 二、充分就业

充分就业不等于全部就业，仍存在**摩擦性失业**和**结构性失业**。

### 就业指标

| 指标 | 说明 |
|------|------|
| 城镇调查失业率 | 核心就业指标 |
| 城镇新增就业人数 | 就业创造能力 |
| 外出务工农村劳动力规模 | 农村就业转移 |
| 16-24岁青年失业率 | 结构性就业问题 |

> [!tip] 2023年数据
> - 城镇调查失业率均值5.2%，比上年下降0.4个百分点
> - 城镇新增就业1244万人，比上年多增38万人

## 三、物价稳定

### 通胀指标

| 指标 | 英文 | 反映领域 |
|------|------|----------|
| 居民消费价格指数 | CPI | 消费领域价格变动 |
| 工业生产者出厂价格指数 | PPI | 生产领域价格变动 |
| GDP平减指数 | — | 全社会物价总水平（最综合） |

### 通货膨胀 vs 通货紧缩

- **通货膨胀**：货币购买力 > 产出供给 → 物价持续普遍上涨
- **通货紧缩**：需求不足 → 物价持续下跌

> [!tip] 2023年数据
> - CPI比上年上涨0.2%
> - 核心CPI上涨0.7%
> - 保持温和上涨态势

## 四、国际收支平衡

### 收支结构

国际收支平衡表包含三个基本部分：
1. **经常账户** — 货物和服务贸易
2. **资本账户** — 资本转移
3. **金融账户** — 直接投资、证券投资

### 辅助指标

| 指标 | 说明 |
|------|------|
| 进出口差额 | 贸易顺差/逆差 |
| 外汇储备 | 对外偿付能力 |
| 人民币汇率 | 币值稳定 |

> [!tip] 2023年数据
> - 货物进出口总额41.8万亿元，逆势增长0.2%
> - 年末外汇储备余额超3.2万亿美元，稳居世界第一

## 指标分类（按领先性）

| 类型 | 指标 | 用途 |
|------|------|------|
| **领先指标** | 社融增速、PMI、M1-M2剪刀差、企业中长期贷款 | 前瞻预判 |
| **同步指标** | 工业增加值、用电量、铁路货运量 | 验证判断 |
| **滞后指标** | CPI、GDP、失业率 | 确认拐点 |

> [!warning] 投资警示
> 绝对不能用CPI、GDP等滞后数据做投资决策，必须用社融、PMI等领先指标做前瞻预判。

## 实战掌握清单

> [!tip] 交易者视角
> 中国宏观经济指标体系 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。宏观分析不是预测新闻，而是理解增长、通胀、利率、信用、汇率和政策反应函数如何影响资产价格。

### 关键判断

- 区分领先、同步和滞后指标。
- 观察政策目标、约束和市场预期差。
- 把宏观结论映射到股票、债券、商品、外汇和现金。

### 落地动作

1. 建立基准情景、上行情景和下行情景。
2. 跟踪利率曲线、信用利差、通胀预期和资金面。
3. 用仓位和对冲表达观点，而不是孤立预测。

### 失效边界

- 把宏观叙事当作交易信号。
- 忽略政策滞后和市场提前定价。
- 用单一指标解释复杂周期。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
