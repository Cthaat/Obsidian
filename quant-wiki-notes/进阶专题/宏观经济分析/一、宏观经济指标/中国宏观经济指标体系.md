# 中国宏观经济指标体系

> [!note] 核心要点
> 分析判断经济形势围绕四大宏观目标进行：**经济增长、充分就业、物价稳定、国际收支平衡**。这四大目标相互联系、相互影响、相互制约，宏观调控的目的在于寻求最佳平衡点。

## 一、经济增长

经济增长通常指一个国家经济总产出或人均产出水平的持续增加，狭义上主要指**国内生产总值（GDP）增长**。

### 供给端指标

| 指标 | 说明 |
|------|------|
| 第一产业增加值 | 农林牧渔业 |
| 第二产业增加值 | 工业增加值 + 建筑业增加值 |
| 服务业增加值 | 第三产业 |

### 需求端指标（三驾马车）

| 指标 | 说明 |
|------|------|
| 社会消费品零售总额 | 消费 |
| 固定资产投资 | 投资 |
| 进出口总额 | 净出口 |

> [!tip] 2023年数据
> - GDP达1260582亿元，增长5.2%
> - 人均GDP达89358元（约1.27万美元）
> - 经济增量约6万亿元，相当于一个中等国家一年的经济总量

## 二、充分就业

充分就业不等于全部就业，仍存在**摩擦性失业**和**结构性失业**。

### 就业指标

| 指标 | 说明 |
|------|------|
| 城镇调查失业率 | 核心就业指标 |
| 城镇新增就业人数 | 就业创造能力 |
| 外出务工农村劳动力规模 | 农村就业转移 |
| 16-24岁青年失业率 | 结构性就业问题 |

> [!tip] 2023年数据
> - 城镇调查失业率均值5.2%，比上年下降0.4个百分点
> - 城镇新增就业1244万人，比上年多增38万人

## 三、物价稳定

### 通胀指标

| 指标 | 英文 | 反映领域 |
|------|------|----------|
| 居民消费价格指数 | CPI | 消费领域价格变动 |
| 工业生产者出厂价格指数 | PPI | 生产领域价格变动 |
| GDP平减指数 | — | 全社会物价总水平（最综合） |

### 通货膨胀 vs 通货紧缩

- **通货膨胀**：货币购买力 > 产出供给 → 物价持续普遍上涨
- **通货紧缩**：需求不足 → 物价持续下跌

> [!tip] 2023年数据
> - CPI比上年上涨0.2%
> - 核心CPI上涨0.7%
> - 保持温和上涨态势

## 四、国际收支平衡

### 收支结构

国际收支平衡表包含三个基本部分：
1. **经常账户** — 货物和服务贸易
2. **资本账户** — 资本转移
3. **金融账户** — 直接投资、证券投资

### 辅助指标

| 指标 | 说明 |
|------|------|
| 进出口差额 | 贸易顺差/逆差 |
| 外汇储备 | 对外偿付能力 |
| 人民币汇率 | 币值稳定 |

> [!tip] 2023年数据
> - 货物进出口总额41.8万亿元，逆势增长0.2%
> - 年末外汇储备余额超3.2万亿美元，稳居世界第一

## 指标分类（按领先性）

| 类型 | 指标 | 用途 |
|------|------|------|
| **领先指标** | 社融增速、PMI、M1-M2剪刀差、企业中长期贷款 | 前瞻预判 |
| **同步指标** | 工业增加值、用电量、铁路货运量 | 验证判断 |
| **滞后指标** | CPI、GDP、失业率 | 确认拐点 |

> [!warning] 投资警示
> 绝对不能用CPI、GDP等滞后数据做投资决策，必须用社融、PMI等领先指标做前瞻预判。

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
