# 国际贸易与汇率

> [!note] 核心要点
> 国际贸易与汇率是宏观经济分析的重要维度，直接影响出口企业盈利、跨境投资收益和资本流动方向。

## 贸易数据指标

| 指标 | 说明 |
|------|------|
| 进出口总额 | 反映对外贸易规模 |
| 贸易差额 | 出口-进口，顺差/逆差 |
| 经常账户 | 货物和服务贸易收支 |
| 资本账户 | 资本流入流出 |

## 汇率指标

| 指标 | 说明 |
|------|------|
| 人民币汇率 | 在岸/离岸汇率 |
| 美元指数 | 衡量美元强弱 |
| 实际有效汇率 | 考虑通胀调整 |
| 外汇储备 | 对外偿付能力 |

## 汇率对投资的影响

### 人民币升值影响
- **利好**：航空、造纸（进口成本降低）
- **利空**：出口企业（产品价格竞争力下降）
- 影响港股通投资收益（汇兑损益）
- 影响QDII基金净值

### 人民币贬值影响
- **利好**：出口企业
- **利空**：外债较高的企业

### 中美利差与汇率

> [!tip] 核心传导机制
> 中美利差持续走阔 → 外资流入中国市场 → 外汇供给增加 → 人民币升值
>
> 美联储加息/美元走强 → 人民币被动贬值

**实操案例**：2022年美联储激进加息425bp，中美利差最大倒挂超150BP，美元指数涨至114，人民币从6.3贬值至7.3（贬值超15%）

## 贸易政策因素

| 因素 | 影响 |
|------|------|
| 关税调整 | 直接影响进出口成本 |
| 贸易协定 | 区域贸易格局变化 |
| 制裁措施 | 特定行业/国家影响 |
| 产业政策 | 进口替代/出口升级 |

## 投资策略

1. **关注贸易敏感行业** — 电子、纺织、机械等出口导向行业
2. **对冲汇率风险** — 使用外汇衍生品
3. **利用贸易数据选股** — 出口数据改善利好相关企业
4. **把握产业链转移机会** — 供应链重构中的投资机会

## 相关概念

| 概念 | 说明 |
|------|------|
| 全球化 | 经济一体化趋势 |
| 供应链重构 | 近岸化/友岸化趋势 |
| 进口替代 | 国产替代逻辑 |
| 出口升级 | 从低端到高端 |
| 汇率传导机制 | 汇率变动→物价→经济 |

## 实战掌握清单

> [!tip] 交易者视角
> 国际贸易与汇率 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。宏观分析不是预测新闻，而是理解增长、通胀、利率、信用、汇率和政策反应函数如何影响资产价格。

### 关键判断

- 区分领先、同步和滞后指标。
- 观察政策目标、约束和市场预期差。
- 把宏观结论映射到股票、债券、商品、外汇和现金。

### 落地动作

1. 建立基准情景、上行情景和下行情景。
2. 跟踪利率曲线、信用利差、通胀预期和资金面。
3. 用仓位和对冲表达观点，而不是孤立预测。

### 失效边界

- 把宏观叙事当作交易信号。
- 忽略政策滞后和市场提前定价。
- 用单一指标解释复杂周期。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 宏观到资产

围绕 国际贸易与汇率 建立一个三情景框架：基准情景、上行情景和下行情景。每个情景都要写出增长、通胀、利率、信用、汇率和政策反应，再映射到股票、债券、商品和外汇。

### 训练动作

- 每周更新一张宏观仪表盘，区分领先、同步和滞后指标。
- 记录市场最关心的变量是否发生切换。
- 检查自己的交易观点是否已经被价格提前反映。

### 风险边界

宏观分析常见问题是看对大方向却做错资产、期限或仓位。

## 最小可执行项目

### 宏观情景表

围绕 国际贸易与汇率 建一张宏观情景表：增长、通胀、利率、信用、汇率、政策和资产价格。每个情景都要对应仓位动作。

| 情景 | 观察指标 | 资产含义 |
|---|---|---|
| 增长改善 | PMI、盈利、信用扩张 | 权益和信用偏强 |
| 通胀升温 | CPI、PPI、商品 | 久期资产承压 |
| 流动性收缩 | 利率、利差、汇率 | 高估值资产降温 |
| 风险偏好下降 | 波动率、信用利差 | 防御和现金价值上升 |

### 验收标准

宏观观点必须落到资产和仓位，不能停留在叙事。
