# 国际贸易与汇率

> [!note] 核心要点
> 国际贸易与汇率是宏观经济分析的重要维度，直接影响出口企业盈利、跨境投资收益和资本流动方向。

## 贸易数据指标

| 指标 | 说明 |
|------|------|
| 进出口总额 | 反映对外贸易规模 |
| 贸易差额 | 出口-进口，顺差/逆差 |
| 经常账户 | 货物和服务贸易收支 |
| 资本账户 | 资本流入流出 |

## 汇率指标

| 指标 | 说明 |
|------|------|
| 人民币汇率 | 在岸/离岸汇率 |
| 美元指数 | 衡量美元强弱 |
| 实际有效汇率 | 考虑通胀调整 |
| 外汇储备 | 对外偿付能力 |

## 汇率对投资的影响

### 人民币升值影响
- **利好**：航空、造纸（进口成本降低）
- **利空**：出口企业（产品价格竞争力下降）
- 影响港股通投资收益（汇兑损益）
- 影响QDII基金净值

### 人民币贬值影响
- **利好**：出口企业
- **利空**：外债较高的企业

### 中美利差与汇率

> [!tip] 核心传导机制
> 中美利差持续走阔 → 外资流入中国市场 → 外汇供给增加 → 人民币升值
> 
> 美联储加息/美元走强 → 人民币被动贬值

**实操案例**：2022年美联储激进加息425bp，中美利差最大倒挂超150BP，美元指数涨至114，人民币从6.3贬值至7.3（贬值超15%）

## 贸易政策因素

| 因素 | 影响 |
|------|------|
| 关税调整 | 直接影响进出口成本 |
| 贸易协定 | 区域贸易格局变化 |
| 制裁措施 | 特定行业/国家影响 |
| 产业政策 | 进口替代/出口升级 |

## 投资策略

1. **关注贸易敏感行业** — 电子、纺织、机械等出口导向行业
2. **对冲汇率风险** — 使用外汇衍生品
3. **利用贸易数据选股** — 出口数据改善利好相关企业
4. **把握产业链转移机会** — 供应链重构中的投资机会

## 相关概念

| 概念 | 说明 |

| 进口替代 | 国产替代逻辑 |
