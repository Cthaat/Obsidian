# 国际贸易与汇率

> [!note] 核心要点
> 国际贸易与汇率是宏观经济分析的重要维度，直接影响出口企业盈利、跨境投资收益和资本流动方向。

## 贸易数据指标

| 指标 | 说明 |
|------|------|
| 进出口总额 | 反映对外贸易规模 |
| 贸易差额 | 出口-进口，顺差/逆差 |
| 经常账户 | 货物和服务贸易收支 |
| 资本账户 | 资本流入流出 |

## 汇率指标

| 指标 | 说明 |
|------|------|
| 人民币汇率 | 在岸/离岸汇率 |
| 美元指数 | 衡量美元强弱 |
| 实际有效汇率 | 考虑通胀调整 |
| 外汇储备 | 对外偿付能力 |

## 汇率对投资的影响

### 人民币升值影响
- **利好**：航空、造纸（进口成本降低）
- **利空**：出口企业（产品价格竞争力下降）
- 影响港股通投资收益（汇兑损益）
- 影响QDII基金净值

### 人民币贬值影响
- **利好**：出口企业
- **利空**：外债较高的企业

### 中美利差与汇率

> [!tip] 核心传导机制
> 中美利差持续走阔 → 外资流入中国市场 → 外汇供给增加 → 人民币升值
>
> 美联储加息/美元走强 → 人民币被动贬值

**实操案例**：2022年美联储激进加息425bp，中美利差最大倒挂超150BP，美元指数涨至114，人民币从6.3贬值至7.3（贬值超15%）

## 贸易政策因素

| 因素 | 影响 |
|------|------|
| 关税调整 | 直接影响进出口成本 |
| 贸易协定 | 区域贸易格局变化 |
| 制裁措施 | 特定行业/国家影响 |
| 产业政策 | 进口替代/出口升级 |

## 投资策略

1. **关注贸易敏感行业** — 电子、纺织、机械等出口导向行业
2. **对冲汇率风险** — 使用外汇衍生品
3. **利用贸易数据选股** — 出口数据改善利好相关企业
4. **把握产业链转移机会** — 供应链重构中的投资机会

## 相关概念

| 概念 | 说明 |
|------|------|
| 全球化 | 经济一体化趋势 |
| 供应链重构 | 近岸化/友岸化趋势 |
| 进口替代 | 国产替代逻辑 |
| 出口升级 | 从低端到高端 |
| 汇率传导机制 | 汇率变动→物价→经济 |

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `国际贸易与汇率` 时，不要只把它当成孤立知识点。把宏观看作资产配置的背景概率，而不是单笔交易的确定答案。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 增长、通胀、利率和信用周期
- 央行反应函数与财政政策
- 汇率、商品和全球资金流
- 预期差和市场定价

### 进阶训练

1. 把一个宏观观点映射到股票、债券、商品和汇率
2. 区分领先、同步和滞后指标
3. 写出观点被证伪的三个数据条件

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
