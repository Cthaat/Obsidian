# 期货市场入门

> [!note] 核心要点
> 期货是标准化的金融衍生品合约，约定在未来特定时间以特定价格买卖标的物。具有**保证金交易、T+0、双向交易、杠杆效应**等特点。

## 期货特点

| 特点 | 说明 |
|------|------|
| 保证金交易 | 只需支付合约价值5-15%作为保证金 |
| T+0交易 | 当日可多次买卖 |
| 双向交易 | 可以做多也可以做空 |
| 杠杆效应 | 以小博大，放大收益也放大风险 |
| 每日无负债结算 | 每日结算盈亏，保证金不足需追加 |

## 主要品种

### 商品期货

| 类别 | 品种 |
|------|------|
| 金属 | 黄金、白银、铜、铝、锌、镍、螺纹钢 |
| 能源 | 原油、燃料油、天然气 |
| 农产品 | 大豆、玉米、棉花、白糖、棕榈油 |
| 化工 | PTA、甲醇、聚丙烯 |

### 金融期货

| 类别 | 品种 |
|------|------|
| 股指期货 | IF(沪深300)、IC(中证500)、IH(上证50)、IM(中证1000) |
| 国债期货 | 2年期、5年期、10年期、30年期 |

## 交易策略

### 套期保值

| 类型 | 场景 |
|------|------|
| 买入套保 | 担心未来价格上涨，在期货市场买入 |
| 卖出套保 | 担心未来价格下跌，在期货市场卖出 |

### 投机交易

- **趋势跟踪** — 顺势交易
- **日内短线** — 当日进出
- **波段操作** — 中短期持仓
- **价差交易** — 利用价差变动

### 套利交易

| 类型 | 说明 |
|------|------|
| 跨期套利 | 不同到期月份合约价差 |
| 跨品种套利 | 相关品种间价差 |
| 跨市场套利 | 不同交易所价差 |
| 期现套利 | 期货与现货价差 |

## 风险管理

> [!warning] 风险提示
> 期货交易风险极高，可能导致本金全部亏损甚至穿仓。新手建议先用模拟盘练习。

1. **严格止损** — 设定止损位
2. **控制仓位** — 建议不超过总资金30%
3. **避免过度交易** — 纪律性操作
4. **关注保证金水平** — 避免强平
5. **了解交割规则** — 注意交割月

## 期货与宏观经济

> [!tip] 宏观分析应用
> - **黄金**：美国实际利率下行 → 金价上涨
> - **原油**：全球经济复苏 → 需求提升 → 油价上涨
> - **铜**：工业需求指标，"铜博士"反映经济景气
> - **国债期货**：经济下行+货币宽松 → 国债期货上涨

## 实战掌握清单

> [!tip] 交易者视角
> 期货市场入门 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。宏观分析不是预测新闻，而是理解增长、通胀、利率、信用、汇率和政策反应函数如何影响资产价格。

### 关键判断

- 区分领先、同步和滞后指标。
- 观察政策目标、约束和市场预期差。
- 把宏观结论映射到股票、债券、商品、外汇和现金。

### 落地动作

1. 建立基准情景、上行情景和下行情景。
2. 跟踪利率曲线、信用利差、通胀预期和资金面。
3. 用仓位和对冲表达观点，而不是孤立预测。

### 失效边界

- 把宏观叙事当作交易信号。
- 忽略政策滞后和市场提前定价。
- 用单一指标解释复杂周期。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 宏观到资产

围绕 期货市场入门 建立一个三情景框架：基准情景、上行情景和下行情景。每个情景都要写出增长、通胀、利率、信用、汇率和政策反应，再映射到股票、债券、商品和外汇。

### 训练动作

- 每周更新一张宏观仪表盘，区分领先、同步和滞后指标。
- 记录市场最关心的变量是否发生切换。
- 检查自己的交易观点是否已经被价格提前反映。

### 风险边界

宏观分析常见问题是看对大方向却做错资产、期限或仓位。

## 最小可执行项目

### 宏观情景表

围绕 期货市场入门 建一张宏观情景表：增长、通胀、利率、信用、汇率、政策和资产价格。每个情景都要对应仓位动作。

| 情景 | 观察指标 | 资产含义 |
|---|---|---|
| 增长改善 | PMI、盈利、信用扩张 | 权益和信用偏强 |
| 通胀升温 | CPI、PPI、商品 | 久期资产承压 |
| 流动性收缩 | 利率、利差、汇率 | 高估值资产降温 |
| 风险偏好下降 | 波动率、信用利差 | 防御和现金价值上升 |

### 验收标准

宏观观点必须落到资产和仓位，不能停留在叙事。
