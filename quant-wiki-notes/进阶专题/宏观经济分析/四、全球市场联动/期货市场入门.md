# 期货市场入门

> [!note] 核心要点
> 期货是标准化的金融衍生品合约，约定在未来特定时间以特定价格买卖标的物。具有**保证金交易、T+0、双向交易、杠杆效应**等特点。

## 期货特点

| 特点 | 说明 |
|------|------|
| 保证金交易 | 只需支付合约价值5-15%作为保证金 |
| T+0交易 | 当日可多次买卖 |
| 双向交易 | 可以做多也可以做空 |
| 杠杆效应 | 以小博大，放大收益也放大风险 |
| 每日无负债结算 | 每日结算盈亏，保证金不足需追加 |

## 主要品种

### 商品期货

| 类别 | 品种 |
|------|------|
| 金属 | 黄金、白银、铜、铝、锌、镍、螺纹钢 |
| 能源 | 原油、燃料油、天然气 |
| 农产品 | 大豆、玉米、棉花、白糖、棕榈油 |
| 化工 | PTA、甲醇、聚丙烯 |

### 金融期货

| 类别 | 品种 |
|------|------|
| 股指期货 | IF(沪深300)、IC(中证500)、IH(上证50)、IM(中证1000) |
| 国债期货 | 2年期、5年期、10年期、30年期 |

## 交易策略

### 套期保值

| 类型 | 场景 |
|------|------|
| 买入套保 | 担心未来价格上涨，在期货市场买入 |
| 卖出套保 | 担心未来价格下跌，在期货市场卖出 |

### 投机交易

- **趋势跟踪** — 顺势交易
- **日内短线** — 当日进出
- **波段操作** — 中短期持仓
- **价差交易** — 利用价差变动

### 套利交易

| 类型 | 说明 |
|------|------|
| 跨期套利 | 不同到期月份合约价差 |
| 跨品种套利 | 相关品种间价差 |
| 跨市场套利 | 不同交易所价差 |
| 期现套利 | 期货与现货价差 |

## 风险管理

> [!warning] 风险提示
> 期货交易风险极高，可能导致本金全部亏损甚至穿仓。新手建议先用模拟盘练习。

1. **严格止损** — 设定止损位
2. **控制仓位** — 建议不超过总资金30%
3. **避免过度交易** — 纪律性操作
4. **关注保证金水平** — 避免强平
5. **了解交割规则** — 注意交割月

## 期货与宏观经济

> [!tip] 宏观分析应用
> - **黄金**：美国实际利率下行 → 金价上涨
> - **原油**：全球经济复苏 → 需求提升 → 油价上涨
> - **铜**：工业需求指标，"铜博士"反映经济景气
> - **国债期货**：经济下行+货币宽松 → 国债期货上涨
