# 期货市场入门

> [!note] 核心要点
> 期货是标准化的金融衍生品合约，约定在未来特定时间以特定价格买卖标的物。具有**保证金交易、T+0、双向交易、杠杆效应**等特点。

## 期货特点

| 特点 | 说明 |
|------|------|
| 保证金交易 | 只需支付合约价值5-15%作为保证金 |
| T+0交易 | 当日可多次买卖 |
| 双向交易 | 可以做多也可以做空 |
| 杠杆效应 | 以小博大，放大收益也放大风险 |
| 每日无负债结算 | 每日结算盈亏，保证金不足需追加 |

## 主要品种

### 商品期货

| 类别 | 品种 |
|------|------|
| 金属 | 黄金、白银、铜、铝、锌、镍、螺纹钢 |
| 能源 | 原油、燃料油、天然气 |
| 农产品 | 大豆、玉米、棉花、白糖、棕榈油 |
| 化工 | PTA、甲醇、聚丙烯 |

### 金融期货

| 类别 | 品种 |
|------|------|
| 股指期货 | IF(沪深300)、IC(中证500)、IH(上证50)、IM(中证1000) |
| 国债期货 | 2年期、5年期、10年期、30年期 |

## 交易策略

### 套期保值

| 类型 | 场景 |
|------|------|
| 买入套保 | 担心未来价格上涨，在期货市场买入 |
| 卖出套保 | 担心未来价格下跌，在期货市场卖出 |

### 投机交易

- **趋势跟踪** — 顺势交易
- **日内短线** — 当日进出
- **波段操作** — 中短期持仓
- **价差交易** — 利用价差变动

### 套利交易

| 类型 | 说明 |
|------|------|
| 跨期套利 | 不同到期月份合约价差 |
| 跨品种套利 | 相关品种间价差 |
| 跨市场套利 | 不同交易所价差 |
| 期现套利 | 期货与现货价差 |

## 风险管理

> [!warning] 风险提示
> 期货交易风险极高，可能导致本金全部亏损甚至穿仓。新手建议先用模拟盘练习。

1. **严格止损** — 设定止损位
2. **控制仓位** — 建议不超过总资金30%
3. **避免过度交易** — 纪律性操作
4. **关注保证金水平** — 避免强平
5. **了解交割规则** — 注意交割月

## 期货与宏观经济

> [!tip] 宏观分析应用
> - **黄金**：美国实际利率下行 → 金价上涨
> - **原油**：全球经济复苏 → 需求提升 → 油价上涨
> - **铜**：工业需求指标，"铜博士"反映经济景气
> - **国债期货**：经济下行+货币宽松 → 国债期货上涨

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
