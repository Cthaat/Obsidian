# 港股市场

> [!note] 核心要点
> 港股市场国际化程度高、机构投资者主导，与A股存在估值差异。2026年有望迎来"戴维斯双击"——盈利修复+估值回升。

## 市场特点

- 国际化程度高（全球资金参与）
- 机构投资者主导
- 与A股存在估值差异（A/H溢价）
- 交易制度灵活

## 主要指数

| 指数 | 说明 |
|------|------|
| 恒生指数 | 香港蓝筹股基准 |
| 恒生科技指数 | 科技股代表 |
| 国企指数（H股） | 内地企业表现 |
| 红筹指数 | 中资背景公司 |

## 投资渠道

| 渠道 | 特点 |
|------|------|
| 港股通 | 最便捷，受额度限制 |
| 直接开户香港券商 | 选择更多，手续较复杂 |
| ETF产品 | 被动投资，费率低 |
| 互认基金 | 基金互认安排 |

## 交易规则

| 规则 | 说明 |
|------|------|
| T+0交易 | 当日可多次买卖 |
| 无涨跌幅限制 | 波动可能较大 |
| 每手股数不固定 | 不同公司每手不同 |
| 可以做空 | 融券卖空 |
| 融资融券灵活 | 杠杆交易 |

## 风险提示

| 风险 | 说明 |
|------|------|
| 汇率风险 | 港币与美元挂钩 |
| 流动性风险 | 部分小盘股流动性差 |
| 政策风险 | 内地政策变化影响 |
| 信息不对称 | 研究覆盖不足 |
| 交易成本 | 印花税等费用较高 |

> [!tip] 2026年投资逻辑
> - **分子端（盈利）**：中国经济复苏带动港股上市公司盈利回升
> - **分母端（估值）**：美联储降息降低全球资金成本，外资回流
> - **首选标的**：恒生科技指数成分股、创新药、A/H折价率较高的高股息国企
