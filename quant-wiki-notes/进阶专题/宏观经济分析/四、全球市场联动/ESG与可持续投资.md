# ESG与可持续投资

> [!note] 核心要点
> ESG是Environmental(环境)、Social(社会)、Governance(治理)的缩写，将非财务因素纳入投资决策。中国双碳目标（2030碳达峰、2060碳中和）推动ESG投资快速发展。

## ESG三大支柱

### 环境（Environmental）
- 气候变化与碳排放
- 资源利用与循环经济
- 污染与废弃物管理
- 生物多样性保护

### 社会（Social）
- 员工权益与多元化
- 产品安全与质量
- 社区关系
- 供应链管理

### 治理（Governance）
- 董事会结构
- 高管薪酬
- 股东权利
- 商业道德与反腐败

## ESG投资策略

| 策略 | 说明 |
|------|------|
| 负面筛选 | 排除不符合ESG标准的行业（烟草、武器、化石燃料） |
| 正面筛选 | 选择ESG表现优于同行的公司 |
| ESG整合 | 将ESG因素系统纳入财务分析 |
| 主题投资 | 专注可持续发展主题（清洁能源、绿色建筑） |
| 影响力投资 | 追求财务回报+可衡量的社会/环境正面影响 |
| 积极股东 | 通过股东权利推动公司改善ESG实践 |

## 负责任投资原则（PRI）

1. 将ESG议题纳入投资分析和决策过程
2. 成为积极所有者，将ESG纳入所有权政策
3. 寻求被投资实体对ESG议题的适当披露
4. 推动投资行业接受和实施PRI
5. 合作提升PRI实施效果
6. 报告PRI实施的活动和进展

## 投资意义

1. **风险管理** — ESG问题可能带来重大财务风险
2. **长期价值** — 可持续企业更具长期竞争力
3. **监管趋势** — 全球ESG监管趋严
4. **投资者需求** — 千禧一代偏好可持续投资

## 课程化学习补充

> [!important] 学习定位
> 宏观分析服务于资产配置和风险预算，重点看边际变化、预期差、政策反应函数和跨资产传导。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 指标是领先、同步还是滞后
- 市场一致预期是什么
- 政策变量如何影响利率和风险偏好
- 传导到哪类资产最直接

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 用宏观结论直接择单票
- 把长期判断用于短线交易
- 忽视汇率和流动性约束
- 把政策口号当已兑现业绩

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `ESG与可持续投资` 时，不要只把它当成孤立知识点。把宏观看作资产配置的背景概率，而不是单笔交易的确定答案。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 增长、通胀、利率和信用周期
- 央行反应函数与财政政策
- 汇率、商品和全球资金流
- 预期差和市场定价

### 进阶训练

1. 把一个宏观观点映射到股票、债券、商品和汇率
2. 区分领先、同步和滞后指标
3. 写出观点被证伪的三个数据条件

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
