# ESG与可持续投资

> [!note] 核心要点
> ESG是Environmental(环境)、Social(社会)、Governance(治理)的缩写，将非财务因素纳入投资决策。中国双碳目标（2030碳达峰、2060碳中和）推动ESG投资快速发展。

## ESG三大支柱

### 环境（Environmental）
- 气候变化与碳排放
- 资源利用与循环经济
- 污染与废弃物管理
- 生物多样性保护

### 社会（Social）
- 员工权益与多元化
- 产品安全与质量
- 社区关系
- 供应链管理

### 治理（Governance）
- 董事会结构
- 高管薪酬
- 股东权利
- 商业道德与反腐败

## ESG投资策略

| 策略 | 说明 |
|------|------|
| 负面筛选 | 排除不符合ESG标准的行业（烟草、武器、化石燃料） |
| 正面筛选 | 选择ESG表现优于同行的公司 |
| ESG整合 | 将ESG因素系统纳入财务分析 |
| 主题投资 | 专注可持续发展主题（清洁能源、绿色建筑） |
| 影响力投资 | 追求财务回报+可衡量的社会/环境正面影响 |
| 积极股东 | 通过股东权利推动公司改善ESG实践 |

## 负责任投资原则（PRI）

1. 将ESG议题纳入投资分析和决策过程
2. 成为积极所有者，将ESG纳入所有权政策
3. 寻求被投资实体对ESG议题的适当披露
4. 推动投资行业接受和实施PRI
5. 合作提升PRI实施效果
6. 报告PRI实施的活动和进展

## 绿色金融产品

| 产品 | 说明 |
|------|------|
| 绿色债券 | 资金用于环保项目 |
| ESG基金 | 按ESG标准筛选投资 |
| 碳中和ETF | 跟踪低碳指数 |
| 可持续发展挂钩贷款 | 利率与企业ESG表现挂钩 |

## 中国ESG发展

- **双碳目标**：2030碳达峰、2060碳中和
- **绿色金融"五大支柱"**
- 上市公司ESG信息披露要求加强
- 全国碳排放权交易市场

## 投资意义

1. **风险管理** — ESG问题可能带来重大财务风险
2. **长期价值** — 可持续企业更具长期竞争力
3. **监管趋势** — 全球ESG监管趋严
4. **投资者需求** — 千禧一代偏好可持续投资
