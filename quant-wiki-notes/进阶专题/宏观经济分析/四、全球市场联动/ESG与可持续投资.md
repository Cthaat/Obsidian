# ESG与可持续投资

> [!note] 核心要点
> ESG是Environmental(环境)、Social(社会)、Governance(治理)的缩写，将非财务因素纳入投资决策。中国双碳目标（2030碳达峰、2060碳中和）推动ESG投资快速发展。

## ESG三大支柱

### 环境（Environmental）
- 气候变化与碳排放
- 资源利用与循环经济
- 污染与废弃物管理
- 生物多样性保护

### 社会（Social）
- 员工权益与多元化
- 产品安全与质量
- 社区关系
- 供应链管理

### 治理（Governance）
- 董事会结构
- 高管薪酬
- 股东权利
- 商业道德与反腐败

## ESG投资策略

| 策略 | 说明 |
|------|------|
| 负面筛选 | 排除不符合ESG标准的行业（烟草、武器、化石燃料） |
| 正面筛选 | 选择ESG表现优于同行的公司 |
| ESG整合 | 将ESG因素系统纳入财务分析 |
| 主题投资 | 专注可持续发展主题（清洁能源、绿色建筑） |
| 影响力投资 | 追求财务回报+可衡量的社会/环境正面影响 |
| 积极股东 | 通过股东权利推动公司改善ESG实践 |

## 负责任投资原则（PRI）

1. 将ESG议题纳入投资分析和决策过程
2. 成为积极所有者，将ESG纳入所有权政策
3. 寻求被投资实体对ESG议题的适当披露
4. 推动投资行业接受和实施PRI
5. 合作提升PRI实施效果
6. 报告PRI实施的活动和进展

1. **风险管理** — ESG问题可能带来重大财务风险

## 实战掌握清单

> [!tip] 交易者视角
> ESG与可持续投资 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。宏观分析不是预测新闻，而是理解增长、通胀、利率、信用、汇率和政策反应函数如何影响资产价格。

### 关键判断

- 区分领先、同步和滞后指标。
- 观察政策目标、约束和市场预期差。
- 把宏观结论映射到股票、债券、商品、外汇和现金。

### 落地动作

1. 建立基准情景、上行情景和下行情景。
2. 跟踪利率曲线、信用利差、通胀预期和资金面。
3. 用仓位和对冲表达观点，而不是孤立预测。

### 失效边界

- 把宏观叙事当作交易信号。
- 忽略政策滞后和市场提前定价。
- 用单一指标解释复杂周期。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 宏观到资产

围绕 ESG与可持续投资 建立一个三情景框架：基准情景、上行情景和下行情景。每个情景都要写出增长、通胀、利率、信用、汇率和政策反应，再映射到股票、债券、商品和外汇。

### 训练动作

- 每周更新一张宏观仪表盘，区分领先、同步和滞后指标。
- 记录市场最关心的变量是否发生切换。
- 检查自己的交易观点是否已经被价格提前反映。

### 风险边界

宏观分析常见问题是看对大方向却做错资产、期限或仓位。

## 最小可执行项目

### 宏观情景表

围绕 ESG与可持续投资 建一张宏观情景表：增长、通胀、利率、信用、汇率、政策和资产价格。每个情景都要对应仓位动作。

| 情景 | 观察指标 | 资产含义 |
|---|---|---|
| 增长改善 | PMI、盈利、信用扩张 | 权益和信用偏强 |
| 通胀升温 | CPI、PPI、商品 | 久期资产承压 |
| 流动性收缩 | 利率、利差、汇率 | 高估值资产降温 |
| 风险偏好下降 | 波动率、信用利差 | 防御和现金价值上升 |

### 验收标准

宏观观点必须落到资产和仓位，不能停留在叙事。
