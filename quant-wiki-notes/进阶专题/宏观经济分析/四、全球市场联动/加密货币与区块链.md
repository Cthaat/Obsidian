# 加密货币与区块链

> [!note] 核心要点
> 加密货币是基于区块链技术的数字资产。比特币(BTC)开创去中心化数字货币先河，以太坊(ETH)是智能合约和DeFi的基础设施。2024年比特币ETF持续流入，RWA代币化加速。

## 主流加密货币

| 币种 | 特点 |
|------|------|
| **比特币(BTC)** | 数字黄金，总量2100万枚，2024年第四次减半 |
| **以太坊(ETH)** | 智能合约平台，2022年完成PoS转型 |
| **BNB** | 币安生态代币 |
| **SOL** | 高性能公链 |
| **USDT/USDC** | 稳定币，锚定美元 |
| **XRP** | 跨境支付 |

## 区块链核心技术

1. **分布式账本** — 去中心化数据存储
2. **共识机制** — PoW（工作量证明）/ PoS（权益证明）
3. **智能合约** — 自动执行的程序化合约
4. **密码学加密** — 保证交易安全

### Layer 1 vs Layer 2

| 层级 | 说明 | 示例 |
|------|------|------|
| L1 | 主链 | 比特币、以太坊 |
| L2 | 扩容方案 | Arbitrum、Optimism |

## DeFi（去中心化金融）

| 应用类型 | 代表项目 |
|----------|----------|
| 去中心化交易所 | Uniswap、Curve |
| 借贷协议 | Aave、Compound |
| 稳定币 | MakerDAO、Frax |
| 衍生品 | dYdX、GMX |
| 收益聚合器 | Yearn Finance |

**DeFi特点**：无需许可、非托管、透明可查、可组合性

## 投资策略

| 策略 | 说明 |
|------|------|
| 定投策略 | 定期买入，摊平成本 |
| 网格交易 | 波动区间低买高卖 |
| 趋势跟踪 | 跟随大趋势 |
| 套利交易 | 跨交易所价差 |
| 质押收益 | PoS币种质押获取利息 |
| DeFi挖矿 | 提供流动性获取收益 |

## 风险管理

> [!warning] 风险提示
> 加密货币市场波动极大，24小时交易，无涨跌幅限制。

1. 仅投资可承受亏损的资金
2. 使用硬件钱包存储大额资产
3. 警惕诈骗和钓鱼
4. 分散投资
5. 了解交易所安全记录
6. 注意私钥安全
