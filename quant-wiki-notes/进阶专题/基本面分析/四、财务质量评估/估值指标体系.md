# 估值指标体系

> [!note] 核心概念
> 估值是投资的核心问题——花100元买一个苹果，总得知道它值不值100元。估值方法分为相对估值法和绝对估值法两大类。

## 一、相对估值指标

### 市盈率（PE）
- **PE = 市值 / 净利润 = 股价 / 每股收益**
- 从盈利角度评估，意味着多少年回本
- 适合盈利稳定的企业，不适合周期股

### 市净率（PB）
- **PB = 市值 / 净资产 = 股价 / 每股净资产**
- 从资产角度评估，以几倍净资产价格买入
- 适合周期股、银行、地产等资产密集型行业
- PB一般在3-10之间比较合理

### 市销率（PS）
- **PS = 市值 / 营业收入**
- 适合亏损但有营收增长的企业（如互联网公司）

### 股息率
- **股息率 = 每股股利 / 股价**
- 类似债券的收益率概念

### PEG
- **PEG = PE / 净利润增长率**
- PEG<1可能被低估，PEG>1可能被高估

## 二、绝对估值指标

### DCF（现金流折现）
- 预测未来自由现金流，折现到当前价值
- 理论上最准确，但对假设敏感

### DDM（股利折现）
- 预测未来股利，折现到当前价值
- 适合稳定分红的企业

## 三、多维度估值框架

| 维度 | 指标 | 适用场景 |
|------|------|---------|
| 盈利估值 | PE、PEG | 盈利稳定的企业 |
| 资产估值 | PB | 资产密集型行业 |
| 收入估值 | PS | 高增长但亏损的企业 |
| 现金流估值 | EV/EBITDA | 资本结构差异大的企业 |
| 股利估值 | 股息率、DDM | 稳定分红的企业 |

## 四、估值注意事项

### 行业差异
- 不同行业适用不同估值方法
- 周期股：PB优于PE（净利润波动大）
- 互联网：PS优于PE（可能亏损）
- 银行：PB为主（关注资产质量）

### 估值陷阱
- 低PE不一定便宜（可能利润不可持续）
- 低PB不一定安全（可能资产质量差）
- 需结合盈利质量、现金流、成长性综合判断

## 五、PB-ROE估值模型

> [!important] 核心公式
> **ROE = PB / PE**
> 
> - ROE越高，资产盈利能力越强，PB应该越高
> - 以低PB买入低ROE企业，等待ROE上升 → 业绩和估值双击
> - 以高PB买入高ROE企业，ROE回落 → 业绩和估值双杀
