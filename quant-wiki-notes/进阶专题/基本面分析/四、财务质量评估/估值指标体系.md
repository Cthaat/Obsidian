# 估值指标体系

> [!note] 核心概念
> 估值是投资的核心问题——花100元买一个苹果，总得知道它值不值100元。估值方法分为相对估值法和绝对估值法两大类。

## 一、相对估值指标

### 市盈率（PE）
- **PE = 市值 / 净利润 = 股价 / 每股收益**
- 从盈利角度评估，意味着多少年回本
- 适合盈利稳定的企业，不适合周期股

### 市净率（PB）
- **PB = 市值 / 净资产 = 股价 / 每股净资产**
- 从资产角度评估，以几倍净资产价格买入
- 适合周期股、银行、地产等资产密集型行业
- PB一般在3-10之间比较合理

### 市销率（PS）
- **PS = 市值 / 营业收入**
- 适合亏损但有营收增长的企业（如互联网公司）

### 股息率
- **股息率 = 每股股利 / 股价**
- 类似债券的收益率概念

### PEG
- **PEG = PE / 净利润增长率**
- PEG<1可能被低估，PEG>1可能被高估

## 二、绝对估值指标

### DCF（现金流折现）
- 预测未来自由现金流，折现到当前价值
- 理论上最准确，但对假设敏感

### DDM（股利折现）
- 预测未来股利，折现到当前价值
- 适合稳定分红的企业

## 三、多维度估值框架

| 维度 | 指标 | 适用场景 |
|------|------|---------|
| 盈利估值 | PE、PEG | 盈利稳定的企业 |
| 资产估值 | PB | 资产密集型行业 |
| 收入估值 | PS | 高增长但亏损的企业 |
| 现金流估值 | EV/EBITDA | 资本结构差异大的企业 |
| 股利估值 | 股息率、DDM | 稳定分红的企业 |

## 四、估值注意事项

### 行业差异
- 不同行业适用不同估值方法
- 周期股：PB优于PE（净利润波动大）
- 互联网：PS优于PE（可能亏损）
- 银行：PB为主（关注资产质量）

### 估值陷阱
- 低PE不一定便宜（可能利润不可持续）
- 低PB不一定安全（可能资产质量差）
- 需结合盈利质量、现金流、成长性综合判断

## 五、PB-ROE估值模型

> [!important] 核心公式
> **ROE = PB / PE**
>
> - ROE越高，资产盈利能力越强，PB应该越高
> - 以低PB买入低ROE企业，等待ROE上升 → 业绩和估值双击
> - 以高PB买入高ROE企业，ROE回落 → 业绩和估值双杀

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
