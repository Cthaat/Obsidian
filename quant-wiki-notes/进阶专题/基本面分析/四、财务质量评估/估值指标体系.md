# 估值指标体系

> [!note] 核心概念
> 估值是投资的核心问题——花100元买一个苹果，总得知道它值不值100元。估值方法分为相对估值法和绝对估值法两大类。

## 一、相对估值指标

### 市盈率（PE）
- **PE = 市值 / 净利润 = 股价 / 每股收益**
- 从盈利角度评估，意味着多少年回本
- 适合盈利稳定的企业，不适合周期股

### 市净率（PB）
- **PB = 市值 / 净资产 = 股价 / 每股净资产**
- 从资产角度评估，以几倍净资产价格买入
- 适合周期股、银行、地产等资产密集型行业
- PB一般在3-10之间比较合理

### 市销率（PS）
- **PS = 市值 / 营业收入**
- 适合亏损但有营收增长的企业（如互联网公司）

### 股息率
- **股息率 = 每股股利 / 股价**
- 类似债券的收益率概念

### PEG
- **PEG = PE / 净利润增长率**
- PEG<1可能被低估，PEG>1可能被高估

## 二、绝对估值指标

### DCF（现金流折现）
- 预测未来自由现金流，折现到当前价值
- 理论上最准确，但对假设敏感

### DDM（股利折现）
- 预测未来股利，折现到当前价值
- 适合稳定分红的企业

## 三、多维度估值框架

| 维度 | 指标 | 适用场景 |
|------|------|---------|
| 盈利估值 | PE、PEG | 盈利稳定的企业 |
| 资产估值 | PB | 资产密集型行业 |
| 收入估值 | PS | 高增长但亏损的企业 |
| 现金流估值 | EV/EBITDA | 资本结构差异大的企业 |
| 股利估值 | 股息率、DDM | 稳定分红的企业 |

## 四、估值注意事项

### 行业差异
- 不同行业适用不同估值方法
- 周期股：PB优于PE（净利润波动大）
- 互联网：PS优于PE（可能亏损）
- 银行：PB为主（关注资产质量）

### 估值陷阱
- 低PE不一定便宜（可能利润不可持续）
- 低PB不一定安全（可能资产质量差）
- 需结合盈利质量、现金流、成长性综合判断

## 五、PB-ROE估值模型

> [!important] 核心公式
> **ROE = PB / PE**
>
> - ROE越高，资产盈利能力越强，PB应该越高
> - 以低PB买入低ROE企业，等待ROE上升 → 业绩和估值双击
> - 以高PB买入高ROE企业，ROE回落 → 业绩和估值双杀

## 实战掌握清单

> [!tip] 交易者视角
> 估值指标体系 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 估值指标体系 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。

## 最小可执行项目

### 公司研究备忘录

围绕 估值指标体系 写一份公司研究备忘录：商业模式、行业格局、三张表、现金流、ROE/ROIC、负债、管理层、估值和风险。

| 证据 | 检查点 |
|---|---|
| 收入 | 是否可持续，是否依赖一次性因素 |
| 利润 | 毛利率和费用率是否合理 |
| 现金流 | 利润是否能变成现金 |
| 资产负债 | 应收、存货、商誉和负债是否异常 |
| 估值 | 是否有安全边际 |

### 验收标准

能用财务证据支持或否定投资假设，而不是只讲行业故事。
