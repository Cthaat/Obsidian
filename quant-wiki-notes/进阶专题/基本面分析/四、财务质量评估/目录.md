# 四、财务质量评估

> [!note] 概述
> 财务质量评估是识别高质量财务报告和盈余的关键技能。能够识别高质量报告的投资者可以更有信心地做出投资决策，识别低质量报告则可以避免损失。

2. [[财报分析核心方法]] — 盈利质量、资产质量与现金流分析
3. [[估值指标体系]] — 相对估值与绝对估值方法

## 学习路径

```
财务报告质量光谱 → 财报分析核心方法 → 估值指标体系
        ↓                  ↓                  ↓
    六级分类           三表分析           估值方法
```

## 核心要点
- 财务报告质量光谱：GAAP合规 → 有偏选择 → 盈余管理 → 不合规 → 虚假交易
- 分析核心：盈利质量（利润表）、资产质量（资产负债表）、现金流（现金流量表）
- 估值需结合行业特性选择合适方法，避免估值陷阱
