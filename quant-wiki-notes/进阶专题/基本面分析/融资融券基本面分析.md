# 融资融券基本面分析

> [!note] 核心要点
> 融资融券数据是反映市场杠杆资金情绪和主力意图的重要指标，可用于趋势确认、主力跟踪、顶部预警和板块轮动分析。

## 一、两融核心数据字段

| 字段 | 含义 | 分析价值 |
|-----|------|---------|
| **融资余额** | 融资买入未偿还总金额 | 市场情绪领先指标 |
| **融资买入额** | 当日新增融资买入 | 杠杆加仓力度 |
| **融资偿还额** | 当日偿还融资金额 | 与买入额对比看净增减 |
| **融券余量** | 融券卖出未偿还股数 | 做空力量指标 |
| **融券卖出量** | 当日新增融券卖出 | 短期抛压信号 |
| **净买入** | 融资买入-偿还+融券偿还-卖出 | 杠杆资金真实方向 |

## 二、典型使用场景

### 1. 趋势确认
筛选近5日融资余额加速净流入的标的，验证强势行情起点。

### 2. 主力跟踪
**融资买入额突然放大 + 股价突破前高 = 杠杆资金入场信号**

### 3. 顶部预警
> [!warning] 预警信号
> 融资余额单日骤降 ≥ 3% 配合放量长上影线 = 阶段性高点警示

### 4. 板块轮动
按行业聚合两融余额变化，找出资金净流入最强的细分赛道。

### 5. 风险规避
融资余额持续下行的标的，通常对应中期下跌趋势。

## 三、基本面分析框架

### 融资余额分析
- **持续攀升**：市场做多情绪高涨，主力资金持续流入
- **快速回落**：市场信心动摇，可能预示调整
- **低位企稳**：底部信号，关注反转机会

### 融券余量分析
- **激增**：短期抛压增大，做空力量增强
- **持续下降**：空头回补，可能见底

### 融资净买入
- 反映杠杆资金的真实博弈意图
- 正值→多头占优，负值→空头占优

## 四、数据应用

- 覆盖范围：支持历史日期查询和月度数据切片
- 支持CSV导出，便于导入量化回测系统
- 每日T+1披露

## 实战掌握清单

> [!tip] 交易者视角
> 融资融券基本面分析 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 融资融券基本面分析 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。

## 最小可执行项目

### 公司研究备忘录

围绕 融资融券基本面分析 写一份公司研究备忘录：商业模式、行业格局、三张表、现金流、ROE/ROIC、负债、管理层、估值和风险。

| 证据 | 检查点 |
|---|---|
| 收入 | 是否可持续，是否依赖一次性因素 |
| 利润 | 毛利率和费用率是否合理 |
| 现金流 | 利润是否能变成现金 |
| 资产负债 | 应收、存货、商誉和负债是否异常 |
| 估值 | 是否有安全边际 |

### 验收标准

能用财务证据支持或否定投资假设，而不是只讲行业故事。

## 进一步实战化

### 财报案例练习

学习 `融资融券基本面分析` 后，选择一家上市公司，连续阅读三期财报，重点看收入、毛利率、费用率、经营现金流、应收、存货、负债和资本开支。不要只摘数字，要解释数字背后的商业变化。

| 检查点 | 需要回答 |
|---|---|
| 收入增长 | 来自销量、价格、并表还是一次性因素 |
| 利润质量 | 毛利率和费用率是否可持续 |
| 现金流 | 利润是否真正变成现金 |
| 资产负债 | 是否存在应收、存货、商誉或杠杆风险 |
| 投资结论 | 估值是否补偿了这些风险 |

验收标准：能写出一个支持买入的证据和一个必须警惕的反方证据。
