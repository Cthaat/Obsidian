# 三、现金流分析

> [!note] 概述
> 投资老鸟看现金流量表。现金流是企业生存的血液，缺乏现金的企业难以存活。现金流分析是检验盈利质量的终极手段。

## 笔记列表

1. [[现金流量表深度解读]] — 四大分类与理想的现金流组合
2. [[自由现金流分析]] — 利润与现金的结构差异，资本支出与营运资本
3. [[现金流与财务质量]] — 现金流重构与盈余质量评估

## 学习路径

```
现金流量表深度解读 → 自由现金流分析 → 现金流与财务质量
        ↓                  ↓                  ↓
    四大分类           FCF分析            质量评估
```

## 核心要点
- **FCF = 经营活动现金流 + 投资活动现金流**
- "利润"是观点，"现金"才是事实
- 收现比≥1、净现比接近1为优秀
- 能带来长期稳定现金流的公司是"印钞机"
