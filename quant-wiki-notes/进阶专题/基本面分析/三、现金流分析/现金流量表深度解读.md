# 现金流量表深度解读

> [!note] 核心概念
> 现金流量表记录企业在一定期间内现金的流入和流出。投资老鸟最看重现金流量表，因为它能看出公司目前的现金来源与使用方向。

## 一、现金流量表的四大分类

### 1. 经营活动现金流（OFC）
- **计算** = 税后净利 + 与现金无关的损益科目 + 营运资金变动
- **正值** = 营运有持续现金流入
- **负值** = 营运现金持续流出（初创期或大幅亏损时需警惕）
- 绝大多数时候应为正值，且数字不应与净利相差太大

### 2. 投资活动现金流
- **正值** = 变卖资产赚钱（多为一次性收入）
- **负值** = 扩大经营投资（正常状况）
- 可以看出企业的资本收入/支出情况

### 3. 筹资活动现金流
- **正值** = 向银行或股东借钱
- **负值** = 还债、付息、发股利、回购股票
- 可以看出企业的筹资情况

### 4. 自由现金流（FCF）
- **FCF = 经营活动现金流 + 投资活动现金流**
- 代表企业当前的可用资金
- 长期为负不代表不赚钱，但可能有资金短缺危机

## 二、理想的现金流组合

| 经营CF | 投资CF | 筹资CF | 企业状态 |
|--------|--------|--------|---------|
| + | - | - | 成熟期：经营造血+投资扩张+偿债 |
| + | - | + | 成长期：经营造血+大量投资+融资补充 |
| + | + | - | 收缩期：经营+变卖资产+偿债 |
| - | - | + | 初创期：全靠融资支撑 |
| - | + | + | 衰退期：变卖资产+借债维持 |

## 三、现金流分析要点

### 经营活动现金流
- 健康企业该指标应持续为正
- 与净利润的匹配程度是判断盈利质量的重要依据
- 净利润可观但经营现金流持续为负 → 利润存在"水分"

### 投资活动现金流
- 可看到企业对资产投资的数据线索
- 重大投资现金流出是否持续 → 展现投资理念和持续获得优质资产的能力

### 筹资活动现金流
- 反映外部输血依赖度
- 长期依赖筹资输血 → 警惕商业模式可持续性

## 实战掌握清单

> [!tip] 交易者视角
> 现金流量表深度解读 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 现金流量表深度解读 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。

## 最小可执行项目

### 公司研究备忘录

围绕 现金流量表深度解读 写一份公司研究备忘录：商业模式、行业格局、三张表、现金流、ROE/ROIC、负债、管理层、估值和风险。

| 证据 | 检查点 |
|---|---|
| 收入 | 是否可持续，是否依赖一次性因素 |
| 利润 | 毛利率和费用率是否合理 |
| 现金流 | 利润是否能变成现金 |
| 资产负债 | 应收、存货、商誉和负债是否异常 |
| 估值 | 是否有安全边际 |

### 验收标准

能用财务证据支持或否定投资假设，而不是只讲行业故事。

## 进一步实战化

### 财报案例练习

学习 `现金流量表深度解读` 后，选择一家上市公司，连续阅读三期财报，重点看收入、毛利率、费用率、经营现金流、应收、存货、负债和资本开支。不要只摘数字，要解释数字背后的商业变化。

| 检查点 | 需要回答 |
|---|---|
| 收入增长 | 来自销量、价格、并表还是一次性因素 |
| 利润质量 | 毛利率和费用率是否可持续 |
| 现金流 | 利润是否真正变成现金 |
| 资产负债 | 是否存在应收、存货、商誉或杠杆风险 |
| 投资结论 | 估值是否补偿了这些风险 |

验收标准：能写出一个支持买入的证据和一个必须警惕的反方证据。
