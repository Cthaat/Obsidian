# 现金流量表深度解读

> [!note] 核心概念
> 现金流量表记录企业在一定期间内现金的流入和流出。投资老鸟最看重现金流量表，因为它能看出公司目前的现金来源与使用方向。

## 一、现金流量表的四大分类

### 1. 经营活动现金流（OFC）
- **计算** = 税后净利 + 与现金无关的损益科目 + 营运资金变动
- **正值** = 营运有持续现金流入
- **负值** = 营运现金持续流出（初创期或大幅亏损时需警惕）
- 绝大多数时候应为正值，且数字不应与净利相差太大

### 2. 投资活动现金流
- **正值** = 变卖资产赚钱（多为一次性收入）
- **负值** = 扩大经营投资（正常状况）
- 可以看出企业的资本收入/支出情况

### 3. 筹资活动现金流
- **正值** = 向银行或股东借钱
- **负值** = 还债、付息、发股利、回购股票
- 可以看出企业的筹资情况

### 4. 自由现金流（FCF）
- **FCF = 经营活动现金流 + 投资活动现金流**
- 代表企业当前的可用资金
- 长期为负不代表不赚钱，但可能有资金短缺危机

## 二、理想的现金流组合

| 经营CF | 投资CF | 筹资CF | 企业状态 |
|--------|--------|--------|---------|
| + | - | - | 成熟期：经营造血+投资扩张+偿债 |
| + | - | + | 成长期：经营造血+大量投资+融资补充 |
| + | + | - | 收缩期：经营+变卖资产+偿债 |
| - | - | + | 初创期：全靠融资支撑 |
| - | + | + | 衰退期：变卖资产+借债维持 |

## 三、现金流分析要点

### 经营活动现金流
- 健康企业该指标应持续为正
- 与净利润的匹配程度是判断盈利质量的重要依据
- 净利润可观但经营现金流持续为负 → 利润存在"水分"

### 投资活动现金流
- 可看到企业对资产投资的数据线索
- 重大投资现金流出是否持续 → 展现投资理念和持续获得优质资产的能力

### 筹资活动现金流
- 反映外部输血依赖度
- 长期依赖筹资输血 → 警惕商业模式可持续性
