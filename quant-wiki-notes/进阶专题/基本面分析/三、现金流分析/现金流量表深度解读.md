# 现金流量表深度解读

> [!note] 核心概念
> 现金流量表记录企业在一定期间内现金的流入和流出。投资老鸟最看重现金流量表，因为它能看出公司目前的现金来源与使用方向。

## 一、现金流量表的四大分类

### 1. 经营活动现金流（OFC）
- **计算** = 税后净利 + 与现金无关的损益科目 + 营运资金变动
- **正值** = 营运有持续现金流入
- **负值** = 营运现金持续流出（初创期或大幅亏损时需警惕）
- 绝大多数时候应为正值，且数字不应与净利相差太大

### 2. 投资活动现金流
- **正值** = 变卖资产赚钱（多为一次性收入）
- **负值** = 扩大经营投资（正常状况）
- 可以看出企业的资本收入/支出情况

### 3. 筹资活动现金流
- **正值** = 向银行或股东借钱
- **负值** = 还债、付息、发股利、回购股票
- 可以看出企业的筹资情况

### 4. 自由现金流（FCF）
- **FCF = 经营活动现金流 + 投资活动现金流**
- 代表企业当前的可用资金
- 长期为负不代表不赚钱，但可能有资金短缺危机

## 二、理想的现金流组合

| 经营CF | 投资CF | 筹资CF | 企业状态 |
|--------|--------|--------|---------|
| + | - | - | 成熟期：经营造血+投资扩张+偿债 |
| + | - | + | 成长期：经营造血+大量投资+融资补充 |
| + | + | - | 收缩期：经营+变卖资产+偿债 |
| - | - | + | 初创期：全靠融资支撑 |
| - | + | + | 衰退期：变卖资产+借债维持 |

## 三、现金流分析要点

### 经营活动现金流
- 健康企业该指标应持续为正
- 与净利润的匹配程度是判断盈利质量的重要依据
- 净利润可观但经营现金流持续为负 → 利润存在"水分"

### 投资活动现金流
- 可看到企业对资产投资的数据线索
- 重大投资现金流出是否持续 → 展现投资理念和持续获得优质资产的能力

### 筹资活动现金流
- 反映外部输血依赖度
- 长期依赖筹资输血 → 警惕商业模式可持续性

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `现金流量表深度解读` 时，不要只把它当成孤立知识点。把企业研究落到现金流、竞争优势、治理和资本回报，不停在故事层面。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 三张报表和杜邦拆解
- 行业周期和议价能力
- 资本开支、营运资本和自由现金流
- 估值、安全边际和仓位

### 进阶训练

1. 用利润表、资产负债表、现金流量表互相验证
2. 写出 ROE 的三层来源
3. 给企业做乐观、基准、悲观三种经营情景

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
