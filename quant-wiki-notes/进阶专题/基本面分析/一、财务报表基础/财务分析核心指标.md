# 财务分析核心指标

> [!note] 核心框架
> 财务报表分析的核心方法论可概括为"**结构拆解 + 比率计算 + 趋势对比 + 行业对标**"四维框架，聚焦偿债能力、盈利能力、运营效率和增长潜力四大核心维度。

## 一、盈利能力指标

| 指标 | 公式 | 含义 | 参考值 |
|------|------|------|--------|
| 扣非归母净利润 | - | 扣除非经常性损益后的净利润 | 越高越好 |
| 息税前利润(EBIT) | 净利润+所得税+利息 | 未扣除利息和税的利润 | - |
| 净资产收益率(ROE) | 净利润/平均净资产 | 股东投资回报率 | 10%-20%优秀 |
| 资产收益率(ROA) | 净利润/平均总资产 | 全部资产盈利能力 | 越高越好 |
| 投入资本回报率(ROIC) | NOPAT/投入资本 | 资本使用效率 | >WACC为创造价值 |
| 人力投入回报率(ROP) | 净利润/人工成本 | 人力资本效率 | - |

## 二、现金流指标

| 指标 | 公式 | 含义 |
|------|------|------|
| EBITDA | 税息折旧摊销前利润 | 经营性现金流的近似 |
| 自由现金流(FCF) | 经营现金流-资本支出 | 可自由分配的现金 |
| 营业现金比率 | 经营现金流/营业收入 | 收入的现金转化率 |
| 净现比 | 经营现金流/净利润 | 利润的含金量 |
| 现金满足投资比率 | 经营现金流/(资本支出+存货增加+现金股利) | 内部融资能力 |

## 三、成长能力指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 营收增长率 | (本期营收-上期营收)/上期营收 | 市场扩张能力 |
| 净利润增长率 | (本期净利-上期净利)/上期净利 | 盈利增长潜力 |
| 扣非净利润增长率 | - | 核心盈利增长 |
| 每股收益增长率 | - | 股东回报增长 |

## 四、营运能力指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 应收账款周转率 | 营收/平均应收账款 | 回款速度 |
| 存货周转率 | 营业成本/平均存货 | 存货管理效率 |
| 总资产周转率 | 营收/平均总资产 | 资产利用效率 |
| 营运资本周转率 | 营收/平均营运资本 | 营运资本效率 |
| 净营业周期 | 存货周转天数+应收周转天数-应付周转天数 | 资金占用周期 |

## 五、偿债能力指标

| 指标 | 公式 | 含义 | 参考值 |
|------|------|------|--------|
| 流动比率 | 流动资产/流动负债 | 短期偿债能力 | 1.5-2.0 |
| 速动比率 | (流动资产-存货)/流动负债 | 严格短期偿债 | 1.0左右 |
| 资产负债率 | 总负债/总资产 | 长期偿债风险 | 40%-60% |
| 利息保障倍数 | EBIT/利息费用 | 利息支付能力 | >3倍 |

## 六、综合分析方法

### 趋势分析法
- 连续数期财务数据的变动趋势
- 判断企业处于成长、稳定还是衰退阶段

### 比较分析法
- **横向比较**：与同行业可比公司对比
- **纵向比较**：与企业历史数据对比
- 毛利率显著高于行业 → 可能具备竞争优势或产品差异化

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
