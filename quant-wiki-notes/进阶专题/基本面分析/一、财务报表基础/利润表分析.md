# 利润表分析

> [!note] 核心概念
> 利润表反映企业在一定会计期间的经营成果，以"收入-成本-费用=净利润"为脉络，分层次揭示盈利来源。

## 一、利润表结构

### 利润层次
1. **营业收入**：主营业务收入 + 其他业务收入
2. **毛利润** = 营业收入 - 营业成本（核心业务盈利能力）
3. **经营利润** = 毛利润 - 销售费用 - 管理费用 - 研发费用 - 财务费用
4. **利润总额** = 经营利润 + 投资收益 + 营业外收支
5. **净利润** = 利润总额 - 所得税费用

### 核心利润概念
> [!important] 核心利润
> 核心利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用
>
> 核心利润强调正常经营、正常核算下所产生的正常利润。越高说明盈利能力越强、盈利质量越高。

## 二、关键盈利指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 毛利率 | (营收-营业成本)/营收 | 产品直接盈利能力 |
| 净利率 | 净利润/营业收入 | 企业整体盈利水平 |
| 核心利润率 | 核心利润/营业收入 | 正常经营盈利能力 |
| 费用率 | 期间费用/营业收入 | 费用控制能力 |

## 三、收入分析要点

- 收入构成与变动原因
- 分业务类别和地区的收入明细
- 主要产品销售数量、价格与结构变化
- 收入波动是否存在季节性等特殊性

## 四、成本费用分析

### 营业成本
- 逐年比较成本分布情况
- 关注主要成本项目构成及变动原因
- 结合原材料、能源采购价格变动分析

### 毛利率分析
- 毛利率较高原因：行业领先地位、产品竞争力强、品牌溢价
- 毛利率较低原因：行业衰退、缺乏竞争力、市场开拓期
- 特殊会计处理方式也会对毛利率造成影响

## 五、盈利质量判断

> [!warning] 注意
> 如果一家公司能够持续产生大量利润却没有持续产生足够的现金，那么这些"利润"就含有水分，是"纸面富贵"。

### 判断方法
- **收现比** = 销售商品收到的现金 / 营业收入（应≥1）
- **净现比** = 经营活动现金流净额 / 净利润（应接近1）
- **现金流量偏离标准比率** = 经营性现金流量 / (净利润+折旧+摊销)（应在1左右）

## 实战掌握清单

> [!tip] 交易者视角
> 利润表分析 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 利润表分析 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。

## 最小可执行项目

### 公司研究备忘录

围绕 利润表分析 写一份公司研究备忘录：商业模式、行业格局、三张表、现金流、ROE/ROIC、负债、管理层、估值和风险。

| 证据 | 检查点 |
|---|---|
| 收入 | 是否可持续，是否依赖一次性因素 |
| 利润 | 毛利率和费用率是否合理 |
| 现金流 | 利润是否能变成现金 |
| 资产负债 | 应收、存货、商誉和负债是否异常 |
| 估值 | 是否有安全边际 |

### 验收标准

能用财务证据支持或否定投资假设，而不是只讲行业故事。
