# 一、财务报表基础

> [!note] 概述
> 财务报表是企业经营状况的"全景图"，掌握三大报表的阅读和分析能力是基本面分析的基础。

## 笔记列表

1. [[三大财务报表概览]] — 资产负债表、利润表、现金流量表的核心框架与勾稽关系
2. [[资产负债表分析]] — 企业"家底"的静态快照，资产质量与偿债能力
3. [[利润表分析]] — 企业"成绩单"的动态记录，盈利质量判断
4. [[现金流量表基础]] — 企业"血液"的流向追踪，四大分类与自由现金流
5. [[财务分析核心指标]] — 四维分析框架与关键比率体系

## 学习路径

```
资产负债表 → 利润表 → 现金流量表 → 综合分析指标
    ↓            ↓           ↓              ↓
  资产质量    盈利质量    现金流质量    四维分析框架
```

## 核心要点
- 资产负债表是**时点值**（静态），利润表和现金流量表是**时期值**（动态）
- 三表环环相扣：净利润影响未分配利润，现金流从净利润调节而来
- 分析核心：**资产质量**（资产负债表）、**盈利质量**（利润表）、**现金流质量**（现金流量表）
