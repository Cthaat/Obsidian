# 三大财务报表概览

> [!note] 核心概念
> 财务报表是企业经营状况的"全景图"，通过三张主表加附注构成企业的"三维立体"财务数据。三张报表环环相扣、互相印证，将企业的成长变化以数据方式形象展现。

## 一、三大报表的功能定位

- **资产负债表** — 企业的"家底"，特定日期的静态财务快照
- **利润表** — 企业的"成绩单"，一定期间的动态盈利记录
- **现金流量表** — 企业的"血液"，一定期间的现金流向追踪

## 二、资产负债表（Balance Sheet）

> [!note] 核心等式
> **资产 = 负债 + 所有者权益**

### 资产端
- **流动资产**：货币资金、应收账款、存货（反映短期运营效率）
- **非流动资产**：固定资产、无形资产、在建工程（体现长期战略投入）

### 负债端
- **流动负债**：应付账款、短期借款（揭示短期偿债压力）
- **非流动负债**：公司债券、长期借款（映射资本结构稳定性）

### 所有者权益
- 股东对企业的所有权，代表企业净资产
- **每股净资产** = 净资产 / 总股本

### 分析要点
- 流动资产占比过高 → 资金利用效率不足
- 资产负债率过高 → 财务风险较大
- 应付账款占流动负债比例 → 对供应商的议价能力

## 三、利润表（Income Statement）

> [!note] 核心结构
> **营业收入 - 营业成本 - 期间费用 + 投资收益 + 营业外收支 = 净利润**

### 关键层次
- **毛利润** = 营业收入 - 营业成本（核心业务盈利能力）
- **经营利润** = 毛利润 - 销售/管理/研发费用（经营效率）
- **净利润** = 综合税负、利息等后的最终盈利

### 关键指标
- **毛利率** = (营业收入 - 营业成本) / 营业收入
- **净利率** = 净利润 / 营业收入
- **费用率** = 期间费用 / 营业收入

### IFRS 18 新变化（2025年）
- 收入和费用按**经营、投资、融资、所得税、终止经营**五大类别列示
- 新增"营业利润"和"融资前利润"等强制性小计项

## 四、现金流量表（Cash Flow Statement）

> [!note] 核心价值
> 现金流量表按照收付实现制编制，弥补了利润表权责发生制的局限，是判断盈利质量的关键工具。

### 三大分类
- **经营活动现金流**：日常运营产生的现金流（健康企业应持续为正）
- **投资活动现金流**：购置设备、处置资产等（扩张期常为净流出）
- **筹资活动现金流**：股权融资、债务偿还（反映外部输血依赖度）

### 核心指标
- **自由现金流（FCF）** = 经营活动现金流 + 投资活动现金流
- **盈利质量** = 经营活动现金流净额 / 净利润（接近1或>1为优秀）

## 五、三表勾稽关系

```
资产负债表 ←→ 利润表：净利润影响未分配利润
利润表 ←→ 现金流量表：净利润是间接法起点
资产负债表 ←→ 现金流量表：现金变动反映在货币资金
```

### 关键验证
- 利润表净利润需与现金流量表"净利润调节项"匹配
- 净利润需与资产负债表未分配利润变动匹配
- 当净利润与经营活动现金流长期背离时，可能预示财务异常

## 六、分析框架总结

| 维度 | 核心问题 | 关键指标 |
|------|---------|---------|
| 偿债能力 | 能否按时还债？ | 流动比率、速动比率、资产负债率 |
| 盈利能力 | 赚钱能力强不强？ | 毛利率、净利率、ROE、ROA |
| 运营效率 | 资产利用效率？ | 应收账款周转率、存货周转率、总资产周转率 |
| 发展潜力 | 未来增长空间？ | 营收增长率、净利润增长率、研发投入比 |

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `三大财务报表概览` 时，不要只把它当成孤立知识点。把企业研究落到现金流、竞争优势、治理和资本回报，不停在故事层面。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 三张报表和杜邦拆解
- 行业周期和议价能力
- 资本开支、营运资本和自由现金流
- 估值、安全边际和仓位

### 进阶训练

1. 用利润表、资产负债表、现金流量表互相验证
2. 写出 ROE 的三层来源
3. 给企业做乐观、基准、悲观三种经营情景

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
