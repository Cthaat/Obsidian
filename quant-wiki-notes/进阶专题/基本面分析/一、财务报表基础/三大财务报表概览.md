# 三大财务报表概览

> [!note] 核心概念
> 财务报表是企业经营状况的"全景图"，通过三张主表加附注构成企业的"三维立体"财务数据。三张报表环环相扣、互相印证，将企业的成长变化以数据方式形象展现。

## 一、三大报表的功能定位

- **资产负债表** — 企业的"家底"，特定日期的静态财务快照
- **利润表** — 企业的"成绩单"，一定期间的动态盈利记录
- **现金流量表** — 企业的"血液"，一定期间的现金流向追踪

## 二、资产负债表（Balance Sheet）

> [!note] 核心等式
> **资产 = 负债 + 所有者权益**

### 资产端
- **流动资产**：货币资金、应收账款、存货（反映短期运营效率）
- **非流动资产**：固定资产、无形资产、在建工程（体现长期战略投入）

### 负债端
- **流动负债**：应付账款、短期借款（揭示短期偿债压力）
- **非流动负债**：公司债券、长期借款（映射资本结构稳定性）

### 所有者权益
- 股东对企业的所有权，代表企业净资产
- **每股净资产** = 净资产 / 总股本

### 分析要点
- 流动资产占比过高 → 资金利用效率不足
- 资产负债率过高 → 财务风险较大
- 应付账款占流动负债比例 → 对供应商的议价能力

## 三、利润表（Income Statement）

> [!note] 核心结构
> **营业收入 - 营业成本 - 期间费用 + 投资收益 + 营业外收支 = 净利润**

### 关键层次
- **毛利润** = 营业收入 - 营业成本（核心业务盈利能力）
- **经营利润** = 毛利润 - 销售/管理/研发费用（经营效率）
- **净利润** = 综合税负、利息等后的最终盈利

### 关键指标
- **毛利率** = (营业收入 - 营业成本) / 营业收入
- **净利率** = 净利润 / 营业收入
- **费用率** = 期间费用 / 营业收入

### IFRS 18 新变化（2025年）
- 收入和费用按**经营、投资、融资、所得税、终止经营**五大类别列示
- 新增"营业利润"和"融资前利润"等强制性小计项

## 四、现金流量表（Cash Flow Statement）

> [!note] 核心价值
> 现金流量表按照收付实现制编制，弥补了利润表权责发生制的局限，是判断盈利质量的关键工具。

### 三大分类
- **经营活动现金流**：日常运营产生的现金流（健康企业应持续为正）
- **投资活动现金流**：购置设备、处置资产等（扩张期常为净流出）
- **筹资活动现金流**：股权融资、债务偿还（反映外部输血依赖度）

### 核心指标
- **自由现金流（FCF）** = 经营活动现金流 + 投资活动现金流
- **盈利质量** = 经营活动现金流净额 / 净利润（接近1或>1为优秀）

## 五、三表勾稽关系

```
资产负债表 ←→ 利润表：净利润影响未分配利润
利润表 ←→ 现金流量表：净利润是间接法起点
资产负债表 ←→ 现金流量表：现金变动反映在货币资金
```

### 关键验证
- 利润表净利润需与现金流量表"净利润调节项"匹配
- 净利润需与资产负债表未分配利润变动匹配
- 当净利润与经营活动现金流长期背离时，可能预示财务异常

## 六、分析框架总结

| 维度 | 核心问题 | 关键指标 |
|------|---------|---------|
| 偿债能力 | 能否按时还债？ | 流动比率、速动比率、资产负债率 |
| 盈利能力 | 赚钱能力强不强？ | 毛利率、净利率、ROE、ROA |
| 运营效率 | 资产利用效率？ | 应收账款周转率、存货周转率、总资产周转率 |
| 发展潜力 | 未来增长空间？ | 营收增长率、净利润增长率、研发投入比 |

## 实战掌握清单

> [!tip] 交易者视角
> 三大财务报表概览 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
