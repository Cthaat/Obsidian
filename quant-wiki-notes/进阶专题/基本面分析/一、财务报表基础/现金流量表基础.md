# 现金流量表基础

> [!note] 核心概念
> 现金流量表记录企业在一定期间内现金的流入和流出，按照收付实现制编制。投资菜鸟看损益表，老鸟都看现金流量表。现金对企业如同血液，缺乏现金的企业难以存活。

## 一、现金流量表的四大分类

### 1. 经营活动现金流（Operating Cash Flow）
- 与营业相关的现金流（生产、销售等）
- **正值** = 营运有持续现金流入
- **负值** = 营运现金持续流出（初创期或大幅亏损时需警惕）
- 绝大多数时候应为正值，且数字不应与净利润相差太大

### 2. 投资活动现金流（Cash Flow from Investing）
- 与投资相关：土地买卖、购入厂房设备、购买股权
- **正值** = 变卖资产赚钱（多为一次性收入）
- **负值** = 扩大经营投资（正常状况）

### 3. 筹资活动现金流（Cash Flows From Financing）
- 与筹资相关：股东募款、银行借款等
- **正值** = 向银行或股东借钱
- **负值** = 还债、付息、发股利、回购股票

### 4. 自由现金流（Free Cash Flow）

> [!important] 自由现金流
> **FCF = 经营活动现金流 + 投资活动现金流**
>
> 代表企业靠运营赚到的现金，扣除投资费用后，可自由运用的资金。长期为负不代表不赚钱，但可能有资金短缺危机。

## 二、现金流分析要点

### 理想的现金流组合
| 经营现金流 | 投资现金流 | 筹资现金流 | 含义 |
|-----------|-----------|-----------|------|
| + | - | - | 成熟企业，经营造血+投资扩张+偿债 |
| + | - | + | 成长期，经营造血+大量投资+融资补充 |
| + | + | - | 收缩期，经营+变卖资产+偿债 |
| - | - | + | 初创期，全靠融资支撑 |

### 关键验证指标
- **盈利质量** = 经营活动现金流净额 / 净利润（接近1或>1为优秀）
- **收现比** = 销售商品收到的现金 / 营业收入（应≥1）
- **净现比** = 经营活动现金流净额 / 净利润（越大越好，1为优秀）

## 三、芒格论自由现金流

> [!quote] 查理·芒格
> "世界上有两种生意，第一种可以每年赚12%的收益，到年底股东可以拿走所有利润；第二种也可以每年赚12%，但是你不得不把赚来的钱重新投资，然后你指着所有的厂房设备对股东们说：'这就是你们的利润'。我恨第二种生意。"

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
