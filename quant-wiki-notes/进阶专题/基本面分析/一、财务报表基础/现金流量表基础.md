# 现金流量表基础

> [!note] 核心概念
> 现金流量表记录企业在一定期间内现金的流入和流出，按照收付实现制编制。投资菜鸟看损益表，老鸟都看现金流量表。现金对企业如同血液，缺乏现金的企业难以存活。

## 一、现金流量表的四大分类

### 1. 经营活动现金流（Operating Cash Flow）
- 与营业相关的现金流（生产、销售等）
- **正值** = 营运有持续现金流入
- **负值** = 营运现金持续流出（初创期或大幅亏损时需警惕）
- 绝大多数时候应为正值，且数字不应与净利润相差太大

### 2. 投资活动现金流（Cash Flow from Investing）
- 与投资相关：土地买卖、购入厂房设备、购买股权
- **正值** = 变卖资产赚钱（多为一次性收入）
- **负值** = 扩大经营投资（正常状况）

### 3. 筹资活动现金流（Cash Flows From Financing）
- 与筹资相关：股东募款、银行借款等
- **正值** = 向银行或股东借钱
- **负值** = 还债、付息、发股利、回购股票

### 4. 自由现金流（Free Cash Flow）

> [!important] 自由现金流
> **FCF = 经营活动现金流 + 投资活动现金流**
>
> 代表企业靠运营赚到的现金，扣除投资费用后，可自由运用的资金。长期为负不代表不赚钱，但可能有资金短缺危机。

## 二、现金流分析要点

### 理想的现金流组合
| 经营现金流 | 投资现金流 | 筹资现金流 | 含义 |
|-----------|-----------|-----------|------|
| + | - | - | 成熟企业，经营造血+投资扩张+偿债 |
| + | - | + | 成长期，经营造血+大量投资+融资补充 |
| + | + | - | 收缩期，经营+变卖资产+偿债 |
| - | - | + | 初创期，全靠融资支撑 |

### 关键验证指标
- **盈利质量** = 经营活动现金流净额 / 净利润（接近1或>1为优秀）
- **收现比** = 销售商品收到的现金 / 营业收入（应≥1）
- **净现比** = 经营活动现金流净额 / 净利润（越大越好，1为优秀）

## 三、芒格论自由现金流

> [!quote] 查理·芒格
> "世界上有两种生意，第一种可以每年赚12%的收益，到年底股东可以拿走所有利润；第二种也可以每年赚12%，但是你不得不把赚来的钱重新投资，然后你指着所有的厂房设备对股东们说：'这就是你们的利润'。我恨第二种生意。"

## 实战掌握清单

> [!tip] 交易者视角
> 现金流量表基础 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 现金流量表基础 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。

## 最小可执行项目

### 公司研究备忘录

围绕 现金流量表基础 写一份公司研究备忘录：商业模式、行业格局、三张表、现金流、ROE/ROIC、负债、管理层、估值和风险。

| 证据 | 检查点 |
|---|---|
| 收入 | 是否可持续，是否依赖一次性因素 |
| 利润 | 毛利率和费用率是否合理 |
| 现金流 | 利润是否能变成现金 |
| 资产负债 | 应收、存货、商誉和负债是否异常 |
| 估值 | 是否有安全边际 |

### 验收标准

能用财务证据支持或否定投资假设，而不是只讲行业故事。
