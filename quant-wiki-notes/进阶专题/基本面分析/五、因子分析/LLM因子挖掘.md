# LLM因子挖掘

> [!note] 核心要点
> 大语言模型（LLM）在量化因子挖掘中展现出巨大潜力，可降低人工成本、提升可解释性、实现人机交互式因子生成。主要分为五种应用方向。

## 一、传统方法的局限与LLM优势

| 传统方法 | 局限 | LLM如何解决 |
|---------|------|------------|
| 人工挖掘 | 效率低、成本高 | 降低人力成本 |
| 遗传规划 | 过拟合、可解释性差 | 逻辑推理+可解释性 |
| 强化学习 | 需大量数据和训练 | Few-Shot高效学习 |

## 二、LLM因子挖掘五大方向

### 1. 底层字段驱动的因子生成

**输入三要素**：
- **底层字段**：量价数据、基本面数据（三表）、高频数据
- **因子示例**：Few-Shot方式提供已有因子的名称、表达式、含义
- **常用算子**：
  - 基本面算子：同比、环比、截面排序、分位数
  - 高频算子：聚合（标准差、均值、偏度、峰度）、滚动窗口

**输出**：因子名称 + 数学表达式 + 因子含义

### 2. 文本与多模态数据驱动

> [!tip] 种子Alpha因子工厂
> 利用LLM分析研报、论文、图表等多模态数据，生成种子Alpha因子，再通过多智能体系统评估和优化。

**流程**：
1. LLM处理文本+视觉+数值数据 → 生成初步Alpha因子
2. 过滤分类 → 动量、波动率、基本面等类别
3. 多智能体评估 → 置信度评分 → 筛选高质量因子

### 3. 人机交互驱动（Alpha-GPT）

```
用户交易想法（自然语言）
    ↓
知识编译器（领域上下文增强）
    ↓
LLM生成因子（数学表达式+含义）
    ↓
因子测试与优化
```

**示例**：
- 用户输入："基于最近10天的价格动量，寻找预测信号"
- 知识编译器补充金融上下文和类似因子示例
- LLM生成：`Alpha_i = MovingAverage(Price, 10) / sqrt(Volume)`

### 4. 情绪与新闻驱动（SKGP方法）

**顺序知识引导提示（SKGP）** 三步骤：
1. **匹配背景知识**：将目标股票与相关新闻匹配，获取关联股票关系
2. **提取影响因素**：从新闻中提取可能影响股价的前k个因素
3. **预测价格走势**：结合关系+因素+历史价格，预测涨跌

> [!important] 优势
> LLM提取的影响因素不局限于新闻词汇，会考虑内容对股票走势的潜在影响，比传统情感分析更精准。

### 5. 因子杂交与优化

| 操作 | 说明 | 示例 |
|-----|------|------|
| **因子杂交** | 将不同因子结合 | 资金流入动量 × 价格动量 |
| **无方向变异** | LLM自行添加改动 | RSI + EMA |
| **有方向变异** | 指定修改方向 | RSI + Volume |
| **因子入库检查** | 语义分类去重 | 降低同质化 |

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `LLM因子挖掘` 时，不要只把它当成孤立知识点。把企业研究落到现金流、竞争优势、治理和资本回报，不停在故事层面。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 三张报表和杜邦拆解
- 行业周期和议价能力
- 资本开支、营运资本和自由现金流
- 估值、安全边际和仓位

### 进阶训练

1. 用利润表、资产负债表、现金流量表互相验证
2. 写出 ROE 的三层来源
3. 给企业做乐观、基准、悲观三种经营情景

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
