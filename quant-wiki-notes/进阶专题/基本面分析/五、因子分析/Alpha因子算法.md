# Alpha因子算法

> [!note] 核心要点
> Alpha因子挖掘是从海量数据中发现具有预测能力的选股因子的过程。主要方法包括人工挖掘、算法自动挖掘（遗传规划等）和LLM辅助挖掘。

## 一、因子挖掘方法分类

| 方法 | 优势 | 劣势 |
|-----|------|------|
| **人工挖掘** | 逻辑清晰、可解释性强 | 效率低、成本高 |
| **遗传规划（GP）** | 自动化搜索、覆盖面广 | 过拟合、可解释性差 |
| **LLM辅助挖掘** | 兼顾效率与可解释性 | 模型依赖、需验证 |

## 二、遗传规划（Genetic Programming）

### 基本原理
遗传规划模拟生物进化过程，通过**选择、交叉、变异**操作迭代优化因子表达式树。

### 算子体系

| 算子类型 | 示例 |
|---------|------|
| 时序算子 | TS_Mean, TS_Std, TS_Max, TS_Min, TS_Rank |
| 截面算子 | CS_Rank, CS_Zscore, CS_Winsorize |
| 元素算子 | Add, Sub, Mul, Div, Abs, Log, Sign |
| 聚合算子 | Mean, Std, Corr, Cov |

### 适应度评估
- **IC（信息系数）**：因子值与未来收益的相关性
- **IC_IR**：IC的均值/标准差，衡量稳定性
- **换手率**：因子信号的稳定性
- **多空收益**：因子分组的收益区分度

## 三、因子生成流程

```
底层数据 → 算子组合 → 因子表达式 → 回测评估 → 筛选入库
```

### 1. 数据层
- 量价数据：开高低收、成交量、成交额
- 基本面数据：财务报表三表字段
- 高频数据：分钟级量价、逐笔委托

### 2. 表达式生成
通过算子组合生成因子表达式，例如：
```
Rank_Mul(Decay_Linear(TS_Return(CLOSE, 10), 10),
         Rank(Div(VOLUME, TS_Mean(VOLUME, 10))))
```

### 3. 回测评估
- 中低频因子：5天滚动，周度调仓
- 高频因子：日频调仓，考虑成本可降至周频

## 四、因子优化技术

| 技术 | 说明 |
|-----|------|
| **因子正交化** | 去除因子间的共线性 |
| **因子中性化** | 剔除行业和市值的影响 |
| **因子衰减处理** | 对因子值进行时间衰减加权 |
| **异常值处理** | Winsorize/截面标准化 |

## 五、因子库管理

> [!tip] 因子入库标准
> - IC均值 > 0.03（或绝对值 > 0.03）
> - IC_IR > 0.5
> - 多空年化收益 > 5%
> - 因子间相关性 < 0.7（保持多样性）

## 六、算法改进方向

1. **深度学习替代GP**：用神经网络自动学习因子表达式
2. **LLM辅助生成**：利用大语言模型的逻辑推理能力
3. **多目标优化**：同时优化IC、换手率、最大回撤等
4. **动态因子库**：根据市场状态自动调整因子权重

## 实战掌握清单

> [!tip] 交易者视角
> Alpha因子算法 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 Alpha因子算法 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。
