# PB-ROE选股模型

> [!note] 核心概念
> PB-ROE策略是经典的金融学选股方法，逻辑简单："**选好公司，在便宜时候买进**"。低PB=便宜，高ROE=好公司。

## 一、市净率（PB）基础

### 定义
- **PB = 市值 / 净资产 = 每股股价 / 每股净资产**
- 理论上代表以几倍的净资产价格买入一家公司
- PB=1 → 以净资产价格买入

### PB的适用范围
- **适合PB估值**：周期股、房地产、银行业、券商、保险
- **不适合PB估值**：以无形资产（商誉、知识产权）为主的公司
- PB一般在**3-10之间**比较合理，接近3或低于3时投资价值较高

## 二、ROE与PB的关系

> [!important] 核心公式
> **ROE = PB / PE = 净利润 / 净资产**
>
> ROE越高 → 资产盈利能力越强 → PB应该越高

### 双击与双杀
- **高PB买入高ROE企业** → ROE回落 → 业绩和估值**双杀**
- **低PB买入低ROE企业** → ROE上升 → 业绩和估值**双击**

> [!tip] 投资法则
> **业绩看ROE，估值看PB**，是投资制胜的不二法则。

## 三、PB-ROE选股模型逻辑

### "便宜"的标准
- **低PB**：每一块钱账面价值对应的价格低
- PB越低，安全系数越高（同一股票）

### "好公司"的标准
- **高ROE**：每一块钱投资得到的回报高
- ROE长期在10%-20%为优秀

### 模型优势
1. 经营者可通过模型变量关系知道如何提高公司估值
2. 基本面分析者能利用模型评价企业管理
3. 能够给股票定价（计算均衡价格）
4. 帮助主动投资者利用价格差异获取超额收益

## 四、实操注意事项

- 市净率越低就越安全的说法**不准确**，需具体分析
- 对同一股票：PB越低，安全系数越高
- 对不同股票：要结合行业和经营情况分析
- 需关注ROE的可持续性和利润含金量

## 实战掌握清单

> [!tip] 交易者视角
> PB-ROE选股模型 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。基本面分析要把商业模式、财报质量、竞争格局、资本配置和估值连接起来。

### 关键判断

- 先看公司如何赚钱、向谁赚钱、为什么能持续赚钱。
- 再看收入、毛利率、费用、现金流、资产负债和资本开支是否匹配。
- 用ROE、ROIC、自由现金流和负债结构识别质量。

### 落地动作

1. 把利润表、资产负债表和现金流量表交叉验证。
2. 为行业景气、竞争格局和管理层资本配置写出假设。
3. 估值前先排除财务舞弊、利润粉饰和周期错觉。

### 失效边界

- 只看利润增长不看现金流。
- 忽略应收、存货和商誉风险。
- 把一次性景气当作长期护城河。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 公司研究练习

学习 PB-ROE选股模型 后，选择一家企业做完整拆解：商业模式、行业格局、三张表、现金流、资本开支、负债、管理层和估值。每个结论都要对应一项财务证据或经营证据。

### 红旗清单

- 利润增长但经营现金流长期跟不上。
- 应收、存货、商誉或资本化费用异常扩张。
- 高ROE来自高杠杆而不是高周转或高利润率。
- 大额关联交易、频繁并购或复杂金融资产掩盖主营质量。

### 输出要求

最终写出买入理由、反方风险、估值区间和退出条件。

## 最小可执行项目

### 公司研究备忘录

围绕 PB-ROE选股模型 写一份公司研究备忘录：商业模式、行业格局、三张表、现金流、ROE/ROIC、负债、管理层、估值和风险。

| 证据 | 检查点 |
|---|---|
| 收入 | 是否可持续，是否依赖一次性因素 |
| 利润 | 毛利率和费用率是否合理 |
| 现金流 | 利润是否能变成现金 |
| 资产负债 | 应收、存货、商誉和负债是否异常 |
| 估值 | 是否有安全边际 |

### 验收标准

能用财务证据支持或否定投资假设，而不是只讲行业故事。

## 进一步实战化

### 财报案例练习

学习 `PB-ROE选股模型` 后，选择一家上市公司，连续阅读三期财报，重点看收入、毛利率、费用率、经营现金流、应收、存货、负债和资本开支。不要只摘数字，要解释数字背后的商业变化。

| 检查点 | 需要回答 |
|---|---|
| 收入增长 | 来自销量、价格、并表还是一次性因素 |
| 利润质量 | 毛利率和费用率是否可持续 |
| 现金流 | 利润是否真正变成现金 |
| 资产负债 | 是否存在应收、存货、商誉或杠杆风险 |
| 投资结论 | 估值是否补偿了这些风险 |

验收标准：能写出一个支持买入的证据和一个必须警惕的反方证据。
