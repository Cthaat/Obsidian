# PB-ROE选股模型

> [!note] 核心概念
> PB-ROE策略是经典的金融学选股方法，逻辑简单："**选好公司，在便宜时候买进**"。低PB=便宜，高ROE=好公司。

## 一、市净率（PB）基础

### 定义
- **PB = 市值 / 净资产 = 每股股价 / 每股净资产**
- 理论上代表以几倍的净资产价格买入一家公司
- PB=1 → 以净资产价格买入

### PB的适用范围
- **适合PB估值**：周期股、房地产、银行业、券商、保险
- **不适合PB估值**：以无形资产（商誉、知识产权）为主的公司
- PB一般在**3-10之间**比较合理，接近3或低于3时投资价值较高

## 二、ROE与PB的关系

> [!important] 核心公式
> **ROE = PB / PE = 净利润 / 净资产**
>
> ROE越高 → 资产盈利能力越强 → PB应该越高

### 双击与双杀
- **高PB买入高ROE企业** → ROE回落 → 业绩和估值**双杀**
- **低PB买入低ROE企业** → ROE上升 → 业绩和估值**双击**

> [!tip] 投资法则
> **业绩看ROE，估值看PB**，是投资制胜的不二法则。

## 三、PB-ROE选股模型逻辑

### "便宜"的标准
- **低PB**：每一块钱账面价值对应的价格低
- PB越低，安全系数越高（同一股票）

### "好公司"的标准
- **高ROE**：每一块钱投资得到的回报高
- ROE长期在10%-20%为优秀

### 模型优势
1. 经营者可通过模型变量关系知道如何提高公司估值
2. 基本面分析者能利用模型评价企业管理
3. 能够给股票定价（计算均衡价格）
4. 帮助主动投资者利用价格差异获取超额收益

## 四、实操注意事项

- 市净率越低就越安全的说法**不准确**，需具体分析
- 对同一股票：PB越低，安全系数越高
- 对不同股票：要结合行业和经营情况分析
- 需关注ROE的可持续性和利润含金量

## 课程化学习补充

> [!important] 学习定位
> 基本面研究把企业经营翻译成可验证的现金流、盈利质量和资本回报，最终落到估值和仓位。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收入和利润是否可持续
- 现金流是否支持利润
- ROE 来自效率还是杠杆
- 估值是否包含足够安全边际

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 财报粉饰
- 周期顶部外推
- 只看利润不看现金流
- 忽视治理和资本开支压力

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
