# 基金与产品尽调实操

> [!note] 核心问题
> 多数人通过基金/ETF 参与市场。尽调重点不是「去年第几名」，而是**费用、跟踪/风格、风险、流动性、是否适合你的说明书**。本文落实 [[基金评价、管理人尽调与产品选择]]，并与 [[专项资产实操导航]] 的 ETF 表对齐。

## 学习目标

1. 用五层评价表拆解一只基金/ETF。  
2. 会找费率、规模、跟踪误差/折溢价（按产品类型）。  
3. 区分 beta、行业、风格与选股贡献（定性即可）。  
4. 完成一张产品尽调卡并给出「配/不配」结论。  
5. 拒绝只看短期排名的决策。  

## 从哪里获取信息（公开渠道）

| 渠道 | 看什么 |
|---|---|
| 基金合同 / 招募说明书 / 产品资料概要 | 投资范围、费率、基准、风险揭示 |
| 定期报告 | 持仓、行业、规模变化 |
| 交易所/行情软件 | 场内成交额、折溢价（ETF） |
| 官方披露网站 | 以监管指定信息披露平台为准 |
| 本库工具 | [[AKShare-基金数据]] · [[xalpha基金工具]]（接口会变） |

教育向费用与风险：[Investor.gov](https://www.investor.gov/introduction-investing)（Understanding Fees、Risk 等主题）。

## 主动基金额外问题

```text
基准是什么？是否长期偏离子风格？
换手是否异常高（成本）？
前十大集中度？
同类排名是否只在风口期好看？
管理人变更？
申购赎回是否限制？
```

收益归因（定性）：beta / 行业 / 风格 / 选股 / 择时 / 杠杆。

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 排名第一就买 | 风格风口与幸存者 |
| 忽略费用 | 费用是确定的负 alpha |
| ETF 无风险 | 仍有市场与跟踪风险 |
| 名称含「稳健」=稳健 | 看合同与持仓 |
