# 期货交易中的应用

> [!note] 期货的特殊性
> 期货交易具有高杠杆特性，仓位管理的重要性远超股票。同样的行情，仓位过大可能因波动过大而爆仓归零，仓位过小则浪费市场机会。凯利公式在期货交易中被称为"杠杆艺术"。

## 期货实战五步法

### 步骤 1：计算原始凯利值

以沪铜策略为例：

| 参数 | 值 | 说明 |
|------|------|------|
| 胜率 $p$ | 44% | 近 3 年 128 次交易 |
| 盈亏比 $b$ | 2.3 | 平均盈利 6200 元 / 亏损 2700 元 |

$$f^* = \frac{bp - q}{b} = \frac{2.3 \times 0.44 - 0.56}{2.3} = \frac{0.452}{2.3} \approx 0.1965$$

原始凯利仓位约为 **19.65%**。

### 步骤 2：应用风控系数

选择半凯利策略：

$$f_{\text{actual}} = f^* \times 0.5 \approx 9.8\%$$

单笔止损额 = 账户净值 × 2%（预设）

> [!warning] 期货必须用半凯利或更低
> 由于期货的高杠杆特性，参数估计的微小误差会被杠杆放大。建议期货交易使用半凯利甚至 1/4 凯利。

### 步骤 3：计算合约手数

假设账户 100 万，沪铜每手止损 2700 元：

$$\text{单笔最大亏损} = 100\万 \times 2\% = 2\万$$

$$\text{最大手数} = \frac{20000}{2700} \approx 7 \text{手}$$

结合凯利仓位：

$$\text{凯利手数} = \frac{100\万 \times 9.8\%}{\text{每手保证金}}$$

取两者中较小值作为实际开仓手数。

### 步骤 4：动态再平衡规则

| 条件 | 动作 | 原因 |
|------|------|------|
| 净值增长 20% | 重新计算仓位 | 适应新的资金规模 |
| 连续 3 次止损 | 仓位降至 50% | 策略可能失效或市场环境变化 |
| 波动率 VIX > 35 | 启动半仓模式 | 市场不确定性增加 |
| 单日亏损 > 5% | 暂停交易 | 情绪管理 |

### 步骤 5：跨品种风险对冲

当同时交易多个期货品种时，需要考虑品种间的相关性：

$$\text{组合 VaR} = \sqrt{\mathbf{w}^T \Sigma \mathbf{w}} \cdot \Phi^{-1}(\alpha)$$

```python
def portfolio_risk_control(positions, cov_matrix, confidence=0.95):
    """组合风险控制"""
    import numpy as np
    from scipy import stats

    w = np.array(list(positions.values()))
    portfolio_var = np.sqrt(w @ cov_matrix @ w) * stats.norm.ppf(confidence)
    return portfolio_var
```

## 期货凯利公式的关键调整

### 1. 合约整数约束

期货交易必须以整手为单位，计算出的仓位需要取整：

```python
def adjust_to_lot(size, contract_size):
    """根据合约规模取整"""
    return round(size / contract_size) * contract_size
```

> [!example] 取整问题
> 计算仓位 0.23 手，实盘只能开 0 或 1 手。对于小资金账户，这个问题尤其突出。

### 2. 保证金管理

期货交易需要考虑保证金占用：

$$\text{最大保证金占用} = \text{账户净值} \times f_{\text{kelly}}$$

$$\text{实际手数} = \min\left(\frac{\text{最大保证金占用}}{\text{每手保证金}}, \frac{\text{单笔最大亏损}}{\text{每手止损}}\right)$$

### 3. 杠杆幻觉

> [!warning] 杠杆幻觉
> 2017年有人用凯利计算出 70% 仓位，忽略券商融资利率侵蚀 3% 年化收益。
>
> **规则：实际杠杆不超过凯利值的 80%**

### 4. 滑点成本

期货交易的滑点通常比股票更大，尤其在流动性差的合约上：

$$b_{\text{actual}} = b_{\text{backtest}} - 2 \times \text{滑点比例}$$

## 索普的期货实践

索普的"可转换对冲基金"在 1962-1988 年间：

| 指标 | 值 | 说明 |
|------|------|------|
| 年化收益 | 20%+ | 持续 26 年 |
| 最大回撤 | 1% | 同期标普 500 最大回撤超 50% |
| 亏损年份 | 0 | 20 年无一年亏损 |

> [!tip] 索普的三大原则
> 1. 统计套利的胜率与赔率：通过历史数据计算价差回归概率和预期幅度
> 2. 动态调整仓位：根据凯利公式计算应持有的头寸
> 3. 引入"半凯利"：为估计误差预留安全边际

## 期货凯利系统的完整架构

```python
class FuturesKellySystem:
    def __init__(self, capital=1000000):
        self.capital = capital
        self.positions = {}

    def calculate_position(self, symbol, contract_size, margin_per_lot, stop_loss_per_lot):
        """期货仓位计算"""
        # 1. 计算基础凯利值
        win_rate = self._calc_win_rate(symbol)
        payoff_ratio = self._calc_payoff_ratio(symbol)
        kelly_f = (payoff_ratio * win_rate - (1 - win_rate)) / payoff_ratio

        # 2. 半凯利
        kelly_f *= 0.5

        # 3. 波动率调节
        vol = self._calc_volatility(symbol)
        vol_adj = min(1.0, 0.25 / vol)
        adj_f = kelly_f * vol_adj

        # 4. 回撤控制
        if self.max_drawdown > 0.2:
            adj_f *= 0.5

        # 5. 黑天鹅熔断
        if vol > 0.4:
            adj_f = min(adj_f, 0.05)

        # 6. 计算手数（两种方法取较小值）
        max_loss_per_trade = self.capital * 0.02  # 单笔最大亏损 2%
        kelly_lots = (self.capital * adj_f) / margin_per_lot
        risk_lots = max_loss_per_trade / stop_loss_per_lot

        final_lots = min(kelly_lots, risk_lots)
        final_lots = round(final_lots)  # 取整

        return int(final_lots)
```

## 课程化学习补充

> [!important] 学习定位
> 仓位管理解决的是在不确定世界中活得足够久；凯利公式提供上限，实盘通常用分数凯利和风险预算。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 胜率和赔率是否有稳定样本
- 最大亏损是否可承受
- 相关仓位是否合并计算
- 连续亏损时是否自动降杠杆

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 高估胜率
- 忽略尾部风险
- 相关性在危机中上升
- 满凯利导致心理和资金双重崩溃

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
