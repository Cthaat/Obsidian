# 量化策略实操导航

> [!note] 核心问题
> `量化策略/` 按选股、择时、多因子、CTA、高频套利分得很细，但容易「收藏策略名」。本篇给你**策略家族地图 + 收益来源检查 + 与 quant-lab/课程的接线**，并规定：每种策略先写一页纸，再谈代码。

## 学习目标

1. 用一张表区分选股/择时/多因子/CTA/套利的收益来源与主要失效。  
2. 把任意策略压成「假设—数据—信号—组合—成本—失效」。  
3. 用阶段零脚本对齐至少一类策略（动量/双均线/因子/组合）。  
4. 知道高频与复杂套利对个人的现实门槛。  
5. 完成「策略一页纸」而不是抄参数。  

## 策略家族地图

| 家族 | 直觉 | 主要风险 | 本库入口 | 最小实操 |
|---|---|---|---|---|
| 选股/多因子 | 横截面排序持有 | 拥挤、风格、PIT | [[因子投资实操导航]] · 本专题多因子 | [[因子打分实操]] |
| 择时/趋势 | 时序强弱 | 震荡反复打脸 | 择时/CTA 子目录 | [[动量策略实操]] · 双均线 |
| 均值回归/配对 | 价差拉回 | 趋势单边、结构破裂 | 高频与套利 · 课程统计套利 | 先读概念，慎实盘 |
| 指数增强 | 基准+超额 | 跟踪误差、暴露失控 | 补充概念/策略文 | 先会因子再谈增强 |
| 高频/做市 | 微观结构 | 技术、牌照、资本 | 高频子目录 | 个人默认**观察** |
| 事件驱动 | 公告/并购等 | 概率、流动性、合规 | 课程事件驱动 | 研究向 |

## 与官方/行业知识框架的对齐（自学用）

全球投管知识体系通常覆盖：权益、固收、衍生品、组合、另类与行为偏差等模块（例如 [CFA Institute 考生资源](https://www.cfainstitute.org/programs/cfa-program/candidate-resources) 的主题地图可作「我缺哪块」对照，不替代考试报名）。  
策略学习时自问：这是在赚 **风险溢价、行为偏差、流动性、还是制度约束**？说不清来源就不要上杠杆。

## 任何策略的六栏规格（强制）

| 栏 | 你必须写清 |
|---|---|
| 收益来源 | 风险/行为/结构？ |
| 数据 | 字段、复权、PIT |
| 信号 | 公式与时点 |
| 组合 | 权重、调仓 |
| 成本 | 佣金、税、滑点、冲击 |
| 失效 | 拥挤、制度变、样本外崩 |

模板可复用 [[阶段三作业打通清单]] 与 [[研究笔记与实验工作流]]。

## 专题内怎么读（抽样）

1. 选股 / 多因子：与因子导航合并，避免两套体系  
2. 择时：与动量、均线实验对照  
3. CTA：理解趋势与回撤，联读 [[组合与仓位实操导航]]  
4. 高频与套利：只建「能力与资本门槛」认知  
5. 策略实战：当案例库，每篇提取一页纸字段
