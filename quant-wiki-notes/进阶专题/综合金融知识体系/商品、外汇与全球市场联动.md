# 商品、外汇与全球市场联动

> [!note] 学习定位
> 商品和外汇连接真实经济、利率差、通胀预期、库存周期和全球资本流动，是资产配置和宏观对冲的重要组成部分。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、商品逻辑

- 能源看供需、库存、运输和地缘冲突。
- 金属看工业周期、地产链、库存和美元流动性。
- 农产品看天气、库存、政策和季节性。

## 二、外汇逻辑

- 汇率受利差、贸易、资本流动、政策可信度和风险偏好影响。
- 短期看预期差和仓位，长期看生产率、通胀和外部平衡。

## 三、组合用途

- 商品可作为通胀和供给冲击的观察窗口。
- 外汇可帮助识别全球资金风险偏好。
- 跨资产联动常在危机中变强，分散不是永远有效。

## 实战掌握清单

### 商品定价框架

- 能源：供给弹性、库存、运输瓶颈、地缘冲突、炼化利润和季节性需求。
- 工业金属：全球制造业周期、地产链、库存、美元流动性和中国需求权重。
- 贵金属：实际利率、美元、避险需求、央行购金和金融系统压力。
- 农产品：天气、播种面积、库存消费比、政策收储和物流扰动。

### 外汇定价框架

汇率短期看预期差、利差变化、仓位和风险偏好，长期看通胀差、生产率、经常账户、资本流动和政策可信度。外汇交易必须特别注意杠杆、隔夜利息、央行干预和流动性时段。

### 跨资产联动训练

1. 当原油上涨时，分别推演通胀、利率、航空/化工利润、能源股和债券的影响。
2. 当美元走强时，检查新兴市场资金流、商品价格、港股/美股估值和外债压力。
3. 当全球避险升温时，观察黄金、美元、美债、日元、股票波动率和信用利差是否同向反应。

### 失效边界

跨资产关系不是永恒公式。危机中相关性会突然上升，政策干预会打断趋势，期货展期成本会让“看对方向”仍然亏钱。

## 深度案例与训练

### 小型研究项目

围绕 商品、外汇与全球市场联动 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 商品、外汇与全球市场联动 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 商品、外汇与全球市场联动 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 商品、外汇与全球市场联动 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。

## 八、实战案例：美元走强的传导链

美元走强通常会影响多类资产，但传导不是机械公式：

1. 对商品：美元计价商品可能承压，但供给冲击可抵消影响。
2. 对新兴市场：外债压力和资本流出风险上升。
3. 对港股和海外成长股：估值可能受全球流动性影响。
4. 对黄金：若美元走强来自实际利率上行，黄金承压；若来自避险，黄金也可能上涨。

跨资产分析要写清“为什么美元走强”，不能只看到美元指数变化就做单向判断。

## 九、期货展期提醒

商品投资不只看现货价格，还要看期货曲线。远月价格高于近月时，滚动持仓可能产生展期损耗；近月高于远月时，可能获得展期收益。商品 ETF 或期货策略必须把展期结构纳入收益来源分析。

## 进一步实战化

### 一页应用练习

围绕 `商品、外汇与全球市场联动` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
