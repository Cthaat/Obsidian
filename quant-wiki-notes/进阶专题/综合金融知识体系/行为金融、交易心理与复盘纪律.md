# 行为金融、交易心理与复盘纪律

> [!note] 学习定位
> 交易心理不是鸡汤，而是流程问题。优秀交易者用制度限制自己，而不是依赖临场自控。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、常见偏差

- 损失厌恶让人过早止盈、亏损死扛。
- 确认偏差让人只看支持自己观点的信息。
- 过度自信让人放大仓位、忽视尾部。
- 锚定效应让人执着于成本价或历史高点。

## 二、流程约束

- 交易前写下理由、无效条件和退出计划。
- 交易中只执行预案，不临时扩大风险。
- 交易后复盘是否按规则行动，而不只复盘盈亏。

## 三、复盘模板

- 我原本的假设是什么？
- 市场实际发生了什么？
- 盈亏来自判断、仓位、执行还是运气？
- 下次要修改模型、规则还是心理流程？

## 实战掌握清单

### 偏差发生的位置

行为偏差可能出现在五个环节：信息收集时只看支持自己观点的证据；判断时过度自信；下单时追涨杀跌；持有时亏损厌恶和锚定成本；复盘时把盈利归功于能力、亏损归因于运气。

### 防偏差流程

1. 交易前写反方观点：如果这笔交易错了，最可能错在哪里。
2. 写失效条件：价格、基本面、宏观、流动性或时间哪一项触发退出。
3. 限制冲动交易：设置冷静期、单日交易次数、最大加仓次数和最大亏损。
4. 交易后复盘执行偏差：该买没买、该卖没卖、仓位过大、止损拖延、盈利过早落袋。

### 情绪日志

复盘时记录当时情绪、信息来源、身体状态、是否疲劳、是否想翻本、是否受社交媒体影响。长期看，情绪日志能暴露最常犯的错误，比单笔盈亏更有价值。

### 纪律标准

优秀交易者不是没有情绪，而是把情绪最容易破坏决策的环节提前制度化。能否长期执行计划，比临场聪明更重要。

## 深度案例与训练

### 小型研究项目

围绕 行为金融、交易心理与复盘纪律 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 行为金融、交易心理与复盘纪律 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 行为金融、交易心理与复盘纪律 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 行为金融、交易心理与复盘纪律 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。

## 八、实战案例：回本心理如何破坏交易

回本心理常见于亏损持仓：投资者不再问资产是否值得持有，而是只想等价格回到成本。这会导致三种错误：

1. 坏资产被长期占用仓位。
2. 新机会因为资金被套而无法参与。
3. 止损规则被成本价替代，风险继续扩大。

破解方法是用“空仓视角”重估：如果今天没有持仓，你是否愿意以当前价格买入？如果答案是否定的，持仓理由就不应只是“还没回本”。

## 九、行为纪律指标

建议每月统计：

- 冲动交易次数。
- 未按计划止损次数。
- 盈利过早卖出次数。
- 亏损后加仓次数。
- 因社交媒体或消息刺激下单次数。

这些指标比单月收益更能说明交易系统是否在变好。

## 进一步实战化

### 一页应用练习

围绕 `行为金融、交易心理与复盘纪律` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
