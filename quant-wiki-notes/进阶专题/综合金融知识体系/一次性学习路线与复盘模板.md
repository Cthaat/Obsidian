# 一次性学习路线与复盘模板

> [!note] 学习定位
> 这是一份把进阶专题压缩成 12 周学习路线的执行表。目标不是读完所有文件，而是把知识变成研究和交易动作。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、12 周路线

- 第 1 周：金融投资全知识地图、投资理念与流派、交易者能力地图。
- 第 2 周：资产类别、市场结构、复利、风险承受力和投资政策书。
- 第 3-4 周：财报、行业、股票研究和估值。
- 第 5-6 周：宏观、利率、资产配置和组合管理。
- 第 7-8 周：因子、量化策略、回测和机器学习。
- 第 9-10 周：期权、可转债、ETF、商品外汇和专项资产。
- 第 11 周：交易系统、执行、数据治理和合规。
- 第 12 周：整理投资政策书、策略清单和复盘模板。

## 二、每日学习动作

- 读一篇主题笔记。
- 写一句话定义、三个关键假设和两个失效条件。
- 找一个真实资产或历史区间做案例。
- 把结论写进交易前检查清单。

## 三、复盘表

- 日期、资产、策略、入场理由、预期收益来源。
- 仓位、止损、退出条件、最大可承受损失。
- 实际结果、偏离原因、下次流程修改。

## 四、每周产出物

为了避免“读了很多但没有变化”，每周都要留下一个可复用产出：

| 周次 | 产出物 |
|---|---|
| 第 1 周 | 一页个人投资理念草稿：我赚什么钱、不赚什么钱 |
| 第 2 周 | 投资政策书 IPS 初稿：目标、期限、风险预算和禁区 |
| 第 3 周 | 一家公司或一个 ETF 的研究卡片 |
| 第 4 周 | 一份三情景估值表和安全边际说明 |
| 第 5 周 | 一张宏观变量观察表：利率、通胀、信用、汇率、商品 |
| 第 6 周 | 一个组合权重表和再平衡规则 |
| 第 7 周 | 一个简单因子或规则策略的假设说明 |
| 第 8 周 | 一次回测质量检查：样本、成本、过拟合、样本外 |
| 第 9 周 | 一份衍生品或可转债风险说明 |
| 第 10 周 | 一份跨资产联动复盘 |
| 第 11 周 | 一张交易执行和合规检查清单 |
| 第 12 周 | 年度复盘模板和下一轮学习缺口 |

## 实战掌握清单

### 四阶段学习路线

1. 基础阶段：资产类别、市场结构、财报、估值、利率、风险指标和交易成本。
2. 策略阶段：价值、成长、动量、低波、红利、CTA、套利、ETF、可转债和期权策略。
3. 系统阶段：回测、因子研究、组合构建、执行系统、数据治理和模型监控。
4. 职业阶段：投资政策书、合规税务、心理纪律、复盘机制和长期能力圈维护。

### 每篇笔记的学习法

- 用一句话概括它解决什么投资问题。
- 写出三个关键变量和三个失效条件。
- 找一个真实标的或策略做案例。
- 把结论转成交易前清单、仓位规则或复盘问题。
- 三天后回看：这个概念是否真的改变了你的决策。

### 复盘模板

| 项目 | 记录内容 |
|---|---|
| 交易假设 | 预期收益来源和核心证据 |
| 风险假设 | 最可能失败的原因 |
| 仓位计划 | 初始仓位、加减仓和退出条件 |
| 执行记录 | 下单价格、成交质量、滑点和成本 |
| 结果归因 | 选股、择时、仓位、市场和运气 |
| 流程改进 | 下次需要新增或删除的规则 |

### 学习节奏

不要试图一天理解全部金融知识。更有效的做法是每天学习一个主题、做一个案例、写一次复盘；每周选一个专题串联；每月回到全知识地图补缺口。

## 深度案例与训练

### 小型研究项目

围绕 一次性学习路线与复盘模板 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 一次性学习路线与复盘模板 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。
