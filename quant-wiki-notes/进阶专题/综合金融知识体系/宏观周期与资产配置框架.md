# 宏观周期与资产配置框架

> [!note] 学习定位
> 宏观分析服务于资产配置：它不能直接告诉你买哪只股票，但能帮助你理解增长、通胀、利率、信用和风险偏好的变化。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、四个核心变量

- 增长决定企业收入和风险偏好。
- 通胀影响利润率、利率和估值倍数。
- 政策决定流动性、信用条件和贴现率。
- 风险偏好决定资产之间的估值切换。

## 二、资产配置步骤

- 确定投资期限、现金流需求和最大可承受回撤。
- 把资产分成增长型、防御型、通胀保护型和流动性储备型。
- 用目标权重、风险预算和再平衡规则替代临时判断。
- 在极端场景下检查组合是否仍能活下去。

## 三、宏观误区

- 把长期宏观观点用于短线交易。
- 只看单个指标，不看市场预期差。
- 只判断方向，不判断价格是否已经反映。

## 实战掌握清单

### 宏观周期四象限

宏观判断可以先压缩成增长、通胀、流动性和政策四个维度。增长上行且通胀温和时，权益和信用资产通常更容易获得风险偏好支持；增长下行且通胀回落时，利率债和高质量资产的防御价值上升；通胀上行伴随供给冲击时，商品、资源股和通胀保护资产更值得观察；流动性急剧收缩时，所有高杠杆和高估值资产都要降低预期。

### 资产配置动作

1. 建立宏观仪表盘：PMI、社融/信贷、通胀、利率曲线、信用利差、汇率、库存和企业利润。
2. 把每个指标标成领先、同步或滞后，不用滞后指标追逐已经发生的行情。
3. 将宏观观点落到仓位：增加什么、减少什么、用什么对冲、错了怎么退。
4. 每次政策变化后，复盘“政策目标、政策约束、市场预期差”三件事。

### 常见误区

- 宏观叙事正确，但资产已经提前定价。
- 用单一指标解释整个市场，例如只看利率或只看汇率。
- 看对大方向，却忽略持仓期限、波动和回撤，导致无法拿到结果。

## 深度案例与训练

### 小型研究项目

围绕 宏观周期与资产配置框架 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 宏观周期与资产配置框架 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 宏观周期与资产配置框架 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 宏观周期与资产配置框架 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。

## 八、实战案例：看对宏观却亏钱

宏观观点正确不等于交易赚钱。常见原因包括：

- 观点已经被市场提前定价。
- 选择了错误资产，例如看多经济却买了估值过高的行业。
- 杠杆或仓位太大，短期波动迫使提前出局。
- 时间太早，资金无法承受等待成本。
- 政策反应函数改变，原逻辑被打断。

宏观交易要同时回答方向、资产、时点、仓位和退出五个问题。

## 九、宏观仪表盘

建议把宏观变量分成四组：

| 组别 | 代表指标 | 资产含义 |
|---|---|---|
| 增长 | PMI、工业增加值、盈利 | 权益和商品需求 |
| 通胀 | CPI、PPI、工资、商品 | 利率和估值压力 |
| 信用 | 社融、贷款、信用利差 | 风险偏好和融资环境 |
| 流动性 | 政策利率、资金利率、汇率 | 估值和杠杆约束 |

仪表盘不是预测器，而是帮助你判断市场最关心的变量是否发生切换。

## 进一步实战化

### 一页应用练习

围绕 `宏观周期与资产配置框架` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
