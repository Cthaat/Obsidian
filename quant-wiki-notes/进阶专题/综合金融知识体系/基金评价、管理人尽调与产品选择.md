# 基金评价、管理人尽调与产品选择

> [!note] 学习定位
> 很多投资者不是直接买股票，而是通过基金、ETF、资管计划或私募产品表达观点。评价基金的重点不是过去涨了多少，而是收益来源、风险暴露、管理人能力、费用结构和是否适合你的组合。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、基金评价的五个层次

| 层次 | 要看什么 | 误区 |
|---|---|---|
| 业绩 | 年化收益、回撤、波动、滚动收益 | 只看最近一年排名 |
| 风险 | 最大回撤、下行波动、风险暴露 | 把低波动当成无风险 |
| 风格 | 价值、成长、行业、大小盘、久期 | 基金名称和真实持仓不一致 |
| 成本 | 管理费、托管费、申赎费、交易成本 | 忽略长期费用侵蚀 |
| 管理人 | 投资流程、团队稳定、风控和透明度 | 只相信明星经理 |

## 二、业绩归因

基金上涨可能来自多种原因：

- 市场 beta：整个市场涨了。
- 行业暴露：押中某个赛道。
- 风格暴露：成长、价值、小盘、红利、低波等风格占优。
- 选股 alpha：同类暴露下选股更好。
- 择时与仓位：市场高低位切换正确。
- 杠杆或集中度：承担更大风险换来更高收益。

只有把收益来源拆开，才能判断管理人是否真的有能力。

## 四、产品选择

不同产品适合不同任务：

| 产品 | 适合任务 | 注意事项 |
|---|---|---|
| 宽基指数基金 | 长期核心 beta | 估值、跟踪误差、费率 |
| 行业/主题基金 | 卫星观点表达 | 周期、拥挤、集中度 |
| 主动权益基金 | 选股 alpha | 风格漂移、经理变动 |
| 债券基金 | 防御、票息、久期管理 | 信用下沉、杠杆、流动性 |
| 量化基金 | 因子或统计 alpha | 模型衰减、容量、换手成本 |
| 私募产品 | 特殊策略或绝对收益 | 流动性、费用、透明度 |

## 五、买基金前检查

1. 这个产品补充了组合哪一块，而不是因为它最近涨。
2. 它的最大回撤是否能承受。
3. 它与已有持仓是否高度重合。
4. 费用是否合理。
5. 如果基金经理离任、风格漂移或业绩回撤，退出规则是什么。

## 八、实战案例：明星基金经理回撤时怎么办

买主动基金时，真正难的不是上涨期买入，而是回撤期判断该不该继续持有。可以用三个问题拆解：

1. 回撤来自市场整体、风格切换，还是管理人判断错误。
2. 基金经理是否仍在执行原策略，还是发生风格漂移。
3. 组合中这只基金承担的是核心仓还是卫星仓。

如果只是市场和风格回撤，且原逻辑未变，可以按再平衡规则处理；如果管理人离任、风格漂移或风险控制失效，就应重新尽调，而不是机械补仓。

## 九、基金复盘指标

| 指标 | 复盘用途 |
|---|---|
| 滚动一年收益 | 看业绩是否只集中在单一时期 |
| 最大回撤 | 判断风险承受是否匹配 |
| 持仓集中度 | 判断是否依赖少数重仓 |
| 行业和风格暴露 | 判断收益来源 |
| 换手率 | 判断成本和策略稳定性 |
| 规模变化 | 判断容量和管理难度 |

基金复盘的目标不是追逐下一只冠军基金，而是确认产品是否仍适合你的组合。

## 进一步实战化

### 一页应用练习

围绕 `基金评价、管理人尽调与产品选择` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
