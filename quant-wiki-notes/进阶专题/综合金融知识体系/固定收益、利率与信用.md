# 固定收益、利率与信用

> [!note] 学习定位
> 固定收益不是低风险的同义词。债券价格由现金流、期限结构、信用利差、流动性和嵌入期权共同决定。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、核心概念

- 久期衡量利率敏感性，凸性衡量久期随利率变化的变化。
- 信用利差反映违约风险、流动性补偿和风险偏好。
- 收益率曲线包含政策预期、期限溢价和经济周期信息。

## 二、投资框架

- 先拆分无风险利率和信用利差。
- 再判断期限、票息、赎回条款和流动性。
- 最后把债券放进组合的防御、收益或对冲角色。

## 三、风险清单

- 利率上行导致价格下跌。
- 信用下沉导致本金损失。
- 流动性枯竭导致账面安全资产无法退出。
- 嵌入期权改变真实久期和收益分布。

## 实战掌握清单

### 固收分析四层

1. 无风险利率：看政策利率、资金面、通胀预期和收益率曲线。
2. 久期与凸性：久期决定利率变化时的价格敏感度，凸性决定大幅波动时的非线性表现。
3. 信用利差：拆分违约概率、回收率、流动性补偿和风险偏好。
4. 条款与流动性：赎回、回售、担保、抵押、次级结构和成交深度会改变真实风险。

### 投资应用

- 防御型组合关注高评级、短久期、流动性和现金替代。
- 收益增强关注信用利差压缩，但要避免为了票息牺牲本金安全。
- 宏观交易关注曲线陡峭化/平坦化、久期切换和利率预期差。
- 可转债和含权债要同时看债底、期权价值和条款触发概率。

### 风险复盘

每次持有债券或债基，都记录收益来自票息、久期、信用利差还是杠杆。若发生亏损，分清是利率风险、信用风险、流动性折价还是产品结构问题，不能笼统说“债券也会跌”。

## 深度案例与训练

### 小型研究项目

围绕 固定收益、利率与信用 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 固定收益、利率与信用 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 固定收益、利率与信用 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 固定收益、利率与信用 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。
