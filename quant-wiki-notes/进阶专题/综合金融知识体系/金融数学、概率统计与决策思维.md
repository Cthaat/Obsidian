# 金融数学、概率统计与决策思维

> [!note] 学习定位
> 优秀投资者不一定每天推公式，但必须有概率、分布、估计误差和决策成本的直觉。金融数学和统计不是为了显得复杂，而是为了避免把偶然盈利误认为能力，把样本内规律误认为真理。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、必须掌握的数学直觉

| 主题 | 投资含义 | 常见误区 |
|---|---|---|
| 复利 | 收益和亏损都会随时间放大 | 只看年化收益，不看回撤恢复难度 |
| 期望值 | 胜率和赔率共同决定结果 | 高胜率不等于好策略 |
| 方差与波动 | 收益路径会影响能否坚持 | 只看平均收益，忽略资金曲线 |
| 相关性 | 多个资产可能在危机中同跌 | 名义分散不等于风险分散 |
| 条件概率 | 新信息会改变原判断 | 看到利好就机械加仓 |
| 贝叶斯更新 | 用新证据更新信心等级 | 坚持旧观点，不愿承认假设改变 |
| 大数定律 | 样本越多，估计越稳定 | 用十几笔交易判断策略好坏 |
| 肥尾分布 | 极端事件比正态假设更常见 | 低估跳空、停牌和流动性断层 |

## 二、从公式到交易

金融数学真正进入交易系统时，通常表现为四类动作：

1. 仓位：用最大可承受亏损、波动和相关性决定买多少。
2. 验证：用样本外、滚动窗口和置信区间判断策略是否稳健。
3. 风控：用 VaR、CVaR、压力测试和流动性模型识别尾部亏损。
4. 复盘：把结果拆成信号贡献、仓位贡献、执行贡献和运气成分。

## 三、统计检验的底线

量化研究里最常见的幻觉是“图形很好看”。最低限度要检查：

- 样本是否足够，是否覆盖牛市、熊市、震荡市和危机期。
- 数据是否真实可得，是否存在前视偏差和幸存者偏差。
- 结果是否被少数极端交易贡献。
- 参数轻微变化后，收益是否大幅坍塌。
- 多次试错后是否做了多重检验修正或样本外验证。

## 四、概率思维训练

每次交易前，用概率语言重写观点：

```text
核心假设：
我认为它成立的概率：
若成立，预期收益：
若不成立，预期亏损：
最大可承受亏损：
需要更新概率的新证据：
```

这个训练能迫使你把“我觉得会涨”改成“在什么证据下，承担多大风险，换取什么赔率”。

## 五、常见失败场景

- 用历史平均收益设计未来仓位，却忽略未来波动可能更大。
- 只看相关性矩阵，不做危机相关性上升测试。
- 用准确率评价模型，忽略收益分布和交易成本。
- 用单次大赚证明策略有效。
- 看不懂置信区间，却相信精确到小数点的回测年化收益。

## 六、最小可执行项目

选择一个策略或基金，计算以下指标：

| 指标 | 用途 |
|---|---|
| 年化收益 | 衡量长期增长 |
| 年化波动 | 衡量路径不确定性 |
| 最大回撤 | 衡量最痛苦亏损 |
| Sharpe/Sortino | 衡量单位风险收益 |
| 胜率与盈亏比 | 拆分期望值 |
| 滚动收益 | 检查不同时期稳定性 |
| 尾部亏损 | 检查极端风险 |

验收标准：不要只输出数字，要写一句话解释“这个策略最怕什么”。

## 八、实战案例：高胜率策略为什么仍会亏钱

假设一个策略胜率 80%，每次盈利 1%，每次亏损 8%。表面上胜率很高，但期望值是：

```text
0.8 * 1% + 0.2 * (-8%) = -0.8%
```

这说明高胜率本身没有意义。卖期权、网格、补仓和部分套利策略都可能呈现“长期小赚、偶尔大亏”的结构。真正要看的是完整收益分布、最大连续亏损、尾部亏损和资金是否能承受。

## 九、训练路径

1. 用一个历史策略计算胜率和盈亏比。
2. 把最大亏损扩大一倍，观察期望值是否仍为正。
3. 把连续亏损次数设为 3、5、8 次，观察账户回撤。
4. 用同一策略分别做固定仓位、波动率目标仓位和半凯利仓位。
5. 写下最终结论：这个策略最怕低胜率、低赔率、连续亏损还是尾部事件。

### 一页应用练习

围绕 `金融数学、概率统计与决策思维` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
