# 资产定价、风险溢价与市场效率

> [!note] 学习定位
> 投资交易最核心的问题是：为什么某类资产长期有收益，为什么某个价格现在成立，以及什么情况下价格会偏离价值。资产定价帮助你区分 beta、alpha、风险补偿、行为偏差和偶然噪音。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、资产价格从哪里来

资产价格可以理解为三层叠加：

1. 现金流：未来能产生多少真实现金、利息、股息或残余价值。
2. 折现率：市场要求多高的风险补偿，包括利率、信用、流动性和不确定性。
3. 预期差：投资者对未来现金流和风险的理解是否发生变化。

股票、债券、商品、外汇和衍生品的定价公式不同，但底层问题相似：未来收益是什么，风险由谁承担，市场现在定价了多少。

## 二、常见风险溢价

| 风险溢价 | 为什么可能存在 | 失效风险 |
|---|---|---|
| 权益风险溢价 | 股东承担企业经营和波动风险 | 估值过高、盈利下行 |
| 信用风险溢价 | 债权人承担违约和流动性风险 | 信用事件集中爆发 |
| 期限溢价 | 长久期资产承担利率变化风险 | 政策利率和通胀预期剧变 |
| 流动性溢价 | 低流动性资产要求补偿 | 危机中无法退出 |
| 波动率溢价 | 卖方承担尾部波动风险 | 跳空和极端行情 |
| 价值溢价 | 市场厌恶低估或困境资产 | 价值陷阱 |
| 动量溢价 | 趋势、信息扩散和行为惯性 | 反转和拥挤踩踏 |
| 小盘溢价 | 小公司信息不充分、流动性差 | 容量小、退市风险 |

## 三、市场效率的正确理解

市场不是永远有效，也不是永远无效。更实用的理解是：越容易获得、越容易交易、越容易验证的信息，越快被价格吸收；越复杂、越难坚持、越受行为约束或制度限制的机会，越可能留下溢价。

因此，寻找机会时要问：

- 我知道的信息是否已经被大多数人知道？
- 我比别人强在哪里：数据、解释、耐心、风险承受、交易执行还是制度约束？
- 这类收益来源是否会被资金涌入压低？
- 我承担的风险是否被充分补偿？

## 四、定价到交易的流程

1. 判断资产类别：现金流资产、信用资产、实物资产、货币资产还是衍生品。
2. 拆分收益来源：beta、风险溢价、alpha、套利、行为偏差或杠杆。
3. 找到关键变量：盈利、利率、利差、库存、汇率、波动率或监管。
4. 建立估值区间：保守、基准、乐观三种情景。
5. 设计仓位：估值越不确定，仓位越应保守。
6. 写出失效条件：什么证据说明市场是对的，而你是错的。

## 五、反例训练

很多亏损不是因为“不懂收益”，而是因为误判收益来源：

- 把牛市 beta 当成选股 alpha。
- 把杠杆收益当成策略能力。
- 把低估值当成安全边际，却忽略商业模式恶化。
- 把高股息当成稳定收益，却忽略分红不可持续。
- 把套利价差当成无风险，却忽略流动性和执行风险。

## 六、最小可执行项目

任选一个资产或策略，写一张风险溢价卡：

```text
资产/策略：
主要收益来源：
承担的风险：
市场为什么愿意支付这笔溢价：
哪些证据说明溢价正在消失：
组合中最大仓位：
退出条件：
```

验收标准：能清楚说出你赚的是“什么钱”，而不是只说“它会涨”。

## 八、实战案例：同一笔收益的三种解释

一只股票上涨 30%，可能至少有三种解释：

1. 市场 beta：指数整体上涨，股票只是跟随市场。
2. 风格溢价：成长、小盘、红利或低波风格在该阶段占优。
3. 个股 alpha：公司基本面改善，市场之前低估了它。

如果不做归因，投资者很容易把 beta 或风格运气当成自己的选股能力。复盘时应对比指数、行业、风格因子和同类公司，判断收益到底来自哪里。

## 九、风险溢价监控表

| 收益来源 | 观察指标 | 失效信号 |
|---|---|---|
| 权益风险溢价 | 估值、盈利、利率 | 高估值叠加盈利下修 |
| 信用风险溢价 | 信用利差、违约率、流动性 | 利差扩大且融资收紧 |
| 动量溢价 | 趋势强度、成交量、拥挤度 | 放量反转、趋势破坏 |
| 价值溢价 | 估值分位、质量、现金流 | 低估值但基本面恶化 |
| 波动率溢价 | 隐含波动率、历史波动率 | 极端事件和跳空频繁 |

这张表的意义是提醒你：任何溢价都有成本，不能只看收益。

## 进一步实战化

### 一页应用练习

围绕 `资产定价、风险溢价与市场效率` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
