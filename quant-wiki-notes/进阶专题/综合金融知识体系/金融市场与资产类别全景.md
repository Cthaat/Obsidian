# 金融市场与资产类别全景

> [!note] 学习定位
> 金融市场由融资者、投资者、中介、交易场所、清算托管和监管制度共同构成；每类资产都有不同现金流、风险来源和交易制度。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、主要资产类别

- 股票代表剩余索取权，核心风险来自盈利、估值、治理和流动性。
- 债券代表约定现金流，核心风险来自利率、信用、再投资和流动性。
- 商品和外汇更受供需、利差、库存、政策和地缘冲击影响。
- 衍生品把方向、波动率、期限、杠杆和尾部风险拆开交易。
- 基金、ETF 和可转债是资产暴露、工具效率和条款结构的组合。

## 二、市场结构

- 一级市场解决融资和发行定价，二级市场解决流动性和价格发现。
- 场内交易强调撮合、透明度和规则，场外交易强调协商、信用和对手方风险。
- 交易所、券商、托管、清算和登记系统共同决定交易能否安全完成。

## 三、学习重点

- 先理解现金流归属，再理解风险来源。
- 先理解交易制度，再做策略回测。
- 先比较资产相关性，再谈组合分散。

## 实战掌握清单

### 资产类别的收益来源

| 资产类别 | 主要收益来源 | 主要风险 |
|---|---|---|
| 股票 | 盈利增长、估值扩张、股息 | 业绩下滑、估值收缩、治理风险 |
| 债券 | 票息、久期收益、信用利差收窄 | 利率上行、违约、流动性折价 |
| 商品 | 供需缺口、库存周期、通胀冲击 | 价格剧烈波动、展期成本、政策干预 |
| 外汇 | 利差、资本流动、避险需求 | 政策突变、杠杆放大、流动性断层 |
| 期权/期货 | 杠杆、对冲、波动率表达 | 保证金、跳空、尾部亏损 |
| 基金/ETF | 分散化、低成本beta、规则化配置 | 跟踪误差、拥挤交易、风格漂移 |

### 建立资产地图的方法

1. 先问这个资产赚的是增长、利息、信用、流动性、波动率还是制度结构的钱。
2. 再问它在组合里承担什么角色：进攻、防御、现金管理、对冲还是战术交易。
3. 最后检查它和已有持仓的相关性，避免名义分散但风险同源。

### 学习训练

- 任选一只股票、一只债券ETF、一只商品ETF和一只货币基金，写出各自收益来源和最大风险。
- 画一张自己的资产暴露表：权益、利率、信用、商品、外汇、现金各占多少。
- 每月复盘一次：哪些资产在同一个宏观冲击下会一起亏钱。

## 深度案例与训练

### 小型研究项目

围绕 金融市场与资产类别全景 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 金融市场与资产类别全景 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 金融市场与资产类别全景 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 金融市场与资产类别全景 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。

## 进一步实战化

### 一页应用练习

围绕 `金融市场与资产类别全景` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。

## 八、最终提醒

不要把资产类别当成简单标签。股票、债券、商品、外汇、基金和衍生品背后，真正重要的是现金流来源、定价变量、流动性结构、杠杆关系和极端情景下的行为。

一个成熟投资者在配置资产时，不只问“哪个资产收益高”，还要问它和现有组合的相关性如何，亏损发生在什么宏观情景下，能否在压力期卖出，是否承担了自己不理解的隐性风险。资产类别学习的最终目的，是把市场拆成可理解、可组合、可控制的风险单元。
