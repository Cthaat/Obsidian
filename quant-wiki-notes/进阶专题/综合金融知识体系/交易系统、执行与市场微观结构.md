# 交易系统、执行与市场微观结构

> [!note] 学习定位
> 很多策略死在执行层：盘口、滑点、冲击成本、限频、断连、重复下单和风控滞后都会吞掉纸面收益。本文仅用于学习、研究与复盘，不构成任何投资建议。
>
> **可执行导航：** [[交易执行与微观结构实操导航]] · 课程深读 [[市场微观结构与交易执行]]

## 一、执行要素

- 订单类型：市价、限价、止损、算法单和条件单。
- 交易成本：佣金、印花税、滑点、买卖价差、冲击成本和机会成本。
- 市场微观结构：撮合规则、最小报价单位、涨跌停、停牌和盘口深度。

## 二、系统链路

- 数据进入信号模块，信号进入风控模块，风控通过后进入订单模块。
- 订单模块必须处理已报、部分成交、撤单、拒单和异常回报。
- 监控模块必须记录延迟、持仓、风险、成交和告警。

## 三、实盘纪律

- 先仿真，再小资金，再逐步放量。
- 先有熔断和回滚，再谈自动化。
- 每笔成交都要能追溯到信号、订单和风控判断。

## 实战掌握清单

### 市场微观结构要点

执行质量取决于撮合规则、盘口深度、买卖价差、最小报价单位、涨跌停、停牌、集合竞价、交易时段和订单优先级。同一个策略在不同市场、不同品种、不同资金规模下，成交成本可能完全不同。

### 执行链路

1. 行情进入数据模块，经过校验后产生特征和信号。
2. 信号进入风控模块，检查仓位、资金、品种、价格、频率和黑名单。
3. 订单模块处理报单、撤单、部分成交、拒单、重复回报和延迟。
4. 持仓模块同步成交、资金、保证金和风险指标。
5. 监控模块发出延迟、断连、异常成交、超限和策略失控告警。

### 交易成本分析

复盘成交时至少拆分佣金税费、买卖价差、滑点、冲击成本、机会成本和未成交成本。策略回测若没有交易成本分析，实盘收益通常会被高估。

### 安全规则

自动化系统必须有 kill switch、资金上限、品种白名单、订单频率限制、异常价格保护、人工接管和完整日志。任何不能解释的订单，都应该视为系统风险事件。

## 深度案例与训练

### 小型研究项目

围绕 交易系统、执行与市场微观结构 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 交易系统、执行与市场微观结构 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 交易系统、执行与市场微观结构 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 交易系统、执行与市场微观结构 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。

## 八、实战案例：纸面 alpha 被滑点吃掉

一个日内或高换手策略在回测中年化很高，但实盘后收益消失，常见原因是：

- 回测按收盘价成交，实盘必须穿过买卖价差。
- 单笔订单过大，冲击盘口。
- 信号太拥挤，很多资金同时交易。
- 撤单、部分成交和拒单没有被回测模拟。
- 市场波动时延迟增加，价格已经离开信号点。

执行研究要把成交质量当成策略的一部分，而不是上线后的细节。

## 九、订单复盘字段

每笔订单至少记录：

| 字段 | 用途 |
|---|---|
| 信号时间 | 判断延迟 |
| 目标价格 | 比较滑点 |
| 下单价格 | 检查执行 |
| 成交价格 | 计算真实成本 |
| 成交比例 | 判断流动性 |
| 撤单/拒单原因 | 发现系统问题 |
| 当时盘口深度 | 估计冲击成本 |

没有订单级复盘，高换手策略很难知道自己到底输在哪里。

## 进一步实战化

### 一页应用练习

围绕 `交易系统、执行与市场微观结构` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
