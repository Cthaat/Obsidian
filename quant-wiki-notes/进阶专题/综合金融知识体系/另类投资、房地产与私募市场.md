# 另类投资、房地产与私募市场

> [!note] 学习定位
> 另类投资包括房地产、REITs、私募股权、私募债、对冲基金、基础设施、商品、艺术品和其他非传统资产。它们的共同点不是“高级”，而是流动性、估值、费用和信息透明度更复杂。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、另类投资的核心问题

| 问题 | 为什么重要 |
|---|---|
| 现金流来源 | 租金、分红、项目退出、利息、套利收益是否真实 |
| 估值方式 | 是否依赖模型估值或管理人主观估值 |
| 流动性 | 能否赎回、转让、退出，是否有封闭期 |
| 费用结构 | 管理费、业绩报酬、交易费、托管费是否侵蚀收益 |
| 杠杆 | 项目或基金是否隐含高杠杆 |
| 信息透明度 | 投资者能否看清底层资产 |
| 法律结构 | 权利、优先级、清算顺序和违约处理 |

## 二、房地产与 REITs

房地产投资的收益来自租金、资产升值、杠杆和税费结构。评估时要看：

- 地段、人口、产业、供给、租金和空置率。
- 贷款利率、还款方式、现金流覆盖和再融资风险。
- 物业维护、税费、交易成本和政策限制。
- REITs 还要看底层资产、分红率、负债率、租户集中度和估值。

房地产最容易被误判的地方是把流动性差和高杠杆包装成“稳定”。

## 三、私募股权与私募债

私募股权通常赚企业成长、治理改善、行业整合和退出估值的钱；私募债赚信用利差和结构保护的钱。关键风险包括：

- 退出周期长，资金被锁定。
- 估值不透明，账面净值可能滞后。
- 管理人能力差异巨大。
- 杠杆收购对利率和现金流敏感。
- 投资者很难独立验证底层资产质量。

## 四、对冲基金与绝对收益

对冲基金不是天然低风险。不同策略风险完全不同：

| 策略 | 收益来源 | 风险 |
|---|---|---|
| 股票多空 | 选股 alpha、风格暴露 | 市场暴露、拥挤交易 |
| CTA | 趋势跟踪、危机 alpha | 震荡期回撤 |
| 套利 | 定价偏差收敛 | 流动性、杠杆、事件失败 |
| 宏观 | 利率、汇率、商品、股债观点 | 观点过早、政策反转 |
| 量化中性 | 因子或统计 alpha | 模型衰减、容量和成本 |

## 六、组合定位

另类资产适合承担某些特定角色：提高分散度、获得流动性溢价、对冲通胀、增加现金流或获取特殊策略收益。但它们不适合用来逃避风险。越不透明、越难退出、费用越复杂的产品，仓位越应保守。

## 七、最小可执行项目

选择一个 REIT、私募基金说明书或另类资产产品，写一页尽调报告：

| 模块 | 输出 |
|---|---|
| 底层资产 | 资产类型、地域、行业、集中度 |
| 收益来源 | 租金、利息、分红、退出或套利 |
| 费用 | 管理费、业绩报酬、隐性成本 |
| 流动性 | 封闭期、赎回规则、转让限制 |
| 风险 | 杠杆、估值、信用、政策和管理人风险 |
| 组合角色 | 核心、防御、收益增强还是卫星 |

验收标准：能说清“为什么值得牺牲流动性”。

## 九、实战案例：高分红资产也可能高风险

某些房地产、基础设施或私募债产品看起来分红稳定，但如果分红来自加杠杆、资产出售、短期融资或估值调节，就不是可持续现金流。判断高分红产品时，要追问三件事：

1. 分红来自经营现金流，还是来自融资和资产处置。
2. 底层资产能否长期产生稳定现金。
3. 如果融资环境收紧，分红和估值会怎样变化。

## 十、另类资产仓位原则

另类资产通常不适合用“看好就重仓”的方式配置。更合理的做法是：

- 先定义流动性预算：这笔钱多久不能动。
- 再定义风险预算：最坏情况下是否接受折价退出。
- 最后定义组合角色：通胀对冲、收益增强、分散化还是特殊机会。

如果说不清组合角色，就不应该因为“稀缺”“高端”“稳健”这些词买入。

### 一页应用练习

围绕 `另类投资、房地产与私募市场` 选择一个真实资产或策略，写一页应用卡：

| 项目 | 输出 |
|---|---|
| 它解决什么问题 | 收益来源、风险约束、执行效率或心理纪律 |
| 需要哪些证据 | 数据、财报、宏观、估值、价格或交易记录 |
| 改变哪个动作 | 买入、卖出、仓位、对冲、观察或不交易 |
| 最大误用风险 | 在什么情况下这个框架会误导你 |
| 复盘指标 | 哪个证据说明原判断应更新 |

这张卡的目的，是把综合知识从“概念地图”变成“决策动作”。
