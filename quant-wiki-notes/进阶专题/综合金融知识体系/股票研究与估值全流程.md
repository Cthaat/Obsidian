# 股票研究与估值全流程

> [!note] 学习定位
> 股票研究的目标不是写漂亮故事，而是判断企业未来现金流、资本回报、竞争优势和估值安全边际。本文仅用于学习、研究与复盘，不构成任何投资建议。

## 一、研究流程

- 理解商业模式：谁付钱、为什么付钱、增长来自哪里。
- 分析行业结构：竞争格局、进入壁垒、供需周期和政策约束。
- 拆解财报：收入质量、利润质量、现金流质量和资产负债表强度。
- 建立估值：DCF、DDM、PE、PB、EV/EBITDA 和情景估值交叉验证。
- 形成交易计划：催化剂、仓位、止损、加仓和退出规则。

## 二、估值三原则

- 估值是区间，不是精确点。
- 安全边际来自保守假设和价格折扣。
- 低估值必须和质量、周期、治理一起看。

## 三、常见陷阱

- 周期顶部利润外推。
- 把会计利润当现金流。
- 忽视股权稀释、资本开支和营运资本压力。

## 四、最小研究报告结构

一份可交易的股票研究，不需要一开始写得很长，但必须覆盖以下部分：

| 模块 | 必须回答 |
|---|---|
| 一句话结论 | 为什么现在值得研究，核心分歧是什么 |
| 商业模式 | 谁付钱、为什么持续付钱、是否有定价权 |
| 行业结构 | 市场空间、竞争格局、进入壁垒、周期位置 |
| 财务质量 | 利润是否变成现金流，资本开支和营运资本是否吞噬利润 |
| 估值区间 | 悲观、基准、乐观三种情景分别值多少 |
| 催化剂 | 什么事件会让市场重新定价 |
| 风险 | 哪个假设错了会造成永久亏损 |
| 交易计划 | 仓位、买入区间、加减仓和退出规则 |

研究报告的底线是可复盘：买入前写清楚的假设，必须能被后续财报、行业数据、价格行为或风控指标验证。

## 五、从研究到仓位

研究越充分，不等于仓位越大。仓位要同时看三件事：

- 置信度：证据是否强，反方证据是否被认真处理。
- 赔率：上涨空间和下跌空间是否不对称。
- 相关性：这笔交易是否和已有持仓暴露在同一个风险上。

可以用一个简单规则约束自己：如果只能解释上涨逻辑，不能解释最大亏损和退出条件，就只能停留在观察池，不能进入真实持仓。

## 实战掌握清单

### 股票研究顺序

1. 商业模式：公司卖什么、卖给谁、凭什么赚钱、收入是否可重复。
2. 行业结构：市场空间、竞争格局、进入壁垒、议价能力和监管环境。
3. 财务质量：收入确认、毛利率、费用率、现金流、资本开支、负债和商誉。
4. 管理层与治理：资本配置、分红回购、关联交易、激励和信息披露质量。
5. 估值与催化：安全边际、增长假设、估值区间、催化剂和退出条件。

### 估值交叉验证

成长股不能只看增速，周期股不能只看低PE，金融股不能只看利润，重资产企业不能忽略折旧和资本开支。至少用两种估值方法交叉验证：例如DCF加PE、PB-ROE、EV/EBITDA加自由现金流、股息率加资产负债质量。

### 研究输出模板

- 一句话投资假设：为什么这家公司未来会创造超额收益。
- 三个关键证据：行业、财务、估值各一个。
- 三个反方风险：基本面、估值、市场情绪各一个。
- 仓位计划：买入区间、加仓条件、减仓条件和彻底退出条件。

## 深度案例与训练

### 小型研究项目

围绕 股票研究与估值全流程 设计一个一页纸研究项目：先说明它属于资产定价、收益来源、风险约束、交易执行还是行为纪律，再选一个真实标的或策略做案例。案例中必须写清楚输入数据、判断逻辑、仓位动作和退出条件。

### 全局连接

- 与宏观连接：增长、通胀、利率、信用和流动性如何影响它。
- 与组合连接：它会增加还是降低组合风险，相关性会在何时变化。
- 与执行连接：交易成本、滑点、流动性和规则限制是否改变结论。
- 与复盘连接：哪些证据说明判断正确，哪些证据要求暂停。

### 训练标准

能把 股票研究与估值全流程 讲清楚还不够，还要能把它写成一条可执行规则、一张风险清单和一个复盘问题。

## 最小可执行项目

### 一页系统卡

围绕 股票研究与估值全流程 写一张系统卡：它属于哪条主线，影响哪类资产，使用哪些数据，改变哪些交易动作，暴露哪些风险。系统卡的目标是把知识从“知道”变成“会用”。

| 模块 | 输出 |
|---|---|
| 收益来源 | 这项知识解释哪类收益 |
| 风险来源 | 最可能亏在哪里 |
| 数据证据 | 需要哪些指标或案例 |
| 组合影响 | 会提高还是降低相关风险 |
| 执行约束 | 成本、流动性和规则限制 |
| 复盘问题 | 哪个证据会推翻原判断 |

### 验收标准

读完 股票研究与估值全流程 后，应能写出一个真实资产案例，并说明买入、卖出、仓位或不交易中的一个动作如何改变。
