# xalpha基金工具

> [!note] 工具概览
> `xalpha` 是专注于**基金投资分析与管理**的 Python 工具，提供基金净值查询、投资组合管理、交易策略回测、QDII 净值预测等功能。支持自然语言驱动的量化分析（xalpha Agent）。由 refraction-ray 开发维护。

## 基本信息

- **GitHub**: [refraction-ray/xalpha](https://github.com/refraction-ray/xalpha)
- **文档**: [xalpha.readthedocs.io](https://xalpha.readthedocs.io)
- **最新版本**: v0.12.3+
- **安装**: `pip install xalpha`
- **新功能**: xalpha Agent（自然语言驱动量化分析）

## 核心功能模块

| 功能 | 说明 |
|------|------|
| 基金/指数信息 | 查询基金净值、持仓、费率等基础信息 |
| 单一标的交易处理 | 记录买卖操作、计算收益、跟踪成本 |
| 投资组合管理 | 多基金组合的资产配置与绩效分析 |
| 交易策略与回测 | 定投策略、均线策略等回测验证 |
| 策略监视与提醒 | 定时监控策略信号、发送提醒通知 |
| 通用数据获取 | 日线和实时数据获取器 |
| QDII净值预测 | 预测QDII基金实时净值 |

## 快速开始

### 基金和指数信息

```python
import xalpha as xa

# 查询基金信息
fund = xa.fundinfo('161725')  # 招商中证白酒指数
fund.price  # 查看净值走势
```

### 单一标的交易处理

```python
# 记录基金买卖交易
myfund = xa.myfund('161725')
myfund.buy('2023-01-01', 1000, 1.5)  # 日期、金额、净值
myfund.sell('2023-06-01', 500, 1.8)
myfund.xirr()  # 计算年化收益
```

### 投资组合管理

```python
# 构建基金投资组合
portfolio = xa.mulfix(['161725', '005827'], [0.6, 0.4])
portfolio.summary()  # 组合概览
```

### 策略回测

```python
# 定投策略回测
strategy = xa.policy.timing('161725', ...)
strategy.backtest(start='2020-01-01', end='2023-12-31')
```

## 高级功能

### 数据缓存与本地化

- 支持本地数据缓存，减少重复网络请求
- 数据本地化存储，方便离线分析

### 数据可视化

- 内置 matplotlib 绑定
- 支持净值走势图、收益曲线、回撤分析等图表

### 数据提供商

支持多个数据源，可通过配置切换：
- 天天基金
- 东方财富
- 其他第三方数据源

### QDII净值预测

> [!tip] 特色功能
> QDII 基金因时差原因净值公布延迟，xalpha 可基于持仓和海外实时行情预测当日净值。

### 动态回测系统

- 支持自定义回测周期
- 支持动态调整策略参数
- 内置多种评价指标

### 记账单格式说明

支持标准格式的交易记录导入导出，便于长期跟踪管理。

## 主要模块关系

```
fundinfo (基金信息)
    ↓
myfund (个人持仓)  ←→  mulfix (组合管理)
    ↓
policy (策略引擎)  ←→  backtest (回测系统)
    ↓
notify (提醒通知)
```

## xalpha Agent（新功能）

> [!tip] 自然语言驱动
> xalpha Agent 支持用自然语言与量化分析交互，降低使用门槛。可直接用中文描述分析需求，自动生成代码并执行。

## 使用场景

1. **个人基金投资管理**: 记录交易、计算收益、跟踪成本
2. **定投策略回测**: 验证各种定投策略的历史表现
3. **QDII净值预测**: 提前预判海外基金当日净值
4. **组合资产配置**: 多基金组合的优化与再平衡
5. **投资研究分析**: 基金持仓变动、业绩归因分析
