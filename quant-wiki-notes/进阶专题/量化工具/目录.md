# 量化工具 — 目录

> [!note] 本专题概览
> 本专题整理主流中文量化工具，覆盖从数据获取（AKShare、efinance）到策略回测（EasyQuant）再到基金分析（xalpha）、实盘交易框架（VnPy）的完整工具链。

## 工具对比总览

| 工具 | 定位 | 语言 | 核心能力 | GitHub Stars |
|------|------|------|---------|-------------|
| [[目录|AKShare]] | 开源财经数据接口库 | Python | 股票/期货/期权/基金/债券/外汇/宏观/加密货币等全覆盖 | 10k+ |
| [[目录|EasyQuant]] | 轻量级量化策略框架 | Python + TS | 事件驱动回测、Web策略工作室、多策略模板、实盘导出 | – |
| [[目录|efinance]] | 股票/基金/期货数据获取 | Python | 简洁API、支持A股及非A股、Docker部署 | 3k+ |
| [[目录|xalpha]] | 基金分析与管理工具 | Python | 基金信息/净值/持仓/交易/回测/组合管理/提醒 | 2k+ |
| [[目录|VnPy/VeighNa]] | 全功能量化交易平台 | Python | 20+交易接口、AI量化模块、GUI、分布式架构 | 41.2k |
