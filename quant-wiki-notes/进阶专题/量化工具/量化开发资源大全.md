## 知识体系结构

### 1. 量化开发基础

入门级知识，适合零基础学习者：
- 金融市场基础概念
- 量化交易基本流程
- 编程语言选择（Python/C++）

### 2. 量化交易系统核心技术栈

| 技术领域 | 关键技术 |
|----------|----------|
| 数据获取 | 行情接口、数据清洗、存储方案 |
| 策略开发 | 信号生成、因子计算、模型训练 |
| 回测系统 | 事件驱动、向量化回测、滑点模拟 |
| 实盘部署 | 接口对接、风控系统、监控告警 |
| 性能优化 | 低延迟、高并发、内存管理 |

### 3. C++量化开发方向

高频/低延迟交易的核心语言：
- C++11/14/17/20 新特性
- 内存管理与性能优化
- 多线程与并发编程
- 网络编程（TCP/UDP/组播）
- C++ API封装与接口设计

### 4. Python量化开发

快速策略研发的首选语言：
- NumPy / Pandas / SciPy
- 数据分析与可视化
- 策略框架（[[VnPy框架详解|VnPy]]、Zipline等）
- 机器学习与深度学习应用

### 5. 系统集成与运维方向

生产环境部署与维护：
- Linux系统管理
- Docker / Kubernetes 容器化
- CI/CD 持续集成
- 监控与日志系统
- 网络安全与风控

### 6. 资源推荐与工具库

| 类别 | 推荐资源 |
|------|----------|
| 开源框架 | VnPy、Backtrader、Zipline |
| 数据源 | TuShare、AKShare、efinance |
| 回测平台 | 聚宽、米筐、优矿 |
| 交易接口 | CTP、XTP、恒生 |

### 7. 进阶话题探讨

高级量化开发主题：
- 高频交易系统设计
- 期权定价与对冲策略
- 机器学习在量化中的应用
- 多因子模型构建
- 风险管理与组合优化

### 8. 面试题

量化开发岗位面试准备：
- C++ 面试高频题
- Python 数据处理题
- 概率统计与数学题
- 系统设计题
- 策略逻辑题

