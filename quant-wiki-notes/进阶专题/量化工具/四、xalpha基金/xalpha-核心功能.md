# xalpha 核心功能

> [!note] 概述
> xalpha 围绕基金投资全流程设计，从信息获取、交易记录到策略回测和组合管理。

## 一、基金和指数信息

### fundinfo — 基金信息

```python
import xalpha as xa

# 获取基金对象
fund = xa.fundinfo("000001")

# 查看净值
fund.price()

# 查看基本信息
fund.info()
```

支持的基金类型：公募基金（普通、ETF、LOF、QDII、FOF）、私募基金。

### indexinfo — 指数信息

```python
# 获取指数对象
index = xa.indexinfo("000300")

# 获取指数成分
index.get_component()
```

## 二、单一标的交易处理

| 功能 | 说明 |
|------|------|
| 交易记录 | 记录每笔买入/卖出交易的时间和份额 |
| 持仓计算 | 自动计算当前持仓和成本 |
| 收益分析 | 计算已实现/未实现收益 |
| 费率计算 | 自动计算申购/赎回费率 |

```python
# 创建交易对象
trade = xa.trade(fund, start="2023-01-01")

# 买入
trade.buy("2023-01-15", amount=10000)

# 卖出
trade.sell("2023-06-15", shares=500)

# 查看收益
trade.profit()
```

## 三、基金投资组合管理

### portfolio — 组合管理

| 功能 | 说明 |
|------|------|
| 多基金组合 | 同时管理多只基金 |
| 权重配置 | 自定义或自动均衡权重 |
| 组合净值 | 计算组合的总净值走势 |
| 再平衡 | 定期再平衡到目标权重 |

```python
# 创建组合
portfolio = xa.portfolio(
    ["000001", "161725", "513100"],
    weights=[0.3, 0.4, 0.3]
)

# 查看组合净值
portfolio.price()

# 回测组合表现
portfolio.backtest(start="2022-01-01")
```

## 四、基金交易策略与回测

### policy — 策略框架

| 概念 | 说明 |
|------|------|
| 策略定义 | 定义买入/卖出信号规则 |
| 数据获取 | 自动获取所需的历史净值数据 |
| 回测执行 | 按历史数据模拟交易 |
| 绩效评估 | 计算收益率、最大回撤、夏普比率等 |

**策略类型**：
- **定投策略**：定期定额买入
- **估值策略**：根据 PE/PB 估值决定买卖
- **趋势策略**：基于均线等技术指标
- **动态平衡**：根据市场波动调整仓位

## 五、监视和定时提醒

| 功能 | 说明 |
|------|------|

| 邮件提醒 | 当满足条件时发送邮件通知 |

## 六、通用数据获取器

```python
# 获取日线数据
xa.get_daily("000001", start="2022-01-01", end="2022-12-31")
```

### get_rt — 实时数据

```python
# 获取实时净值估算
xa.get_rt("161725")
```

## 课程化学习补充

> [!important] 学习定位
> 工具服务于研究闭环：获取数据、清洗数据、验证策略、执行交易和复盘监控；选择工具要看稳定性和可维护性。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 接口是否稳定
- 数据字段和频率是否满足策略
- 版本是否锁定
- 是否有替代数据源交叉验证

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 免费数据缺口
- 网页结构变化导致接口失效
- 版本升级破坏兼容
- 忽视数据授权和合规

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
