# xalpha 核心功能

> [!note] 概述
> xalpha 围绕基金投资全流程设计，从信息获取、交易记录到策略回测和组合管理。

## 一、基金和指数信息

### fundinfo — 基金信息

```python
import xalpha as xa

# 获取基金对象
fund = xa.fundinfo("000001")

# 查看净值
fund.price()

# 查看基本信息
fund.info()
```

支持的基金类型：公募基金（普通、ETF、LOF、QDII、FOF）、私募基金。

### indexinfo — 指数信息

```python
# 获取指数对象
index = xa.indexinfo("000300")

# 获取指数成分
index.get_component()
```

## 二、单一标的交易处理

| 功能 | 说明 |
|------|------|
| 交易记录 | 记录每笔买入/卖出交易的时间和份额 |
| 持仓计算 | 自动计算当前持仓和成本 |
| 收益分析 | 计算已实现/未实现收益 |
| 费率计算 | 自动计算申购/赎回费率 |

```python
# 创建交易对象
trade = xa.trade(fund, start="2023-01-01")

# 买入
trade.buy("2023-01-15", amount=10000)

# 卖出
trade.sell("2023-06-15", shares=500)

# 查看收益
trade.profit()
```

## 三、基金投资组合管理

### portfolio — 组合管理

| 功能 | 说明 |
|------|------|
| 多基金组合 | 同时管理多只基金 |
| 权重配置 | 自定义或自动均衡权重 |
| 组合净值 | 计算组合的总净值走势 |
| 再平衡 | 定期再平衡到目标权重 |

```python
# 创建组合
portfolio = xa.portfolio(
    ["000001", "161725", "513100"],
    weights=[0.3, 0.4, 0.3]
)

# 查看组合净值
portfolio.price()

# 回测组合表现
portfolio.backtest(start="2022-01-01")
```

## 四、基金交易策略与回测

### policy — 策略框架

| 概念 | 说明 |
|------|------|
| 策略定义 | 定义买入/卖出信号规则 |
| 数据获取 | 自动获取所需的历史净值数据 |
| 回测执行 | 按历史数据模拟交易 |
| 绩效评估 | 计算收益率、最大回撤、夏普比率等 |

**策略类型**：
- **定投策略**：定期定额买入
- **估值策略**：根据 PE/PB 估值决定买卖
- **趋势策略**：基于均线等技术指标
- **动态平衡**：根据市场波动调整仓位

## 五、监视和定时提醒

| 功能 | 说明 |
|------|------|
| 信号监控 | 实时监控策略信号触发 |
| 邮件提醒 | 当满足条件时发送邮件通知 |
| 定时任务 | 可配合 cron 定时执行检查 |

## 六、通用数据获取器

### get_daily — 日线数据

```python
# 获取日线数据
xa.get_daily("000001", start="2022-01-01", end="2022-12-31")
```

### get_rt — 实时数据

```python
# 获取实时净值估算
xa.get_rt("161725")
```
