# 网格交易实践汇总

> [!note] 本篇定位
> 把网格从"懂原理"（见 [[网格交易赚钱逻辑]]）落到"能开干"：选什么标的、区间怎么定、几格、每格多少钱、留多少补仓、何时收网。是一张**实操参数清单**。

## 一、四步开一张网

```mermaid
graph LR
    A[1 选标的: 高波动+高流动+低费+不归零] --> B[2 定区间: 当前价±一定幅度]
    B --> C[3 分格: 等差/等比, 定格数]
    C --> D[4 配资金: 每格金额+补仓预留]
    D --> E[运行+动态调整]
```

## 二、选标的

| 要求 | 原因 |
|---|---|
| 波动适中偏高 | 没波动网格不转 |
| 流动性好 | 价差小、能成交 |
| 费率低 | 满足"间距 > 成本" |
| 不会归零 | 敢越跌越买（宽基/行业 ETF） |
| 最好 T+0（跨境 ETF） | 当日多次触发，效率高 |

## 三、参数经验值（示例）

| 参数 | 经验范围（示例） | 说明 |
|---|---|---|
| 价格区间 | 当前价 ±10%–15% | 太窄易突破，太宽资金利用低 |
| 网格数 | 8–15 格 | 兼顾利润厚度与触发频率 |
| 单格仓位 | 总投入 / 格数 | 等分 |
| 最大持仓 | 不超过总资金的一定比例 | 给单边下跌留余地 |
| 补仓预留 | 留足现金（如 ≥30%） | 跌破下沿后还能接 |

> [!important] 区间要"以丰补歉"地设
> 区间下沿要设在你**愿意越跌越买**的位置，并预留补仓资金应对跌破下沿。把全部资金一次性铺进网格，是单边下跌中爆仓的主因。

## 四、等差 vs 等比网格

| 类型 | 间距 | 适合 |
|---|---|---|
| 等差网格 | 每格价差固定（如每跌 0.1 元买一份） | 价格区间窄、习惯绝对价位 |
| 等比网格 | 每格比例固定（如每跌 1% 买一份） | 区间宽、更符合百分比波动直觉 |

等比网格间距 $r=(H/L)^{1/n}-1$（见 [[网格交易赚钱逻辑]]）。

## 五、运行纪律与动态调整

| 情形 | 应对 |
|---|---|
| 区间内震荡 | 机械执行，不追高不恐慌 |
| 触及上沿 | 分批止盈或上移区间 |
| 跌破下沿 | 用预留补仓资金接，或暂停、重设更低区间 |
| 趋势确立 | 趋势向上可"收网"转持有，向下应果断止损/重设 |

> [!warning] 网格不是"挂上就不管"
> 趋势改变时必须人工介入（收网/移网/止损）。把网格当全自动印钞机，是最常见的亏损方式。

## 六、网格实操检查清单

- [ ] 标的是否高波动 + 高流动 + 低费 + 不归零？
- [ ] 区间下沿是否在我愿意接盘的位置？
- [ ] 单格利润是否明显 > 双边成本？
- [ ] 是否预留了足够补仓现金？
- [ ] 跌破下沿/涨破上沿的预案写好了吗？
- [ ] 最大持仓是否在可承受范围内？

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 全仓铺网格 | 必须留补仓资金应对单边下跌 |
| 区间随便设 | 下沿要设在愿意接盘处 |
| 挂上就不管 | 趋势变了要收网/移网/止损 |
| 越密越好 | 间距要大于双边成本 |
