# 网格交易实践汇总

> [!note] 概述
> 整合上交所官方指南、天天基金平台、雪球社区等多方网格交易实践资源。

## 上交所官方指南要点

- 网格交易是震荡市的有效策略
- 适用于波动率适中的ETF品种
- 建议结合止盈止损设置

## 天天基金版网格交易

- 场外基金也可实现类网格操作
- 利用定投+手动补仓实现网格效果
- 费率高于场内ETF，需注意成本

## 雪球社区实践精华

### 网格交易标的推荐（雪球共识）

| 品种 | 优势 |
|------|------|
| 沪深300ETF | 流动性好、波动适中 |
| 中证500ETF | 弹性高、机会多 |
| 创业板ETF | 波动大、网格空间大 |
| 恒生科技ETF | T+0、高弹性 |
| 芯片ETF | 高波动、政策红利 |

### 网格参数经验值

| 参数 | 经验范围 |
|------|----------|
| 价格区间 | 当前价±10%-15% |
| 网格密度 | 8-15格 |
| 单格仓位 | 等分资金的1/N |
| 最大持仓 | 不超总资金30% |
| 补仓预留 | ≥30% |

## 多方共识

> [!note] 网格交易成功关键
> 1. 选对品种：高波动+高流动性+低费率
> 2. 设好参数：区间合理、间距适当、仓位可控
> 3. 严格纪律：不追高、不恐慌、机械执行
> 4. 动态调整：趋势改变时及时收网或重新设网
