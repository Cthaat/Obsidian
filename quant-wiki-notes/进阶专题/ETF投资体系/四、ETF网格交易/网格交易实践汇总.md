# 网格交易实践汇总

> [!note] 本篇定位
> 把网格从"懂原理"（见 [[网格交易赚钱逻辑]]）落到"能开干"：选什么标的、区间怎么定、几格、每格多少钱、留多少补仓、何时收网。是一张**实操参数清单**。

## 一、四步开一张网

```mermaid
graph LR
    A[1 选标的: 高波动+高流动+低费+不归零] --> B[2 定区间: 当前价±一定幅度]
    B --> C[3 分格: 等差/等比, 定格数]
    C --> D[4 配资金: 每格金额+补仓预留]
    D --> E[运行+动态调整]
```

## 二、选标的

| 要求 | 原因 |
|---|---|
| 波动适中偏高 | 没波动网格不转 |
| 流动性好 | 价差小、能成交 |
| 费率低 | 满足"间距 > 成本" |
| 不会归零 | 敢越跌越买（宽基/行业 ETF） |
| 最好 T+0（跨境 ETF） | 当日多次触发，效率高 |

## 三、参数经验值（示例）

| 参数 | 经验范围（示例） | 说明 |
|---|---|---|
| 价格区间 | 当前价 ±10%–15% | 太窄易突破，太宽资金利用低 |
| 网格数 | 8–15 格 | 兼顾利润厚度与触发频率 |
| 单格仓位 | 总投入 / 格数 | 等分 |
| 最大持仓 | 不超过总资金的一定比例 | 给单边下跌留余地 |
| 补仓预留 | 留足现金（如 ≥30%） | 跌破下沿后还能接 |

> [!important] 区间要"以丰补歉"地设
> 区间下沿要设在你**愿意越跌越买**的位置，并预留补仓资金应对跌破下沿。把全部资金一次性铺进网格，是单边下跌中爆仓的主因。

## 四、等差 vs 等比网格

| 类型 | 间距 | 适合 |
|---|---|---|
| 等差网格 | 每格价差固定（如每跌 0.1 元买一份） | 价格区间窄、习惯绝对价位 |
| 等比网格 | 每格比例固定（如每跌 1% 买一份） | 区间宽、更符合百分比波动直觉 |

等比网格间距 $r=(H/L)^{1/n}-1$（见 [[网格交易赚钱逻辑]]）。

## 五、运行纪律与动态调整

| 情形 | 应对 |
|---|---|
| 区间内震荡 | 机械执行，不追高不恐慌 |
| 触及上沿 | 分批止盈或上移区间 |
| 跌破下沿 | 用预留补仓资金接，或暂停、重设更低区间 |
| 趋势确立 | 趋势向上可"收网"转持有，向下应果断止损/重设 |

> [!warning] 网格不是"挂上就不管"
> 趋势改变时必须人工介入（收网/移网/止损）。把网格当全自动印钞机，是最常见的亏损方式。

## 六、网格实操检查清单

- [ ] 标的是否高波动 + 高流动 + 低费 + 不归零？
- [ ] 区间下沿是否在我愿意接盘的位置？
- [ ] 单格利润是否明显 > 双边成本？
- [ ] 是否预留了足够补仓现金？
- [ ] 跌破下沿/涨破上沿的预案写好了吗？
- [ ] 最大持仓是否在可承受范围内？

## 常见误区

| 误区 | 更好的理解 |
|---|---|
| 全仓铺网格 | 必须留补仓资金应对单边下跌 |
| 区间随便设 | 下沿要设在愿意接盘处 |
| 挂上就不管 | 趋势变了要收网/移网/止损 |
| 越密越好 | 间距要大于双边成本 |

## 课程化学习补充

> [!important] 学习定位
> 用 ETF 把大类资产、行业主题和策略工具模块化，重点不是猜单只产品，而是把指数暴露、费率、流动性和再平衡纪律放进同一张决策表。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 底层指数是否清楚
- 规模与成交额是否足以承载仓位
- 跟踪误差和折溢价是否可接受
- 是否有清晰的再平衡和止盈规则

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 主题拥挤后估值回撤
- 小规模 ETF 流动性不足
- 跨境 ETF 汇率与时差风险
- 杠杆/反向产品路径依赖

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `网格交易实践汇总` 时，不要只把它当成孤立知识点。把 ETF 当成资产配置和策略表达工具，而不是低门槛的普通基金。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 指数编制和成分股暴露
- 费率、跟踪误差和折溢价
- 成交额、盘口深度和冲击成本
- 再平衡纪律和税费影响

### 进阶训练

1. 比较同类 ETF 的指数、规模、换手和跟踪误差
2. 用行业/宽基/跨境 ETF 做一版风险预算
3. 记录一次折溢价或流动性异常并写出处理规则

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
