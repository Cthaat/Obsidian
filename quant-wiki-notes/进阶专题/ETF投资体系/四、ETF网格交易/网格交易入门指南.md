# 网格交易入门指南

> [!note] 概述
> 网格交易像一张"渔网"，在价格波动中自动低买高卖赚差价。本文从原理到实操，帮助新手快速上手。

## 一、网格交易原理

### 核心概念

网格交易 = ETF（价格波动）+ 自动化买卖（低买高卖）

```
设定价格区间 → 分网格 → 跌到格子买、涨到格子卖 → 反复循环
```

### 具体操作步骤

1. **设定价格区间**：如ETF当前10元，预测震荡区间9元-11元
2. **分网格**：每0.5元一格（9 / 9.5 / 10 / 10.5 / 11）
3. **挂单买卖**：
   - 跌到格子价 → 自动买入
   - 涨到格子价 → 自动卖出
4. **反复循环**：波动越大，买卖次数越多，收益越多

## 二、实战案例

### 以沪深300ETF为例

| 参数 | 设定值 |
|------|--------|
| 当前价格 | 5元 |
| 价格区间 | 4.5元-5.5元 |
| 每格间距 | 0.1元 |
| 总网格数 | 10格 |
| 初始买入 | 5手（每手100股） |
| 每格操作 | 跌0.1加仓1手，涨0.1卖出1手 |

## 三、优缺点分析

### ✅ 优点

- **自动化**：设定条件单后不用盯盘
- **震荡市赚钱**：价格来回波动时收益明显
- **分散风险**：分批买卖，避免一次性高位接盘

### ❌ 缺点

- **单边行情亏损**：持续下跌可能不断买入被套
- **踏空风险**：持续上涨可能卖光后踏空
- **手续费成本**：频繁交易可能增加手续费
- **需要资金管理**：需预留足够资金补仓

## 四、操作步骤

1. **选ETF**：流动性好、波动适中的ETF
2. **定区间**：根据历史波动设定±10%
3. **分网格**：每格间距1%-3%
4. **挂条件单**：用券商APP的"网格交易工具"
5. **监控调整**：若ETF趋势改变需手动干预

## 五、注意事项

> [!note] 八大要点
> ① 网格交易 = 震荡市自动收割机
> ② 选对平台：场内基金，不是场外基金
> ③ 选对ETF：品种活跃、T+0等
> ④ 合理分格：降低欲望，千分之一收益就是暴利
> ⑤ 严守纪律：目标必须坚定
> ⑥ 手工操盘强于APP自动挂单（但需要盯盘时间）
> ⑦ 解套好工具：忘记成本
> ⑧ 具备超短线操盘技术和盘口判断能力
