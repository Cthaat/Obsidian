# ETF轮动策略构建与改进

> [!note] 来源
> 知乎量化藏经阁专栏：手把手教你构建与改进ETF轮动策略（十年19倍，附源码）

## 轮动策略概述

### 三种常见轮动

| 类型 | 说明 | 代表策略 |
|------|------|----------|
| 资产轮动 | 大类资产间轮动 | 股债轮动、美林时钟 |
| 行业轮动 | 各行业/板块间轮动 | 行业动量轮动 |
| 风格轮动 | 投资风格间轮动 | 大小盘轮动、成长价值轮动 |

### 核心两要素

1. **候选池**：选择哪些品种参与轮动
2. **强弱排序**：用什么标准判断强弱

### 数据获取（Python）

```python
import akshare as ak

code_list = ['510300', '510500', '510880', '159915']
start_date = '20150101'
end_date = '20250828'

for code in code_list:
    df = ak.fund_etf_hist_em(symbol=code, period='daily',
        start_date=start_date, end_date=end_date, adjust='hfq')
```

### 动量计算方法

| 方法 | 公式 | 特点 |
|------|------|------|
| ROC | (当日收盘/前N日收盘) - 1 | 经典动量 |
| 收益率 | 多周期复合 | 更平滑 |
| 线性回归斜率 | sklearn.LinearRegression | 趋势方向 |

### 调仓逻辑

- 每N个交易日排序一次
- 买入动量最强的1-2个品种
- 卖出不再最强的品种

## 策略回测结果

| 指标 | 数值 |
|------|------|

| 策略年化 | 显著超越单一持有 |

## 策略改进方向

1. **多周期动量**：结合短/中/长周期动量信号
2. **趋势过滤**：在下跌市场中转持货币ETF
3. **波动率调整**：风险平价加权
4. **多品种扩展**：加入行业ETF、跨境ETF
