# ETF期权策略

> [!note] 概述
> ETF期权是在ETF基础上衍生出的金融工具，允许投资者在特定时间内以特定价格买入或卖出ETF份额的权利。

## 基础概念

| 术语 | 定义 |
|------|------|
| 看涨期权 (Call) | 赋予买入权利 |
| 看跌期权 (Put) | 赋予卖出权利 |
| 行权价 | 期权执行价格 |
| 到期日 | 期权有效期截止日 |

## 常见策略

### 1. 保护性看跌策略 (Protective Put)

```
持有ETF + 买入看跌期权
```

- **目的**：为ETF持仓购买"保险"
- **适用**：担心短期下跌但不想卖出ETF
- **成本**：支付期权金

### 2. 备兑看涨策略 (Covered Call)

```
持有ETF + 卖出看涨期权
```

- **目的**：在持仓基础上增收期权金
- **适用**：预期ETF不会大幅上涨时
- **风险**：如果大涨，ETF可能被行权卖出

### 3. 领口策略 (Collar)

```
持有ETF + 买入看跌 + 卖出看涨
```

- **目的**：低成本对冲，同时增收
- **适用**：希望对冲下跌风险但不想支付高额期权金
- **特征**：用卖出Call的期权金抵消买入Put的成本

### 4. 牛市价差 (Bull Spread)

```
买入低行权价Call + 卖出高行权价Call
```

- **目的**：温和看涨时降低期权成本
- **收益上限**：行权价差 - 净期权金

### 5. 熊市价差 (Bear Spread)

```
买入高行权价Put + 卖出低行权价Put
```

- **目的**：温和看跌时降低期权成本

## 风险管理

| 风险维度 | 管理方法 |
|----------|----------|
| Delta（方向） | Delta对冲，控制方向性敞口 |
| Gamma（变化率） | 控制Gamma风险 |
| Theta（时间） | 管理时间价值衰减 |
| Vega（波动率） | 控制波动率风险敞口 |

## 适用场景

- **投资组合对冲**：保护性看跌策略
- **收益增强**：备兑看涨策略
- **波动率交易**：利用隐含波动率定价
- **方向性投机**：牛市/熊市价差
