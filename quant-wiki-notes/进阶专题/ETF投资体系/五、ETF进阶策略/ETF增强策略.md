# ETF增强策略

> [!note] 来源
> 中国基金报：追求超额收益还能灵活交易，ETF也有增强版（2025年5月）

## 增强ETF概述

### 什么是增强ETF

指数增强ETF = 被动指数跟踪 + 主动管理增强

- 兼具ETF的交易灵活、持仓透明、低费率优势
- 通过主动管理争取超越标的指数的回报

### 市场现状

| 指标 | 数据（截至2025年5月） |
|------|----------------------|
| 指增ETF数量 | 35只 |
| 指增ETF总规模 | 67.20亿元 |
| 首只指增ETF成立 | 2021年12月 |
| 美国主动ETF规模 | 8579亿美元（2024年底） |

## 增强策略实现方式

### 主动+量化结合

1. **主动选股因子**：捕捉传统量化因子无法捕获的Alpha
2. **量化选股因子**：大类因子综合建模
3. **量化风险模型**：严控行业、风格风险暴露

### 代表产品：景顺长城沪深300增强策略ETF（159238）

| 特征 | 说明 |
|------|------|
| 跟踪指数 | 沪深300 |
| 指数质地 | 6%成分股贡献55.82%总市值、80.68%净利润 |
| 基金经理配置 | "被动+主动"搭配 |
| 增强效果 | 2024年中证500增强策略ETF超额约5% |

## 增强ETF vs 纯被动ETF

| 对比维度 | 纯被动ETF | 增强ETF |
|----------|----------|---------|
| 跟踪目标 | 完全复制指数 | 超越指数 |
| 管理方式 | 被动 | 主动+量化 |
| 费率 | 最低0.15% | 略高 |
| 超额收益潜力 | 无 | 有 |
| 跟踪误差 | 最小 | 可接受范围 |
