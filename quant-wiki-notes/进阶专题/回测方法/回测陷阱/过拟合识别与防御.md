# 过拟合识别与防御

> [!note] 核心概念
> 过拟合（Overfitting）是量化交易的头号杀手。你的模型不是在学真实规律，而是在记忆历史数据的随机噪声。

## 什么是过拟合？

**类比**：假设你要预测明天天气，翻看过去100天记录发现了"规律"——每次穿红色衣服的人超过30%，第二天就下雨。这个规律在过去100天成立（纯巧合），但预测未来显然荒谬。量化交易中的过拟合就是找到这种"看起来很强，实际上毫无因果关系"的规律。

## 为什么量化特别容易过拟合？

| 原因 | 说明 |
|------|------|
| 参数太多 | 100个参数，只有1000个样本 |
| 多重检验 | 测试1000个策略，总有几个"有效" |
| 信噪比低 | 市场噪音大，真实信号弱 |
| 分布不稳定 | 过去的规律未来可能失效 |

## 过拟合的识别

### 识别信号

1. **策略参数异常敏感**：均线20日效果好，换成19日或21日就大幅恶化
2. **收益曲线过于完美**：回测净值几乎是一条直线，没有明显回撤区
3. **策略逻辑缺乏经济学解释**："每周三下午2:30买入"——没有逻辑
4. **样本内/外表现差异巨大**：训练集年化50%，测试集年化5%

### 常见过拟合模式

| 模式 | 症状 | 风险等级 |
|------|------|---------|
| 完美曲线 | 无比平滑，无任何回撤 | 🔴 极高 |
| 极端参数敏感 | 参数微小变化导致结果剧变 | 🔴 高 |
| 信号数量正相关 | 信号越多收益越高（反常） | 🔴 高 |
| 训练/测试差距大 | 训练集 >> 测试集 | 🟡 中 |

## 防御方法

### 1. 减少参数（奥卡姆剃刀）
在同等解释力下，参数越少的模型越好。3个参数的策略比15个参数的更可靠。

### 2. 严格的样本外测试
```
完整数据划分：
├─ 训练集 (60%)：参数优化、策略开发
├─ 验证集 (20%)：参数最终选择
├─ 测试集 (20%)：最终绩效评估 ← 只用一次！
```

**关键原则**：测试集**仅使用一次**用于最终评估，不可回看也不可调参。

### 3. Walk-Forward 验证
```
第1轮：训练[1月-6月] → 测试[7月]
第2轮：训练[2月-7月] → 测试[8月]
第3轮：训练[3月-8月] → 测试[9月]
...
每轮都是严格的样本外测试
```

### 4. Monte Carlo 模拟
随机扰动测试策略鲁棒性：
- 随机打乱交易顺序
- 随机调整入场时间 ±1天
- 随机调整成本 ±20%
- → 如果90%模拟结果为正且分布集中，策略可靠

### 5. 正则化
- **Ridge (L2)**：惩罚大系数，使因子系数趋于平滑
- **Lasso (L1)**：稀疏化系数，自动特征选择

### 6. 跨市场/跨时期验证
- 美股发现的策略，在A股、港股、欧洲市场也有效吗？
- 2015-2019训练的策略，在2020-2023表现如何？

## 实战清单

- [ ] 参数数量控制在最小（≤5个核心参数）
- [ ] 使用了严格的样本外测试（训练/验证/测试三段划分）
- [ ] 策略有清晰的经济逻辑
- [ ] t统计量超过3.0（而非2.0）
- [ ] 参数±20%变化，收益变化<30%
- [ ] 回测涵盖至少1个牛熊周期
- [ ] 如果回测收益减半后仍然可接受，再考虑实盘

## 行业经验

> **回测到实盘性能衰减平均30-50%**——这不是悲观估计，而是无数实盘验证后的行业共识。

| 衰减幅度 | 占比 | 主要原因 |
|---------|------|---------|
| <20% | 10% | 低频策略、优秀成本建模 |
| 20-30% | 25% | 正常衰减 |
| 30-50% | 40% | 典型情况 |
| 50-70% | 15% | 成本低估、轻度过拟合 |
| >70% | 10% | 严重过拟合 |

## 实战掌握清单

> [!tip] 交易者视角
> 过拟合识别与防御 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。回测是研究假设的压力测试，不是寻找漂亮收益曲线的工具。

### 关键判断

- 确认数据时间戳、复权、停牌、退市、公告日和交易日历。
- 分清样本内开发、样本外验证和实盘跟踪。
- 把手续费、滑点、冲击成本和容量限制前置。

### 落地动作

1. 先建立最简单基准，再逐步加入过滤条件。
2. 检查参数稳定性、收益来源和不同市场阶段表现。
3. 输出回撤、换手、暴露、交易笔数和失败案例。

### 失效边界

- 前视偏差和幸存者偏差。
- 过度调参。
- 忽略资金规模放大后的成交约束。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
