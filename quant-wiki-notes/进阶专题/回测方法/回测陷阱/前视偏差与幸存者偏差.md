# 前视偏差与幸存者偏差

> [!note] 核心概念
> 前视偏差和幸存者偏差是回测中最具破坏性的两大陷阱，也是"回测年化100%，实盘亏成狗"的主要元凶。

## 前视偏差（Look-ahead Bias）

### 本质
**时间维度的信息泄露**——策略在t时刻的决策不应基于t时刻之后才公布的数据，但回测中往往无意中使用了这些未来信息。

### 五大产生原因

| 原因类别 | 具体情形 | 典型案例 |
|---------|---------|---------|
| 数据对齐问题 | 日内数据与日数据混合 | 用盘中最高/低价计算收益，实际只能收盘价成交 |
| 财务报表延迟 | 使用立即可得的财务数据 | 在财报发布日期前使用EPS数据，实际滞后2-4周 |
| 除权除息处理 | 未正确处理分红派息 | 在股权登记日前计入分红收益 |
| 技术指标计算 | 使用偷价价格 | 用未平滑的HIGH/LOW计算指标 |
| 事件日期误用 | 事件生效日期错误 | 把公告日期当作事件发生日期 |

### 代码示例

```python
# ❌ 错误：前视偏差
for i in range(len(data)):
    if data['close'][i] > data['open'][i]:  # 今天收阳
        buy_price = data['open'][i]  # 错！收盘后才知道收阳

# ✅ 正确：滞后一期
for i in range(1, len(data)):
    if data['close'][i-1] > data['open'][i-1]:  # 昨天收阳
        buy_price = data['open'][i]  # 今天开盘价入场

# ✅ 正确：特征工程中用 shift(1)
df['normalized'] = (df['close'] - df['close'].rolling(20).mean()) / df['close'].rolling(20).std()
```

### 避免原则

| 原则 | 实现方式 |
|------|---------|
| 信号与执行分离 | T日信号，T+1日执行 |
| 只用过去数据 | 计算指标用 `shift(1)` |
| 假设最坏成交价 | 买入用最高价，卖出用最低价 |
| 时间戳校验 | 所有数据标注正确的可获得时间 |

## 幸存者偏差（Survivorship Bias）

### 本质
分析只看到"活下来的"样本（幸存者），没看到"死掉的"样本，从而得出错误结论。

> 经典比喻：二战飞机弹孔分析——只看到了返航飞机的弹孔，那些发动机中弹的飞机没有回来。

### 量化投资中的三层表现

**1. 数据层面**：用当前沪深300成分股回测过去10年，遗漏了已退市/被剔除的股票。学术研究表明收益每年虚高**1%-3%**。

**2. 策略层面**：看到社交媒体上的"年化80%策略"，但没看到：
- 99个类似策略亏损的作者不会发帖
- 可能同时跑了10个策略，只展示最好的

**3. 基金层面**：修正幸存者偏差后，对冲基金行业真实年化超额收益仅**2%-3%**，远低于宣传数字。

### 五层纵深分析

| 层次 | 发现 | 影响 |
|------|------|------|
| **退市股票** | NASDAQ年退市率5.6%，退市前跌-55% | 年化收益高估1.6% |
| **指数调整** | 被剔除标普500的股票，5年后跑赢市场5%+ | 收益被低估 |
| **小市值因子** | 修正退市偏差后效应消失 | 经典因子被颠覆 |
| **价值因子** | 价值溢价每年被高估0.60% | Alpha虚高 |
| **基金回测** | 存活基金Alpha比真实高估0.60%/年 | 行业水平被美化 |

### 解决方案

1. **使用完整历史数据库**：CRSP、Compustat、Bloomberg等包含退市证券
2. **进行敏感性分析**：分别用当前宇宙和完整宇宙回测，量化偏差
3. **动态成分股回测**：使用每个历史时点真实的成分股名单
4. **添加失败成本模拟**：将退市股票的损失计入结果

## 实战掌握清单

> [!tip] 交易者视角
> 前视偏差与幸存者偏差 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。回测是研究假设的压力测试，不是寻找漂亮收益曲线的工具。

### 关键判断

- 确认数据时间戳、复权、停牌、退市、公告日和交易日历。
- 分清样本内开发、样本外验证和实盘跟踪。
- 把手续费、滑点、冲击成本和容量限制前置。

### 落地动作

1. 先建立最简单基准，再逐步加入过滤条件。
2. 检查参数稳定性、收益来源和不同市场阶段表现。
3. 输出回撤、换手、暴露、交易笔数和失败案例。

### 失效边界

- 前视偏差和幸存者偏差。
- 过度调参。
- 忽略资金规模放大后的成交约束。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
