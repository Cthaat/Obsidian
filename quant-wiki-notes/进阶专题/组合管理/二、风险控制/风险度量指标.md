# 风险度量指标

> [!note] 核心要点
> 单纯的累计收益率极具误导性。一个短期收益惊人的策略，可能隐藏着足以让账户归零的毁灭性风险。本文系统剖析衡量策略优劣的核心**风险调整后收益指标**：夏普比率、索提诺比率、最大回撤和MAR比率。

## 一、夏普比率 (Sharpe Ratio)

### 1.1 定义

由诺贝尔奖得主威廉·夏普提出，衡量每单位**总风险**获得的**超额回报**。

$$\text{Sharpe} = \frac{E[R_p] - R_f}{\sigma_p}$$

| 参数 | 含义 |
|------|------|
| $E[R_p]$ | 策略期望收益率 |
| $R_f$ | 无风险利率 |
| $\sigma_p$ | 收益率标准差（总风险） |

### 1.2 年化计算

```python
def calculate_sharpe(returns, rf=0.0, periods=252):
    daily_rf = (1 + rf) ** (1/periods) - 1
    excess = returns - daily_rf
    return np.sqrt(periods) * (excess.mean() / excess.std())
```

### 1.3 评判标准

| 夏普比率 | 评价 |
|----------|------|
| < 0 | 策略不如无风险资产 |
| 0 ~ 1 | 一般 |
| 1 ~ 2 | 良好 |
| > 2 | 优秀 |
| > 3 | 罕见（警惕过拟合） |

### 1.4 局限性

将向上波动（盈利）和向下波动（亏损）视为同等"风险"，不符合投资者真实感受。

## 二、索提诺比率 (Sortino Ratio)

### 2.1 定义

后现代投资组合理论（PMPT）的产物，只考虑**下行风险**。

$$\text{Sortino} = \frac{E[R_p] - MAR}{\sigma_d}$$

- **MAR**：最低可接受回报率（Minimum Acceptable Return）
- **σ_d**：下行标准差（只计算低于MAR的样本）

### 2.2 与夏普比率的对比

| 维度 | 夏普比率 | 索提诺比率 |
|------|----------|------------|
| 风险度量 | 总波动（上下都算） | 仅下行波动 |
| 适用场景 | 收益对称分布 | 收益非对称分布 |
| 典型策略 | 均值回归 | 趋势跟踪、期权卖方 |

### 2.3 Python实现

```python
def calculate_sortino(returns, mar=0.0, periods=252):
    daily_mar = (1 + mar) ** (1/periods) - 1
    excess = returns - daily_mar
    downside = excess[excess < 0]
    downside_std = np.sqrt((downside**2).mean())
    return np.sqrt(periods) * (excess.mean() / downside_std)
```

## 三、最大回撤 (Maximum Drawdown)

### 3.1 定义

在选定时间周期内，策略净值从任意历史高点下跌到其后低点的**最大跌幅**。

$$MDD = \max_{t \in [0,T]} \left(\frac{\text{Peak}_t - \text{Trough}_t}{\text{Peak}_t}\right)$$

### 3.2 意义

- 揭示策略的**尾部风险**
- 回答"最倒霉时点买入最多亏多少"
- 杠杆交易中可能直接导致爆仓

### 3.3 Python实现

```python
def calculate_max_drawdown(equity_curve):
    peak = equity_curve.expanding(min_periods=1).max()
    drawdown = (equity_curve - peak) / peak
    return drawdown.min()
```

### 3.4 实战参考

| 最大回撤 | 评价 | 影响 |
|----------|------|------|
| < 10% | 优秀 | 心理压力小 |
| 10%~20% | 良好 | 可承受 |
| 20%~30% | 一般 | 需要减仓 |
| > 30% | 警告 | 可能触发清盘 |
| > 50% | 危险 | 几乎无法恢复 |

## 四、MAR比率

### 4.1 定义

$$MAR = \frac{\text{预期收益}}{\text{最大回撤}}$$

### 4.2 核心思想

风险可以简单理解为**预期最大回撤**。MAR比率揭示的是：在可承受风险之下的最大收益能力。

> [!tip] 实战经验
> 历史最大回撤 × 1.5 ≈ 预期最大回撤。选择MAR更高的系统，长期复利效果差异巨大。

### 4.3 策略选择示例

| 系统 | 最大回撤 | 盈利 | MAR |
|------|----------|------|-----|
| 系统一 | 10万 | 200万 | 20 |
| 系统二 | 20万 | 300万 | 15 |

虽然系统二盈利更多，但系统一的MAR更高，长期复利下系统一必然胜出。

## 五、Calmar比率

$$\text{Calmar} = \frac{\text{年化收益}}{\text{最大回撤}}$$

与MAR类似，但使用年化收益，更适合跨期比较。

## 六、指标综合对比

| 指标 | 分子 | 分母 | 关注点 |
|------|------|------|--------|
| 夏普比率 | 超额收益 | 总波动率 | 单位总风险回报 |
| 索提诺比率 | 超额收益 | 下行波动率 | 单位下行风险回报 |
| 最大回撤 | — | 净值跌幅 | 极端损失 |
| MAR比率 | 预期收益 | 最大回撤 | 收益/极端风险 |
| Calmar比率 | 年化收益 | 最大回撤 | 年化收益/极端风险 |
| 信息比率 | 超额收益 | 跟踪误差 | 主动管理能力 |

## 七、风险指标组合仪表盘

单个风险指标都可能误导。优秀交易者不会只看夏普，也不会只看最大回撤，而是用一组指标分别回答“波动是否可承受、尾部是否会毁灭账户、收益是否来自主动能力、组合是否过度集中、压力期是否还能交易”。

| 指标 | 回答的问题 | 容易误导的情况 | 应搭配 |
|---|---|---|---|
| 年化波动率 | 日常净值波动有多大 | 收益分布厚尾、跳空、停牌时低估风险 | 最大回撤、VaR/CVaR |
| 夏普比率 | 每单位总波动带来多少超额收益 | 趋势策略和期权策略收益分布不对称 | 索提诺、偏度、最大回撤 |
| 索提诺比率 | 下行波动是否被充分补偿 | 样本太短时下行样本不足 | 最大回撤、亏损天数 |
| 最大回撤 | 最痛苦的历史亏损是多少 | 历史没有覆盖真正压力期 | 压力测试、杠杆率 |
| Calmar/MAR | 收益是否足以补偿回撤 | 回撤尚未暴露时会虚高 | 回撤持续时间、恢复时间 |
| VaR | 常规置信水平下可能亏多少 | 对极端尾部不敏感 | CVaR、压力测试 |
| CVaR/ES | 极端亏损发生后平均会亏多少 | 依赖尾部样本质量 | 情景分析、流动性折价 |
| Beta | 对市场系统性风险暴露多大 | 非线性策略、择时策略中不稳定 | 因子暴露、相关性 |
| 跟踪误差 | 偏离基准的主动风险有多大 | 基准选错时没有意义 | 信息比率、主动持仓 |
| 换手率 | 策略依赖交易频率到什么程度 | 不直接反映冲击成本 | 滑点、成交量占比 |
| 单票/单策略权重 | 是否过度集中 | 相关资产会伪装成分散 | 风险贡献、相关性矩阵 |
| 流动性指标 | 想卖时能否卖出去 | 牛市成交量会高估真实容量 | 成交冲击、赎回压力 |

日常看板可以分为三层：

- 账户层：净值、最大回撤、回撤持续时间、杠杆、现金比例。
- 组合层：资产权重、因子暴露、行业集中度、相关性、风险贡献。
- 执行层：成交滑点、换手率、订单失败、成交量占比、异常波动。

## 八、压力测试与流动性风险

风险管理不能只问“过去发生过什么”，还要问“没有发生但合理可能发生什么”。压力测试的价值，是把抽象数字还原成可以想象、可以行动的场景。

| 压力场景 | 需要检查 | 典型应对 |
|---|---|---|
| 市场快速下跌 | 净敞口、杠杆、保证金、止损触发 | 降低杠杆、保留现金、减少同向拥挤 |
| 利率快速上行 | 久期、成长股估值、债券价格、融资成本 | 缩短久期、检查折现率敏感性 |
| 流动性枯竭 | 持仓成交量占比、买卖价差、赎回压力 | 降低容量假设、分批交易 |
| 汇率大幅波动 | 外币资产、海外收入、对冲成本 | 明确是否对冲和对冲比例 |
| 行业监管冲击 | 政策敏感行业、商业模式合规性 | 控制单行业暴露、设定事件退出条件 |
| 模型失效 | 信号衰减、成交成本上升、拥挤交易 | 降低仓位、重新做样本外验证 |

流动性风险尤其容易被忽视。一个策略在回测里可以瞬间成交，在实盘里却可能因为盘口深度不足、价格跳空、停牌、涨跌停、赎回压力或交易对手撤单而无法退出。因此容量评估至少要写清楚：

- 单日成交量占比上限是多少。
- 建仓和清仓分别需要多少个交易日。
- 压力期成交量下降、价差扩大时，滑点会扩大到多少。
- 如果同时遭遇回撤和赎回，是否会被迫在最差价格卖出。

## 九、风险复盘模板

每次发生明显回撤或风控触发，都应写一页风险复盘，而不是只看账户亏了多少。

| 复盘项 | 记录内容 |
|---|---|
| 触发事件 | 是市场下跌、个股事件、模型失效、执行错误还是流动性问题 |
| 原始预案 | 交易前是否写过最大亏损、退出规则和仓位上限 |
| 实际损失 | 金额、比例、回撤持续天数、恢复所需收益 |
| 风险来源 | Beta、行业、单票、杠杆、相关性、滑点、情绪决策 |
| 是否可避免 | 是研究错误、执行错误、纪律错误，还是必须承受的风险补偿 |
| 后续动作 | 降仓、停止交易、修正规则、增加监控、补充数据 |

风险复盘的关键不是追求“永远不亏”，而是区分三类损失：

- 计划内损失：策略为获得长期收益必须承受，且规模在预算内。
- 可改进损失：研究、执行、数据、仓位或纪律存在缺陷，下次应减少。
- 不可接受损失：超过承受能力、违反投资政策书或可能导致无法继续交易。

如果一笔亏损无法被归类，就说明交易前计划不够清楚，应回到[[综合金融知识体系/投资政策书IPS与交易计划模板|投资政策书和交易计划]]重新定义风险预算。

## 实战掌握清单

> [!tip] 交易者视角
> 风险度量指标 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。组合管理把单个机会放进总风险预算，核心是分散、相关性、再平衡、流动性和尾部风险。

### 关键判断

- 区分资产配置、策略配置、行业配置和单券配置。
- 观察波动、相关性、回撤、杠杆和流动性是否集中。
- 用风险贡献而不是只用资金权重看组合。

### 落地动作

1. 设定目标权重、偏离阈值和再平衡频率。
2. 做压力测试和情景分析，尤其是相关性同时上升的情形。
3. 把组合复盘拆成选股贡献、择时贡献、仓位贡献和成本贡献。

### 失效边界

- 名义分散但风险同源。
- 回撤后被动砍仓。
- 忽略流动性和赎回压力。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？
