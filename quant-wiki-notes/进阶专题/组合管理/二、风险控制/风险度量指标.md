# 风险度量指标

> [!note] 核心要点
> 单纯的累计收益率极具误导性。一个短期收益惊人的策略，可能隐藏着足以让账户归零的毁灭性风险。本文系统剖析衡量策略优劣的核心**风险调整后收益指标**：夏普比率、索提诺比率、最大回撤和MAR比率。

## 一、夏普比率 (Sharpe Ratio)

### 1.1 定义

由诺贝尔奖得主威廉·夏普提出，衡量每单位**总风险**获得的**超额回报**。

$$\text{Sharpe} = \frac{E[R_p] - R_f}{\sigma_p}$$

| 参数 | 含义 |
|------|------|
| $E[R_p]$ | 策略期望收益率 |
| $R_f$ | 无风险利率 |
| $\sigma_p$ | 收益率标准差（总风险） |

### 1.2 年化计算

```python
def calculate_sharpe(returns, rf=0.0, periods=252):
    daily_rf = (1 + rf) ** (1/periods) - 1
    excess = returns - daily_rf
    return np.sqrt(periods) * (excess.mean() / excess.std())
```

### 1.3 评判标准

| 夏普比率 | 评价 |
|----------|------|
| < 0 | 策略不如无风险资产 |
| 0 ~ 1 | 一般 |
| 1 ~ 2 | 良好 |
| > 2 | 优秀 |
| > 3 | 罕见（警惕过拟合） |

### 1.4 局限性

将向上波动（盈利）和向下波动（亏损）视为同等"风险"，不符合投资者真实感受。

## 二、索提诺比率 (Sortino Ratio)

### 2.1 定义

后现代投资组合理论（PMPT）的产物，只考虑**下行风险**。

$$\text{Sortino} = \frac{E[R_p] - MAR}{\sigma_d}$$

- **MAR**：最低可接受回报率（Minimum Acceptable Return）
- **σ_d**：下行标准差（只计算低于MAR的样本）

### 2.2 与夏普比率的对比

| 维度 | 夏普比率 | 索提诺比率 |
|------|----------|------------|
| 风险度量 | 总波动（上下都算） | 仅下行波动 |
| 适用场景 | 收益对称分布 | 收益非对称分布 |
| 典型策略 | 均值回归 | 趋势跟踪、期权卖方 |

### 2.3 Python实现

```python
def calculate_sortino(returns, mar=0.0, periods=252):
    daily_mar = (1 + mar) ** (1/periods) - 1
    excess = returns - daily_mar
    downside = excess[excess < 0]
    downside_std = np.sqrt((downside**2).mean())
    return np.sqrt(periods) * (excess.mean() / downside_std)
```

## 三、最大回撤 (Maximum Drawdown)

### 3.1 定义

在选定时间周期内，策略净值从任意历史高点下跌到其后低点的**最大跌幅**。

$$MDD = \max_{t \in [0,T]} \left(\frac{\text{Peak}_t - \text{Trough}_t}{\text{Peak}_t}\right)$$

### 3.2 意义

- 揭示策略的**尾部风险**
- 回答"最倒霉时点买入最多亏多少"
- 杠杆交易中可能直接导致爆仓

### 3.3 Python实现

```python
def calculate_max_drawdown(equity_curve):
    peak = equity_curve.expanding(min_periods=1).max()
    drawdown = (equity_curve - peak) / peak
    return drawdown.min()
```

### 3.4 实战参考

| 最大回撤 | 评价 | 影响 |
|----------|------|------|
| < 10% | 优秀 | 心理压力小 |
| 10%~20% | 良好 | 可承受 |
| 20%~30% | 一般 | 需要减仓 |
| > 30% | 警告 | 可能触发清盘 |
| > 50% | 危险 | 几乎无法恢复 |

## 四、MAR比率

### 4.1 定义

$$MAR = \frac{\text{预期收益}}{\text{最大回撤}}$$

### 4.2 核心思想

风险可以简单理解为**预期最大回撤**。MAR比率揭示的是：在可承受风险之下的最大收益能力。

> [!tip] 实战经验
> 历史最大回撤 × 1.5 ≈ 预期最大回撤。选择MAR更高的系统，长期复利效果差异巨大。

### 4.3 策略选择示例

| 系统 | 最大回撤 | 盈利 | MAR |
|------|----------|------|-----|
| 系统一 | 10万 | 200万 | 20 |
| 系统二 | 20万 | 300万 | 15 |

虽然系统二盈利更多，但系统一的MAR更高，长期复利下系统一必然胜出。

## 五、Calmar比率

$$\text{Calmar} = \frac{\text{年化收益}}{\text{最大回撤}}$$

与MAR类似，但使用年化收益，更适合跨期比较。

## 六、指标综合对比

| 指标 | 分子 | 分母 | 关注点 |
|------|------|------|--------|
| 夏普比率 | 超额收益 | 总波动率 | 单位总风险回报 |
| 索提诺比率 | 超额收益 | 下行波动率 | 单位下行风险回报 |
| 最大回撤 | — | 净值跌幅 | 极端损失 |
| MAR比率 | 预期收益 | 最大回撤 | 收益/极端风险 |
| Calmar比率 | 年化收益 | 最大回撤 | 年化收益/极端风险 |
| 信息比率 | 超额收益 | 跟踪误差 | 主动管理能力 |

## 课程化学习补充

> [!important] 学习定位
> 组合管理把单个机会变成可长期复利的资金曲线，核心是相关性、风险预算、再平衡和极端情景。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收益来源是否足够分散
- 相关性在压力期如何变化
- 再平衡规则是否机械执行
- 最大回撤是否在承受范围内

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 表面分散实则同一风险
- 低估相关性突变
- 杠杆叠加
- 回撤期停止执行纪律

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
