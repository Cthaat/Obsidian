# 风险平价模型 (Risk Parity)

> [!note] 核心要点
> 风险平价（Risk Parity）是一种强调**均衡分配各类资产风险贡献**的资产配置方法。突破传统按资本权重分配的逻辑，主张按风险贡献均衡分配，避免单一资产（如股票）主导风险来源。相比"60/40"股债组合中股票承担超90%的风险，风险平价通过杠杆调节低波动资产，实现真正的风险分散。

## 一、核心理念

### 1.1 与传统配置的差异

| 维度 | 传统60/40 | 风险平价 |
|------|-----------|----------|
| 分配依据 | 资本权重 | 风险贡献 |
| 股票风险占比 | >90% | ~50% |
| 分散化程度 | 低 | 高 |
| 单位风险收益 | 较低 | 较高 |

### 1.2 数学表达

设组合权重为 $w_i$，资产边际风险贡献（MRC）为 $\frac{\partial \sigma_p}{\partial w_i}$，则风险平价求解：

$$w_i \cdot \frac{\partial \sigma_p}{\partial w_i} = w_j \cdot \frac{\partial \sigma_p}{\partial w_j}, \quad \forall i,j$$

即各资产的**风险贡献（RC）相等**。

## 二、理论基础

### 2.1 组合收益与风险

$$\mu_p = \mathbf{w}^T \boldsymbol{\mu}$$

$$\sigma_p^2 = \mathbf{w}^T \Sigma \mathbf{w}$$

### 2.2 风险分解

组合总风险可分解为各资产的加权边际贡献：

$$RC_i = w_i \cdot \frac{\partial \sigma_p}{\partial w_i} = \frac{w_i (\Sigma w)_i}{\sigma_p}$$

根据欧拉定理，$\sum RC_i = \sigma_p$，保证了风险分解的可加性。

### 2.3 三资产示例

| 资产 | 预期年化收益 | 年化波动率 |
|------|-------------|-----------|
| 股票A | 10% | 20% |
| 债券B | 4% | 6% |
| 商品C | 8% | 15% |

等权重组合：收益7.33%，波动率10.2%

## 三、实现方法

### 3.1 风险预算模型

$$\min_w \quad \sigma_p = \sqrt{w^T \Sigma w}$$
$$\text{s.t.} \quad RC_i = b_i \cdot \sigma_p, \quad \sum w_i = 1, \quad w_i \geq 0$$

其中 $b_i$ 是目标风险预算（风险平价时 $b_i = 1/N$）。

### 3.2 迭代求解算法

```python
import numpy as np
from scipy.optimize import minimize

def risk_parity(cov_matrix):
    n = len(cov_matrix)
    target_risk = np.ones(n) / n  # 等风险贡献
    
    def objective(w):
        portfolio_vol = np.sqrt(w @ cov_matrix @ w)
        marginal_risk = cov_matrix @ w / portfolio_vol
        risk_contrib = w * marginal_risk
        risk_ratio = risk_contrib / portfolio_vol
        return np.sum((risk_ratio - target_risk)**2)
    
    constraints = {'type': 'eq', 'fun': lambda x: np.sum(x) - 1}
    bounds = tuple((0.01, 1) for _ in range(n))
    w0 = np.ones(n) / n
    
    result = minimize(objective, w0, method='SLSQP',
                     bounds=bounds, constraints=constraints)
    return result.x
```

## 四、与其他策略对比

| 策略 | 核心思想 | 风险控制 | 收益潜力 |
|------|----------|----------|----------|
| 等权重 | 权重相等 | 间接 | 中等 |
| 最小方差 | 最小化波动 | 强 | 较低 |
| 最大夏普 | 最大化风险调整收益 | 中等 | 较高 |
| 风险平价 | 风险贡献相等 | 强 | 中等 |
| CVaR风险平价 | 尾部风险贡献相等 | 最强 | 中等 |

## 五、实践要点

### 5.1 协方差矩阵估计

- **样本估计**：简单但不稳定（维度灾难）
- **Ledoit-Wolf收缩**：在样本和目标矩阵间收缩
- **指数加权**：近期数据权重更大
- **DCC-GARCH**：动态条件相关模型

### 5.2 杠杆调节

风险平价通常需要对低波动资产（如债券）加杠杆，使其风险贡献与高波动资产（如股票）匹配。

### 5.3 动态再平衡

定期或基于阈值重新调整权重，维持等风险贡献状态。

## 六、历史表现

- 2008年金融危机期间表现优于传统60/40组合
- 长期风险调整后收益稳定
- 在利率上升环境中可能面临挑战
