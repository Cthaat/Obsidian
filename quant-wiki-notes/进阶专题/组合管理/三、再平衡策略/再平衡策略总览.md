# 再平衡策略总览

> [!note] 核心要点
> 投资组合再平衡是维持目标风险特征的最简单杠杆。若不管理，60/40配置在长期牛市中可能偏离至80/20+的股票权重，增加未预期的下行风险。再平衡的主要目标是**风险控制**，而非通过市场时机来打败市场。

## 一、再平衡的本质

### 1.1 为什么需要再平衡

- 各类资产表现差异导致配置**漂移**
- 漂移可能使组合风险超出投资者承受能力
- 再平衡恢复预期风险暴露，实现"高抛低吸"

### 1.2 再平衡溢价

> [!tip] 重要认知
> 所谓"再平衡溢价"（买入表现不佳者、卖出赢家）的可衡量收益大多来自**纪律性**（降低波动性和避免行为错误），而非持续alpha。纪律性再平衡带来**行为性alpha**。

### 1.3 实例

| 阶段 | 股票 | 债券 | 总计 | 股票占比 |
|------|------|------|------|----------|
| 初始 | $6,000,000 | $4,000,000 | $10,000,000 | 60% |
| 股票涨25% | $7,500,000 | $4,000,000 | $11,500,000 | 65.2% |
| 偏离 | — | — | — | +5.2% |
| 再平衡 | $6,900,000 | $4,600,000 | $11,500,000 | 60% |

## 二、三种再平衡触发方式

### 2.1 日历型（Calendar Rebalancing）

在设定的时间间隔进行再平衡。

| 维度 | 描述 |
|------|------|
| 典型设定 | 按季度或年度 |
| 优点 | 简单、可预测、易于审计 |
| 缺点 | 漂移小时引发不必要交易 |
| 适用 | 小型团队、简单账户 |

### 2.2 阈值型（Threshold Rebalancing）

当资产权重偏离预设带宽时再平衡。

| 维度 | 描述 |
|------|------|
| 典型设定 | ±2%、±5%、±10% |
| 优点 | 对实际漂移灵敏，总体交易较低 |
| 缺点 | 需要持续监控，波动期集中交易 |
| 适用 | 税收优惠账户、顾问模型 |

### 2.3 混合型（Calendar + Threshold）

固定节奏检查，只有漂移超过阈值才再平衡。

| 维度 | 描述 |
|------|------|
| 典型设定 | 季度检查 + 5%区间 |
| 优点 | 可预测性与敏感性结合 |
| 缺点 | 需要系统支持 |
| 适用 | 机构模型组合、智能投顾 |

## 三、对比分析

| 方法 | 典型设定 | 主要好处 | 成本结构 | 使用场景 |
|------|----------|----------|----------|----------|
| 日历型 | 季度/年度 | 易于实施 | 频繁时交易成本高 | 简单账户 |
| 阈值型 | ±5% | 限制漂移 | 波动期集中交易 | 税优账户 |
| 混合型 | 季度+5% | 最佳平衡 | 可预测且高效 | 机构/智能投顾 |

> [!tip] 经验法则
> Vanguard实证：对于多元化股债组合，不同再平衡频率下风险调整后回报无显著差异——但再平衡事件数量（成本）差异很大。年度/半年度+5%区间是务实低成本折中。

## 四、再平衡成本

### 4.1 交易摩擦

- 显性佣金
- 买卖价差（点差）
- 市场冲击

### 4.2 税务摩擦

- 已实现收益的税负
- 洗售规则（前后30天限制）
- 税损收割机会

### 4.3 成本衡量框架

1. 估计预期换手率（回测或蒙特卡洛）
2. 应用交易成本假设（点差+佣金+冲击）
3. 应用税率假设
4. 比较：再平衡成本 vs 风险控制价值

## 五、大资金池特殊考虑

- 可预测再平衡时间表产生**订单流可预测性**
- 大型养老金/目标日期基金的再平衡成本可能是实质性的
- 使用**衍生品叠加**（期货、掉期）避免现货市场冲击
- 执行算法：VWAP/TWAP、Almgren-Chriss框架

## 六、运营设计要点

| 环节 | 要求 |
|------|------|
| 策略语言 | 明确再平衡规则和阈值 |
| 执行控制 | 订单路由、滑点监控 |
| 税务管理 | 税务批次元数据、跨账户监控 |
| 自动化 | 系统化触发和执行 |
| 审计追踪 | 记录每次再平衡详情 |

## 课程化学习补充

> [!important] 学习定位
> 组合管理把单个机会变成可长期复利的资金曲线，核心是相关性、风险预算、再平衡和极端情景。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收益来源是否足够分散
- 相关性在压力期如何变化
- 再平衡规则是否机械执行
- 最大回撤是否在承受范围内

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 表面分散实则同一风险
- 低估相关性突变
- 杠杆叠加
- 回撤期停止执行纪律

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `再平衡策略总览` 时，不要只把它当成孤立知识点。把单笔机会放进组合曲线，目标是可长期执行的风险收益结构。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 资产相关性和风险贡献
- 波动率、回撤和尾部风险
- 再平衡、杠杆和流动性
- 投资期限和行为承受力

### 进阶训练

1. 计算每个资产或策略的边际风险贡献
2. 做压力测试和相关性上行情景
3. 制定再平衡和止损纪律

### 能力验收

- 能否说清楚这个主题影响的是收益来源、风险来源、交易成本、流动性还是心理纪律？
- 能否指出它在什么市场环境、资产类别或交易周期中更有效？
- 能否把它写成一条可复盘的研究或交易规则？
- 能否说明如果判断错误，组合最大损失和退出机制是什么？
