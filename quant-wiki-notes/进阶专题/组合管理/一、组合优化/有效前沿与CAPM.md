# 有效前沿与CAPM

> [!note] 核心要点
> 有效前沿是市场能提供的最优风险资产组合集合，CAPM在此基础上引入无风险资产和市场均衡，建立了资产定价的理论基石。理解有效前沿→效用函数→CAL→CML的逻辑链条，是掌握现代金融理论的关键。

## 一、最小方差前沿

### 1.1 定义

马科维茨的现代投资组合理论揭示：通过多元化构建**最小方差前沿**，投资者能在特定回报下实现风险最小化。

- 所有风险资产的组合落在弧线包围的区域内
- 弧线称为"最小方差前沿"
- **Z点**：全局最小方差组合（风险最小，但收益非最低）
- Z点上方为**有效前沿**：相同风险下收益最高的组合

### 1.2 两资产组合

$$R_p = w_1 R_1 + (1-w_1) R_2$$

$$\sigma_p = \sqrt{w_1^2\sigma_1^2 + (1-w_1)^2\sigma_2^2 + 2w_1(1-w_1)\rho\sigma_1\sigma_2}$$

相关系数ρ的影响：
- ρ = 1：线性关系
- ρ < 1：曲线向左收缩
- ρ = -1：可构建零风险组合

## 二、效用函数

### 2.1 定义

$$U = E(R) - \frac{1}{2}\lambda\sigma^2$$

- λ > 0：风险厌恶（λ越大，越厌恶）
- λ = 0：风险中性
- λ < 0：风险喜好

### 2.2 无差异曲线

在相同效用下，不同风险对应不同收益的点连成的线。对于风险厌恶者，曲线越往左上方，效用越大。

## 三、资本资产配置线（CAL）

### 3.1 构建

将风险资产组合p与无风险资产组合：

$$R_P = w_1 R_1 + (1-w_1) R_f$$

$$\sigma_P = w_1 \sigma_1$$

推导得：
$$R_P = R_f + \frac{R_1 - R_f}{\sigma_1}\sigma_P$$

这就是**资本市场线（CML）**的前身。

### 3.2 最优组合

- CAL与无差异曲线的切点 = **最优投资者组合**（Optimal Investor Portfolio）
- CAL与有效前沿的切点 = **最优风险组合**（Optimal Risky Portfolio）

## 四、资本资产定价模型（CAPM）

### 4.1 公式

$$E(r_i) = r_f + \beta_{im}(E(r_m) - r_f)$$

| 参数 | 含义 |
|------|------|
| $E(r_i)$ | 资产i的预期回报率 |
| $r_f$ | 无风险利率 |
| $\beta_{im}$ | 资产i的系统性风险 |
| $E(r_m) - r_f$ | 市场风险溢价 |

### 4.2 β系数

- β = 1：与市场同步
- β > 1：波动大于市场（进攻型）
- β < 1：波动小于市场（防御型）

### 4.3 系统风险与非系统风险

| 风险类型 | 特征 | 能否分散 |
|----------|------|----------|
| 系统风险 | 利率、经济衰退、战争 | ❌ 不能 |
| 非系统风险 | 个别股票自有风险 | ✅ 可以 |

### 4.4 资本市场线（CML）

当满足**同质预期**假设后，最优风险组合 = 市场组合，CAL(M) = CML：

$$E(R_p) = R_f + \frac{E(R_m) - R_f}{\sigma_m}\sigma_p$$

## 五、应用意义

| 应用领域 | 方法 |
|----------|------|
| 资产估值 | 根据β确定贴现率 |
| 资金成本 | 估算权益资本成本 |
| 资源配置 | 牛市选高β，熊市选低β |
| 业绩评估 | 超额收益是否合理补偿了系统风险 |
