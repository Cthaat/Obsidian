# 现代投资组合理论 (Modern Portfolio Theory)

> [!note] 核心要点
> 现代投资组合理论（MPT）由Harry Markowitz于1952年提出，是金融学领域的重要突破。该理论通过数学模型证明投资者可以通过资产组合选择，在风险和收益之间找到最佳平衡点。MPT是 [[均值-方差模型]]、[[有效前沿与CAPM]] 等理论的基础框架。

## 一、理论基础

### 1.1 核心假设

- 投资者是**风险厌恶**的
- 投资者追求效用最大化
- 投资者基于收益率的**均值**和**方差**做决策
- 单期投资决策
- 无交易成本和税收

### 1.2 核心思想

1. **风险与收益的权衡**：组合预期收益是各资产收益的加权平均，风险由资产间协方差决定
2. **有效前沿**：给定风险水平下收益最高（或给定收益下风险最低）的组合集合
3. **分散化**：通过资产组合降低非系统性风险

## 二、关键概念

### 2.1 收益与风险度量

| 概念 | 公式 | 说明 |
|------|------|------|
| 组合收益 | $\mu_p = \sum w_i \mu_i$ | 加权平均 |
| 组合方差 | $\sigma_p^2 = w^T \Sigma w$ | 考虑协方差 |
| 协方差 | $\sigma_{ij} = \rho_{ij}\sigma_i\sigma_j$ | 相关性度量 |

### 2.2 分散化效应

- 非系统性风险可通过增加资产数量降低
- 系统性风险无法通过分散化消除
- 资产间相关性越低，分散化效果越好

### 2.3 有效前沿

所有风险资产组合中，理性投资者只选择有效前沿上的组合：
- 相同风险下收益最高
- 相同收益下风险最低

## 三、理论发展脉络

```
1952 Markowitz → 均值-方差模型 / 有效前沿
    ↓
1964 Sharpe/Lintner/Treynor/Mossin → CAPM
    ↓
1966 Sharpe → 夏普比率
    ↓
1976 Ross → APT 套利定价理论
    ↓
1990 Black-Litterman → 贝叶斯资产配置
    ↓
1993 Risk Parity → 风险平价策略
```

## 四、MPT的局限与改进

| 局限 | 描述 | 改进方向 |
|------|------|----------|
| 正态假设 | 现实收益呈偏态、肥尾 | CVaR、ES |
| 参数敏感 | 输入微变导致输出大变 | BL模型、收缩估计 |
| 单期假设 | 忽视多期动态 | 动态规划 |
| 无交易成本 | 实际交易有摩擦 | 带约束优化 |
| 静态框架 | 不适应市场变化 | 动态再平衡 |

## 五、实际应用

### 资产配置策略类型

| 策略 | 核心思想 |
|------|----------|
| 核心-卫星策略 | 核心资产控风险，卫星资产增收益 |
| 风险平价策略 | 各资产风险贡献相等 |
| 动态再平衡策略 | 根据市场变化调整配置 |
| BL模型策略 | 融合市场均衡与主观观点 |

## 实战掌握清单

> [!tip] 交易者视角
> 现代投资组合理论 (Modern Portfolio Theory) 的学习重点不是记住术语，而是把它放进研究、组合、执行和复盘的闭环。组合管理把单个机会放进总风险预算，核心是分散、相关性、再平衡、流动性和尾部风险。

### 关键判断

- 区分资产配置、策略配置、行业配置和单券配置。
- 观察波动、相关性、回撤、杠杆和流动性是否集中。
- 用风险贡献而不是只用资金权重看组合。

### 落地动作

1. 设定目标权重、偏离阈值和再平衡频率。
2. 做压力测试和情景分析，尤其是相关性同时上升的情形。
3. 把组合复盘拆成选股贡献、择时贡献、仓位贡献和成本贡献。

### 失效边界

- 名义分散但风险同源。
- 回撤后被动砍仓。
- 忽略流动性和赎回压力。

### 复盘问题

- 这项知识改变了哪一个具体决策：标的、方向、仓位、退出、对冲还是不交易？
- 如果判断相反，最大亏损、最长恢复期和退出触发条件是什么？
- 有没有一个更简单的基准方法可以取得相近结果？

## 深度案例与训练

### 组合诊断

围绕 现代投资组合理论 (Modern Portfolio Theory) 做一次组合体检：资金权重、风险权重、行业暴露、因子暴露、相关性、流动性和最大回撤来源。

### 压力测试

- 股票下跌、利率上行、信用利差扩大、商品波动和汇率冲击分别会怎样影响组合。
- 相关性同时上升时，哪些仓位会一起亏钱。
- 如果需要三天内降仓，哪些资产最难卖出。

### 再平衡纪律

再平衡要服务于风险结构，而不是单纯追求低买高卖。
