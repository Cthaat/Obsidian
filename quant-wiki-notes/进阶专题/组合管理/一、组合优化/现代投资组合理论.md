# 现代投资组合理论 (Modern Portfolio Theory)

> [!note] 核心要点
> 现代投资组合理论（MPT）由Harry Markowitz于1952年提出，是金融学领域的重要突破。该理论通过数学模型证明投资者可以通过资产组合选择，在风险和收益之间找到最佳平衡点。MPT是 [[均值-方差模型]]、[[有效前沿与CAPM]] 等理论的基础框架。

## 一、理论基础

### 1.1 核心假设

- 投资者是**风险厌恶**的
- 投资者追求效用最大化
- 投资者基于收益率的**均值**和**方差**做决策
- 单期投资决策
- 无交易成本和税收

### 1.2 核心思想

1. **风险与收益的权衡**：组合预期收益是各资产收益的加权平均，风险由资产间协方差决定
2. **有效前沿**：给定风险水平下收益最高（或给定收益下风险最低）的组合集合
3. **分散化**：通过资产组合降低非系统性风险

## 二、关键概念

### 2.1 收益与风险度量

| 概念 | 公式 | 说明 |
|------|------|------|
| 组合收益 | $\mu_p = \sum w_i \mu_i$ | 加权平均 |
| 组合方差 | $\sigma_p^2 = w^T \Sigma w$ | 考虑协方差 |
| 协方差 | $\sigma_{ij} = \rho_{ij}\sigma_i\sigma_j$ | 相关性度量 |

### 2.2 分散化效应

- 非系统性风险可通过增加资产数量降低
- 系统性风险无法通过分散化消除
- 资产间相关性越低，分散化效果越好

### 2.3 有效前沿

所有风险资产组合中，理性投资者只选择有效前沿上的组合：
- 相同风险下收益最高
- 相同收益下风险最低

## 三、理论发展脉络

```
1952 Markowitz → 均值-方差模型 / 有效前沿
    ↓
1964 Sharpe/Lintner/Treynor/Mossin → CAPM
    ↓
1966 Sharpe → 夏普比率
    ↓
1976 Ross → APT 套利定价理论
    ↓
1990 Black-Litterman → 贝叶斯资产配置
    ↓
1993 Risk Parity → 风险平价策略
```

## 四、MPT的局限与改进

| 局限 | 描述 | 改进方向 |
|------|------|----------|
| 正态假设 | 现实收益呈偏态、肥尾 | CVaR、ES |
| 参数敏感 | 输入微变导致输出大变 | BL模型、收缩估计 |
| 单期假设 | 忽视多期动态 | 动态规划 |
| 无交易成本 | 实际交易有摩擦 | 带约束优化 |
| 静态框架 | 不适应市场变化 | 动态再平衡 |

## 五、实际应用

### 资产配置策略类型

| 策略 | 核心思想 |
|------|----------|
| 核心-卫星策略 | 核心资产控风险，卫星资产增收益 |
| 风险平价策略 | 各资产风险贡献相等 |
| 动态再平衡策略 | 根据市场变化调整配置 |
| BL模型策略 | 融合市场均衡与主观观点 |

## 课程化学习补充

> [!important] 学习定位
> 组合管理把单个机会变成可长期复利的资金曲线，核心是相关性、风险预算、再平衡和极端情景。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收益来源是否足够分散
- 相关性在压力期如何变化
- 再平衡规则是否机械执行
- 最大回撤是否在承受范围内

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 表面分散实则同一风险
- 低估相关性突变
- 杠杆叠加
- 回撤期停止执行纪律

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？
