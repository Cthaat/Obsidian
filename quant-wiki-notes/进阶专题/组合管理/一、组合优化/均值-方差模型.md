# 均值-方差模型 (Mean-Variance Model)

> [!note] 核心要点
> 均值-方差模型由Harry Markowitz于1952年提出，是现代投资组合理论（MPT）的基石。通过量化风险与收益的权衡关系，为投资者提供系统化的决策工具，首次将"分散化"作为降低非系统性风险的有效手段。

## 一、模型基本概念

### 核心假设
- 投资者是**风险厌恶**的
- 投资者根据风险偏好选择最优组合
- 资产收益的均值和方差足以描述投资机会

### 两个关键变量

**均值（期望收益）**：
$$\mu_p = \sum_{i=1}^{n} w_i \mu_i$$

**方差（风险）**：
$$\sigma_p^2 = \sum_{i=1}^{n}\sum_{j=1}^{n} w_i w_j \sigma_{ij}$$

## 二、有效前沿

### 2.1 最小方差前沿

在所有可能的组合中，能同时实现最大收益与最小风险的组合集合构成**有效前沿**。

- **最小方差组合**（Global Minimum Variance Portfolio）：全市场风险最小的组合
- **有效前沿**：最小方差组合上方的前沿线，理性投资者的最优选择区域

### 2.2 有效前沿的性质

| 性质 | 描述 |
|------|------|
| 下凸性 | 在风险-收益平面上呈下凸曲线 |
| 分散化效应 | 增加资产数量可降低风险而不牺牲收益 |
| 相关性影响 | 资产间相关系数越低，前沿越靠左 |

### 2.3 两资产情形

当相关系数ρ变化时：
- ρ = 1：有效前沿为直线
- ρ < 1：前沿向左弯曲
- ρ = -1：可构建零风险组合

## 三、优化问题

### 3.1 最小方差组合

$$\min_w \quad w^T \Sigma w$$
$$\text{s.t.} \quad w^T \mathbf{1} = 1$$

### 3.2 最大夏普比率组合

$$\max_w \quad \frac{w^T \mu - r_f}{\sqrt{w^T \Sigma w}}$$

### 3.3 约束条件

- 权重之和为1：$\sum w_i = 1$
- 不允许卖空：$w_i \geq 0$
- 单资产权重上限：$w_i \leq u_i$

## 四、Python实现

```python
import numpy as np
from scipy.optimize import minimize

def portfolio_return(weights, returns):
    return np.dot(weights, returns)

def portfolio_volatility(weights, cov_matrix):
    return np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))

def min_variance(cov_matrix, n_assets):
    constraints = {'type': 'eq', 'fun': lambda x: np.sum(x) - 1}
    bounds = tuple((0, 1) for _ in range(n_assets))
    init_weights = np.array([1/n_assets] * n_assets)
    result = minimize(lambda w: portfolio_volatility(w, cov_matrix),
                     init_weights, method='SLSQP',
                     bounds=bounds, constraints=constraints)
    return result.x
```

## 五、模型局限性

| 局限 | 描述 | 改进方向 |
|------|------|----------|
| 参数敏感 | 微小收益预测偏差导致权重大幅波动 | Black-Litterman模型 |
| 极端权重 | 容易出现过度集中或大幅做空 | 添加约束条件 |
| 正态假设 | 假设收益服从正态分布 | 使用CVaR等 |
| 静态模型 | 未考虑市场动态变化 | 动态再平衡 |
| 估计误差 | 协方差矩阵估计不稳定 | 收缩估计 |

## 六、实证验证

### R语言实证结果

使用AAPL、MSFT、JPM、PG、XOM五只股票（2018-2023年）：

| 组合 | 年化收益 | 年化波动率 | 夏普比率 |
|------|----------|------------|----------|
| 最小方差组合 | 13.38% | 31.62% | 0.0423 |
| 最大夏普组合 | 22.45% | 39.99% | 0.0561 |
| 等权重组合 | 17.30% | 29.97% | 0.0577 |

> [!tip] 分散化效应
> 对比2只股票和5只股票的可行集，增加证券数量能显著降低风险且不牺牲收益。

## 课程化学习补充

> [!important] 学习定位
> 组合管理把单个机会变成可长期复利的资金曲线，核心是相关性、风险预算、再平衡和极端情景。本文仅用于学习、研究与复盘，不构成任何投资建议。

### 必须掌握的问题

- 收益来源是否足够分散
- 相关性在压力期如何变化
- 再平衡规则是否机械执行
- 最大回撤是否在承受范围内

### 实战应用流程

1. 先写清楚你的投资假设：为什么这个信号、资产或方法应该产生收益。
2. 明确数据口径：样本范围、更新时间、复权/分红/停牌处理和交易日历。
3. 做最小可行验证：先用简单规则验证方向，再逐步加入复杂模型。
4. 把成本和约束前置：手续费、滑点、冲击成本、保证金、流动性和容量都要进入测算。
5. 上线后持续复盘：记录信号、下单、成交、持仓、回撤和失效原因。

### 风险与失效条件

- 表面分散实则同一风险
- 低估相关性突变
- 杠杆叠加
- 回撤期停止执行纪律

### 复盘问题

- 这笔交易或这套模型赚的是什么钱：风险补偿、行为偏差、流动性溢价，还是偶然噪音？
- 如果市场环境反过来，最大亏损和最长恢复期会是多少？
- 当前结论是否依赖某个不可持续假设，例如低利率、低波动、充裕流动性或监管套利？
- 有没有一个更简单的基准策略能取得接近效果？

## 跨领域进阶扩展

> [!tip] 交易者视角
> 学到 `均值-方差模型 (Mean-Variance Model)` 时，不要只把它当成孤立知识点。把单笔机会放进组合曲线，目标是可长期执行的风险收益结构。优秀投资交易者会把它放入“宏观背景 - 资产选择 - 估值/信号 - 组合风险 - 交易执行 - 复盘反馈”的闭环。

### 与其他知识的连接

- 资产相关性和风险贡献
- 波动率、回撤和尾部风险
- 再平衡、杠杆和流动性
- 投资期限和行为承受力

### 进阶训练

1. 计算每个资产或策略的边际风险贡献
2. 做压力测试和相关性上行情景
3. 制定再平衡和止损纪律
