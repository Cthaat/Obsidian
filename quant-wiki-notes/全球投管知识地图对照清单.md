# 全球投管知识地图对照清单

> [!note] 核心问题
> 国际投管教育（如 [CFA Program Level I 主题结构](https://www.cfainstitute.org/programs/cfa-program/candidate-resources/level-i-exam)）给出一套「工具 → 资产 → 组合」的通用骨架。本清单把它**映射到本 Vault 笔记与实操导航**，用于查缺补漏；**不是**报考、押题或官方教材替代。

## 学习目标

1. 用 10 个主题自检知识缺口。  
2. 每个主题至少链到 1 篇课程 + 1 条实操路径。  
3. 区分「描述概念」与「可交付作业」。  
4. 把伦理/合规当成硬约束，而非选修。  
5. 完成自检表并选定下周补缺主题。

## 程序化交易合规意识（中国）

公开规定（例）：《证券市场程序化交易管理规定（试行）》自 **2024-10-08** 起施行（文号与全文以 [证监会页面](https://www.csrc.gov.cn/csrc/c100028/c7480577/content.shtml) 为准）。要点印象：

| 概念 | 自学含义 |
|---|---|
| 程序化交易定义 | 计算机程序自动生成或下达指令在交易所交易 |
| 先报告、后交易 | 报告相关信息后再交易的监管取向 |
| 公平性 | 不得影响系统安全与正常交易秩序 |
| 高频差异化 | 对高频有报告、收费、监控等差异安排（细则看交易所） |

个人用量化研究工具 ≠ 已满足实盘程序化报备；上线前读规定全文与券商要求。库内：[[A股交易规则实操要点]] · 程序化交易相关进阶笔记。

## 建议补缺顺序（个人量化）

```text
1. 伦理/禁止事项（说明书）
2. 定量 + 回测方法论
3. 财报 + 权益估值
4. 组合与风控卡
5. 经济/宏观情景
6. 固收基础
7. 衍生品/另类（能力圈内）
```
