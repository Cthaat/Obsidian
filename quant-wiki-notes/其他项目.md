# 相关项目

本页保留与 AI 量化研究相关的项目线索，删除招募、申请和推广内容，仅作为学习索引。

## Alpha Agent

多智能体因子挖掘与验证框架，适用于 alpha 研究、因子回测和策略原型验证。

## Sentiment Agent

基于非结构化文本生成交易信号的情感分析框架，可用于新闻、研报和社交媒体数据处理。

## LocalQuant

面向金融场景的本地化大模型和 RAG 部署方案，重点关注数据隐私、知识检索和本地推理。
