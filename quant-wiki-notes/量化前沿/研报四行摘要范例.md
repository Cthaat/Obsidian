# 研报四行摘要范例

> [!note] 核心问题
> [[前沿研报与论文学习导航]] 要求四行摘要；本页给**可模仿结构**。下列「虚拟样例」用于演示写法，**不是**对某篇真实研报的转述或荐股。

## 模板（复制）

```markdown
## 材料：标题 / 机构或作者 / 日期 / URL
- 假设：
- 数据与样本：
- 结论（作者声称）：
- 可复现性与我的质疑：
- 管道决定：灵感 / 开 EXP / 否决库
```

## 范例 A：因子类（教学虚拟）

```markdown
## 材料：某《价值因子在 A 股的再检验》（虚拟样例）/ 教学 / 无真实链接
- 假设：低 PE/PB 组合相对高估值组合存在长期溢价。
- 数据与样本：声称全 A 股 2010–2020 月度调仓；未说明是否含退市股票。
- 结论（作者声称）：多空年化差异为正，回撤可接受。
- 可复现性与我的质疑：
  1) 是否 PIT/财务对齐公告日？
  2) 成本与涨跌停？
  3) 行业市值中性？
  4) 2021 后样本外？
- 管道决定：灵感 → 对照 [[因子投资实操导航]] 写一页纸；不跟单。
```

## 范例 B：AI/ML 类（教学虚拟）

```markdown
## 材料：某《LSTM 预测收益》（虚拟样例）
- 假设：滞后价量可预测次日涨跌方向。
- 数据与样本：指数成分日频；准确率 57%。
- 结论（作者声称）：模型有效。
- 可复现性与我的质疑：
  1) 是否随机 K 折打乱时间？
  2) 标准化是否用全样本？
  3) 换手与费用后是否仍为正？
  4) 与线性基线比如何？
- 管道决定：否决为可交易；仅作 [[机器学习交易实操导航]] 反面教材清单。
```
