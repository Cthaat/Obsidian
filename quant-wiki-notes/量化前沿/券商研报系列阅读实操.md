# 券商研报系列阅读实操

> [!note] 核心问题
> 本库 `量化前沿/券商研报/` 按系列堆了大量目录。本篇教你**选一条系列、怎么读、如何四行摘要、如何接到因子/ML 作业**，避免系列收藏症。

## 学习目标

1. 在系列目录中只选定 1 条主线跟 2 周。  
2. 区分多因子 / AI / 高频系列的用途与门槛。  
3. 每篇产出四行摘要（模板见 [[研报四行摘要范例]]）。  
4. 结论只进「灵感 / EXP / 否决」，不进跟单。  
5. 遵守版权：优先正式渠道 PDF，本库目录作索引。

## 2 周阅读协议

| 规则 | 内容 |
|---|---|
| 主系列 | 只选 1 个 |
| 深读 | 每周 ≤2 篇全文级 |
| 产出 | 每篇四行摘要 + 管道决定 |
| 禁止 | 同时开 5 个系列；把图表当交易指令 |
| 对接 | 有启发 → 因子一页纸或 EXP；坑多 → 否决库 |

## 四行摘要（强制）

见 [[研报四行摘要范例]]。必须问：

1. 样本与 PIT？  
2. 成本与涨跌停？  
3. 中性化？  
4. 样本外？  
5. 拥挤？  

## 推荐路径 A：多因子系列

```text
Day1 打开某多因子系列目录，浏览标题年表
Day2 选「因子测试框架/评价」类 1 篇
Day3 四行摘要 + 对照 [[因子检验与评价]]
Day4 选「某个风格因子」1 篇
Day5 摘要 + 与 [[因子打分实操]] 边界对比
Day6-7 写：我是否要做该因子的教学级实验（默认否或仅笔记）
```

## 推荐路径 B：AI 系列

```text
先完成 [[机器学习交易实操导航]] 防泄露清单
再读 AI 系列 1 篇 → 用清单挑刺 ≥3
默认管道：否决为可交易 / 仅工具向灵感
```
